## About

The ACP-220 is Atlassian's associate-level certification for Confluence Cloud administrators. It validates the ability to manage Confluence Cloud instances at the organizational level, including user access, space configuration, security, content governance, and performance monitoring. Designed for IT professionals and knowledge management administrators, the certification demonstrates competency in deploying, configuring, and maintaining Confluence in cloud environments. Released in 2020, it remains current and widely recognized in organizations using Atlassian tooling for collaboration and documentation.

Collaboration platforms enable teams to coordinate work, share knowledge, and maintain organizational institutional memory through centralized documentation, commenting, and workflow automation. Confluence is the dominant knowledge management platform in the Atlassian ecosystem.

## Topics covered

- **Access and Permissions (21%)** — User and group management, permission hierarchies, space-level and global access control, directory synchronization
- **Space Setup and General Settings (15%)** — Space creation and configuration, space templates, organizational structure, settings and governance
- **Site Configuration (13%)** — Global site settings, instance administration, user directory integration, site-wide permissions
- **Content Administration (13%)** — Content lifecycle management, pages, attachments, versioning, content organization
- **Security and Data Management (11%)** — Security policies, audit logging, data retention, compliance requirements, GDPR considerations
- **Automation Rules (11%)** — Creating and managing automation rules to streamline administrative tasks and workflows
- **Look and Feel (8%)** — Theme customization, branding, interface configuration, user experience
- **Monitor Confluence Site Performance (7%)** — Health checks, performance monitoring, analytics, reporting
