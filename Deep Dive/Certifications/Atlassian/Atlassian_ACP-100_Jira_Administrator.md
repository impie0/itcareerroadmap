## About

The ACP-100 Jira Administrator certification validates hands-on expertise in configuring, deploying, and maintaining Jira across small-to-enterprise environments. Launched in 2016 as Atlassian's response to growing demand for certified Jira ops staff, it remains the primary professional-level credential for IT operations, project-management tooling, and DevOps platform engineers who manage Jira infrastructure. The exam covers on-premise (Server/Data Center) and cloud deployments, though cloud-specific material dominates post-2022 updates. Status: active and widely recognized by enterprise IT hiring managers.

## Topics covered

- **User and Project Configurations** (10%) — User/group management, project creation, roles, notification schemes
- **Schemes and Workflows** (30%) — Workflow design, scheme association, transitions, post-functions, validators
- **Permissions and Notifications** (20%) — Permission matrix, global permissions, issue-level security, notification schemes
- **Custom Fields and Screens** (15%) — Field creation, screen layouts, issue-type schemes, field associations
- **Boards, Reporting, and Search** (15%) — Board types (Scrum/Kanban), advanced issue searches (JQL), gadgets, reporting
- **Integrations and Operations** (10%) — API basics, webhooks, plugins, backup/restore, upgrade paths

## Skills validated

*Jira-specific technical competencies tested by this exam.*

- User, group, and permission scheme configuration
- Workflow design, post-function scripting, validators, and condition logic
- Custom field creation, screen layout, and issue-type association
- Notification scheme triggers and email template customization
- JQL (Jira Query Language) for advanced issue filtering and reporting
- Board configuration (Scrum, Kanban, and board settings)
- Plugin/app discovery and configuration (Atlassian Marketplace)
- Backup, restore, and version upgrade procedures
- REST API basics (authentication, query construction)
- Webhook setup for external integrations
