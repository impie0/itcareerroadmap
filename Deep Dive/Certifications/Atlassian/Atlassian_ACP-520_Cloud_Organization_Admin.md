## About

The ACP-520 Atlassian Cloud Organization Admin certification validates expertise in configuring, managing, and securing Atlassian Cloud sites and products at the organizational level. Launched in 2021 as enterprise cloud adoption accelerated, it has grown in prominence alongside Atlassian's shift away from Server products (retired February 2024). Spring 2026 update emphasized identity governance (SCIM provisioning, conditional access) and regulatory compliance (GDPR, audit trails, data residency). This credential is essential for IT operations, enterprise identity engineers, and cloud platform teams managing Atlassian at scale. Status: active and increasingly required by mid-to-large enterprises using Jira Cloud, Confluence Cloud, or Service Management Cloud.

## Topics covered

**Spring 2026 exam topics (updated blueprint):**

- **Atlassian Cloud Subscription & Site Management** (20%) — Organization structure, multi-site provisioning, product bundles (Jira, Confluence, Service Management), billing, licensing, site migration from Server/Data Center
- **Identity & Access Management** (35%) — Atlassian Access, SAML SSO, SCIM user provisioning, identity provider integrations (Okta, Azure AD/Entra ID, Okta, Google Workspace), domain verification, conditional access policies, app passwords, API token management
- **User Provisioning & Group Management** (15%) — User lifecycle (onboarding/offboarding), group synchronization via SCIM, role-based access, team structures, deprovisioning workflows
- **Security Policies & Controls** (15%) — Two-factor authentication (2FA) enforcement, IP allowlisting, session management, API security, webhook security, audit logging, activity monitoring
- **Compliance & Data Protection** (10%) — GDPR compliance, audit trails, data residency, privacy policies, backup/recovery, incident response, regulatory reporting
- **Troubleshooting & Operational Support** (5%) — Common admin issues, API debugging, support ticket handling, documentation
