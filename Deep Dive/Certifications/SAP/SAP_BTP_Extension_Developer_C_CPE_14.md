## About

SAP Certified Development Associate – SAP BTP Extension Developer is an Associate-level certification for backend developers building cloud-native extensions on SAP Business Technology Platform (BTP) using the Cloud Application Programming Model (CAP). Candidates demonstrate knowledge of side-by-side and in-app extensibility patterns, Core Data Services (CDS), CAP runtimes (Node.js and Java), authorization/authentication, and integration with SAP HANA Cloud and Fiori Elements. First released in 2021, it remains the primary gateway credential for SAP BTP extension developers in enterprise environments.

## Topics covered

*Based on official exam syllabus and community resources:*

- **SAP Business Technology Platform (BTP) fundamentals** (10–15%)
  - BTP services, environments (Cloud Foundry / Kyma), global account structure
- **SAP Cloud Application Programming Model (CAP)** (25–30%)
  - Core Data Services (CDS), domain modeling, persistence, service definitions
- **Service development & implementation** (20–25%)
  - Node.js and Java runtimes, custom logic, RESTful and OData services
- **Data persistence & databases** (10–15%)
  - CAP schema, SQLite / SAP HANA Cloud integration, migrations
- **User interfaces & Fiori Elements** (10–15%)
  - Fiori Elements UI generation, SAP UI5, basic frontend integration
- **Security & authorization** (10–15%)
  - Identity Authentication (IAS), role-based access control, scopes, JWT tokens
- **Integration & deployment** (10–15%)
  - SAP Integration Suite (Cloud Integration / CPI), CI/CD pipelines, build tools
- **Testing & debugging** (5–10%)
  - Unit tests, debugging techniques, VS Code tooling

*Source:* [Official ERPPrep Syllabus ↗](https://www.erpprep.com/sap-btp-extensions-sap-cloud-application-programming-model-certification-exam-syllabus)

## Skills validated

*Specific to SAP BTP Extension Developer exam:*

- **SAP Cloud Application Programming Model (CAP)** — domain modeling, CDS (Core Data Services), service definitions, generic handlers
- **Data persistence** — database schema design, ORM patterns, migration strategies, SAP HANA Cloud integration
- **Service development** — Node.js and Java CAP runtimes, custom business logic, metering, resilience
- **OData & REST APIs** — CRUD operations, query filtering, composition, semantic versioning
- **User interface development** — Fiori Elements, SAP UI5 basics, form/list page generation
- **Security & identity** — SAP Identity Authentication Service, role/scope design, JWT token handling, RBAC
- **Cloud integration** — Integration Suite (CPI), message flows, adapter patterns, API connectivity
- **Deployment & DevOps** — Cloud Foundry CLI, CI/CD pipelines (GitHub Actions, SAP Cloud Build), health checks
- **Testing** — Unit testing frameworks (JavaScript/Java), integration test patterns, mock objects
- **Extensibility patterns** — side-by-side vs. in-app extensions, custom code, business events
