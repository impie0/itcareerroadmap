## About

The C_TAW12_750 certifies developers in classical SAP NetWeaver 7.50 ABAP fundamentals and the ABAP Workbench. It is the established entry-level credential for ABAP developers in traditional SAP systems (ECC, legacy S/4HANA deployments). Since its release in 2015, it has remained the baseline certification for ABAP development roles. SAP continues to offer it alongside newer certifications (C_ABAPD for cloud-native ABAP). The exam validates competency in ABAP language syntax, Data Dictionary, ALV reporting, ABAP Object-Oriented programming, and system integration patterns (RFCs, BAPIs, Enhancements).

## Domain context — Software Engineering · SAP

SAP is a vertically integrated enterprise resource planning (ERP) ecosystem. ABAP (Advanced Business Application Programming) is SAP's proprietary fourth-generation language, used exclusively for SAP system customization and development. The ABAP domain is a specialty discipline within Software Engineering, requiring deep knowledge of both programming and ERP domain logic.

## Topics covered

- **ABAP Language Fundamentals** (15%) — data types, control structures, modularization, object-oriented programming
- **ABAP Dictionary / Repository** (15%) — database tables, structures, domains, search helps, lock objects
- **ALV (ABAP List Viewer) and Reports** (12%) — classic list reports, ALV grid, interactive reports, print layouts
- **ABAP Objects** (12%) — classes, interfaces, inheritance, polymorphism, exception handling
- **ABAP Workbench Tools** (12%) — Debugger, ABAP Editor, SE24 (Class Builder), transaction codes, transport system
- **SQL and Data Access** (10%) — ABAP Open SQL, internal tables, joins, SELECT statements, performance optimization
- **Enhancements and Modifications** (10%) — user exits, BAdi (Business Add-In), implicit enhancements, custom code lifecycle
- **BAPI and RFC (Remote Function Call)** (8%) — remote communication, synchronous / asynchronous calls, error handling
- **Web Services and Integration** (4%) — ABAP Web services, XML parsing, integration patterns

## Skills validated

*Concrete technologies and practices tested by the C_TAW12_750 exam.*

- **ABAP Language** — data types, control flow (PERFORM, CALL, IF/ELSE), string/table manipulation
- **Data Dictionary** — table creation, views, domains, check tables, search helps
- **ALV Reporting** — SALV model, classic ALV grid, event handling, interactive reports
- **ABAP Objects** — class definition, method implementation, inheritance, polymorphic code
- **SQL and Open SQL** — SELECT, JOIN, INSERT, UPDATE, DELETE; performance considerations (buffer awareness, index usage)
- **Enhancements** — user exits (pre/post), BAdi implementation, custom logic injection
- **RFCs and BAPIs** — synchronous/asynchronous communication, error handling, SAP standard APIs
- **ABAP Workbench Tools** — SE38 (Editor), SE24 (Class Builder), SE91 (Messages), Debugger (SE37)
- **Web Services** — ABAP XSD, XML parsing, HTTP communication
- **Transport Management** — understanding change request / task workflow
