## About

The SAP Certified Development Associate — SAP Customer Data Cloud (C_CDCDEV_01) validates core technical competencies required of developers and implementation consultants building customer identity and data management solutions on SAP Customer Data Cloud (formerly SAP Gigya). This certification confirms hands-on experience with platform configuration, customer data schema design, authentication flows, consent management, and JavaScript/SDK integration. Released in 2022 to align with SAP's cloud-native Customer Experience (CX) strategy, this credential serves developer practitioners joining or supporting SAP CDC projects in a mentored capacity.

---

## Domain context — Customer Data Platform & Identity

Customer Data Platforms (CDPs) and Identity & Access Management (IAM) form the backbone of modern customer experience and zero-trust security architectures. SAP Customer Data Cloud is a SaaS-based CDP with integrated CIAM (Customer Identity and Access Management) that consolidates customer profiles, preferences, and consents across enterprise systems.

## Topics covered

The exam blueprint emphasizes platform engineering, data modeling, and identity federation, with approximate weightings:

- **Customer Identity & Site Groups Configuration** – 21–30% : Site setup, screen-sets, registration flows, identity federation (SAML/OAuth/OIDC), multi-tenant configuration
- **Data Schema & Profile Management** – 11–20% : User profile design, schema extensions, data binding, field-level consent, profile APIs, custom attributes
- **Extensibility & Data Flows** – 11–20% : JavaScript SDK integration, webhooks, event streaming, serverless functions, third-party API integration, custom scripts
- **Platform Console — Administration & Configuration** – ≤10% : User management, audit logs, API key provisioning, permission scoping, debugging tools
- **Managing Clean Core & Compliance** – ≤10% : Data governance best practices, PII handling, GDPR / CCPA consent enforcement, data retention policies
- **Federation & Single Sign-On (SSO)** – ≤10% : Identity provider integration, social login (Google, Facebook, Apple), enterprise directory sync
- **Mobile Applications** – ≤10% : Mobile SDK implementation, native app authentication, secure token handling, offline capabilities
- **B2B Customer Identity** – ≤10% : Account-based identity, role-based access (RBAC), organizational hierarchies, partner authentication
- **Enterprise Preference Management** – ≤10% : Consent preferences, communication channels, preference center UI, GDPR right-to-be-forgotten workflows

*Source:* [SAP Learning — C_C4H62_2408 Exam Syllabus](https://www.erpprep.com/sap-cloud-solutions/sap-c-c4h62-2408-certification-exam-syllabus)
