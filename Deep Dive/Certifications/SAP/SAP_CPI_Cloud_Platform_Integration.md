## About

The SAP Certified Development Associate – SAP Integration Suite (C_CPI_2506, formerly C_CPI_15) validates core competency in designing, developing, and deploying integration solutions on SAP Business Technology Platform. Released in 2020 as part of SAP's shift to cloud-native applications, this certification is intended for integration developers, consultants, and architects responsible for building hybrid-cloud iPaaS flows, managing APIs, and integrating on-premises and cloud applications. In 2026, SAP adopted a 12-month validity cycle with annual renewal via a lighter "Stay Certified" assessment, aligning all certifications with continuous learning expectations.

---

## Domain context — Cloud Integration

Cloud integration encompasses the architecture, development, and operational patterns required to connect applications, data, and services across hybrid and multi-cloud environments. On SAP Business Technology Platform, this includes designing integration content (flows, adapters, mappings), managing APIs, implementing event-driven architecture, and securing integrations.

## Topics covered

The C_CPI_2506 exam blueprint is weighted across four major topic areas:

- **SAP Integration Suite Overview** (11–20%)
  - BTP platform architecture and services
  - Integration Suite components (Cloud Integration, API Management, Trading Partner Management, Event Mesh)
  - Tenancy and security model
  - Monitoring and alerting concepts

- **Managing APIs** (21–30%)
  - API Management capabilities
  - API Business Hub and developer portal
  - API policy configuration
  - OAuth 2.0 and certificate-based security
  - Rate limiting and quotas

- **Implementing Cloud Integration** (31–40%)
  - Integration flow design and development
  - Message flow processing and context handling
  - Adapter configuration (HTTP, SOAP, OData, IDoc, JMS, SFTP, RFC, file systems)
  - Payload mappings (XSLT, graphical mapping, operation mapping)
  - Groovy scripting and JavaScript in message processing

- **SAP Event Mesh** (41–50%)
  - Event-driven integration patterns
  - Topic and queue management
  - Event schema and payload handling
  - Integration with Cloud Integration flows

## Skills validated

*Cert-specific — what this exam actually tests.*

- **Integration flow design** — creating stateful, stateless, and event-driven flows in SAP Integration Suite
- **Adapter configuration** — HTTP, SOAP, OData, IDoc, JMS, SFTP, RFC, file system, SAP connectors
- **Message mapping and transformation** — XSLT stylesheets, graphical mapping, operation mapping, message routing
- **Groovy scripting** — in-flow scripting for message enrichment, context handling, conditional logic
- **API management lifecycle** — creating, versioning, securing, and publishing APIs via API Business Hub
- **OAuth 2.0 and certificate security** — configuring authentication and authorization for integrations
- **Event Mesh integration** — topic/queue design, event schema, asynchronous event processing
- **Monitoring and troubleshooting** — log analysis, message tracing, error handling and retry logic
- **Trading Partner Management** — B2B document exchange (EDI, EDIFACT) over cloud integration
- **Open Connectors** — third-party SaaS connectivity (Salesforce, ServiceNow, Workday, Oracle, etc.)
