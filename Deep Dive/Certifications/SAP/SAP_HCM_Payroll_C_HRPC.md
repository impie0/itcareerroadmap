## About

The SAP Certified Application Associate — SAP HCM Payroll (C_HRPC_2305) validates intermediate-level knowledge of payroll processing, payroll master data configuration, wage type management, payroll schemas and rules, tax calculation, and payroll reporting integration in SAP Human Capital Management (HCM). This certification is designed for functional consultants, payroll specialists, HR analysts, and business process specialists implementing or supporting payroll operations on SAP HCM and SAP SuccessFactors platforms. The exam covers both on-premise SAP ECC/S/4HANA payroll and cloud-native payroll architectures, with emphasis on payroll run execution, statutory compliance, and country-specific payroll variations.

## Topics covered

The exam blueprint spans payroll configuration, execution, and reporting with the following weightings:

- **Payroll Master Data** – 12–16% : Employee master data (INFOTYPES 0000–0008), organizational assignment, personnel master data structure, employment terms, wage earner status
- **Payroll Schemas and Rules** – 12–16% : Wage type determination, payroll schemas (basic and advanced), transaction codes (PMCO, SMIT, SMPL), schema maintenance, condition techniques
- **Wage Types** – 10–14% : Wage type definition, classification (pay scale, statutory deduction, voluntary deduction), wage type groups, cumulative settings, tax key assignment
- **Garnishments and Statutory Deductions** – 8–12% : Garnishment orders, attachments, child support orders, tax withholding, social security integration
- **Time Management Integration** – 8–12% : Time data capture, attendance records, time evaluation, integration with payroll run, quota management, time type setup
- **Tax Calculation and Compliance** – 12–16% : Income tax withholding (US, EU, APAC variants), social contribution calculation, tax exemptions, tax compliance for multiple countries
- **Payroll Run Processing** – 8–12% : Payroll release, simulate payroll, final payroll run, error handling, correction procedures, payroll archiving
- **Reporting and Posting to Finance** – 8–12% : Payroll reports (RPCLST00, RPCLFCH, TCURR), cost allocation, posting to Financial Accounting (FI), accrual entries, reconciliation
- **Country-Specific Payroll** – 6–10% : US Federal/State/Local taxes, EU payroll directives, APAC tax variations, statutory reporting requirements by country
- **Payroll Integration and Interfaces** – 6–10% : Integration with SuccessFactors Employee Central Payroll (ECP), third-party payroll feeds, tax authority reporting, bank file generation

*Source:* [SAP Learning Center — Exam Objectives](https://learning.sap.com/certifications)

## Skills validated

*Specific technologies, processes, and tools this exam tests, distinct from shared HR/Payroll competencies above.*

- SAP HCM transaction codes (PPMCO, SMIT, SMPL, PC00, RPCLST00)
- Payroll schema design and wage type determination logic
- MOLGA (Multiple Organizational Levels Global Architecture) payroll customization
- Time data integration via IT0007 (Planned Working Time) and IT2000–IT2010 (Time Management)
- Tax and social contribution calculation engines (country-specific configuration)
- Payroll posting to FI (Financial Accounting) with accrual reversal
- SAP SuccessFactors Employee Central Payroll (ECP) cloud integration
- Payroll variant maintenance (PERNR, PERSK, MOLGA hierarchies)
- Reconciliation and variance resolution (RPCLFCH, RPCLFIN)
- Electronic banking file generation and tax authority reporting (AUE, ASC X12 formats)
- Data archiving and retention compliance
