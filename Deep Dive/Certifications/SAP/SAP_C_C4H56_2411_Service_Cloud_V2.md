## About

The C_C4H56_2411 validates the ability to implement and configure SAP Service Cloud Version 2, a cloud-native customer service platform. Released in 2024, this credential replaced the older C_C4H51_2405 (Service Cloud Implementation Consultant) and reflects SAP's enhanced omnichannel ticketing, AI-driven case routing, and tighter S/4HANA integration. Aimed at functional consultants, implementation advisors, and CX solution architects deploying Service Cloud in mid-market and enterprise settings.

## Domain context — Customer Experience / Service Cloud

SAP Service Cloud is a cloud-based omnichannel service management suite within the SAP Customer Experience ecosystem. It handles ticket lifecycle, knowledge base authoring, SLA automation, agent productivity, and analytics—bridging customer service operations and wider CX strategy.

## Topics covered

Based on the official SAP exam blueprint:

- **Master Data Management (20%)** — Entity setup (products, service categories, service teams), configuration of customer, contact, and employee hierarchies
- **Communication Channels (20%)** — Email, chat, phone, social media integration; omnichannel ticketing; channel routing rules
- **Case & Ticket Management (18%)** — Ticket creation, status workflows, escalation logic, involvement parties, assignment rules
- **Service Level Agreements (SLAs) (15%)** — SLA definition, determination rules, KPI metrics (response time, resolution time), breach notifications
- **Knowledge Base & Self-Service (12%)** — Article creation, publication, FAQs, customer self-service portal, search optimization
- **Personalization & Extensibility (10%)** — UI customization, field extensions, role-based access, workflow automation, approvals
- **Analytics & Reporting (5%)** — Dashboards, KPI tracking, queue performance, agent productivity metrics
