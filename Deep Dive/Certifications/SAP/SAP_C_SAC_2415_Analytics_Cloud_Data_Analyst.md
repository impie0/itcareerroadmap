## About

SAP Certified Associate - Data Analyst - SAP Analytics Cloud validates hands-on proficiency in designing analytical stories, modeling data, connecting live and imported data sources, and implementing planning solutions within SAP Analytics Cloud. Released in 2021 as part of SAP's enterprise analytics suite, it remains actively maintained by SAP with exam code updates (e.g., C_SAC_2415, C_SAC_2501). The certification targets data analysts, BI professionals, and SAP Analytics Cloud developers who build dashboards, conduct analytics, and support planning workflows. SAP transitioned to performance-based, open-book, unproctored exams in 2026.

## Topics covered

- **Story Design & Visualization** (20–25%): Design and configure stories, pages, tables, charts, input controls, filters, formatting, interactivity, and story optimization.
- **Data Modeling & Connections** (25–30%): Create analytic and planning models; configure live data connections and import data; manage data quality and transformations.
- **Planning & Forecasting** (20–25%): Implement planning models, budgeting, forecasting, version management, and what-if simulation.
- **Data Analysis & Smart Predict** (15–20%): Exploratory analysis, augmented analytics, predictive algorithms, anomaly detection, and recommendation engine.
- **Security, Administration & Mobile** (10–15%): Role-based access, audit logging, mobile app support, and system configuration.
- **Data Wrangling & Calculation Views** (5–10%): Transform and prepare data; build calculated fields and analytic expressions.
