## About

The SAP Certified Application Associate — SAP HANA Cloud Modeling certification verifies that candidates possess fundamental knowledge and proven practical skills in data modeling with SAP HANA Cloud. The exam confirms proficiency in implementing calculation view graphical modeling, managing modeling content in SAP Business Application Studio, and applying SQLScript for data manipulation. This certification is targeted at SAP HANA Cloud application consultants and data engineers who work on SAP HANA Cloud projects and need to validate their modeling expertise. The certification remains valid for 5 years, after which retesting is required.

## Topics covered

The C_HCMOD exam blueprint comprises six primary domains:

- **Domain 1: Build calculation views** (> 12%) — graphical modeling, node-based architecture, data source configuration
- **Domain 2: Configure modeling functions** (> 12%) — filters, parameters, hierarchies, aggregations, and expression functions
- **Domain 3: Manage and administer models** (> 12%) — model organization, metadata management, SAP Business Application Studio workflows
- **Domain 4: Optimize performance of models** (> 12%) — query execution, indexing strategies, partitioning, materialization
- **Domain 5: Secure data models** (8–12%) — row-level security (RLS), cell-level security (CLS), authorization concepts
- **Domain 6: SQLScript in models** (8–12%) — table functions, stored procedures, script-based calculations
