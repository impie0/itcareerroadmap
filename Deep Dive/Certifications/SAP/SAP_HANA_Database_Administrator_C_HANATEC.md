## About

The C_HANATEC was a vendor-specific Associate-level certification that validated technical knowledge of SAP HANA database administration, architecture, and operations. Launched in 2015 as SAP's flagship in-memory database platform matured, it evolved through 19 versions (C_HANATEC_10 through C_HANATEC_19) to track SAP HANA 2.0 service pack releases (SPS02 through SPS07). 

In April 2024, SAP transitioned all product-based certifications to role-based profiles, retiring C_HANATEC in favor of the new C_DBADM, which broadens scope to include both SAP HANA and SAP HANA Cloud. Holders of C_HANATEC remain industry-recognized as SAP HANA database professionals, and the cert remains a clear indicator of hands-on database administration competency. Organizations with mature SAP HANA on-premises deployments still value C_HANATEC credentials when hiring legacy-system administrators.

## Domain context – Database Administration

**SAP HANA is a proprietary in-memory relational database platform owned by SAP, widely deployed in enterprises running SAP ERP, S/4HANA, and analytics workloads.** Database administration at the SAP HANA level spans system architecture, backup/recovery, performance tuning, multi-tenant container management, and security. SAP HANA professionals straddle infrastructure and application layers.

## Topics covered

Official exam blueprint from SAP Training (C_HANATEC_19):

- **SAP HANA Architecture & Components** (15–20%)
  - Database structure, in-memory storage, columnar/row engine
  - SAP HANA Cockpit and DBA Cockpit usage
  
- **Installation & Administration** (20–25%)
  - Installation procedures
  - Parameter configuration
  - Client and system administration
  
- **Backup and Recovery** (15–20%)
  - Backup types and strategies
  - Recovery procedures
  - Backint integration
  
- **Performance Monitoring & Tuning** (15–20%)
  - Monitoring tables and metrics
  - Trace analysis
  - Performance diagnosis
  
- **High Availability & Disaster Recovery** (10–15%)
  - SAP HANA System Replication
  - HANA Failover/Takeover scenarios
  - Datacenter setup
  
- **Multitenant Database Containers (MDC)** (10%)
  - Tenant management
  - System vs. tenant databases
  - Lifecycle operations
  
- **Security & Authentication** (10–15%)
  - User and role management
  - Authentication methods (LDAP, SAML, mTLS)
  - Encryption (data at rest, in transit)
  - Auditing and compliance

*Source:* [SAP Training sample questions archive ↗](https://cdn.training.sap.com/uploads/C_HANATEC_10_Sample_Questions.pdf)

## Skills validated

*C_HANATEC-specific technical competencies tested on the exam.*

- SAP HANA database architecture (in-memory columnar/row storage, indexing)
- Installation, configuration, and patching of SAP HANA 2.0 systems
- Backup and recovery procedures (complete, differential, incremental strategies)
- SAP HANA Cockpit and DBA Cockpit navigation and diagnostics
- System Replication (SR) configuration for HA/DR
- Multitenant Database Container (MDC) design and lifecycle management
