# SAP Certified Professional - Solution Architect SAP BTP

**`P_BTPA`** · ● **Active** · **Professional** · _SAP_

This certification validates the ability to design and architect enterprise solutions on the SAP Business Technology Platform (BTP), encompassing cloud strategy, multi-cloud integration, identity & security, database services, and solution methodologies. Targeted at experienced architects and senior consultants transitioning to SAP BTP or expanding their BTP expertise.

## About

The P_BTPA certification measures competency in designing enterprise solution architectures using SAP Business Technology Platform. The exam uses scenario-based assessment to evaluate candidates' ability to interpret business requirements, select appropriate BTP services (integration, extension, analytics, AI), and architect sustainable multi-cloud solutions aligned with enterprise governance. Released in 2022 and modernized in 2024, it serves as the vendor-recognized credential for BTP Solution Architects and is required for SAP Partner accreditation in cloud architecture roles.

## Domain context — Cloud Architecture

SAP BTP is a hyperscale platform-as-a-service (PaaS) offering that extends SAP ERP ecosystems into modern cloud, covering integration, analytics, AI/ML, application development, and identity management across multiple cloud providers (AWS, Azure, Google Cloud, Alibaba Cloud).

## Topics covered

The exam blueprint covers five major competency areas:

- **Solution Architecture & Design Methodology (41–50%)** — SAP reference architecture, Intelligent Enterprise framework, BTP account model, integration patterns, extension strategy, analytics positioning
- **Platform Fundamentals & Services (30–40%)** — BTP environments (Cloud Foundry, Kyma, Neo legacy), account hierarchy, subaccounts, resource management, SAP AI Core, AI Launchpad
- **Integration & Connectivity (15–25%)** — SAP Integration Suite, API Management, destination management, custom code lifecycle, integration patterns (point-to-point, hub-and-spoke, API-led)
- **Data, Analytics & AI (10–15%)** — SAP Data Intelligence, SAP Analytics Cloud positioning, master data governance, data lakes, AI service selection and placement
- **Security, Identity & Operations (10–15%)** — SAP Cloud Identity Services (IAS, XSUAA), single sign-on (SSO), role-based access control (RBAC), audit & compliance, DevOps architecture, multi-tenancy

## Skills validated

*Cert-specific — what this exam actually tests, distinct from the shared "Common skills" above.*

- **SAP BTP architecture & design patterns** — Multi-cloud strategy, environment separation, account hierarchy, DevOps integration
- **SAP Integration Suite configuration** — iPaaS design, API management, event-driven architecture, B2B integrations
- **Cloud Foundry & Kyma runtimes** — Application lifecycle management, container orchestration, service binding, sidecars
- **Identity & Access Management** — XSUAA, SAP Cloud Identity Services (IAS), multi-factor authentication (MFA), identity federation
- **SAP Data Intelligence & Analytics** — Pipeline design, data flow governance, SAP Analytics Cloud positioning, HANA Cloud
- **SAP AI Core & ML services** — Model deployment, inference endpoints, MLOps on BTP, generative AI (SAP AI Foundation)
- **Database services** — SAP HANA Cloud (multi-cloud), PostgreSQL on BTP, data lake design, backup strategies
- **Mobile services & Progressive Web Apps (PWAs)** — Deployment on BTP, offline capability, push notifications
- **Destination & Connectivity** — Custom connectivity, destination management, secure communication patterns
- **Security & compliance** — Audit logging, encryption at rest/in-transit, data residency, industry compliance (PCI-DSS, GDPR, HIPAA)
- **Cost modeling & resource governance** — Metering, unit economics, quota management, chargeback
