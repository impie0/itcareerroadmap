## About

The CCFA validates hands-on ability to deploy, configure, and manage the CrowdStrike Falcon platform in enterprise environments. Designed for security operations, systems administration, and IT teams responsible for endpoint detection and response (EDR) deployment and day-to-day administration. This is the entry-level associate certification in the CrowdStrike credential pathway and serves as the foundation for advanced CCFR (Responder) and CCFH (Hunter) specializations.

---

## Domain context — Security/EDR

Endpoint Detection and Response (EDR) is the real-time telemetry and forensics layer that sits between antivirus (prevention) and SIEM (correlation). EDR platforms like Falcon prioritize visibility: they capture process trees, network flows, file operations, and memory execution to enable post-breach investigation and threat hunting. At the associate level, practitioners focus on operational administration: rolling out sensors to enterprise hosts, tuning prevention and detection policies, triaging alerts, and ensuring compliance requirements.

## Topics covered

- **Falcon Console Navigation** — Platform UI, menus, dashboards, information architecture
- **User and Role Management** — User provisioning, role assignment, permission models, RBAC
- **Policy Configuration** — Prevention policies, response policies, detection tuning, sensor grouping
- **Sensor Deployment** — Installation and provisioning strategies, deployment groups, Windows/Linux/macOS agents
- **Host Management** — Host onboarding, asset organization, grouping, tagging, host isolation
- **Real Time Response Basics** — Interactive endpoint access, common commands, response techniques
- **Alert and Detection Triage** — Alert classification, severity assessment, false-positive filtering, investigation workflows
- **Indicator of Compromise (IOC) Management** — Custom IOC creation, threat matching, rule application
- **Falcon Insight Basics** — Threat intelligence integration, detection logic, indicator matching
- **Charlotte AI for Admins** — AI-assisted platform navigation, configuration recommendations
- **Falcon Reporting and Dashboards** — Custom reporting, dashboard creation, compliance metrics, data export
