## About

The Grafana Certified Associate (GCA) validates foundational competency in building, configuring, and managing observability solutions using the Grafana platform. Released in 2023, this entry-level certification is designed for observability engineers, SREs, and DevOps professionals implementing Grafana in production environments. It covers platform fundamentals, dashboard design, data source integration, and alerting workflows.

---

## Domain context — Observability & Visualization

Modern infrastructure requires real-time visibility into system health, performance, and user impact across distributed applications. Observability (logs, metrics, traces) differs from monitoring by enabling investigation of unknown failure modes. Visualization tools like Grafana translate observability data into actionable dashboards and alerts.

## Topics covered

- **Grafana platform fundamentals** (core architecture, editions, deployment options)
- **Dashboard design & creation** (panel types, variables, transformations, templating)
- **Data sources & connectors** (Prometheus, Loki, Tempo, Mimir, Elasticsearch, SQL databases, MongoDB)
- **Alerting & notification management** (alert rules, contact points, notification channels)
- **User management & RBAC** (roles, permissions, team organization, data source access)
- **Plugins & extensibility** (installing, configuring, and managing Grafana plugins)
- **Grafana Cloud overview** (managed SaaS platform, cloud provisioning)
- **OpenTelemetry integration** (OTel collectors, instrumentation, trace correlation)

## Common skills at Observability & Visualization · Associate

- Configuring and managing multiple data source types (metrics, logs, traces)

- Interpreting time-series metrics and logs in real time
- Role-based access control and team/org multi-tenancy setup
- Troubleshooting data source connectivity and query performance
- Understanding observability concepts (cardinality, sampling, retention)
