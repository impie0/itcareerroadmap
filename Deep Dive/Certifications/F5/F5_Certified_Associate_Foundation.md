## About

The F5 Certified Administrator, BIG-IP (F5-CA, BIG-IP) is an associate-level certification launched in 2025, designed for IT professionals responsible for day-to-day operations and basic administration of F5 BIG-IP systems. The certification replaces the legacy single-exam model with a modular five-exam pathway (F5CAB1–F5CAB5), each focusing on a specific domain of BIG-IP administration. Candidates can take exams in any order, making the certification more accessible for working professionals. This credential validates foundational knowledge of virtual server management, traffic steering, data plane configuration, control plane operations, and troubleshooting techniques critical for application delivery networks.

## Domain context — Application Delivery / Load Balancing

Application Delivery Networking (ADN) focuses on optimizing, securing, and accelerating application performance through load balancing, SSL/TLS termination, traffic management, and distributed denial-of-service (DDoS) mitigation. F5 BIG-IP is the market-leading Application Delivery Controller (ADC), widely deployed in enterprise, cloud, and hybrid environments.

## Topics covered

The F5-CA certification spans five modular exams, each with distinct objectives:

**F5CAB1: BIG-IP Administration Install, Initial Configuration, and Upgrade**
- Initial system setup and network configuration
- Interface and VLAN management
- Self-IP and routing configuration
- BIG-IP system upgrades and unit ID assignment

**F5CAB2: BIG-IP Administration Data Plane Concepts**
- VLAN, interface, and trunking concepts
- Self-IP, floating-IP, and routing fundamentals
- High Availability (HA) and failover concepts
- Health monitoring and virtual server pool relationships

**F5CAB3: BIG-IP Administration Data Plane Configuration**
- Virtual server creation and management
- Pool and pool member configuration
- Monitor creation and health check assignment
- Traffic steering and load-balancing algorithms
- Connection rate limiting and traffic control

**F5CAB4: BIG-IP Administration Control Plane**
- Administrative authentication and authorization (AAA)
- High Availability operations and synchronization (Config-Sync)
- NTP, DNS, SNMP, and syslog configuration
- UCS backup and recovery
- System services and resource management

**F5CAB5: BIG-IP Administration Support and Troubleshooting**
- Performance metrics and statistics collection
- PCAP (packet capture) analysis
- Load-balancing misconfiguration diagnosis
- Virtual server and pool-level troubleshooting
- Log analysis and common issue resolution
