## About

The **Application Delivery Fundamentals (101)** exam validates fundamental knowledge of F5 products, technologies, and application delivery networking (ADN) concepts. Candidates demonstrate understanding of OSI model layers, traffic management, load balancing, security, and F5 solutions architecture. This exam was the entry point to the F5 Certified BIG-IP Administrator credential and all specialist pathways. Though officially retired April 30, 2025, it remains relevant as historical context for existing F5-CA certified professionals and those studying the transition to the new modular exam structure.

---

## Domain context — Application Delivery / Load Balancing

Application delivery encompasses the technologies, platforms, and practices that optimize how applications are deployed, scaled, secured, and managed across distributed infrastructure. Load balancing distributes client traffic across multiple backend servers to improve availability, resilience, and performance. F5 dominates this space with BIG-IP Local Traffic Manager (LTM), Global Traffic Manager (GTM), and integrated security modules.

## Topics covered

The 101 exam blueprint covers the following core topics:

- OSI Model and Data Link, Network, Transport, Application Layers
- Switching & Routing fundamentals
- F5 Solutions, BIG-IP architecture, and TMOS platform overview
- Load Balancing concepts (virtual servers, pools, load-balancing methods)
- Traffic Management and application delivery networking
- Security fundamentals (DoS, DDoS, authentication, SSL/TLS)
- Application Delivery Platforms and integration
- DNS, Global Traffic Management, and high availability
- Basic troubleshooting and operational concepts

## Skills validated

_Concrete technologies and concepts this exam tests, distinct from the shared domain-level skills._

- **F5 BIG-IP architecture** — TMOS operating system, module structure (LTM, GTM, ASM, etc.)
- **Virtual servers and pools** — Configuration, management, and monitoring
- **Load-balancing algorithms** — Round robin, least connections, weighted methods, dynamic ratios
- **Connection and session persistence** — Cookie-based, source IP, SSL session persistence
- **Health monitoring** — Monitor types, health check concepts, health status
- **Traffic policies and iRules fundamentals** — Basic policy application
- **SSL/TLS termination and key management** — Certificate handling, cipher suites
- **Global Traffic Management (GTM) overview** — Wide IP, intelligent DNS
- **DoS and DDoS protection basics** — Conceptual understanding
- **High availability and redundancy** — Failover, active–active clustering
- **Operational tasks** — Backups, upgrades, basic troubleshooting, log review
