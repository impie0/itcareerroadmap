## About

The F5 Certified Administrator BIG-IP (201 TMOS Administration) exam validates associate-level competency in day-to-day administration, troubleshooting, and operational management of TMOS-based application delivery systems (BIG-IP platform). Candidates must first pass the 101 Application Delivery Fundamentals exam. The 201 covers configuration, management, monitoring, backup/restore, high-availability synchronization, SSL/TLS profiles, iRules basics, persistence, health monitors, and diagnostic procedures. This certification is part of F5's legacy two-exam administrator track, which transitions to a five-exam model after April 30, 2025.

## Domain context — Networking and Application Delivery

Application delivery platforms handle load balancing, SSL termination, content acceleration, and traffic management across distributed infrastructure. BIG-IP (TMOS) is F5's flagship appliance for this function, deployed in enterprises, cloud providers, and service providers globally.

## Topics covered

Based on the official exam blueprint:

- **TMOS Architecture & System Administration** — System configuration, licensing, TMOS resource provisioning, module management
- **Virtual Server Configuration** — Virtual server creation, profiles, rules, policies, status monitoring
- **Networking Fundamentals** — Interfaces, VLANs, self IPs, routes, network failover, HA pairs
- **Local Traffic Manager (LTM)** — Pools, pool members, persistence profiles, health monitors, load balancing algorithms
- **SSL/TLS & Security Profiles** — Client/server SSL profiles, certificate management, cipher suites
- **iRules Basics** — Introduction to event-driven scripting for traffic manipulation (manipulation, logging, routing)
- **High Availability (HA) & Sync** — Active-standby configuration, ConfigSync, state synchronization, failover
- **Backup, Licensing & Upgrade** — System backup/restore, UCS files, license management, software upgrade procedures
- **Troubleshooting & Diagnostics** — Log analysis, tmsh commands, performance monitoring, connectivity diagnostics
- **Administrative Tasks** — Authentication methods, SNMP, syslog, NTP, sysctl settings

*Source:* [F5 201 TMOS Administration Blueprint ↗](https://www.f5.com/pdf/certification/exams/blueprint-tmos-administration.pdf)

## Transition notice (April 2025)

Effective **May 1, 2025**, F5 transitioned from the two-exam administrator path (101+201) to a five-exam model. Candidates who passed 101 between April 2023 and April 30, 2025 may still take and pass 201 to earn the F5 Certified Administrator, BIG-IP credential. After April 30, 2025, the 201 exam is only available to those who previously passed the 101. New candidates must pursue the five-exam certification path instead. Consult [F5 Education Services](https://education.f5.com/learning-path/view/9) for current eligibility.
