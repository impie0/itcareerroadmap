## About

The ServiceNow Certified Application Specialist - Performance Analytics (CAS-PA) is an advanced professional credential validating deep expertise in ServiceNow's performance measurement and analytics platform. Launched circa 2018 and updated quarterly to align with ServiceNow releases, it builds on the foundational CSA (Certified System Administrator) to certify hands-on mastery of Performance Analytics (PA) module components: KPI indicators, data collection jobs, breakdowns and aggregations, dashboard design, forecasting and thresholds, mobile analytics, and the Analytics Hub. Suited for ServiceNow functional consultants, performance analytics specialists, BI analysts, reporting engineers, and IT operations professionals responsible for designing and maintaining measurement systems. CSA prerequisite reflects the technical platform foundation required; PA Fundamentals course recommended; PA Advanced course optional but beneficial for hands-on scenario preparation.

## Topics covered

The ServiceNow CAS-PA exam blueprint covers the following domain areas:

- **Performance Analytics Architecture and Foundation (12%)** — PA module architecture, platform navigation, role-based access control (RBAC) for analytics users, PA user/developer workspace distinction, licensing, performance optimization.
- **Indicators (KPIs) and Measurement Design (22%)** — KPI definition and lifecycle, indicator creation and configuration, breakdowns (multi-dimensional slicing), aggregates and totals, time series data management, threshold configuration and notifications, indicator dependencies.
- **Data Collection and Integration (18%)** — Data collection job types (scheduled, on-demand, event-triggered), job scheduling and frequency, data validation and error handling, metric aggregation formulas, ETL concepts within PA, data quality and reconciliation.
- **Dashboards and Visualizations (18%)** — Dashboard creation and layout, widget types (gauge, pie, bar, line, trend, table, scorecard), widget configuration and drill-down, sharing and access control, widget interaction and filtering, best practices in data visualization for operations.
- **Forecasting, Targets, and Thresholds (12%)** — Forecast model setup, trend analysis, target vs. actual comparison, threshold and SLA breach detection, alert and notification automation, remediation action triggers.
- **Mobile Analytics (8%)** — Mobile-responsive dashboard design, touch-friendly widget interaction, mobile-optimized navigation, offline capability considerations, mobile user personas.
- **Advanced Features and Integration (10%)** — Analytics Hub integration, API access for external reporting, role and permission delegation, reporting best practices, troubleshooting common PA issues, data export and integration with external BI tools.

*Source:* [Official CAS-PA Exam Blueprint ↗](https://www.servicenow.com/content/dam/servicenow/other-documents/training/mainline-blueprint-cas-pa.pdf)

## Skills validated

*What the CAS-PA exam specifically tests, distinct from shared Professional-tier ITSM skills.*

- **Indicator (KPI) lifecycle management** — Creation, editing, publishing, lifecycle states, versioning, indicator families and hierarchies.
- **Breakdown configuration** — Multi-dimensional slicing, hierarchy definition, drill-down paths, breakdown performance optimization.
- **Data collection job design** — Scheduled jobs, event-triggered jobs, job frequency and execution, error handling and retry logic, data validation rules.
- **Metric aggregation formulas** — Sum, average, count, percentile, custom formula syntax, formula dependencies and optimization.
- **Dashboard and widget design** — Dashboard layouts, widget types and appropriate use cases, widget filtering and interactions, drill-down configuration, sharing and governance.
- **Threshold and forecasting** — Threshold configuration with breach conditions, SLA and target comparison, trend forecasting, anomaly detection.
- **Mobile analytics** — Touch-friendly widget design, mobile dashboard optimization, mobile user role configuration.
- **Performance Analytics Hub** — Hub navigation, report access, mobile app consumption, role and permission delegation.
- **Data integration** — Data source configuration, external API integration, data quality reconciliation, export to BI tools.
- **Analytics security and access control** — Role-based access to indicators and dashboards, user and group assignment, data visibility rules.
