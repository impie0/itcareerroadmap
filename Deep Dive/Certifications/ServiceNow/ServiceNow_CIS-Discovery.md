## About

The ServiceNow Certified Implementation Specialist - Discovery (CIS-Discovery) is a professional-level credential for IT Operations Management specialists who design and implement automated discovery of IT infrastructure assets. Launched circa 2014 and updated three times yearly to align with ServiceNow release cycles, it validates expertise in MID Server configuration, discovery pattern design, probe and sensor behavior, CMDB reconciliation logic, credential management, IP-based and cloud discovery techniques, service mapping integration, and troubleshooting of discovery pipelines. Prerequisite: CSA (foundational) plus hands-on Discovery Implementation course. Suitable for ServiceNow ITOM engineers, CMDB specialists, infrastructure automation consultants, and solution architects leading discovery initiatives in mid-to-large IT environments.

## Domain context — IT Operations Management (ITOM)

Automated discovery, mapping, and management of IT infrastructure across physical, cloud, and hybrid environments. ServiceNow Discovery module: MID Server agents, discovery patterns, CI/service mapping, CMDB data population, reconciliation workflows, IP-based scanning, cloud connector discovery (AWS/Azure/GCP), credential and schedule management, performance tuning.

## Topics covered

The official ServiceNow CIS-Discovery exam blueprint covers these domain areas:

- **Discovery Architecture & MID Servers** — MID Server deployment, clustering, high availability, networking, credential management, probe execution environment.
- **Discovery Patterns & Probe Configuration** — Pattern selection and design, probe behavior, sensor execution, trigger conditions, map rules, CI class mapping.
- **Credentials & Authentication** — Credential storage, multi-hop authentication, SSH/SNMP/WMI/API credential setup, credential reuse across patterns.
- **Discovery Schedules & Execution** — Schedule creation, frequency tuning, exclusion windows, priority queuing, impact on network and CMDB.
- **IP-based Discovery** — Network scanning (NMAP, ICMP), service enumeration, OS detection, port analysis, VLAN and subnet discovery.
- **Cloud Discovery** — AWS, Azure, GCP connector setup, instance/resource discovery, cloud service mapping, cross-cloud environments.
- **Service Mapping & CMDB Integration** — Service-to-CI relationships, business service definition, dependency mapping, CMDB reconciliation workflows.
- **CMDB Population & Reconciliation** — Duplicate detection and merging, identification rules, reconciliation logic, data quality and deduplication.
- **Troubleshooting & Performance** — Discovery logs analysis, probe debugging, slow pattern detection, network path tracing, common failure modes.

## Skills validated

*What the CIS-Discovery exam specifically tests, distinct from shared ITOM · Professional skills.*

- **MID Server architecture and deployment** — Installation, clustering, high availability, networking prerequisites, probe environment setup.
- **Discovery pattern design and selection** — Pattern catalog navigation, probe behavior prediction, sensor execution, rule mapping, CI class alignment.
- **Credential management** — Secure credential storage, multi-hop authentication chains, SSH/SNMP/WMI/API credentials, credential validation and reuse.
- **Discovery schedules and optimization** — Schedule creation, frequency tuning, network impact assessment, CMDB load balancing.
- **IP-based discovery techniques** — NMAP scanning, ICMP sweep, service enumeration (port scanning), OS fingerprinting, VLAN/subnet mapping.
- **Cloud discovery integration** — AWS / Azure / GCP connectors, instance discovery, tag-based filtering, cross-cloud asset correlation.
- **Service mapping and dependency discovery** — Service-to-CI relationships, business service definition, dependency chaining, impact analysis.
- **CMDB reconciliation and data quality** — Duplicate detection algorithms, identification rules, reconciliation workflows, data merge/split operations, CI lifecycle.
- **Discovery troubleshooting and analysis** — Probe log interpretation, discovery failure diagnosis, slow pattern detection, network path analysis, common error resolution.
