## About

The Google Cloud Professional Cloud Developer certification validates expertise in building highly scalable, secure, and reliable cloud-native applications on Google Cloud Platform. This certification demonstrates proficiency with application design patterns, deployment across multiple GCP platforms (App Engine, Cloud Run, GKE, Cloud Functions), CI/CD pipelines, and integration of GCP services. It is ideal for software engineers and developers who design and deploy production applications on GCP.

## Domain context — Cloud

Hyperscale public cloud (GCP). Part of the Google Cloud certification track for cloud application specialists.

## Topics covered

**Exam Domain Distribution:**

- **Designing Cloud-Native Applications** (~25%) – Design scalable, highly available, disaster-resistant applications; evaluate architectural patterns; optimize for performance and cost
- **Building and Testing Applications** (~25%) – Develop cloud-native code; implement error handling and logging; create and run unit and integration tests; optimize code performance
- **Deploying Applications** (~20%) – Deploy to various GCP platforms (App Engine, Cloud Run, GKE, Cloud Functions); manage versions and traffic; implement blue-green and canary deployments
- **Integrating Google Cloud Services** (~20%) – Integrate messaging (Pub/Sub), APIs, databases (Firestore, Spanner), authentication (IAM, OAuth); manage service discovery
- **Managing Application Performance and Reliability** (~10%) – Monitor applications; implement observability; debug and optimize; manage incidents; apply security practices

**Key Technologies Covered:**
- Cloud Run (serverless containers)
- App Engine (managed platform)
- Google Kubernetes Engine (GKE)
- Cloud Functions (event-driven serverless)
- Cloud Build (CI/CD automation)
- Artifact Registry (container registry)
- Cloud Deploy (deployment automation)
- Cloud Pub/Sub (event streaming)
- Firestore, Spanner (databases)
- Firebase (mobile/web backend)
- Cloud Monitoring, Cloud Logging, Cloud Trace, Cloud Debugger
- IAM, Cloud KMS (security)
- Service Mesh (Istio), networking

## Common job-ready skills

- Design cloud-native, scalable applications using GCP services
