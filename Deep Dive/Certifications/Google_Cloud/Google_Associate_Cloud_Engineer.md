## About

The Google Cloud Associate Cloud Engineer (ACE) certification validates foundational hands-on skills in deploying, managing, and operating GCP infrastructure. Released circa 2018, it remains Google's primary entry-level cloud cert, bridging the Cloud Digital Leader (CDL) foundational level and the Professional Cloud Architect (PCA) expert tier. ACE holders demonstrate competency across Compute Engine VMs, Cloud Storage, networking, IAM, and operational monitoring—the core pillars of self-service cloud administration. The exam is actively maintained with quarterly content reviews by Google; no retirement is announced.

## Topics covered

The exam weights knowledge across five major domains (official Google blueprint):

- **Setting up a cloud solution environment** (~17%)
  - Creating/managing GCP projects and billing accounts
  - Enabling/disabling APIs
  - Managing service accounts and IAM bindings
  
- **Planning and configuring a cloud solution** (~17%)
  - Selecting appropriate Compute Engine machine types
  - Planning storage architecture (Cloud Storage buckets, object lifecycle)
  - Designing networks (VPCs, subnets, routes, firewalls)
  
- **Deploying and implementing a cloud solution** (~25%)
  - Launching and configuring Compute Engine VMs
  - Creating and managing Cloud Storage buckets + objects
  - Deploying App Engine, Cloud Run, Cloud Functions applications
  - Setting up Cloud SQL, Cloud Datastore, Cloud Firestore databases
  - Creating managed instance groups and autoscaling
  
- **Ensuring successful operation of a cloud solution** (~17%)
  - Monitoring VMs and applications (Cloud Monitoring, Logging)
  - Managing instance groups and load balancers
  - Performing backups and snapshots
  - Troubleshooting common GCP service failures
  
- **Configuring access and security** (~24%)
  - Managing IAM roles and custom roles
  - Configuring service accounts and key rotation
  - Setting up Cloud VPN, Cloud Interconnect
  - Implementing Cloud Armor and DDoS protection
  - Encrypting data at rest and in transit

*Source:* [Google Cloud ACE Exam Guide](https://cloud.google.com/learn/certification/guides/cloud-engineer)

## Skills validated

*Specific to the ACE exam—what hands-on capabilities the exam measures.*

- **Compute Engine:** VM creation, lifecycle management, custom images, instance groups, autoscaling, SSH key setup
- **Cloud Storage:** Bucket creation, object versioning, lifecycle policies, access controls, signed URLs, Cloud CDN integration
- **Networking:** VPC creation, subnets, static/dynamic routing, firewall rules, Cloud NAT, VPN / Interconnect concepts
- **Cloud SQL & Firestore:** Database instance provisioning, backups, replication, user access control, basic query patterns
- **IAM:** Service accounts, custom roles, bindings, key rotation, understanding principle of least privilege
- **Monitoring & Logging:** Cloud Monitoring dashboards, alerting policies, Cloud Logging log sinks, basic troubleshooting
- **App Engine & Cloud Functions:** Deploying standard / flexible environments, scaling, traffic splitting, function triggers
- **Infrastructure-as-Code:** Deployment Manager templates, Terraform basics (reading/modifying manifests)
- **Security:** Cloud Armor, DDoS protection, VPC Service Controls, encryption key management (Cloud KMS)
- **Cost management:** Budgets, billing APIs, resource labeling for cost attribution
