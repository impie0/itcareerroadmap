## About

The Professional Cloud Architect (PCA) certification validates advanced expertise in designing, planning, and managing Google Cloud solutions for complex, multi-tier enterprise environments. Launched in 2018 and continuously updated, this professional-level cert represents the pinnacle of Google Cloud architecture certifications. Candidates demonstrate mastery across infrastructure design, security/compliance architecture, cost optimization, and operational reliability. The exam includes **five real-world case studies** (Mountkirk Games, Helicopter Racing League, Dress4Win, EHR Healthcare, TerramEarth — updated 2026) that test decision-making under constraints. PCA is the credential most directly comparable to AWS Solutions Architect Professional (SAP-C02) and Microsoft Azure Solutions Architect Expert (AZ-305) in scope and difficulty.

## Topics covered

Exam blueprint (official exam guide, 2026 release):

- **Designing and planning cloud architecture** (~24%) — Design patterns, reference architectures, multi-tier deployment, hybrid/multi-cloud integration, Google Cloud services selection, API design.
- **Managing and provisioning solution infrastructure** (~15%) — Compute (GCE, GKE, App Engine, Cloud Run), networking (VPC, load balancing, CDN), storage (Cloud Storage, Firestore, BigTable, SQL).
- **Designing for security and compliance** (~18%) — Identity/access (IAM, service accounts, Workload Identity), data protection (encryption, DLP, Secret Manager), audit logging, regulatory compliance (HIPAA, SOC 2, PCI-DSS).
- **Analyzing and optimizing technical and business processes** (~18%) — Cost optimization (Commitment Discounts, Rightsizing, Labels), performance tuning, SLO/SLI design, monitoring/alerting (Cloud Monitoring, Cloud Logging).
- **Managing implementation** (~11%) — Infrastructure as Code (Terraform), deployment pipelines, change management, rollback strategies.
- **Ensuring solution and operations reliability** (~14%) — Disaster recovery, backup/restore, high availability architecture, operational runbooks, incident response.

## Case studies (exam scenarios)

Candidates must apply architecture knowledge to solve business problems for **five real-world case studies**:

1. **Mountkirk Games** — Gaming company; latency-sensitive global multiplayer. Tests: scaling, multi-region architecture, CDN, database decisions.
2. **Helicopter Racing League** — Real-time video streaming and telemetry. Tests: high-throughput Pub/Sub, BigQuery analytics, live streaming infrastructure.
3. **Dress4Win** — E-commerce platform migration from on-prem. Tests: hybrid deployment, VM migration, database lift-and-shift, cost optimization.
4. **EHR Healthcare** — Healthcare provider managing HIPAA-regulated data. Tests: compliance architecture, data encryption, audit logging, access controls, multi-environment (dev/staging/prod).
5. **TerramEarth** — IoT fleet management (heavy machinery). Tests: edge computing, time-series data handling (BigTable), high-volume ingestion (Dataflow), analytics at scale.

Each case study presents constraints (budget, latency, compliance, team skills) and requires candidates to prioritize tradeoffs.
