## About

The Google Cloud Professional Google Workspace Administrator certification (formerly G Suite Administrator) validated expertise in managing Google Workspace deployments, including Gmail, Drive, Meet, Calendar, and the Admin Console. This certification is **no longer available** as of 31 December 2024. Google has retired the Professional-level credential and replaced it with the **Associate Google Workspace Administrator** certification, which covers the same domains but at the Associate level. Professionals seeking Workspace administration credentials should pursue the Associate certification instead.

## Domain context — Cloud

Hyperscale public cloud (GCP). Part of the Google Cloud certification track for business/productivity administrators.

## Topics covered (Historical)

**Exam Domain Distribution (when active):**

- **Managing User Accounts and Settings** (~20%) – Create, manage, and deactivate user accounts; configure user account settings; manage organizational units; implement user provisioning and deprovisioning workflows
- **Core Workspace Services** (~22%) – Manage Gmail, Drive, Calendar, Meet, Chat, Classroom, and other core services; configure service settings; enable/disable services; manage sharing and permissions
- **Data Governance and Compliance** (~18%) – Implement data retention policies; configure eDiscovery and vault features; manage data export; ensure GDPR, HIPAA, and other compliance requirements
- **Security Policies** (~20%) – Configure security settings; manage authentication and 2-factor authentication; implement security protocols; manage device policies; configure endpoint management
- **Managing Endpoints** (~10%) – Manage mobile devices and endpoints using MDM; configure device compliance; manage app policies
- **Troubleshooting** (~10%) – Troubleshoot common issues; analyze audit logs; diagnose service problems; use Admin Toolbox; manage support cases

**Key Topics Covered (when active):**
- Google Workspace Admin Console
- User and organizational structure management
- Gmail administration and security
- Google Drive and file sharing policies
- Calendar and video conferencing (Meet)
- Mobile device management (MDM)
- Security keys and multi-factor authentication (MFA)
- Data loss prevention (DLP)
- Audit logs and monitoring
- Recovery and business continuity

## Status: Retired

**This certification was retired on 31 December 2024 and is no longer available for new candidates.**

Professionals who previously earned this credential retain their certification until its 2-year validity period expires. However, new candidates should pursue the **Associate Google Workspace Administrator** certification instead, which is the current Google Cloud offering for Workspace administration.
