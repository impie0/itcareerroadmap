## About

The NCIE–SAN Specialist, ONTAP certification validates that you can plan, design, implement, integrate, and troubleshoot NetApp ONTAP SAN environments spanning FC, FCoE, iSCSI, and NVMe protocols across heterogeneous host operating systems (Windows, Linux, ESXi, UNIX). This is a professional-level credential, second-tier in NetApp's hierarchy, requiring foundational NCDA certification and 6–12 months of real SAN implementation experience. The current exam code NS0-521 replaced earlier versions (NS0-519, NS0-520) and is actively maintained as of 2026.

---

## Domain context — Storage / SAN

Enterprise-scale block storage protocol design, configuration, and optimization for Fibre Channel, iSCSI, and converged fabrics. SAN engineers specialize in zoning, LUN mapping, multipath I/O, performance tuning, and disaster recovery in production environments.

## Topics covered

- Planning and requirements analysis (site surveys, capacity, performance targets)
- ONTAP SAN architecture and dual-controller high-availability concepts
- FC (Fibre Channel) protocol, zoning, fabric design, ALUA (Asymmetric Logical Unit Access)
- FCoE (Fibre Channel over Ethernet) configuration on unified switches
- iSCSI configuration, authentication (CHAP), network optimization, igroup creation
- NVMe over FC (NVMe/FC) and NVMe over TCP (NVMe/TCP) protocol fundamentals
- LUN creation, mapping, igroup management, and WWPN / IQN configuration
- Snapshot policies, SnapDrive, and SnapManager for application integration
- Host operating system integration (Windows MPIO, Linux multipath, ESXi ALUA)
- SAN performance tuning, thin provisioning, deduplication, and compression
- Data protection and disaster recovery (SnapMirror, MetroCluster basics)
- Troubleshooting common SAN issues (connectivity, zoning, path availability)
