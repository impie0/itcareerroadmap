## About

The NetApp Certified Hybrid Cloud Administrator (NCHCA) certification validates expertise in managing and administering hybrid cloud environments that integrate on-premises and cloud-based NetApp storage solutions. This certification demonstrates proficiency in deploying, managing, and operating NetApp hybrid cloud technologies across both cloud and on-premises environments.

The NCHCA is designed for IT professionals with significant hands-on experience managing NetApp cloud solutions and demonstrates readiness for senior storage administration roles in hybrid cloud environments. This certification is ideal for professionals managing multi-cloud deployments leveraging NetApp technologies.

## Domain context

The NCHCA certification focuses on hybrid cloud storage architecture and administration:

- **ONTAP Foundation**: On-premises ONTAP cluster administration
- **Cloud Volumes ONTAP**: Cloud-based ONTAP instances (AWS, Azure, GCP)
- **Cloud Volumes Service**: Fully managed cloud storage
- **Cloud Manager/BlueXP**: Cloud management and orchestration
- **Data Replication**: SnapMirror, Cloud Sync, replication management
- **Hybrid Data Protection**: Backup, recovery, data tiering
- **Performance Optimization**: Cloud storage optimization
- **Compliance & Governance**: Data residency, regulatory requirements
