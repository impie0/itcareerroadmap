## About

The NetApp Certified Associate — Data Administrator (NCDA) is a vendor-specific certification that validates hands-on competency in administering NetApp ONTAP clustered storage systems. Candidates must demonstrate proficiency in storage platform management, ONTAP core concepts, data protection, security, networking, and performance optimization. The certification is designed for IT storage professionals, systems administrators, and data engineers with practical experience in multiprotocol storage environments. The current exam (NS0-163) was updated to reflect modern ONTAP cluster features and is aligned with NetApp's broader certification ecosystem spanning specialist and professional levels.

## Topics covered

The NCDA exam covers eight primary domains (no official weights published by NetApp):

- **Storage Platforms** — NetApp FAS and AFF hardware architectures, cluster management, and ONTAP licensing
- **Core ONTAP** — ONTAP system management, Storage Virtual Machines (SVMs), high availability (HA) configurations, and cluster setup
- **ONTAP Storage** — logical storage provisioning, volumes, LUNs, data efficiency features (deduplication, compression), and QoS
- **Networking** — ONTAP-relevant network configuration, LAG/LACP, VLAN tagging, network troubleshooting
- **Storage Protocols and Connectivity** — NFS, SMB/CIFS, iSCSI, Fibre Channel, and S3 protocol implementation and troubleshooting
- **Data Protection** — SnapVault, SnapMirror, Snap Lock, backup strategies, and disaster recovery concepts
- **Security** — ONTAP hardening, role-based access control (RBAC), SSL/TLS, encryption at rest and in transit, anti-ransomware
- **Performance** — performance monitoring, bottleneck identification, workload optimization, and tuning
