## About

The Salesforce Certified Platform App Builder certification validates the ability to build custom applications on the Salesforce platform using point-and-click declarative tools (not code). Candidates demonstrate competency in data modeling, business logic automation, user interface design, and process automation. It is the foundational associate-level cert for Salesforce developers and configuration specialists, released in 2013 and remains the entry point for hands-on Salesforce builders. Most holders move toward either Salesforce Administrator (dual cert) or Platform Developer I (PD I) for advanced declarative and programmatic skills.

## Topics covered

Official exam blueprint weights (as of 2026):

- Salesforce Fundamentals (8%) — org structure, licensing, Salesforce UI basics
- Data Modeling and Management (20%) — objects, fields, relationships, data integrity, schema design
- Security (10%) — organization security, field-level security, record-level security (sharing, OLS)
- Business Logic and Process Automation (27%) — workflows, flows, validation rules, triggers, automation
- Social (3%) — Chatter, collaboration features, feed tracking
- User Interface (14%) — page layouts, record types, lightning components, page customization

- Mobile (5%) — Salesforce mobile app, responsive design
- App Deployment (8%) — change sets, sandboxes, managed packages, release management

## Skills validated

*Concrete capabilities this exam tests, distinct from general CRM / Low-Code Associate skills.*

- **Salesforce objects and fields** — creating custom objects, field types (text, number, lookup, formula, roll-up), field validation
- **Relationships** — master-detail, lookups, junctions, understanding cascade behavior
- **Sharing and security** — org-wide defaults, role hierarchies, sharing rules, field-level security, permission sets
- **Workflows and Flow Builder** — declarative automation, scheduled flows, approval processes, email alerts
- **Validation rules** — APEX formula-free validation logic
- **Page layouts and record types** — conditional page layouts, field-level customization, record-type-specific layouts
- **Lightning components and Aura** — understanding component architecture, embedding custom components on pages
- **Dashboards and reports** — report types, report builder, dashboard creation and refresh
- **Change management** — sandboxes, change sets, deployment best practices
- **Mobile app customization** — responsive design, Salesforce mobile considerations
