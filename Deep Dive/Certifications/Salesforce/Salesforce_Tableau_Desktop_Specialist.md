## About

The Tableau Desktop Specialist certification validates hands-on competency with Tableau Desktop, the primary self-service BI tool in the Salesforce ecosystem. Launched in 2019 after Salesforce's acquisition of Tableau, the entry-level TDS credential covers data connection, field manipulation, chart type selection, dashboard design, and publishing. It is the gateway to advanced Tableau certifications (Certified Associate Architect, Certified Professional Architect) and a core credential for data analysts, junior BI developers, and business analysts adopting Tableau for analytics.

## Topics covered

- **Connecting to & Preparing Data (~30%)** — Live vs. extract connections, file formats (CSV, Excel, SQL databases, cloud connectors), authentication, removing/adding fields, calculated fields, grouping and binning, data type assignment, parameters
- **Exploring & Analyzing Data (~30%)** — Dimension vs. measure, continuous vs. discrete, aggregation functions, filtering (quick, conditional, context), sorting, date hierarchies, trend analysis, basic statistical interpretation
- **Understanding Tableau Concepts (~20%)** — Chart type selection by data shape, formatting marks (size, color, label, detail), dual-axis charts, shelves and encoding, reference lines and bands
- **Sharing Insights (~20%)** — Dashboard design (layout, filters, dashboard actions, interactive elements), publishing to Tableau Server / Tableau Public, permissions, scheduled refreshes, email subscriptions

*Source:* [Tableau Certification Objectives ↗](https://www.tableau.com/learn/certification/desktop-specialist)

## Skills validated

- **Data connectivity:** Live and extract connections, authentication, connection troubleshooting
- **Data preparation:** Calculated fields, binning, grouping, date hierarchies, parameters, field renaming
- **Visualization design:** Mark types, channels (size, color, label), dual-axis charts, trend lines, reference lines
- **Dashboard creation:** Filtering, dashboard actions, worksheet-level vs. dashboard-level filters, layout containers
- **Data analysis:** Aggregation choices, sorting, filtering by criteria, time-series analysis
- **Publishing & sharing:** Permissions, data source refresh, scheduled extracts, content organization
- **Troubleshooting:** Chart interpretation, debugging null values, performance optimization of extracts
