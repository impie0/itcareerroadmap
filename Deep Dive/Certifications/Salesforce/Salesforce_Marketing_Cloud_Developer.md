## About

The Salesforce Certified Marketing Cloud Developer certification validates expertise in building, deploying, and maintaining custom solutions within Salesforce Marketing Cloud. This Professional-level credential is for developers who write server-side and client-side code (AMPscript, SSJS, SQL) to extend Marketing Cloud capabilities beyond standard features—custom activities, Cloud Pages, preference centers, subscriber management automation, and API integrations. Released in 2021, it positions developers as technical practitioners capable of translating business requirements into programmatic solutions. Requires foundational Marketing Cloud knowledge (Email Specialist or Engagement Consultant certification).

---

## Domain context — Software Engineering / Marketing Cloud

Marketing Cloud Developer sits at the intersection of marketing technology and software engineering: it combines programmatic marketing automation with full-stack web development. Developers design data models, write cloud-based scripts, build REST/SOAP API integrations, and create customer-facing interactive content. Unlike the Engagement Consultant role (architecture/strategy), the Developer role is hands-on execution and custom code ownership.

## Topics covered

Exam blueprint breakdown (per Salesforce Trailhead):

- **AMPscript Fundamentals** (18%) — syntax, variables, functions, control flow, string/date manipulation, personalization logic
- **Server-Side JavaScript (SSJS)** (16%) — HTTP requests, JSON handling, Platform.request() patterns, error handling, async operations
- **SQL and Data Management** (15%) — data extension queries, joins, subqueries, subscriber data modeling, ETL logic
- **Marketing Cloud REST and SOAP APIs** (14%) — authentication, endpoint calls, rate limiting, sync/async operations, error handling
- **Cloud Pages and Web Development** (12%) — HTML/CSS/JavaScript, form handling, dynamic content rendering, file uploads, security
- **Custom Activities** (12%) — activity framework, drag-drop interactions, configuration JSON, event publishing
- **Preference Centers and Subscriber Management** (8%) — dynamic preference building, consent tracking, subscription updates, data governance
- **Integration Patterns and Best Practices** (5%) — data loading strategies, API rate-limit handling, logging, performance optimization

## Skills validated

*Concrete competencies this exam specifically measures.*

- AMPscript syntax, functions, and personalization logic (string, date, math operations)
- Server-side JavaScript (SSJS): HTTP requests, JSON parsing, async error handling
- Data extension modeling: primary keys, data relationships, attribute configuration
- SQL queries: SELECT, JOIN, WHERE, aggregate functions, subqueries in Marketing Cloud context
- Marketing Cloud REST API: authentication (OAuth2), common endpoints (create subscriber, upsert, retrieve), rate limiting, response handling
- Marketing Cloud SOAP API: authentication, endpoint invocation, fault handling
- Cloud Pages: HTML/CSS form creation, dynamic content rendering, file handling, JavaScript interactivity
- Custom Activities: event framework, configuration JSON schema, drag-drop interface, event publishing
- Preference Centers: dynamic question builders, subscriber consent tracking, data governance
- Content Blocks and Code Snippets: reusable template logic, library management
- Debugging: logs, error messages, API response validation
- Performance optimization: script efficiency, API call batching, data load strategies
- Security best practices: credential handling, input validation, XSS prevention
