## About

The Tableau Server Certified Associate credential validates hands-on competency in deploying, configuring, and administering Tableau Server in single-machine and distributed environments. Intended for IT professionals, system administrators, and BI operations staff with 4–6 months of practical Tableau Server experience. The exam covers installation, user/group management, permissions, site administration, monitoring, backup/restore, and troubleshooting. Valid for 2 years; recertification requires retaking the exam.

## Domain context — Business Intelligence & Data

Server administration for analytics platforms — installation, multi-user access control, performance tuning, security policy, and operational resilience of Tableau Server deployments.

## Topics covered

- **Tableau Server Architecture & Deployment** — Single-machine vs. distributed architectures, Linux/Windows environments, component dependencies
- **Installation & Configuration** — Server setup, repository configuration, licensing, port/firewall settings
- **User & Group Management** — User creation, groups, authentication methods (local, SAML, OIDC, LDAP)
- **Permissions & Access Control** — Project-level, workbook-level, data-source permissions; capability matrix
- **Site Administration** — Site creation/management, content ownership, data-source publishing, refresh schedules
- **Monitoring & Performance** — Activity monitoring, server logs, resource utilization, query performance
- **Backup & Disaster Recovery** — Full/incremental backup procedures, restore scenarios, high availability
- **Troubleshooting & Support** — Log interpretation, common failure modes, connectivity diagnostics
- **Data Sources & Connectivity** — Data-source configuration, refresh frequency, connection security (SSL/TLS)
- **Security Best Practices** — Password policies, SSO implementation, audit logging, content encryption
