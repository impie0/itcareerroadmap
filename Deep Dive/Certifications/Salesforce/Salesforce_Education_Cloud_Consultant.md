## About

The Salesforce Certified Education Cloud Consultant validates expertise in designing and implementing Education Cloud solutions that meet institutional requirements for recruitment, admissions, student success, and advancement. This professional-level credential is for consultants with customer-facing experience implementing Education Cloud in K-12 and higher-education environments. The exam was substantially revised in March 2023 to focus exclusively on the unified Education Cloud product platform (no longer separate K-12 and higher-education variants). Prerequisite: valid Salesforce Certified Administrator credential.

## Topics covered

The exam covers four equally weighted sections (25% each):

- **Education Cloud Data Model (EDA)** — Education Data Architecture objects (Account, Contact, Term, Course, Course Connection, Program Plan), entity relationships, record hierarchies, multi-tenant configuration
- **Admissions & Recruitment** — Admissions Connect application workflows, application review processes, decision management, term-based processing, recruitment integrations
- **Student Success** — Student Success Hub configuration, Early Alerts system, intervention workflows, academic performance tracking, advising tools, retention best practices
- **Administration & Governance** — Implementation methodology (plan, gather requirements, design, build, test, deploy, optimize), change management, data governance, security, compliance, integration with Sales Cloud / Service Cloud / Marketing Cloud
