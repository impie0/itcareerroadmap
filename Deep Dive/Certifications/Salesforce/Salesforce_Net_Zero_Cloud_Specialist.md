## About

The Salesforce Net Zero Cloud Accredited Professional certification validates your ability to implement Net Zero Cloud solutions that help enterprises track, analyze, and report environmental data for ESG compliance. This credential proves you can design and deploy configurations for GHG emissions tracking, manage sustainability data models, and support organizational ESG reporting requirements. Released in 2024, it is part of Salesforce's Professional certification tier for specialized cloud domains and is intended for implementation partners, functional consultants, and solution architects with Salesforce core competency.

## Topics covered

The exam blueprints covers three major domains:

- **Discovery (24%)** — Identify and prioritize use cases for Net Zero Cloud implementation; map business requirements to product features; understand calculation methodologies; align reporting boundaries with compliance frameworks.
- **Design (31%)** — Translate customer requirements into system configurations; design user experiences for different personas; set up analytics dashboards using CRMA and Tableau; configure approval processes and workflows; plan data governance.
- **Implementation (45%)** — Execute Net Zero Cloud deployments; configure GHG emissions tracking (Scope 1/2/3); integrate third-party data sources; manage data models; enable ESG reporting; support compliance with CDP, GRI, TCFD, SBTi frameworks; troubleshoot configuration issues.
