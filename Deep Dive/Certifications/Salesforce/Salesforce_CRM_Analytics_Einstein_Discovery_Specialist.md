## About

Salesforce Certified CRM Analytics and Einstein Discovery Consultant (Analytics-Con-201) is a professional-level certification for analytics professionals, consultants, and architects who design and implement data-driven insights using Salesforce's unified CRM Analytics platform (formerly Tableau CRM) and Einstein Discovery predictive AI. The certification validates hands-on expertise across datasets, data flows, recipes (ETL), lens creation, dashboard design, and predictive modeling. It bridges analytics infrastructure (data modeling, security, performance) with end-user experience (dashboard usability, storytelling). As of May 2026, this cert remains a core credential for Salesforce analytics specialists seeking to advance into architect or advisory roles, with documented salary premiums of $8K–$20K+ over baseline CRM admin roles.

## Topics covered

*Exam blueprint (6 core areas, with weightings from official Salesforce exam guide):*

- **Admin Configuration & Data Layer Management (17%)** — CRM Analytics org setup, security, data layer architecture, field types, entity relationships, custom fields, data modeling.
- **Data Integration & ETL (23%)** — Datasets, dataflows, data sources, connectors, transformations, field mappings, recipe creation, incremental refresh logic, scheduling.
- **Security & Access Management (16%)** — Sharing rules, predicate-based access, field-level security, RLS (row-level security), Salesforce data cloud integration, compliance.
- **Lens Creation & Analytics Dashboard Design (13%)** — Lens building (KPI, table, gauge, chart types), interactions, drill-downs, filters, sorting, best practices for exploratory analytics.
- **Analytics Dashboard Implementation & Optimization (19%)** — Dashboard layout, page design, performance tuning, embedded analytics, mobile responsiveness, user adoption patterns.
- **Einstein Discovery & Predictive Modeling (12%)** — Recipe creation, model training, accuracy metrics, SAQL (Salesforce Analytics Query Language) fundamentals, prediction insights, action recommendations.

## Skills validated

*Concrete, testable capabilities this exam assesses — distinct from general "analytics professional" skills.*

- Designing and implementing data layers from relational or cloud data sources (multi-source connectors, field mapping, relationship modeling).
- Creating and optimizing dataflows and recipes for incremental ETL and data transformation (SOQL-to-CRM Analytics pipeline).
- Applying row-level and field-level security to protect sensitive customer data in analytics contexts.
- Authoring multi-type lenses (KPI, table, chart, gauge) with drill-down and interactive filtering.
- Designing responsive dashboards for web and mobile with performance optimization.
- Configuring Einstein Discovery recipes to train predictive models on cleaned datasets.
- Interpreting model quality metrics (R², confusion matrix, feature importance) and selecting appropriate algorithms.
- Writing SAQL (Salesforce Analytics Query Language) for custom queries and advanced calculations.
- Troubleshooting data refresh failures, performance bottlenecks, and access issues.
- Implementing embedded analytics and Salesforce Data Cloud integration for unified customer profiles.
