## About

The Salesforce Accredited B2B Commerce Developer credential validates professional-level expertise in designing and implementing B2B Commerce solutions on Salesforce. This certification targets experienced developers (system integrators, consultants, customer employees) who build custom storefronts, configure complex pricing and checkout flows, integrate payments and external systems, and optimize storefront performance using Lightning Web Components and Apex. Accredited Professional tier launched in 2023 as part of Salesforce's tiered certification model, positioning it above foundational knowledge and reflecting hands-on production experience.

---

## Domain context — B2B Commerce

B2B Commerce is a specialized Salesforce product line for business-to-business digital selling, spanning complex pricing models, multi-tenant subscriptions, custom catalogs, and buyer-specific ordering experiences. Distinct from B2C Commerce Cloud (consumer ecommerce), B2B Commerce is tightly integrated with Sales Cloud, Service Cloud, and Order-to-Cash workflows, making it relevant for enterprise digitalization and omnichannel sales strategies.

## Topics covered

- **B2B Commerce Architecture & Data Model** — Product catalog, inventory, pricing tiers, buyer groups, inventory allocation, order-to-cash integration
- **Custom Storefront Development** — Lightning Web Component (LWC) design patterns, component lifecycle, state management, API integration
- **Cart & Checkout Flows** — Cart data models, checkout step customization, cart validation, order placement, cart pricing calculations
- **Pricing & Quote Management** — Tiered pricing, volume discounts, contract pricing, quote generation, pricing rule configuration
- **Payment & Order Integration** — Payment gateway integration, order creation, order status management, external system connectors
- **Inventory & Fulfillment** — Inventory tracking, allocation logic, backorder handling, fulfillment workflows
- **Security & Permissions** — Store roles, buyer group access control, data visibility, compliance with B2B security patterns
- **Error Handling & Diagnostives** — Debugging LWC components, Apex logging, transaction flow analysis
- **Reference Implementations & APIs** — Salesforce Commerce Cloud APIs, reference store projects, sample configurations
