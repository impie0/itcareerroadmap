## About

The Salesforce Certified Tableau Architect certification validates expertise in designing, deploying, monitoring, and maintaining scalable Tableau infrastructure at enterprise scale. Intended for experienced professionals who lead Tableau Server deployments or Tableau Cloud migrations, this Expert-level credential demonstrates hands-on knowledge of multi-node architectures, performance optimization, data governance, backup/restore procedures, site management, and platform migration strategies. Launched as part of Salesforce's portfolio following the acquisition of Tableau, it remains the gold standard for infrastructure architects in the business intelligence space.

## Topics covered

The exam evaluates expertise across these architectural domains (weights approximate based on official blueprint):

- **Design a Tableau Infrastructure** (22%) — Architecture planning, scalability, high availability, disaster recovery
- **Deploy Tableau Server** (15%) — Multi-node setup, clustering, SSL/TLS, identity store integration
- **Design Tableau Cloud migrations** (18%) — Tableau Cloud architecture, Tableau Bridge configuration, migration planning
- **Site and Project Hierarchy** (12%) — Site administration, content governance, permissions, user provisioning
- **Monitor and Maintain** (18%) — Tableau Resource Monitor, performance tuning, logging, upgrade planning
- **Backup, Restore, and Disaster Recovery** (8%) — Data protection, failover, recovery procedures
- **Tableau REST API and Embedded Analytics** (7%) — API design, authentication, embedded workbooks

*Source:* [Salesforce Tableau Architect Exam Syllabus ↗](https://www.analyticsexam.com/salesforce/salesforce-analytics-arch-201-certification-exam-syllabus)
