## About

The Salesforce Certified Marketing Cloud Administrator (MCAA) certification validates expertise in setting up, configuring, and managing Salesforce Marketing Cloud platforms and accounts. This Associate-level credential, released in 2015, is designed for IT professionals, operations engineers, and administrators responsible for Marketing Cloud account infrastructure, user management, data architecture, and technical administration. Unlike the Marketing Cloud Email Specialist (MCES), which focuses on email campaign execution, the MCAA emphasizes platform governance, setup, and cross-channel administration including email, mobile push, social, and advertising integrations.

---

## Domain context — Marketing Cloud

Salesforce Marketing Cloud is a customer experience marketing platform spanning email, SMS/mobile push, social, display advertising, and journey orchestration. The Administrator credential covers account setup, multi-cloud integration, data architecture, user and role management, security, and technical governance across all channels.

## Topics covered

Exam blueprint breakdown (per Salesforce Trailhead):

- **Marketing Cloud Account Setup (15%)** — Multi-cloud setup, business units, account hierarchy, feature licensing, brand domains.
- **User and Role Management (12%)** — User provisioning, role-based permissions, delegated administration, two-factor authentication.
- **Data Management Setup (18%)** — Data extensions, attributes, import workflows, data retention policies, data governance.
- **Subscriber Data Architecture (15%)** — Subscriber profiles, attributes, list design, mobile subscribers, preference centers.
- **Account Tracking and Reporting (12%)** — Tracking tags, analytics, reporting setup, custom reporting objects, data warehouse.
- **Sender Authentication (7%)** — DKIM, SPF, brand domains, CNAME setup, authentication package (SAP).
- **Mobile Push & SMS Configuration (10%)** — Mobile Connect setup, SMS administration, channel configuration, app management.
- **Sender Reputation and Deliverability (6%)** — IP warming, bounce handling, suppression lists, complaint monitoring, sender score.
- **Marketing Cloud APIs Fundamentals (5%)** — API authentication, REST/SOAP overview, integration patterns, data sync basics.
