## About

The Oracle Cloud Infrastructure 2025 Security Professional certification validates expertise in designing, implementing, and managing comprehensive security strategies across OCI environments. Aimed at cloud security professionals, security architects, and infrastructure engineers, this exam tests hands-on capability and theoretical knowledge across identity management, network security, data protection, threat detection, and compliance monitoring. The certification launched as part of Oracle's 2025 cloud credential updates and serves security teams adopting or expanding within Oracle's hyperscale public cloud platform.

## Topics covered

- **OCI Security Fundamentals** — Shared Security Responsibility model, security design principles, core OCI security services overview
- **Identity and Access Management (IAM)** — IAM domains, user and group management, policy design, dynamic groups, network sources, tag-based access control, multi-factor authentication, sign-on policies
- **Network and Application Security** — Virtual Cloud Networks (VCN), Network Security Groups (NSGs), Security Lists, Network Firewall configuration, load balancing, SSL/TLS certificate management, Web Application Firewall (WAF)
- **OS and Workload Protection** — Time-limited access via OCI Bastion, vulnerability scanning, Container Image scanning
- **Data Security** — Database encryption, Transparent Data Encryption (TDE), Data Safe features, encryption key management
- **Security Monitoring and Compliance** — Cloud Guard, Security Zones, Security Advisor, OCI Monitoring, Logging, Audit, Event Services, threat detection, compliance reporting

## Common skills at Cloud Security · Professional

- Design zero-trust network architectures and apply least-privilege access controls
- Implement identity federation and single sign-on (SSO) across hybrid and multicloud environments
- Deploy and manage encryption at rest and in transit; rotate keys per compliance requirements
- Configure cloud-native SIEM and threat detection; orchestrate response workflows
- Conduct security posture assessments and remediate cloud infrastructure misconfigurations
- Build security compliance frameworks aligned with regulatory mandates (SOC 2, PCI-DSS, HIPAA)
- Perform threat modeling and risk assessment for cloud workloads
- Manage cloud security budgets and vendor risk
