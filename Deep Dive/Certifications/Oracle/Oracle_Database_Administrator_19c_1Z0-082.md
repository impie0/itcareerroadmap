## About

The Oracle Database Administration I (1Z0-082) certification validates fundamental and intermediate skills in managing Oracle Database 19c, including instance configuration, user and privilege management, storage allocation, schema object lifecycle, and data dictionary operations. This Associate-level exam establishes readiness for database administrator roles and serves as the prerequisite pathway to the Professional-level OCP Administrator certification (1Z0-083). The exam reflects 19c release features and is actively maintained by Oracle University.

## Skills validated

*What this exam specifically measures — distinct from the shared "Common skills" above.*

- Oracle Database architecture components (SGA, PGA, processes, listener)
- Instance startup, shutdown, and parameter configuration
- User creation, role assignment, and system/object privilege management
- Tablespace, datafile, and storage management
- Table, view, index, sequence, and synonym creation and modification
- Data dictionary views (DBA_, USER_, ALL_ prefixes) and dynamic performance views
- Oracle-specific SQL concepts (ROWNUM, pseudocolumns, flashback queries)
- Backup and recovery fundamentals (RMAN architecture)
- Security policies and audit trail configuration
