## About

The Oracle Cloud Infrastructure 2025 Architect Associate (1Z0-1072-25) validates foundational and practical expertise in designing, deploying, and managing infrastructure solutions on Oracle Cloud Infrastructure (OCI). This is Oracle's entry-level cloud architect certification, ideal for professionals transitioning to cloud roles or validating hands-on OCI experience. The exam emphasizes Compute, Networking, Storage, and Identity & Access Management (IAM)—the core services underpinning most OCI architectures.

## Topics covered

The exam blueprintassesses the following domains and approximate weights:

- **Plan and Design Solutions on OCI** (~30%) — Selecting compute shapes, network topology, storage architecture, and security posture for business requirements
- **Implement Infrastructure as a Service (IaaS) Solutions** (~25%) — Deploying bare-metal and virtual compute instances, configuring load balancers, implementing auto-scaling
- **Build Cloud-Native Solutions** (~10%) — Designing containerized and microservices architectures on OCI
- **Implement Security and Governance** (~15%) — Configuring IAM policies, compartments, audit logging, and encryption
- **Migration and Storage Solutions** (~10%) — Selecting and provisioning block storage, object storage, file storage; designing backup and disaster recovery
- **Observability and Disaster Recovery** (~10%) — Implementing monitoring, logging, high-availability strategies, and recovery procedures

## Skills validated

*This exam assesses practical, hands-on capability in these OCI-specific technologies:*

- **Compute:** Bare Metal Instances, Virtual Machine Shapes, Instance Families, Auto Scaling
- **Networking:** Virtual Cloud Networks (VCNs), Subnets, Security Lists, Network Security Groups, Internet Gateway, NAT Gateway, VPN, FastConnect, Load Balancers, Traffic Management
- **Storage:** Block Storage Volumes, Object Storage (Buckets), File Storage (NFS), Archive Storage, Snapshots, Backup strategies
- **Identity & Access Management:** Compartments, IAM Policies, Groups, Dynamic Groups, Federated Identity, Principle of Least Privilege
- **Database Services:** Autonomous Databases, MySQL Database Service, PostgreSQL, NoSQL
- **High Availability & Disaster Recovery:** Availability Domains, Failover strategies, Cross-region replication
- **Observability:** Logging, Monitoring, Events, Health Checks
- **Security:** Encryption, Key Management, Data Safe, WAF, DDoS protection
