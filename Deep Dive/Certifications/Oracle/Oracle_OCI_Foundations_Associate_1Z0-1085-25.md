## About

The Oracle Cloud Infrastructure (OCI) Foundations Associate certification validates baseline knowledge of Oracle's cloud platform, encompassing core concepts, service architecture, identity management, networking, compute, storage, and security. Released in 2025, this entry-level certification is designed for IT professionals beginning a cloud career path with Oracle or transitioning from on-premises infrastructure to cloud-native environments. Passing earns the badge and entry point into the OCI professional stack (leading to Architect, Database Admin, or Cloud Engineer specializations).

Oracle Cloud Infrastructure (OCI) is a hyperscale public cloud platform competing with AWS, Microsoft Azure, and Google Cloud Platform. This cert validates foundational competency across multi-cloud architectures, identity systems, and infrastructure-as-a-service (IaaS) operations.

## Topics covered

Exam blueprint weightings per Oracle's official exam guide (2025):

- **OCI Concepts** (5%) — tenancy, compartment model, identity domains, free-tier resources
- **Architecture and OCI Services** (40%) — compute (VMs, bare metal), storage (block, object, file), databases, middleware, developer services, networking topology
- **Identity and Access Management (IAM)** (10%) — users, groups, policies, authentication methods (SSO, MFA), OCI Identity Cloud Service
- **Networking** (15%) — VCN fundamentals, subnets, security lists, network gateways, routing, VPN, FastConnect
- **Compute and Storage** (15%) — Compute instances (VM shapes), block volumes, object storage operations, backup and disaster recovery
- **Database and Developer Services** (10%) — Autonomous Database, Database Cloud Service, MySQL, NoSQL, Functions, API Gateway
- **Observability and Security** (5%) — monitoring, logging, security posture, compliance, encryption at rest/in-transit

*Source:* [Oracle OCI Foundations Associate Exam Blueprint ↗](https://www.oracle.com/education/certifications/foundations-details/)

## Skills validated

*Cert-specific — what this exam actually tests, distinct from the shared "Common skills" above.*

- OCI tenancy and compartment model (resource organization, isolation)
- OCI Compute shapes and instance types (VM.Standard, VM.Optimized, Bare Metal)
- Block Storage (Boot volumes, block volumes, snapshots)
- Object Storage (buckets, objects, lifecycle policies, versioning)
- Autonomous Database (serverless, self-healing)
- OCI IAM (users, groups, dynamic groups, policies, authentication providers)
- Virtual Cloud Network (VCN) design and operations
- Security Lists and Network Security Groups (NSGs)
- OCI Monitoring, Logging, and Observability
- Cost tracking, budgets, and billing in OCI
- Disaster recovery and backup strategies (Database Backup Service, block volume backups)
- API Gateway and Functions (serverless compute)
- MySQL Database Service and Oracle Database Cloud Service deployment models
