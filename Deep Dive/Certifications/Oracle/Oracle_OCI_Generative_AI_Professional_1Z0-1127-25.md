## About

The Oracle Cloud Infrastructure 2025 Generative AI Professional certification validates expert-level competency in designing, building, and deploying generative AI applications on Oracle Cloud. Launched in 2024 alongside OCI's native generative AI services (Cohere and Meta Llama models), this professional-level exam is designed for software developers, ML engineers, and AI professionals with foundational OCI experience who need to demonstrate mastery of LLM architectures, prompt engineering, retrieval-augmented generation (RAG), and intelligent agent orchestration on OCI. Passing earns the credential and positions holders for specialized AI engineering and architecture roles in cloud-native organizations.

## Domain context — AI / Machine Learning

Generative AI and Large Language Models (LLMs) are rapidly becoming foundational to cloud platform strategy. This certification validates hands-on expertise in OCI's managed generative AI services—a domain spanning prompt engineering, vector databases, RAG pipelines, LangChain integration, fine-tuning, and responsible AI governance. OCI competes with AWS Bedrock, Azure OpenAI, and Google Vertex AI in this hyperscale market.

## Topics covered

Exam blueprint weightings per Oracle's official learning objectives (2025):

- **Fundamentals of Large Language Models** (20%) — transformer architecture, attention mechanisms, token encoding, in-context learning, model training pipelines, capabilities and limitations of LLMs
- **OCI Generative AI Service Configuration & Usage** (25%) — provisioning inference endpoints, selecting Cohere and Meta Llama models, managing API calls, authentication, rate limiting, cost optimization, monitoring and logging
- **Prompt Engineering & In-Context Learning** (15%) — prompt design patterns, few-shot learning, chain-of-thought reasoning, prompt injection risks and mitigation, model parameter tuning (temperature, top-k, top-p)
- **Retrieval-Augmented Generation (RAG)** (20%) — building RAG pipelines, vector embeddings, semantic search, knowledge base integration, chunking strategies, retrieval quality optimization, integrating external data sources
- **LangChain & OCI Integration** (10%) — LangChain framework on OCI, chains, agents, memory management, tool integration, evaluating LLM applications
- **OCI Generative AI RAG Agents Service** (10%) — agent architecture, autonomous reasoning, tool orchestration, multi-step workflows, handling agent failures, monitoring agent performance
- **Fine-Tuning, Responsible AI & Governance** (included in above) — custom model fine-tuning, bias detection, content moderation, responsible AI practices, security governance, compliance

*Source:* [Oracle OCI Generative AI Professional Learning Path ↗](https://learn.oracle.com/ords/product?p_code=OCI_GENAI_PRO)
