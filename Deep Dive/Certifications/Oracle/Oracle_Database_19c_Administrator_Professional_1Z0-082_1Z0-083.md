## About

The Oracle Database 19c Administrator Professional certification is a two-part exam (1Z0-082 and 1Z0-083) that validates enterprise-grade database administration skills on Oracle Database 19c. Exam 1Z0-082 (Administration I) tests foundational skills: database architecture, instance startup/shutdown, user/privilege management, tablespace and datafile administration, and basic backup/recovery. Exam 1Z0-083 (Administration II) covers advanced topics: multitenant container databases (CDB), pluggable databases (PDB), RMAN backup strategies, Flashback technologies, Data Guard, performance tuning, and patching/upgrades. Both exams became the Professional-level certification in 2019 and remain the primary Oracle DBA credential through 2026.

## Domain context — Data/Database

Relational and non-relational database administration, backup/recovery, security, and performance tuning across commercial, open-source, and vendor-specific platforms (Oracle, SQL Server, PostgreSQL, MongoDB, etc.). Oracle Database 19c focuses on enterprise-grade relational administration at the Professional level.

## Topics covered (1Z0-082: Administration I)

- Database architecture and Oracle instance components (15–20%)
- Creating and managing databases (10–15%)
- Managing database users and security (roles, privileges, profiles) (10–15%)
- Managing tablespaces and datafiles (10–15%)
- Undo management and transactions (5–10%)
- Database backup and recovery (RMAN basics) (15–20%)
- Oracle network configuration (SQL\*Net, listener) (5–10%)
- Database monitoring and performance (5–10%)

*Source:* [Oracle 1Z0-082 Study Guide ↗](https://www.oracle.com/a/ocom/docs/dc/ww-ou-5297-database2019-studyguide-5.pdf)

## Topics covered (1Z0-083: Administration II)

- Multitenant container database (CDB) architecture and administration
- Pluggable database (PDB) creation, configuration, and cloning
- Advanced backup and recovery (RMAN): incremental backups, compression, active duplication, TSPITR
- Flashback Database, Flashback PDB, and point-in-time recovery
- Oracle Data Guard: physical standby, logical standby, protection modes
- Real Application Clusters (RAC) and Oracle Automatic Storage Management (ASM) fundamentals
- Performance tuning: AWR, ADDM, SQL Tuning Advisor, wait events, SGA/PGA advisors
- Patching and database upgrades (Release Updates, patches)
- Database security: encryption, auditing, virtual private database (VPD)
- Resource Manager and metric thresholds in multitenant environments

*Source:* [Oracle 1Z0-083 Study Guide ↗](https://www.oracle.com/a/ocom/docs/dc/ww-ou-5297-database2019-studyguide-5.pdf)

## Skills validated

*Cert-specific — what this exam actually tests, distinct from the shared "Common skills" above.*

- Oracle Database instance architecture (SGA, PGA, background processes)
- RMAN (Recovery Manager): full, incremental, compressed backups; active duplication; TSPITR
- Multitenant Container Database (CDB) and Pluggable Database (PDB) administration
- Oracle Data Guard: physical standby, logical standby, protection modes, switchover/failover
- Flashback Database and point-in-time recovery
- Oracle Real Application Clusters (RAC) and Oracle Automatic Storage Management (ASM) concepts
- Performance tuning: AWR, ADDM, SQL Tuning Advisor, wait events, memory advisors
- Database patching and upgrades (Release Updates, one-off patches)
- User and privilege management (RBAC, roles, system/object privileges, profiles)
- Tablespace and datafile management, undo management
- Oracle Net (SQL\*Net) listener and connection configuration
- SQL and PL/SQL scripting for automation
- Encryption, auditing, virtual private database (VPD)
- Resource Manager for multitenant environments
