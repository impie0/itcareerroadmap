## About

The Oracle Cloud Infrastructure 2025 Multicloud Architect Associate (1Z0-1115-25) validates foundational expertise in designing, deploying, and managing multicloud solutions that integrate Oracle Cloud Infrastructure with public cloud providers (Azure, Google Cloud Platform, AWS). This certification is ideal for architects and engineers transitioning to multicloud roles or validating hands-on experience integrating OCI with heterogeneous cloud environments. The exam emphasizes Oracle Database@Azure, Oracle Database@Google Cloud, OCI FastConnect, cross-cloud networking, identity federation, and multicloud security posture.

## Topics covered

The exam blueprint assesses the following domains and approximate weights:

- **Oracle Cloud Infrastructure Fundamentals** (~15%) — OCI regions, availability domains, compartments, resource hierarchy, IAM
- **Multicloud Architecture Design Patterns** (~25%) — Hybrid cloud topology, interconnect strategies, workload placement, latency-aware design, failover scenarios
- **Oracle Database@Azure Integration** (~20%) — Deploying Oracle Autonomous Database in Azure datacenters, Exadata on Azure, Azure networking integration (ExpressRoute + FastConnect), database replication across clouds
- **Oracle Database@Google Cloud Integration** (~15%) — Deploying Oracle Autonomous Database in Google Cloud, interconnection via Cloud Interconnect, cross-cloud data synchronization
- **Cross-Cloud Networking and Security** (~15%) — OCI FastConnect, hybrid DNS, VPN tunneling, identity federation (IAM + Azure AD/Entra ID), encryption, security groups and network security lists, Data Safe
- **Multicloud Observability and Cost Management** (~10%) — Cross-cloud monitoring, logging aggregation, cost allocation across clouds, license mobility

## Skills validated

*This exam assesses practical, hands-on capability in these multicloud technologies and integration patterns:*

- **Oracle Database@Azure:** Autonomous Database deployment in Azure datacenters, Exadata Cloud@Customer integration with Azure ExpressRoute, cross-cloud replication, network connectivity
- **Oracle Database@Google Cloud:** Autonomous Database provisioning on GCP, Cloud Interconnect integration, cross-cloud data synchronization
- **OCI FastConnect:** Dedicated network connections from OCI to Azure (ExpressRoute pairing), Google Cloud (Cloud Interconnect pairing), AWS (AWS Direct Connect pairing)
- **Hybrid Networking:** VPN tunneling, site-to-site connectivity, multicloud VPC/VCN peering, latency-aware routing, failover strategies
- **Identity Federation:** OCI IAM integration with Azure AD/Entra ID, federation policies, single sign-on (SSO) across clouds, federated groups and roles
- **Multicloud Security Posture:** Encryption key management across clouds, Data Safe for database security, WAF and DDoS protection, network segmentation, compliance monitoring
- **Workload Portability:** Containerized workloads on Kubernetes (OKE), serverless functions (Oracle Functions), database migration patterns, Oracle MySQL HeatWave on multicloud
- **Cost and License Management:** Bring-Your-Own-License (BYOL), cross-cloud cost allocation, resource metering, license mobility across OCI and on-premises
- **Multicloud Observability:** Cross-cloud logging and monitoring (OCI Logging, Monitoring; Azure Monitor integration), event-driven architectures, health checks and alerting
