# Oracle Fusion Cloud Financials: General Ledger Implementation Professional

**`1Z0-1054`** · ● **Active** · **Professional** · _Oracle_

This certification validates expertise in implementing Oracle Fusion Cloud General Ledger solutions. Candidates must pass both hands-on challenges and multiple-choice questions, demonstrating proficiency in configuring enterprise structures, financial reporting structures, ledgers, journal processing, intercompany transactions, period close operations, and financial reporting. This credential is essential for Oracle Fusion Cloud consultants, implementers, and administrators.

## About

The 1Z0-1054 certification is part of the Oracle Fusion Cloud Financials certification path and is designed for SaaS implementers, administrators, managers, and consultants who deploy Oracle Cloud General Ledger solutions. It is the anchor certification in the Financials track, typically followed by specialized certifications in Accounts Payable (1Z0-1055) or Accounts Receivable (1Z0-1056). The exam validates hands-on experience with enterprise structure design, ledger configuration, and financial close procedures essential to Fusion implementations.
