## About

The Kubernetes and Cloud Native Associate (KCNA) is a pre-professional certification introduced by CNCF in 2021 to serve as an entry point for those seeking to learn about Kubernetes and cloud-native technologies. Unlike hands-on performance-based exams (CKA, CKAD), KCNA is knowledge-based, testing conceptual understanding and decision-making through multiple-choice questions. It prepares candidates to advance to professional-level certifications and is commonly sought by developers, systems administrators, and IT professionals transitioning into cloud-native roles.

## Topics covered

Official exam blueprint (CNCF published):

- **Kubernetes Fundamentals** (46%) — Kubernetes architecture, resources, API objects, scheduling, container runtime, networking basics
- **Container Orchestration** (22%) — Runtime selection, container security, networking policies, service mesh fundamentals, storage provisioning
- **Cloud Native Architecture** (16%) — Microservices principles, serverless computing, autoscaling, high availability, distributed systems concepts
- **Cloud Native Observability** (8%) — Prometheus, logging, distributed tracing, telemetry, monitoring strategy, cost tracking
- **Cloud Native Application Delivery** (8%) — GitOps, CI/CD pipelines, deployment strategies (blue-green, canary), application lifecycle management

*Source:* [CNCF KCNA ↗](https://www.cncf.io/training/certification/kcna/)
