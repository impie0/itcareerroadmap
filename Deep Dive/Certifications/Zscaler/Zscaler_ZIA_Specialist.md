## About

The Zscaler Certified Cloud Professional - Internet Access (ZCCP-IA) is an advanced professional-level certification that validates expertise in deploying, configuring, and troubleshooting Zscaler Internet Access solutions at enterprise scale. This certification builds directly on ZIA Administrator knowledge, diving into advanced topics such as Client Connector deployment, bandwidth control, DNS security, Nanolog Streaming Service (NSS), and complex authentication scenarios. Candidates must complete a hands-on lab component in addition to passing the written exam, ensuring practical competency with real ZIA deployments. Valid for three years, ZCCP-IA is designed for network engineers, security administrators, and Zscaler implementation specialists responsible for enterprise ZIA deployments in zero trust architectures.

## Domain context — Security / SASE / Cloud Security

Secure Access Service Edge (SASE) converges cloud-delivered network and security functions into a single platform, eliminating legacy hub-and-spoke network models. Zscaler Internet Access leads this transition with cloud-native secure web gateway (SWG), firewall, DLP, and advanced threat prevention. ZIA is the foundation of zero trust network access—every user, device, and application is verified and validated before access is granted.

## Topics covered

The ZCCP-IA exam covers advanced ZIA architecture, configuration, and deployment across six main domains:

- **Traffic Forwarding Methods** (15–20%)
  - GRE (Generic Routing Encapsulation) tunnel configuration and optimization
  - IPSec tunneling and key management
  - Zscaler Client Connector (ZCC) deployment models and proxy chaining
  - Port forwarding and direct routing scenarios
  - Traffic steering and failover strategies

- **Authentication & User Identification** (15–20%)
  - SAML 2.0 integration with identity providers (Azure AD, Okta, etc.)
  - LDAP and Active Directory authentication
  - Zscaler Authentication Bridge deployment
  - Kerberos and mutual authentication
  - Device posture assessment and conditional access

- **Advanced ZIA Features** (20–25%)
  - Client Connector architecture, deployment, and management
  - Bandwidth control and traffic management policies
  - DNS security and DoH (DNS over HTTPS) configuration
  - SSL/TLS inspection, certificate pinning, and bypass policies
  - Nanolog Streaming Service (NSS) for log collection and SIEM integration

- **Web Security & Filtering** (10–15%)
  - URL filtering and custom URL categories
  - Web DLP (Data Loss Prevention) policy configuration
  - Advanced malware and threat protection (sandboxing, detonation)
  - Cloud App Security (CASB) and Shadow IT detection
  - Safe search enforcement and content inspection

- **Cloud Firewall & Network Security** (10–15%)
  - Cloud firewall rule configuration
  - IPS (Intrusion Prevention System) and threat prevention tuning
  - Geo-blocking and IP reputation filtering
  - Advanced threat protection (ATP) and threat intelligence integration
  - Network segmentation via ZIA policies

- **Deployment, Integration & Troubleshooting** (15–20%)
  - Enterprise deployment architecture and best practices
  - Integration with third-party security tools and SIEM platforms
  - Connectivity troubleshooting: tunnels, PAC files, routing
  - Performance optimization and bandwidth efficiency
  - High availability and disaster recovery configurations
  - NSS troubleshooting and log analysis
