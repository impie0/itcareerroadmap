## About

The Arista Certified Associate Network Engineer validates foundational competency with Arista EOS (Extensible Operating System), Layer 2 switching, basic Layer 3 routing, and core routing protocols (BGP, OSPF, MLAG). Launched June 1, 2025 as the entry-level credential within Arista's Enhanced Certification Program, it replaces the deprecated legacy ACE-L1 (which retired December 31, 2025). The exam emphasizes hands-on CLI configuration of VLAN switching, Multi-Chassis LAG (MLAG) redundancy, VXLAN overlay basics, and CloudVision platform fundamentals. Designed for network operators, junior network engineers, infrastructure technicians, and career-changers new to data center or enterprise Arista environments.

Enterprise and data center switching fabrics spanning Layer 2 switching through Layer 3 routing and overlay networking (VXLAN/EVPN). ## Topics covered

Exam blueprint (from official Arista Academy Detailed Overview and community publications):

- **EOS Fundamentals** (15–20%) — Command syntax, configuration file management, operational modes, logging, account management, user privilege levels
- **Layer 2 Switching & VLANs** (15–20%) — VLAN creation and trunk configuration, MAC address table inspection, port channel (LAG) basics, Spanning Tree Protocol operation
- **Layer 3 Routing Basics** (20–25%) — IPv4 addressing and subnetting, static routing, default gateways, routing table inspection, interface configuration
- **OSPF & BGP Essentials** (15–20%) — Single-area OSPF neighbor formation and area configuration, EBGP neighbor adjacency, route advertisement, basic route filtering
- **MLAG (Multi-Chassis LAG)** (10–15%) — MLAG domain setup, peer-link configuration, MLAG interface assignments, redundancy validation, failure scenarios
- **VXLAN & EVPN Basics** (10–15%) — VXLAN tunnel encapsulation, VTEP (Virtual Tunnel End Point) role and assignment, basic EVPN control plane with MP-BGP, head-end replication
- **CloudVision Platform** (5–10%) — CloudVision UI navigation, device onboarding and registration, telemetry overview, event log interpretation, basic alerting
- **Troubleshooting & Validation** (10–15%) — Packet capture and frame analysis, show command interpretation (routing, BGP, interface, MAC, VLAN status), ping/traceroute, traffic flow verification

*Source:* [Arista Academy Training Overview ↗](https://www.training.arista.com/hubfs/Guides/Arista%20Academy%20Overview%20Detailed.pdf)
