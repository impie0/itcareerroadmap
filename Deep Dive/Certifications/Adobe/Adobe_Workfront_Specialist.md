## About

The **AD0-E908** is Adobe's mid-level technical certification for Workfront implementation specialists, system administrators, and developers. Launched as part of Adobe's unified Experience Cloud credentials (after acquiring Workfront in 2020), it validates your ability to configure projects, programs, portfolios, custom forms, approval workflows, templates, reporting, Fusion integrations, and Workfront Proof. Unlike the Project Manager certification (AD0-E903), this path focuses on technical setup and customization rather than project execution. Requires 3–6 months of direct Workfront experience and is valid for 3 years.

## Skills validated

*What the AD0-E908 exam specifically tests — distinct from shared Work Management competencies.*

- Workfront Project, Portfolio, and Program object configuration
- Custom forms: field types, conditional display logic, field inheritance
- Approval workflows: single and multi-stage approvals, approval delegation
- Project templates: task templates, timeline templates, resource assignment
- Workfront Proof: proof setup, review workflows, version management
- Reporting: report builder, custom fields in reports, dashboard creation
- Access levels: role-based permissions, group security, layout template design
- Fusion integrations: Adobe Workfront Fusion connector setup, basic scenario building
- Workfront API: REST API fundamentals, authentication, common endpoints
- User and group management: bulk user operations, team hierarchies
- Status and custom statuses: workflow states, approval-based status changes
