## About

The Adobe Certified Professional in Web Authoring Using Adobe Dreamweaver validates competency in creating, publishing, and managing websites using Adobe Dreamweaver, the industry-standard visual web development environment. This Associate-level certification demonstrates practical knowledge of HTML, CSS, JavaScript integration, responsive design, and modern web authoring workflows. 

Introduced in its current form in 2021, it replaced the legacy Adobe Certified Expert (ACE) Dreamweaver CC credential and remains the primary Adobe certification for web designers and front-end developers using Dreamweaver. Unlike its predecessor, the ACP exam (delivered exclusively through Certiport's OnVUE platform and authorized testing centers) includes a significant performance-based component: candidates work directly in the live Dreamweaver application to demonstrate real-world skills alongside traditional multiple-choice questions.

The certification targets professionals with approximately 150 hours of hands-on Dreamweaver experience and validates practical competency in site architecture, information organization, semantic HTML structure, CSS-based responsive layouts, accessibility best practices, and site deployment workflows. The exam is particularly valued for entry-level web designer and junior front-end developer roles in creative agencies, freelance work, and in-house marketing/web departments.

## Domain context — Web Design & Development

Web design and development encompasses the creation, styling, and interactive enhancement of websites and web applications. It bridges visual design (layout, color, typography) and front-end development (HTML, CSS, JavaScript, responsive patterns). This domain ranges from entry-level web designers focused on visual composition to senior front-end developers managing complex interactive systems and frameworks.

Dreamweaver occupies a unique position as a WYSIWYG (visual) editor that also allows direct code editing—appealing to both designers transitioning to development and developers seeking rapid prototyping. While competitors like VS Code have gained market share for pure development work, Dreamweaver remains strong for teams that prioritize visual design-to-code workflows and integrated site management features (FTP publishing, asset organization, template systems). The ACP certification validates this specific toolset and the modern web standards knowledge required to use it effectively.

## Topics covered

The ACP-DW exam blueprint covers web authoring fundamentals across six main domains:

- **Workspace and Basic Navigation** — Dreamweaver interface, project setup, workspace customization, file and folder management
- **HTML Content Creation** — Semantic HTML structure, headings (h1–h6), paragraphs, lists (ordered/unordered), emphasis, line breaks, blockquotes, spans, div containers, forms and data tables, accessibility attributes
- **Media and Images** — Image insertion and optimization, alt text, responsive image sizing, video and audio embedding, image scaling, SVG basics
- **Links and Navigation** — Internal links (relative and absolute paths), external links, email links, telephone links, named anchors, link targets
- **CSS Styling and Layout** — External stylesheets, internal styles, inline styles, CSS selectors and properties, @media queries, responsive design patterns, CSS Grid, Flexbox, Bootstrap integration
- **JavaScript and Interactivity** — Basic JavaScript integration, event handlers, form validation, DOM manipulation basics, third-party library integration (jQuery, Bootstrap components)

## Skills validated

*Specific technologies and workflows tested by the ACP-DW exam — distinct from general web design fundamentals.*

- Adobe Dreamweaver visual editing and code view workflows (Design panel, Code view synchronization, Split view)
- HTML5 semantic elements and document structure (DOCTYPE, head metadata, semantic tags vs div soup)
- CSS external stylesheets and media queries (linking best practices, cascade rules, mobile-responsive breakpoints)
- Responsive design with Flexbox and CSS Grid (flexible containers, auto-fit/auto-fill patterns, gap properties)
- Bootstrap framework integration (grid system, utility classes, component copy-paste integration into Dreamweaver)
- Image optimization (file formats: PNG vs JPG vs WebP, alt text authoring, srcset for responsive images)
- Form creation and accessibility (label association, input types, ARIA landmarks, error messaging patterns)
- JavaScript event handling and DOM manipulation in Dreamweaver (inline event attributes, external script linking, basic DOM queries)
- FTP/SFTP publishing workflows (Dreamweaver's built-in site management, file sync, remote connection setup)
- Accessibility compliance (WCAG 2.1 Level A/AA, screen reader testing, keyboard-only navigation)
- Cross-browser testing practices (how to identify browser-specific rendering issues, testing tools)
- File and site management in Dreamweaver (templates, library items, asset organization, site definitions)
