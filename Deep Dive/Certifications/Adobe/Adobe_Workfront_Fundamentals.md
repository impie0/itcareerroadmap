## About

Adobe Workfront Fundamentals is an entry-level foundational course designed for users new to Workfront who need to understand the platform's core features, key capabilities, and basic functionality. It serves as the gateway course for both individual contributors (Worker license users) and administrators preparing for more advanced Workfront certifications. The course introduces the product, use cases, core workflow concepts, and provides reference materials supporting hands-on learning in the platform.

## Domain context — Project/Work Management

Workfront is Adobe's enterprise project and work management platform enabling teams to plan, execute, track, and deliver work collaboratively across organizations. This domain encompasses work request intake, project lifecycle management, resource planning, and cross-functional team coordination.

Based on the official Workfront Foundations course, learners gain competency in:

- Viewing and managing personal work queues

- Basic reporting and dashboard usage
- Workfront best practices and initial setup context
