## About

The Adobe Certified Professional: Adobe Captivate certification validates proficiency in designing and developing interactive eLearning courses using Adobe Captivate (current version 12.3+). Aimed at instructional designers, eLearning developers, and course authors, this professional-level certification demonstrates hands-on competency in branching scenarios, software simulations, assessments, multimedia integration, and SCORM/xAPI compliance. Adobe transitioned from Captivate Classic to the new Captivate in mid-2023; the Classic exam was discontinued April 12, 2024. Current as of 2026; vendor actively releases quarterly feature updates to the platform.

## Topics covered

Exam blueprint weighted as follows:

- **Simulations** (13%) — Creating software simulations, recording workflows, editing slides, adding interactions to capture real-world tasks
- **Non-Interactive Content** (21%) — Building informative, visually engaging static slides; typography, imagery, animations, themes
- **Assessment** (21%) — Designing quizzes, question pools, result slides, reporting to LMS, pass/fail logic, branching on score
- **Interactions** (24%) — Advanced actions, variables, conditional logic, widgets, buttons, drag-and-drop, animations, timeline-based interactions
- **Media and Accessibility** (21%) — Audio/video integration, image optimization, responsive design, WCAG 2.1 compliance, closed captions, keyboard navigation

## Skills validated

*This exam specifically tests the following technologies and concepts distinct from general course design:*

- **Captivate Interface & Workflow** — Project management, slide editor, timeline, properties panel, object library, import/export
- **Slide Creation & Editing** — Master slides, theme application, responsive breakpoints, slide animation, segment interactions
- **Screen Recordings & Simulations** — Recording workflows, auto-play object editing, custom overlays, callout editing, slide insertions post-recording
- **Interactive Elements** — Buttons, click boxes, text entry boxes, drag-and-drop, images as interactive objects, advanced actions (conditional if/else logic)
- **Assessment Objects** — Question slides (multiple choice, true/false, matching, drag-and-drop), question pools, random question generation, result slides, scoring logic
- **Variables & Advanced Actions** — Creating variables, setting/getting values, branching based on conditions, state persistence across slides
- **Media Integration** — Audio editing (narration, background music, sound effects), video playback, captions/subtitles, image optimization, SVG support
- **Accessibility** — WCAG 2.1 AA compliance, closed captions, keyboard tab order, color contrast verification, screen reader testing, text alternatives
- **LMS Integration** — SCORM 1.2 and 2004 compliance, xAPI (Tin Can) statement creation, AICC support, learning record store (LRS) connectivity, success/completion definitions
- **Responsive Design** — Breakpoint creation, mobile optimization, tablet-specific design, viewport testing, landscape/portrait handling
- **Publishing & Delivery** — HTML5 output, MP4 video export, package creation for LMS upload, browser compatibility testing, offline playback
