## About

Adobe Certified Expert - Adobe Commerce Developer (AD0-E716) is a scenario-based, backend-focused certification designed to validate expert-level skills in customizing and extending the Adobe Commerce platform (formerly Magento 2). Released in 2022 as part of the modernized Adobe Digital Experience Certification Program, this exam tests architects, senior developers, and technical leads responsible for designing custom modules, managing cloud deployments, optimizing performance, and integrating third-party services at enterprise scale. Approximately 25% of the exam covers Adobe Commerce Cloud architecture, reflecting the shift toward cloud-first deployment models.

## Topics covered

- **Adobe Commerce Architecture** — file structure, module system, DI configuration, plugin/interceptor patterns
- **Database & EAV** — Magento's entity-attribute-value model, custom attributes, attribute sets, schema extensions
- **Admin & UI** — grid/form components, admin panel customization, controllers
- **Catalog & Product Management** — product types, pricing rules, layered navigation, import/export
- **Sales Operations** — cart/checkout customization, orders, shipments, payment gateway integration
- **REST & GraphQL APIs** — API architecture, custom endpoints, request/response handling, rate limiting
- **Caching & Performance** — full-page caching (FPC), application caching, indexing strategies
- **Configuration & Localization** — config.php, environment variables, multi-store setup, translations
- **Composer & Dependencies** — package management, version constraints, vendor integration
- **Cloud Architecture (25%)** — Cloud architecture, project structure, environment variables, ECE tools, patches
- **Cloud CLI & Deployment** — branch management, build/deploy pipeline, log access, troubleshooting
