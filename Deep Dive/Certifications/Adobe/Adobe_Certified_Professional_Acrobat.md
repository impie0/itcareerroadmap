## About

The Adobe Certified Professional (ACP) in Document Creation and Management Using Adobe Acrobat Pro is an industry-recognized entry-level certification that validates competency in PDF creation, editing, security, accessibility, and document management using Adobe's leading PDF solution. The exam tests practical skills within the actual Acrobat Pro application, combining selected-response questions with hands-on tasks. This certification demonstrates mastery of document workflows, collaboration features, e-signature integration, and document security standards required in professional document management roles.

## Topics covered

Based on official Adobe Certified Professional exam objectives:

- **Application Workspace and Navigation** — Navigating the Acrobat Pro interface, workspace customization, preference settings, and efficient workflow configuration
- **PDF Creation and Organization** — Creating PDFs from various document types, combining files into PDF portfolios, page organization, and batch processing
- **Page Management and Manipulation** — Inserting, deleting, rotating, extracting, and reordering pages; managing page properties and thumbnails
- **Document Editing** — Text and content editing within PDFs, replacing images, updating properties, and managing document metadata
- **Forms and Signatures** — Creating interactive forms, distributing forms, collecting form data, digital signature workflows, and e-signature preparation
- **Document Security and Protection** — Password protection, permission settings, redacting sensitive information, removing hidden metadata, and preventing unauthorized copying/printing
- **Accessibility Standards** — Ensuring PDFs meet WCAG and Section 508 accessibility requirements, adding alt text, tagging content, and testing accessibility
- **Collaboration Features** — Commenting, review workflows, track changes, shared reviews, and team collaboration tools
- **Exporting and Output** — Exporting PDFs to multiple formats (Word, Excel, PowerPoint), printing to file, and optimizing for different purposes
- **Document Analysis and Optimization** — File compression, redaction workflows, security audits, and document cleanup

*Source:* [Adobe Certified Professional Exam Objectives ↗](https://edex.adobe.com/adobe-certified-professional/exams)
