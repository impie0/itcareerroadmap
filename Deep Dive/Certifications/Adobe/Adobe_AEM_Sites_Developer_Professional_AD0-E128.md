## About

The AD0-E128 Adobe Certified Professional – AEM Sites Developer credential recognizes practical proficiency in building and configuring AEM Sites solutions. It assumes foundational knowledge of component development, HTL templating, Sling Models, OSGi configurations, Dispatcher caching, and AEM workflows. Candidates demonstrate ability to extend AEM core components, implement content fragments and templates, manage replication, configure caching strategies, and troubleshoot AEM issues. Valid for three years; renewal via passing modules or retaking the exam at no additional cost. This is the entry Professional-level AEM credential, preceding the Expert-level AD0-E137 (AEM Sites Developer Expert).

## Topics covered

Per the official Adobe AD0-E128 exam blueprint:

- **Configurations** (15%)
  - AEM Dispatcher configuration and caching mechanisms
  - OSGi and Oak index configuration and management
  - Multi-site and multi-tenant management
  - Custom logging and diagnostics

- **AEM Development** (37%)
  - Extend core components using proxying techniques
  - Develop components with HTL, Sling Models, and OSGi services
  - Build and customize AEM workflows
  - Work with content fragments and experience fragments
  - Implement editable templates for flexible content authoring
  - Understand AEM page architecture and component hierarchy

- **Build and Deployment** (17%)
  - Maven project structure and AEM archetypes
  - Configure build and deployment pipelines
  - Understand AEM Cloud Manager CI/CD integration
  - Repository structure and module organization

- **Environment Maintenance** (13%)
  - Apply maintenance releases and patches
  - Configure replication agents and topologies
  - Diagnose replication issues
  - Manage Oak indexes and query optimization
  - Monitor instance health

- **Analyzing and Debugging** (18%)
  - Troubleshoot replication failures
  - Debug Sling requests and component rendering
  - Use Developer Console and error logs
  - Performance analysis and optimization
  - Security considerations and access controls
