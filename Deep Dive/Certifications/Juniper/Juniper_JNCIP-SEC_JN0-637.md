## About

JNCIP-SEC is the professional-level advanced security certification in Juniper Networks' security track. It verifies advanced professional understanding of Junos OS for SRX Series firewalls, including advanced security policies, complex VPN architectures, advanced threat prevention, multi-tenant security models, high-availability clustering, advanced NAT configurations, and hands-on troubleshooting at scale. The exam targets network and security engineers moving beyond specialist knowledge (JNCIS-SEC) into advanced architecture and production operations. JN0-637, released in June 2024, is the current standard replacing JN0-636. This certification is a prerequisite for JNCIE-SEC (expert lab exam).

## Topics covered

- Advanced SRX Series architecture and design patterns
- Advanced security policies and policy templates (inheritance, jumbo frames, dynamic address books)
- Advanced Network Address Translation (NAT) — policy-based NAT, NAT translation order, DNS doctoring
- Advanced IPsec VPN configuration — site-to-site, remote access, auto-discovery, redundancy
- Advanced threat prevention — IDP policy tuning, signature management at scale
- Advanced Threat Prevention (ATP) Cloud integration and threat feeds
- SSL Proxy and TLS inspection (decryption, re-encryption, certificate management)
- Layer 2 security and spanning tree
- SRX Cluster (chassis clustering) — multi-node HA, synchronization, failover, scaling
- Logical systems and multi-tenant security architecture
- Virtual systems (vsys) configuration and tenant isolation
- Automated threat mitigation and automated DDoS protection
- Advanced troubleshooting — security policy evaluation, NAT troubleshooting, VPN negotiation debugging
- Policy-based routing (PBR) integration with security policies
- Application Layer Gateway (ALG) for protocol inspection
- Security zone functions and inter-zone traffic flow
