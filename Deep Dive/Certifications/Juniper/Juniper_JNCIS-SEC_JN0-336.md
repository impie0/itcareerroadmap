## About

JNCIS-SEC is the specialist-level security certification in Juniper Networks' security track. It verifies intermediate-to-professional understanding of Junios OS for SRX Series firewalls, including security policies, VPN configuration, threat prevention technologies, and real-world firewall operations. The exam targets network and security engineers moving beyond foundational knowledge (JNCIA-SEC) into advanced configuration and troubleshooting. JN0-336, released in September 2025, is the current standard replacing JN0-335.

## Topics covered

- Juniper Junos OS for SRX Series architecture and operational concepts
- Security policies (address books, security policies, policy templates)
- Network Address Translation (NAT) — source, destination, static NAT
- IPsec VPN site-to-site and remote-access configuration and troubleshooting
- Intrusion Detection and Prevention (IDP) policies and signatures
- Juniper Advanced Threat Prevention (ATP) Cloud integration
- Application Security and SSL Proxy / TLS Inspection
- High Availability (HA) clustering and failover configuration
- Unified Threat Management (UTM) — antivirus, web filtering, anti-spam
- User authentication and identity-aware security policies
- Logging, monitoring, and security event analysis
- Juniper Security Director (centralized management)
- Flow logging, session analysis, and troubleshooting
