## About

JNCIA-Junos is Juniper Networks' entry-level certification validating foundational knowledge of Junos OS, CLI navigation, network fundamentals, and basic troubleshooting. Released in 2015 as the foundation of the Juniper Networks Certification Program (JNCP) pyramid, it serves network administrators and junior engineers transitioning to Juniper or building core competency in Junos-based devices. Redesigned with the HPE acquisition (July 2025), it remains the gateway certification for all six JNCP tracks (Enterprise Routing, Service Provider, Security, Data Center, Cloud, and Mist AI). No retirement announced; actively maintained.

## Topics covered

*Drawn from official Juniper exam blueprint and Juniper Open Learning curriculum.*

- **Junos OS Fundamentals** (15%)
  - Control plane vs. forwarding plane architecture
  - Routing engine and line card roles
  - System software versions and upgrades
  - Boot sequences and recovery

- **User Interfaces** (10%)
  - Command-line interface (CLI) navigation and modes (operational vs. configuration)
  - Junos Space GUI (basic overview)
  - Configuration commit, rollback, and save mechanisms

- **Network Fundamentals** (15%)
  - IP addressing (IPv4, IPv6, subnetting)
  - Collision domains, broadcast domains
  - MAC addresses and switching concepts
  - Class of Service (CoS) principles

- **Interface Configuration and Operations** (20%)
  - Interface types (ge-, xe-, et-)
  - Logical interface configuration (units, families)
  - Interface operational states (up/down, link-level diagnostics)
  - Interface monitoring and statistics

- **Routing Fundamentals** (15%)
  - Static routing configuration
  - Dynamic routing protocols (OSPF, BGP, RIP, IS-IS neighbor establishment)
  - Routing table structure and route selection
  - Default routes and route aggregation

- **Basic Security Concepts** (10%)
  - Security zones and policies (firewall filter basics)
  - Network Address Translation (NAT) overview
  - Stateful firewall concepts
  - User authentication basics

- **Operational Monitoring and Maintenance** (10%)
  - System monitoring (CPU, memory, disk)
  - Interface and protocol troubleshooting
  - Log analysis and syslog configuration
  - SNMP basics for network monitoring

- **Troubleshooting Fundamentals** (5%)
  - Ping, traceroute, and connectivity diagnostics
  - Show command syntax and output interpretation
  - Basic packet tracing and filtering
  - Common operational issues and resolution

*Source:* [Juniper Open Learning Course Materials ↗](https://learningportal.juniper.net/)

## Common skills at Networking · Associate

*Shared skill set for entry-level networking professionals — not specific to Juniper.*

- **IP Routing Fundamentals** — Understanding routing tables, path selection, static vs. dynamic routing, and basic BGP/OSPF neighbor adjacency
- **Switching and VLAN Configuration** — Layer 2 concepts, VLAN tagging, spanning tree, and interface management across vendors
- **Network Troubleshooting** — Using ping, traceroute, packet captures, and show commands to diagnose connectivity issues
- **Interface Management** — Configuring physical and logical interfaces, monitoring link state, and interpreting interface counters
- **Basic Network Security** — Firewall rules, NAT, access control lists (ACLs), and user authentication concepts
- **Network Monitoring and Logging** — SNMP, syslog, and NetFlow for visibility; interpreting system logs for operational health

## Skills validated

*Concrete technologies and protocols this exam tests.*

- **Junos CLI** — Navigation, configuration modes, commit/rollback, operational commands
- **Junos Device Architecture** — Control plane (RE), forwarding plane (PFE), memory hierarchy
- **Interface Configuration** — Physical (ge-*, xe-*), logical units, address families (inet, inet6)
- **Routing Protocols** — OSPF, BGP, RIP, IS-IS neighbor establishment and route propagation
- **Static Routing** — Default routes, floating static routes, route aggregation
- **VLAN and Switching** — VLAN configuration, trunk ports, spanning tree concepts
- **Basic Firewall Filters** — Stateless firewall rules, filter order, action (accept/discard)
- **Network Address Translation (NAT)** — Source NAT, destination NAT, overload concepts
- **Interface Troubleshooting** — Using show commands to diagnose carrier loss, flapping, errors
- **System Monitoring** — CPU, memory, disk usage; syslog configuration; SNMP basics
- **Connectivity Diagnostics** — Ping, traceroute, DNS resolution, route tracing

## Summary

**JNCIA-Junos (JN0-104)** is Juniper's entry-level certification validating foundational knowledge of Junos OS, CLI operations, network fundamentals, and basic troubleshooting. No prerequisites; open to all learners. Free training pathway via Juniper Open Learning reduces exam cost from $200 to $100–$150 if candidate scores 70%+ on voucher assessment. Typical job outcome: Junior Network Engineer ($65K–$90K USD). Serves as prerequisite for six specialist JNCP tracks (Enterprise, SP, Security, Data Center, Cloud, Mist AI). Active and maintained; no retirement announced. HPE acquisition (July 2025) has not disrupted certification program. High demand in enterprises and service providers upgrading or maintaining Juniper infrastructure.
