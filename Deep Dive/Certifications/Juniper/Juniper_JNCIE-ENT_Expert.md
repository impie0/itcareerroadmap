## About

The JNCIE-ENT (Juniper Certified Internet Expert – Enterprise) is Juniper Networks' pinnacle expert-level certification for enterprise routing and switching technologies. It is a single 8-hour hands-on lab exam—not multiple choice—requiring candidates to build, configure, and troubleshoot complex enterprise network scenarios using Juniper's MX/EX/SRX platforms in real time. This certification validates deep expertise in enterprise network design, BGP/OSPF/IS-IS routing, MPLS Layer 2 and Layer 3 VPNs, advanced switching with VLAN/STP/MAC mobility, High Availability, QoS/CoS, multicast, and network security. JNCIE-ENT is the Juniper equivalent to Cisco's CCIE Routing & Switching (expert tier) and is recognized globally as a gold-standard credential for senior network engineers and architects.

## Topics covered

The JNCIE-ENT lab exam tests operational mastery across the full enterprise routing and switching domain:

- **Routing Protocols & Scaling** (15–20% of lab tasks)
  - BGP: eBGP peering, route aggregation, path selection, community filtering, route refresh
  - OSPF: areas, route summarization, network types, convergence tuning
  - IS-IS: level hierarchy, routing policies, TLV extensions
  - Routing policy language (RPL) and filter construction

- **MPLS & VPN Technologies** (15–20%)
  - MPLS label switching, LSP setup and monitoring
  - Layer 3 VPNs (L3VPN): VRF configuration, route distinguishers, route targets
  - Layer 2 VPNs (L2VPN): VPLS, CCC (Circuit Cross-Connect), extended VLANs
  - LDP and RSVP-TE protocols and interaction

- **Advanced Switching** (15–20%)
  - VLAN tagging, trunk configuration, private VLANs
  - Spanning Tree Protocol (STP, RSTP, MSTP) convergence and optimization
  - MAC address learning, MAC mobility, IRB (Integrated Routing & Bridging)
  - Storm control and BPDU guard

- **High Availability & Redundancy** (10–15%)
  - Graceful restart, BFD failover detection
  - Virtual Chassis technology (EX and MX)
  - Redundancy Group (RG) and Routing Engine failover
  - NSR (Non-Stop Routing) on MX platforms
  - VRRP and session redundancy

- **Quality of Service (QoS) & Class of Service (CoS)** (10–15%)
  - Forwarding classes, code points, scheduling
  - Policers and shapers
  - Loss priority and drop policies
  - Per-hop behavior (PHB) configuration
  - Traffic classification and queue management

- **Multicast Routing** (5–10%)
  - PIM (Protocol Independent Multicast) configurations
  - IGMP snooping on switches
  - Rendezvous points and multicast trees
  - Fast convergence in redundant topologies

- **Network Security (Basics)** (5–10%)
  - SRX firewall fundamentals (if relevant to enterprise topology)
  - Access lists and firewall filters
  - Attack prevention (DDoS, spoofing, rate-limiting)
  - Secure management and device hardening

- **Troubleshooting & Optimization** (Integrated across all tasks)
  - Packet tracing and debug output interpretation
  - Performance analysis and baseline tuning
  - Network verification and monitoring

## Skills validated

*Cert-specific technologies and protocols tested in the JNCIE-ENT lab exam.*

- Border Gateway Protocol (BGP): eBGP multihop, route aggregation, community-based filtering, route refresh, BFD integration
- Open Shortest Path First (OSPF): stub areas, NSSA, summarization, SPF optimization, graceful restart
- Intermediate System to Intermediate System (IS-IS): multiple levels, wide metrics, routing policies, graceful restart
- Multiprotocol Label Switching (MPLS): label assignment, LDP, RSVP-TE, LSP troubleshooting
- Layer 3 VPN (L3VPN): VRF configuration, route targets, route distinguishers, multipath
- Layer 2 VPN (L2VPN): VPLS, MAC learning in provider networks, pseudowire configuration
- Advanced Switching: VLAN tagging, trunk protocols, MAC address table manipulation, Private VLAN (PVLAN)
- Spanning Tree Protocol (STP/RSTP/MSTP): BPDU processing, port roles, convergence tuning
- Integrated Routing and Bridging (IRB): subnetting across switch/router interfaces, dual-stack
- High Availability: Graceful Restart, BFD, Virtual Chassis, Routing Engine failover, NSR
- Class of Service (CoS): code point marking, scheduler queues, policer/shaper design, drop policies
- Multicast: PIM-DM/PIM-SM, IGMP, multicast source discovery protocol (MSDP)
- Junos OS CLI: configuration commit architecture, rollback, system logging, performance monitoring
- Firewall Filters: stateless and stateful packet filtering, rate limiting, attack detection (on SRX/MX)
- Juniper Routing Policy Language (RPL): prefix-list, route-filter, policy conditions and actions
- Packet Forwarding: packet classification, next-hop determination, load balancing, ECMP
