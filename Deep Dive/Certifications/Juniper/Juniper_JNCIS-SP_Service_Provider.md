## About

JNCIS-SP is the specialist-level certification in Juniper's Service Provider routing and switching track, designed for networking professionals with beginner-to-intermediate knowledge of Junos platform configuration and troubleshooting in large-scale carrier environments. The exam validates understanding of Junos OS for service provider networks, routing protocols (OSPF, IS-IS, BGP), MPLS fundamentals, layer 3 VPNs (RFC 4364 BGP/MPLS VPN), layer 2 VPNs (VPLS), class of service (CoS), IP multicast, and high-availability architectures in service provider deployments.

JNCIA-Junos is the recommended prerequisite; JNCIA-SP is preferred. Pass/fail status is available immediately after the exam; certification remains valid for three years. This cert sits between the associate-level JNCIA-SP (entry) and the professional-level JNCIP-SP (intermediate).

## Domain context — Networking / Service Provider

Large-scale carrier-grade routing, switching, and MPLS-based service provisioning on Juniper MX/SRX platforms. Focuses on inter-domain routing (BGP, OSPF, IS-IS), traffic engineering, layer 2 and layer 3 VPN architectures, class of service, and high-availability design patterns for internet service providers (ISPs), telecom operators, and carrier networks.

## Topics covered

Official exam blueprint topics verified as of May 2026:

- **Junos OS for Service Providers** — routing engine, forwarding plane, chassis cluster, control plane protection, SP-specific system features
- **OSPF in Service Provider Networks** — multi-area design, area types (backbone, stub, NSSA), traffic engineering extensions, neighbor adjacency, metric tuning, graceful restart
- **IS-IS for Service Providers** — level 1/level 2 routing, Type–Length–Value (TLV) encoding, SPF algorithm, area setup, TE extensions, multi-level design
- **BGP for Service Providers** — advanced BGP, route reflection, confederations, community tags, AS-path manipulation, route filtering, aggregate routes, graceful restart
- **MPLS Fundamentals** — label switching, label forwarding information base (LFIB), label distribution protocol (LDP), RSVP-TE for traffic engineering, LSP creation and maintenance
- **Layer 3 VPNs (BGP/MPLS VPN / RFC 4364)** — VRF (Virtual Routing and Forwarding) concepts, route distinguishers (RD), route targets (RT), PE–CE routing protocols, VPN route propagation, site-to-site connectivity
- **Layer 2 VPNs** — LDP-based VPLS (Virtual Private LAN Service), BGP VPLS, service interface configuration, bridge domains, VPLS forwarding
- **Class of Service (CoS)** — policing, queue scheduling, traffic shaping, forwarding classes, drop profiles, per-hop behavior (PHB) in SP networks
- **IP Multicast in Service Providers** — PIM sparse and dense modes, IGMP, multicast VPN (MVPN), multicast forwarding tables, RPF checks
- **Junos Automation for Service Providers** — Junos PyEZ basics, NAPALM fundamentals, template-driven configuration, operational scripting

## Skills validated

*Concrete technologies, protocols, and tools this exam tests.*

- **Junos OS** — configuration, operational mode, system administration, chassis cluster, redundancy
- **Routing Fundamentals** — routing tables, static routes, protocol-independent features, system routes
- **OSPF** — multi-area design, area types, neighbor adjacency, interface costs, OSPF extensions for TE
- **IS-IS** — level 1/level 2 design, TLV encoding, metric calculation, area setup, SPF algorithm, TE extensions
- **BGP** — EBGP/IBGP, neighbor establishment, path selection, route filtering, communities, aggregation, confederations, route reflection
- **MPLS and LDP** — label allocation, LDP discovery, LSP forwarding, LDP-over-TCP, label retention modes
- **RSVP-TE** — RSVP as signaling protocol, RSVP-TE LSPs, traffic engineering, bandwidth constraints
- **BGP/MPLS L3VPN** — VRF configuration, route distinguishers, route targets, PE–CE routing (static, RIP, OSPF, EBGP), VPN import/export
- **VPLS (Layer 2 VPN)** — LDP-based VPLS, BGP VPLS, service interfaces, bridge domains, VPLS forwarding
- **IPv6 in SP** — IPv6 address configuration, IPv6 routing, DHCPv6, IPv6-over-IPv4 tunnels, dual-stack deployment
- **IP Multicast** — PIM sparse/dense mode, IGMP, multicast forwarding, RPF checks, multicast VPN (MVPN)
- **Class of Service (CoS)** — traffic classification, policing, queue scheduling, drop profiles, forwarding classes
- **Tunneling** — GRE, IP-in-IP, tunnel endpoint setup, tunneling behavior, tunnel encapsulation
- **High Availability** — graceful restart (OSPF, BGP), BFID (Bidirectional Forwarding Detection), warm standby, dual routing engines, system redundancy
- **Layer 2** — spanning tree (STP/RSTP), VLAN configuration, bridge domains, LAG (Link Aggregation Group), LACP
- **Junos Automation** — Junos PyEZ basics, NAPALM fundamentals, template-driven configuration, JunOS Script basics
