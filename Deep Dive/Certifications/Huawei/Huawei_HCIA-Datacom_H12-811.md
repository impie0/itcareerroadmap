# Huawei Certified ICT Associate — Datacom

**`H12-811`** · ● **Active** · **Associate** · _Huawei_

Entry-level Huawei certification validating foundational competency in datacom networking fundamentals: TCP/IP, VLAN, switching, routing (OSPF, RIP), NAT/DHCP, network security, WLAN, and SDN basics. Designed as the first rung in Huawei's datacom career path (HCIA → HCIP → HCIE). Widely recognized in EMEA, Asia-Pacific, and African markets where Huawei infrastructure is deployed. No prerequisites; suitable for early-career network engineers and career changers.

## About

HCIA-Datacom is Huawei's associate-level entry point for IT professionals entering or pivoting to networking. It bridges the gap between foundational networking knowledge (CompTIA Network+ / Cisco CCENT concepts) and intermediate professional credentials. The exam was originally launched as HCNA (Huawei Certified Network Associate) in 2018 and rebranded to HCIA (ICT Associate) in 2020 to align with Huawei's global career certification framework. As of 2026, it remains actively updated with V1.0 being the current standard. The certification validates hands-on competency in planning, deploying, and managing small-to-medium datacom networks typical of enterprise campuses, branch offices, and service provider edge deployments.

## Topics covered

*From official exam blueprint (April 2026):*

- Data communication and network basics (8%)
- Build an IP network with interconnection and interworking — subnetting, IP services, VLAN, inter-VLAN routing (27%)
- Ethernet switching network construction — STP, VLAN switching, port security (28%)
- Cyber security infrastructure and network access — ACL, AAA, basic defense (8%)
- Network services and applications — DHCP, DNS, NAT, IPv6 basics (5%)
- WLAN basics — 802.11 principles, SSID, authentication (10%)
- WAN basics — PPP, frame relay (3%)
- Network management and operations — SNMP, syslog, Telnet, SSH (3%)
- IPv6 basics — address format, ICMP6, DHCPv6 (5%)
- SDN and automation basics — OpenFlow, network programmability concepts (3%)

## Skills validated

*Concrete technologies and protocols this exam specifically tests:*

- Huawei AR/S series router CLI and configuration
- Huawei S series switch CLI and configuration
- OSPF dynamic routing protocol implementation
- RIP routing protocol (legacy but still tested)
- Spanning Tree Protocol (STP) and RSTP
- VLAN trunking and switch security
- NAT (network address translation) — static and dynamic
- DHCP server configuration and lease management
- Access Control Lists (ACL) for security and QoS
- 802.1X port-based network access control
- WLAN configuration and security (WEP, WPA, WPA2)
- eNSP (Huawei network simulator) hands-on lab skills
- Basic network troubleshooting and diagnostics
- SNMP monitoring and basic automation concepts
