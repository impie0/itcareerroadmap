## About

The HCIA-Cloud Service (H13-811) is a vendor-specific, associate-level certification that validates foundational competency in deploying, configuring, and managing Huawei Cloud platform services. Designed for IT professionals transitioning into cloud engineering roles or deepening expertise in Huawei's hyperscale public cloud infrastructure. The current version (V3.5, released post-April 2024) covers compute, networking, storage, security, and operations across Huawei Cloud services. No formal prerequisites; accessible to professionals with baseline IT knowledge.

## Domain context — Cloud Computing/Infrastructure

Entry-level cloud certification spanning Huawei's native cloud platform: compute (Elastic Cloud Server / ECS), networking (VPC, load balancing), storage (Object Storage Service / OBS), identity and access management (IMS), security, and cloud operations. Complements but does not replace hyperscaler dominants (AWS, Azure, GCP); valuable in Asia-Pacific deployments and enterprises standardized on Huawei infrastructure.

## Topics covered

- **Cloud Computing Fundamentals** (15%): Service models (IaaS, PaaS, SaaS), deployment models, virtualization concepts, cloud benefits and challenges.
- **Huawei Cloud Overview** (5%): Platform positioning, regional availability, service catalog, pricing models.
- **Compute Cloud Services** (20%): Elastic Cloud Server (ECS) provisioning, instance lifecycle, flavors, image management, auto-scaling, bare-metal servers (BMS).
- **Network Cloud Services** (20%): Virtual Private Cloud (VPC), subnets, routing, security groups, Network ACLs, Elastic IP, load balancing, NAT gateway concepts.
- **Storage Cloud Services** (20%): Object Storage Service (OBS) bucket creation, access policies, lifecycle rules, encryption, versioning, cross-region replication, Elastic Volume Service (EVS).
- **Identity and Access Management (IMS)** (5%): RBAC design, IAM policies, service accounts, federated identity, least-privilege principles.
- **Huawei Cloud Operations & Monitoring** (5%): CloudWatch basics, resource monitoring, cost management, optimization strategies.
- **Security Services** (5%): Data encryption (in-transit, at-rest), key management, DDoS protection, Web Application Firewall (WAF), anti-spam, compliance frameworks.
- **Database Services** (3%): RDS (Managed Relational DB), DCS (Redis-equivalent), basic operational patterns.
- **Advanced Services** (2%): Content Delivery Network (CDN), AI/ML services overview, edge computing basics.
