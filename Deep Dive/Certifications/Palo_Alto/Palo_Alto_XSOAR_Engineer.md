## About

The Palo Alto Networks Certified XSOAR Engineer validates the ability to deploy, configure, manage, integrate, and troubleshoot Cortex XSOAR (Security Orchestration, Automation & Response) solutions across a full SOC automation lifecycle. Launched in 2023, this professional-level certification confirms competency in planning installations, designing use cases, building playbooks, managing incidents, handling threat intelligence, and supporting security operations teams in enterprise environments. Cortex XSOAR v8 SaaS represents the current platform generation (v6 on-premise reached end-of-support).

---

## Domain context — Security Orchestration & Automation (SOAR)

SOAR platforms consolidate disparate security tools, automate incident response workflows, and reduce mean-time-to-resolution (MTTR) in security operations centers. This certification validates depth in Cortex XSOAR, Palo Alto's enterprise SOAR product trusted by Fortune 500 security teams.
