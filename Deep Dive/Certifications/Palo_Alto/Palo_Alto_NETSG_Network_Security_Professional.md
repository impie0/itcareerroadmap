## About

The Palo Alto Networks Certified Network Security Professional (NETSG) is a modern, role-based professional-level credential that validates hands-on knowledge of Palo Alto Networks' integrated network security solution portfolio. Launched August 1, 2025, as part of a comprehensive certification restructure, NETSG replaced legacy generalist credentials and aligns directly with mid-tier security operations, network administration, and threat prevention roles. The exam covers baseline configuration and deployment across Palo Alto's Strata product family (next-generation firewalls, cloud-native CN-Series, VM-Series virtual firewalls), Strata Cloud Manager (centralized policy and device management), and foundational threat prevention concepts. Unlike the retired PCNSA (associate-level) and PCNSE (specialist-level) legacy certs, NETSG assumes 1–2 years hands-on platform experience and tests breadth across the entire product ecosystem rather than depth in a single vertical. The credential is valid for 2 years and is the primary stepping stone toward specialist certifications (NGFW Engineer, SOC Professional) and architect-level credentials.

## Role-based framework context (August 2025 restructure)

**Timeline:**
- **Announced:** June 2024 — Palo Alto Networks published comprehensive certification modernization roadmap
- **Legacy sunset:** March 31, 2025 (PCNSA, PCCET, PCSFE) and July 31, 2025 (PCNSE, PCCSE, PCSAE, PCDRA)
- **New framework live:** August 1, 2025 — Four-tier role-based ladder across three tracks: Network Security, Cloud Security, Security Operations

**New role-based framework structure:**
1. **Foundational** (entry; no experience required) — Network Security Foundational, Cloud Security Foundational, SOC Foundational
2. **Professional** (mid-tier breadth; 1–2 years experience) — **Network Security Professional (NETSG)**, Cloud Security Professional, SOC Professional
3. **Specialist** (deep expertise in a domain) — NGFW Engineer, Incident Responder, Threat Intelligence Analyst, Cloud Network Architect
4. **Architect** (5+ years experience; design + leadership) — Network Security Architect, Cloud Security Architect, Security Operations Architect

**NETSG positioning:** Mid-tier breadth credential ideal for network security analysts, junior network engineers, firewall administrators transitioning from legacy certs, and hands-on security operations professionals with 1–2 years Palo Alto platforms experience.

## Exam objectives (role-based framework blueprint)

The NETSG exam is organized into four primary domains with weighted coverage:

**1. Installation, Configuration, and Baseline Deployment (30%)**
- Firewall hardware and virtual form factors (PA-Series, CN-Series, VM-Series)
- Initial setup, licensing, and management interface access
- Interface configuration, zones, and network addressing
- Basic routing and network connectivity
- Device registration and Strata Cloud Manager onboarding
- Platform fundamentals: stateful inspection, threat prevention concepts

**2. Security Objects and Policies (35%)**
- Address, service, and URL category object creation and management
- Policy rule design, ordering, and enforcement
- Application-based policy and User-ID integration
- Security profiles (antivirus, anti-spyware, vulnerability protection, file blocking)
- Threat prevention subscription selection and application
- NAT policies and network address translation concepts
- Policy logging and traffic analysis

**3. Management and Monitoring (20%)**
- Strata Cloud Manager interface and workflows
- Device management, policy deployment, and updates
- Logging, reporting, and alerting fundamentals
- Admin roles, authentication, and access control
- Backup and restore procedures
- Basic troubleshooting workflows and support tools

**4. Cloud and Advanced Deployments (15%)**
- Cloud-native deployments (Prisma Cloud integration concepts)
- SASE architecture overview
- Multi-site management and policy consistency
- SSL/TLS inspection basics
- Threat prevention in hybrid cloud environments

## Transition guidance (May 2026)

**If you're studying for NETSG (new certification track):**
1. NETSG assumes 1–2 years hands-on Palo Alto Networks experience (firewall deployment, policy management, cloud platform familiarity).
2. If you're a complete newcomer, consider **Network Security Foundational** (no experience required; 90 min, $150 USD) first.
3. Primary study resources: Palo Alto Networks' official course (EDU-115 or equivalent), Beacon free labs, and hands-on firewall configuration (Palo Alto's free VM-Series eval license).
4. Exam emphasizes modern platform features: Strata Cloud Manager, cloud-native deployments, API integration, and PAN-OS 12.x concepts.

**If you're transitioning from legacy PCNSA/PCNSE:**
1. Your PCNSA/PCNSE hands-on knowledge is directly applicable; NETSG tests breadth across modern Strata portfolio.
2. Focus new study on: Strata Cloud Manager (centralized management paradigm shift from Panorama), cloud deployments (Prisma integration), and modern threat prevention.
3. Expect questions on containerized CN-Series and VM-Series virtual firewalls alongside traditional PA-Series hardware.
4. Beacon learning portal bridges legacy and new curriculum; many concepts transfer, but tool UIs have changed.

## Exam environment & delivery (May 2026)

As of August 1, 2025, all Palo Alto Networks certification exams, including NETSG, are delivered exclusively at in-person Pearson VUE test centers. Remote proctoring (OnVUE) is no longer offered. Candidates must schedule an exam appointment at a Pearson VUE center near them using their legal name and government-issued ID. Photo identification and center-administered photograph verification are mandatory. Exam scheduling may be done in advance or same-day, subject to test center availability. Vouchers expire 12 months after purchase; ensure exam is scheduled and completed within that window.
