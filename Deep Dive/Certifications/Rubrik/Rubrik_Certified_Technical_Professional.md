## About

The Rubrik Certified Technical Professional (RTP) certifies advanced ability to design, architect, and deploy Rubrik Cloud Data Management (CDM) solutions and modern Polaris-based infrastructure at scale. RTP holders typically design SLA domain hierarchies, architect multi-cluster replication strategies, configure cloud archive tiers, integrate Rubrik with enterprise backup/DR platforms, and advise customers on ransomware resilience. The cert targets technical pre-sales professionals, solutions architects, and implementation specialists; it sits above the RCSA (Administrator) level and is often a stepping stone toward Rubrik Expert or vendor-agnostic architect roles. The exam reflects current Rubrik CDM topology and Polaris (cloud-native) capabilities introduced from 2019 onward.

## Domain context — Backup, Disaster Recovery & Cyber Resilience

Backup, disaster recovery, and cyber resilience form a converged domain where data protection, ransomware defense, and business continuity overlap. Rubrik pioneered "immutable snapshots" and "anomaly detection" as cornerstones of ransomware resilience. The RTP sits within Rubrik's certification ladder above the RCSA (Associate/Administrator) and serves as a bridge to domain expert credentials.

## Topics covered

Based on official Rubrik exam blueprints, partner training materials, and candidate preparation guides, the RTP exam spans these technical domains:

- **Rubrik CDM Architecture & Topology** — Cluster node types (master, data, metadata), hybrid deployment models, multi-cluster federation, edge appliances
- **Polaris Platform & Cloud-Native Backup** — Rubrik Security Cloud (RSC) architecture, Polaris agent deployment, cloud-native workload protection, serverless/containerized application backup
- **SLA Policy Design & Hierarchy** — Multi-tier SLA domain structures, policy inheritance, advanced snapshot retention logic, tiered backup-to-cloud workflows
- **Cloud Archive & Storage Tiers** — Configuration of AWS S3, Azure Blob, GCP Cloud Storage for archive; Cloud Vault immutable storage; air-gapped backup strategies
- **Ransomware Detection & Threat Hunting** — Polaris ML-based anomaly detection; Sonar sensitive data discovery; threat hunting in backup metadata and snapshots
- **Application-Consistent Backup** — Database protection (Oracle, SQL Server, SAP HANA) with block-level consistency; log shipping and log-forwarding integration
- **Replication & Geo-Distribution** — Cluster-to-cluster replication design; cross-region failover; load balancing and bandwidth optimization
- **Integration & Advanced Features** — Ruby API scripting for automation; ServiceNow / Slack / Splunk webhook integration; RBAC and compliance reporting; Rubrik Radar threat intelligence

## Skills validated

*Specific technologies and competencies tested by the RTP exam.*

- Rubrik Cloud Data Management (CDM) cluster architecture: node roles, hybrid on-premises/cloud topology, federation, edge appliances
- Rubrik Polaris Security Cloud (RSC) and modern cloud-native backup: agent deployment, serverless workload protection, continuous monitoring
- Advanced SLA policy design: hierarchical domain structures, inheritance logic, dynamic retention, policy-driven protection at scale
- Cloud archive and immutable storage: AWS S3, Azure Blob, GCP Cloud Storage integration; Cloud Vault air-gapped backups; S3-compatible storage targets
- Ransomware threat detection and response: Polaris anomaly detection tuning, Sonar sensitive data discovery, threat hunting in backups, rapid recovery from attacks
- Application-consistent database backup: Oracle, SQL Server, SAP HANA backup with transactional consistency; log shipping; database cloning and instant recovery
- Multi-site replication and disaster recovery: cluster-to-cluster replication, cross-region failover, bandwidth optimization, RPO/RTO targeting
- Integration and automation: Ruby API, webhooks (ServiceNow, Slack, Splunk), policy-as-code, RBAC design, compliance reporting (GDPR, HIPAA, SOX)
- Troubleshooting and optimization: log analysis, performance tuning, Rubrik Support tools, cluster health diagnostics
