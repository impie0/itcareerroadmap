## About

AB-100 is Microsoft's expert-level certification for solution architects, senior consultants, and technical leaders designing and governing enterprise-scale AI solutions built on Microsoft platforms. Released in beta in Q1 2026, it focuses on multi-agent orchestration, generative AI integration, secure agentic deployment, and measurable business outcomes. Candidates design scalable, secure, compliant AI-driven systems using Azure OpenAI, Copilot Studio, Agent Framework, and related Microsoft services. General availability is targeted for Q3 2026.

---

## Domain context — AI Security Architecture

Design, deployment, and governance of intelligent agent systems and generative AI workloads in secure, compliant enterprise environments. Architects ensure Zero Trust principles, data protection, compliance posture management, and responsible AI guardrails across agentic workflows.

## Topics covered

- **Design agentic AI solutions** (25–30%) — Multi-agent orchestration, role-based agent design, conversation flow, handoff patterns, state management, semantic kernel architecture
- **Secure agentic AI workloads** (20–25%) — Authentication/authorization for agents, Defender for AI, AI Security Posture Management (AI-SPM), prompt injection mitigation, content filtering, responsible AI governance
- **Integrate business processes** (15–20%) — Agent integration with Azure services, data sources, APIs, enterprise applications; Copilot Studio orchestration
- **Implement governance & compliance** (15–20%) — Data residency, audit logging, model monitoring, fine-tuning guardrails, regulatory alignment (SOC 2, HIPAA, GDPR)
- **Measure & optimize outcomes** (10–15%) — Cost optimization, agent performance metrics, user feedback loops, iteration strategies

## Cybersecurity AI Path: Combining AB-100 with SC-100/SC-500

If your goal is **cybersecurity AI architect** specifically, Microsoft recommends:

1. **Start with SC-500** (Cloud and AI Security Engineer) — covers AI workload security, Defender for AI, compliance basics
2. **Then AB-100** (Agentic AI Architect) — design patterns, governance, multi-agent orchestration
3. **Conclude with SC-100** (Cybersecurity Architect Expert) — enterprise security strategy, Zero Trust, cross-domain governance

This **SC-500 → AB-100 → SC-100** sequence aligns with Microsoft's 2026 AI security strategy: foundation in cloud/AI security, then architectural breadth, then executive-level strategy.

**Estimated timeline:** 6–9 months (3–4 months per cert, with overlap study possible).
