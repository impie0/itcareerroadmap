## About

PL-300 is Microsoft's Associate-level certification for Power BI practitioners, launched August 2021, targeting data analysts and BI professionals building analytics solutions on the Power BI platform. Unlike entry-level Power BI fundamentals (PL-900), PL-300 demands hands-on competency across the full analytics lifecycle: connecting to data sources, cleaning and transforming data via Power Query, designing dimensional models with relationships and calculated columns, writing DAX measures (CALCULATE, FILTER, time-intelligence functions), creating interactive visualizations and dashboards, implementing row-level security (RLS), publishing to Power BI Service, and managing workspaces and deployment pipelines. The exam balances theory with performance-based tasks where candidates must configure Power BI solutions in a sandbox environment. Passing PL-300 establishes intermediate-to-senior analyst credentials and unlocks pathways to PL-500 (Power Automate RPA), PL-600 (Solutions Architect Expert), or specializations in Fabric Data Engineering (DP-700).

## Topics covered

The exam blueprint organizes topics across four weighted domains:

- **Prepare the data (25–30%)** — identify and connect to data sources (SQL Server, Azure SQL Database, Excel, SharePoint, APIs), use Power Query for data profiling and cleansing, apply transformations (merge, append, pivot, unpivot), handle missing or duplicate data
- **Model the data (25–30%)** — design dimensional models (star schema, snowflake schema), define table relationships (one-to-many, many-to-many), manage cardinality and cross-filtering direction, create calculated columns and calculated tables using DAX, write DAX measures (SUM, AVERAGE, CALCULATE, FILTER, ALL, RELATED, ROW_NUMBER), implement time-intelligence functions (YEAR-TO-DATE, MONTH-TO-DATE, prior-year comparisons)
- **Visualize and analyze the data (25–30%)** — create reports with multiple visualizations (tables, charts, maps, KPIs), choose appropriate visuals for data type and narrative, apply conditional formatting, create drill-through, bookmarks, and slicers, use Q&A natural-language queries, leverage AI visuals (decomposition tree, key influencers, Smart Narrative), implement tooltips and report-level filters, design mobile-optimized layouts
- **Deploy and maintain assets (15–20%)** — manage Power BI workspaces and users (Viewer, Contributor, Admin roles), configure row-level security (RLS) using DAX, publish reports to Power BI Service, manage datasets and refresh schedules (import, DirectQuery, dual-mode), create and manage deployment pipelines (Dev/Test/Prod), understand Power BI Premium vs. Pro licensing, monitor usage and performance

## Skills validated

- Power Query data transformation (M language, connectors, steps)
- DAX formula authoring (calculated columns, measures, calculated tables)
- DAX functions: CALCULATE, FILTER, ALL, RELATED, ROW_NUMBER, SUMX, DISTINCTCOUNT
- Time-intelligence DAX: DATEADD, DATESYTD, PARALLELPERIOD, TOTALMTD
- Star schema and dimensional modeling (fact/dimension tables, relationships, cardinality)
- Power BI Desktop report and visualization creation
- Conditional formatting and data bars in tables
- Drill-through, bookmarks, and report-level interactions
- Power BI Service workspace management and sharing
- Row-level security (RLS) implementation via DAX
- Refresh scheduling (import, DirectQuery, dual-mode)
- Deployment pipelines (Dev/Test/Prod environments)
- Power BI Premium vs. Pro licensing and capacity management
