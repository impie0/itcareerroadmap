## About

The **SC-900** is Microsoft's entry-level certification for security, compliance, and identity (SCI) concepts across Azure and Microsoft 365 environments. Released in 2021, it serves as a foundational stepping stone for IT professionals, business analysts, and non-technical stakeholders who want to understand cloud security principles, identity management, compliance frameworks, and Microsoft's integrated security solutions—without requiring hands-on lab experience or prior security certification. The exam does not expire and requires no renewal, making it an ideal low-risk entry point into the Microsoft security ecosystem.

## Topics covered

The SC-900 exam blueprint is organized into four domains with the following weights:

- **Describe the concepts of security, compliance, and identity (10–15%)**
  - Shared responsibility model, Zero Trust architecture, defense in depth, encryption fundamentals, data residency and sovereignty, common security threats (phishing, ransomware, DDoS)

- **Describe the capabilities of Microsoft Entra (25–30%)**
  - Authentication and identity models, single sign-on (SSO), multi-factor authentication (MFA), Conditional Access, Azure AD (now Microsoft Entra ID), external identities (B2B, B2C), Azure AD Premium features

- **Describe the capabilities of Microsoft security solutions (35–40%)**
  - Microsoft Defender (Defender for Cloud, Defender for Endpoint, Defender for Identity, Defender for Cloud Apps), Microsoft 365 Defender, DLP (data loss prevention), email/collaboration security, threat intelligence, incident response

- **Describe the capabilities of Microsoft compliance solutions (20–25%)**
  - Microsoft Purview (compliance portal), Compliance Manager, Information Protection, eDiscovery, audit logging, regulatory compliance (GDPR, HIPAA, PCI-DSS, SOC 2), insider risk management
