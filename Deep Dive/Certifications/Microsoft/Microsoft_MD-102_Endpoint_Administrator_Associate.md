## About

MD-102 certifies endpoint administrators responsible for deploying, configuring, securing, managing, and monitoring devices and client applications in Microsoft 365 environments. Launched March 28, 2023, it is the standalone Associate-level credential for modern endpoint management via Intune, Windows Autopilot, Azure Virtual Desktop, and Windows 365. Candidates typically have hands-on experience with Microsoft Entra ID, Microsoft Defender for Endpoint, and Windows client deployment.

---

## Domain context — Endpoint/M365

Modern device and application management across Windows, cloud, and hybrid environments. Endpoint administrators bridge IT operations and identity/security teams to enforce policy, patch, and compliance at scale.

## Topics covered

- **Deploy Windows Client (25–30%)** — Windows 10/11 deployment via Autopilot, manual installation, provisioning packages, modern driver management
- **Manage Identity and Compliance (15–20%)** — Entra ID device registration, device compliance policies, conditional access integration, compliance monitoring
- **Manage, Maintain, and Protect Devices (40–45%)** — Intune enrollment and onboarding, device lifecycle, updates and patches, configuration profiles, endpoint protection
- **Manage Applications (15–20%)** — App deployment via Intune, Win32 app packaging, Microsoft Store for Business, app protection policies, app-based conditional access
