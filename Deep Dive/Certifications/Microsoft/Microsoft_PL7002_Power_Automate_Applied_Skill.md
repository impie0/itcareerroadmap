## About

Microsoft Applied Skills credentials validate the ability to solve real-world business problems using Power Automate. This credential demonstrates competency in creating cloud flows, building automated workflows, configuring approvals, testing automation logic, and managing process automation at scale. Unlike traditional certification exams, PL-7002 uses interactive lab-based assessment where candidates perform actual automation tasks in a live Power Automate environment. This credential aligns with the Microsoft Power Platform ecosystem and serves as an entry-level validation for process automation professionals, RPA practitioners, and business analysts working with Power Automate.

## Domain context — Automation & RPA

Robotic Process Automation (RPA) and low-code automation are core competencies in modern IT operations, enabling organizations to streamline repetitive tasks, reduce manual effort, and improve process efficiency. Process automation spans cloud flows, desktop RPA, approval workflows, and business process optimization.

## Topics covered

The PL-7002 assessment evaluates the following skill areas based on real-world automation scenarios:

- **Creating and configuring cloud flows** (triggers, actions, conditional logic)
- **Building approval workflows** (single and multi-stage approvals)
- **Implementing automation logic** (expressions, variables, scope conditions)
- **Testing and troubleshooting flows** (run history, error handling)
- **Sharing and managing cloud flows** (permissions, access control)
- **Connecting to external systems** (connectors, API calls, data integration)
- **Implementing error handling and retries**
- **Managing flow performance and optimization**
