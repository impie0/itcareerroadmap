## About

The AZ-140 is Microsoft's **specialty-level** certification for Azure Virtual Desktop (AVD). It validates the ability to architect, configure, and operate multi-user desktop and app virtualization on Microsoft Azure. Launched in September 2021 as Azure Hyperscale Remote Desktop replaced on-premises VDI strategies, AZ-140 is the dedicated exam for professionals building and maintaining cloud-hosted desktop ecosystems integrated with M365, identity services (Entra ID), and Microsoft Intune. It is narrower than AZ-104 (general Azure infrastructure) but deeper in the VDI/endpoint computing domain.

## Topics covered

The official exam blueprint weights the following domains:

- **Plan and implement Azure Virtual Desktop infrastructure (40–45%)** — Deploy session host pools, configure host pool properties, manage scaling plans, configure storage for user profiles (FSLogix), plan for resilience and high availability.
- **Plan and implement identities and security profiles (10–15%)** — Entra ID integration, hybrid identity scenarios, Conditional Access policies, multi-factor authentication, single sign-on for AVD.
- **Manage access and security (10–15%)** — Azure role-based access control (RBAC), Azure Security Center recommendations for AVD, Network security groups, private endpoints, monitoring security posture.
- **Manage user environments and applications (20–25%)** — Deploy RemoteApp applications, manage user sessions, configure application groups, license management (per-user or per-seat), user profile management (FSLogix containers).
- **Monitor and maintain Azure Virtual Desktop (10–15%)** — Azure Monitor for AVD diagnostics, Log Analytics, performance metrics, troubleshoot connectivity and performance issues, backup and disaster recovery.

- Azure Virtual Desktop (AVD) host pool deployment and configuration

- FSLogix Profile Container and Office Container configuration

- Conditional Access and device compliance policies

- RemoteApp application publishing and user assignment
