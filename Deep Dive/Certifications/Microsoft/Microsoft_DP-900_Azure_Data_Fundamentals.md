## About

DP-900 is Microsoft's foundational data certification, launched January 2021, targeting professionals beginning their journey into cloud data services. Entry-level by design — no prior cloud or data experience required — it covers the core concepts needed to understand relational databases, non-relational data stores, analytics workloads, and Azure's data platform services. The exam validates ability to describe what data is, how it's structured, and how Azure manages it; popular with career changers, recent graduates, and IT generalists transitioning into data roles. Passing DP-900 unlocks eligibility for advanced data certifications: DP-100 (Data Scientist), DP-300 (Database Administrator), or DP-700 (Fabric Data Engineer).

## Topics covered

The exam blueprint organizes topics across four domains, weighted as follows:

- **Describe core data concepts** (25–30%) — data formats (structured, semi-structured, unstructured), transactional vs. analytical workloads, roles (data engineer, analyst, architect)
- **Identify considerations for relational data on Azure** (20–25%) — SQL databases, schemas, indexes, queries, Azure SQL Database, managed instances, pools
- **Describe considerations for working with non-relational data on Azure** (15–20%) — NoSQL principles, document/key-value/graph stores, Azure Cosmos DB, time-series data, Azure Table Storage
- **Describe an analytics workload on Azure** (20–25%) — OLAP, data warehouses, Azure Synapse Analytics, data lakes, visualization, Power BI, real-time analytics

## Skills validated

*Cert-specific — what this exam actually tests.*

- Azure SQL Database setup and querying (T-SQL basics)
- Azure Cosmos DB document model and NoSQL concepts
- Azure Data Lake Storage (ADLS) and Parquet format
- Azure Synapse Analytics workspaces and SQL pools
- Power BI report and dashboard fundamentals
- Data ingestion patterns (batch vs. streaming)
- Star schema and data warehouse design principles
- Relational vs. non-relational trade-offs
- Time-series and IoT data handling in Azure
- Data governance and compliance in cloud
