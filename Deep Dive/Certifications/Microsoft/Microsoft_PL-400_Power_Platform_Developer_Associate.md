## About

PL-400 is an associate-level certification validating professional expertise in designing, developing, testing, and troubleshooting solution components using the extension points of Microsoft Power Platform. Candidates typically possess development experience with Power Apps, Power Automate, Dataverse, and related Microsoft cloud services. The exam emphasizes custom user experiences, business logic implementation, system integrations, and complex automation workflows. Updated March 19, 2026, the current version focuses on advanced Power Fx logic, plugin development, cloud flows, and AI-augmented solutions.

## Topics covered

Official exam blueprint (Microsoft Learn):

- **Create a technical design (10–15%)**
  - Evaluate custom development requirements
  - Design data models and entity schemas
  - Design security architecture and access control
  - Design solution architecture and cloud integration patterns

- **Configure Common Data Service / Dataverse (15–20%)**
  - Create and manage tables, columns, relationships
  - Configure option sets, choice columns, and calculated fields
  - Manage business rules and form configurations
  - Implement choice columns and hierarchical data

- **Create and manage Power Apps (15–20%)**
  - Build model-driven and canvas apps
  - Create and configure forms, views, dashboards
  - Implement custom controls and PCF components
  - Manage app navigation and user experience
  - Implement responsive design and accessibility

- **Create and manage Power Automate flows (15–20%)**
  - Build automated, instant, and scheduled flows
  - Configure cloud flows and process flows
  - Implement error handling and flow logic
  - Integrate with connectors and APIs
  - Design complex flow patterns (async, approval, RPA)

- **Extend Power Apps (15–20%)**
  - Develop custom controls using Power Apps Component Framework (PCF)
  - Build custom connectors and connection references
  - Extend model-driven apps with JavaScript and plugins
  - Implement embedded Power BI and Power Virtual Agents
  - Create reusable code components

- **Develop integrations (10–15%)**
  - Integrate with external systems via APIs and webhooks
  - Configure service principals and authentication
  - Build custom connectors
  - Implement data migration and synchronization patterns
  - Manage integration security and compliance

*Source:* [Study guide for Exam PL-400 ↗](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/pl-400)

## Skills validated

*Cert-specific technologies and tools tested by PL-400.*

- **Languages & frameworks:** JavaScript, TypeScript, C#, Power Fx (Power Platform formula language)
- **Power Platform services:** Power Apps (canvas & model-driven), Power Automate, Power Virtual Agents, Power BI embedded
- **Dataverse:** Table and column design, relationships, calculated columns, business rules, security
- **Extension points:** Power Apps Component Framework (PCF), custom controls, JavaScript event handlers, plugins
- **Connectors & integration:** Standard and custom connectors, webhook design, API authentication (OAuth, service principals)
- **Cloud platforms:** Microsoft Azure (App Service, Functions, Logic Apps), Azure AD / Entra ID
- **Developer tools:** Visual Studio Code, Power Platform CLI, solution deployment (ALM)
- **Automation patterns:** Cloud flows, desktop flows (RPA), approval flows, async patterns
- **Security & compliance:** RBAC, environment isolation, data loss prevention (DLP), audit logging
