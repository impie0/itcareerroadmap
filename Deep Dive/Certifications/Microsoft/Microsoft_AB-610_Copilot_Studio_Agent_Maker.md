## About

The Copilot Studio Agent Maker (AB-610) is a new entry-level certification for business users, makers, IT professionals, and citizen developers building low-code AI agents in Microsoft Copilot Studio. Released in 2026, it positions accessibility and no-code agent design as core competencies for the Power Platform ecosystem. Unlike the concurrent AB-620 (Agent Developer) aimed at professional developers, the Agent Maker certification focuses on practical scenario-based design using pre-built connectors, knowledge sources, and topic configuration within the Copilot Studio UI.
