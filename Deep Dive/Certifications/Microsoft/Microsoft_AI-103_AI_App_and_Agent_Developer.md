## About

Exam AI-103 validates expertise in building AI applications and agentic solutions on Microsoft Azure, with emphasis on generative AI, multi-agent orchestration, and responsible AI deployment. Launched in April 2026 as beta, AI-103 replaces the retiring AI-102 (Azure AI Engineer) effective June 30, 2026, reflecting Microsoft's shift toward agentic AI frameworks and Azure AI Foundry. Target audience: software developers with Python experience building intelligent solutions using Azure OpenAI Service, Cognitive Services, and Microsoft Foundry. As of May 2026, beta testers receive 80% discount (code AI103Claxton) until May 7; general availability planned for June 2026.

## Topics covered

- **Plan and Manage Azure AI Solutions** (25–30% weight)
  - Azure AI Foundry environment provisioning and governance
  - Solution architecture using Azure OpenAI, Cognitive Services, and AI Search
  - Responsible AI principles and content filtering
  
- **Implement Generative AI and Agentic Solutions** (30–35% weight)
  - Generative AI application development (prompt engineering, LLM integration)
  - Multi-agent orchestration and tool integration
  - RAG (Retrieval-Augmented Generation) pipelines with Azure AI Search
  - Deployment and monitoring in production environments
  
- **Implement Computer Vision Solutions** (10–15% weight)
  - Azure Computer Vision API for image analysis and classification
  - Document Intelligence (OCR, form recognition, layout analysis)
  - Multimodal vision-language model integration
  
- **Implement Text Analysis Solutions** (10–15% weight)
  - Azure Language service (NLP, sentiment analysis, entity recognition)
  - Custom text classification and Named Entity Recognition (NER)
  
- **Implement Information Extraction Solutions** (10–15% weight)
  - Document extraction and knowledge mining
  - Azure Cognitive Search for semantic retrieval
