## About

MS-900 was Microsoft's foundational entry-point certification for cloud and productivity professionals, validating baseline understanding of Microsoft 365 services, cloud concepts, security, and licensing. Released in 2020, it became the standard stepping-stone for IT support staff, business users, and career-changers entering the Microsoft ecosystem. The exam was retired on March 31, 2026, as Microsoft shifted focus toward AI-enabled certifications (AB-900), though credentials earned before retirement remain valid indefinitely.

## Topics covered

- **Describe cloud concepts (10–15%)**
  - Types of cloud services (SaaS, PaaS, IaaS)
  - Shared responsibility model
  - Microsoft cloud services overview

- **Describe Microsoft 365 apps and services (30–35%)**
  - Core productivity apps (Word, Excel, PowerPoint, OneNote)
  - Collaboration tools (Teams, SharePoint, OneDrive)
  - Management and analytics (Planner, Project, Forms, Lists)
  - Email and calendaring (Exchange, Outlook)
  - Stream, Loop, and content services

- **Describe security, compliance, privacy, and trust in Microsoft 365 (30–35%)**
  - Identity and access (Microsoft Entra ID / Azure AD)
  - Security capabilities (threat protection, data loss prevention)
  - Compliance and regulatory frameworks
  - Privacy and data governance

- **Describe Microsoft 365 pricing, licensing, and support (5–10%)**
  - Microsoft 365 subscription plans and licensing models
  - Billing and subscription management
  - Support options and service level agreements
