## About

PL-200 validates intermediate competency in configuring and customizing Microsoft Power Platform solutions for business organizations. Introduced in 2020, it is role-based (functional consultant) rather than entry-level, requiring demonstrated hands-on experience with Power Apps, Power Automate, and Microsoft Dataverse. Holders configure business processes, create canvas and model-driven apps, design automations, implement integrations, and facilitate user adoption within enterprise environments. **Critical:** This certification retires August 31, 2026. Microsoft recommends transitioning to the new AB-410 Intelligent Applications Builder Associate certification for future upskilling in low-code application development.

## Topics covered

- **Configure Microsoft Dataverse (25–30%)**
  - Create and manage tables, columns, relationships, and calculated/rollup fields
  - Model data for business requirements; enforce data types and formats
  - Apply row-level and organization-level security
  - Manage business rules, workflows, and process flows

- **Create apps using Microsoft Power Apps (25–30%)**
  - Design and build canvas apps (responsive, form-driven, gallery-based)
  - Create and customize model-driven apps (forms, views, dashboards)
  - Configure app navigation, components, and embedded Power BI tiles
  - Apply formula language (Power Fx) in canvas apps
  - Implement data validation and user experience patterns

- **Create and manage Power Automate logic and process automation (25–30%)**
  - Build cloud flows (automated, instant/button, scheduled)
  - Design multi-step workflows with conditions, loops, and error handling
  - Select and configure connectors; integrate with third-party APIs
  - Implement approval flows and process mining (process advisor)
  - Monitor and troubleshoot automations

- **Manage environments (15–20%)**
  - Create and configure Power Platform environments
  - Deploy solutions (managed/unmanaged) across environments
  - Apply environment security and data policies
  - Manage analytics and reporting on Power Platform usage
  - Monitor performance and optimize resource consumption

## Important notice: Certification retirement

**This certification retires on August 31, 2026.** Microsoft has announced this as part of its strategic pivot toward AI-first and intelligent applications certifications. Candidates should plan accordingly:

- If you plan to pursue PL-200, complete the exam by August 31, 2026.
- If you already hold PL-200, you have until August 31, 2026 to renew; after this date, the cert is no longer valid.
- **Microsoft's recommended transition path:** AB-410 Intelligent Applications Builder Associate (available April 2026) or AB-100 Agentic AI Business Solutions Architect for expert-track progression.
