## About

AZ-040 is a Microsoft official training course (not a formal exam-based certification, but rather an Applied Skills credential) that teaches IT professionals to automate administration of Windows servers and Azure resources using PowerShell. The course is designed for IT professionals with experience in Windows Server, Windows Client, Azure, or Microsoft 365 administration who want to add scripting and automation capabilities. Upon completion, learners gain proficiency in PowerShell fundamentals, cmdlets, scripting, remoting, Azure automation, and Microsoft 365 service management. The course is actively maintained and widely available through Microsoft Learn (free) and certified training partners globally.

---

## Domain context — Automation/Scripting

Automation and scripting form the backbone of modern IT operations, enabling systems administrators and infrastructure engineers to reduce manual effort, improve consistency, and scale management across hundreds or thousands of servers. PowerShell is Microsoft's native scripting and automation platform, spanning Windows Server, Azure, Microsoft 365, Exchange, SharePoint, and SQL Server environments.

## Topics covered

Based on Microsoft Learn course structure and training partner outlines:

- PowerShell fundamentals and the execution environment
- Cmdlets and the pipeline architecture
- Variables, operators, and control flow
- Scripts and functions (creation, parameters, scope)
- Modules and module discovery
- Object handling and property access
- Windows Management Instrumentation (WMI) and CIM cmdlets
- Local and remote command execution
- Persistent remoting sessions and advanced remoting techniques
- PowerShell Desired State Configuration (DSC) concepts
- Active Directory administration via PowerShell
- Managing Azure resources with the Azure PowerShell module (Az)
- Microsoft 365 service administration (Exchange, SharePoint, Microsoft Teams)
- Background jobs and scheduled jobs
- Error handling and debugging
