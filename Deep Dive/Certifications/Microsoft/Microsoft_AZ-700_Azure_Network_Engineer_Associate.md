## About

The AZ-700 validates expertise in designing, implementing, and managing Azure networking infrastructure at scale. Released in March 2022, this Associate-level exam is the primary credential for cloud network engineers specializing in hybrid connectivity, VPNs, ExpressRoute, network security, routing, and private access to Azure services. It is distinct from AZ-104 (Administrator) and AZ-305 (Solutions Architect), focusing specifically on networking depth across the Azure platform. Target audience: network professionals transitioning to Azure, Azure administrators specializing in networking, and infrastructure engineers building enterprise-grade hybrid connectivity solutions. Certification validity is 1 year; renewal is free if you pass the updated renewal exam, or you can recertify by passing AZ-305.

## Topics covered

**Exam objectives by domain weight (from official blueprint):**

- Design and implement core networking infrastructure (20–25%)
- Design, implement, and manage connectivity services (20–25%)
- Design and implement application delivery services (20–25%)
- Secure network connectivity to Azure resources (15–20%)
- Design and implement private access to Azure services (5–10%)

**Core topics include:**
- Virtual networks, subnets, IP addressing, and network segmentation
- Hybrid connectivity (VPN Gateway, ExpressRoute, site-to-site/point-to-site VPNs)
- Routing and traffic engineering (user-defined routes, Network Watcher)
- Load balancing and application delivery (Azure Load Balancer, Application Gateway, Traffic Manager, Azure Front Door)
- Network security (Network Security Groups, Azure Firewall, DDoS Protection Standard, Web Application Firewall)
- Private access patterns (Private Link, Private Endpoints, service endpoints)
- Network monitoring and diagnostics (Network Watcher, Traffic Analytics, Connection Monitor)

## Skills validated

*Cert-specific — what this exam tests, distinct from general Cloud Associate skills.*

- Azure Virtual Network (vNets) design and management
- Hybrid connectivity technologies: VPN Gateway, ExpressRoute, site-to-site and point-to-site VPNs
- Advanced routing: User-Defined Routes (UDRs), Border Gateway Protocol (BGP), route optimization
- Load balancing: Azure Load Balancer, Application Gateway, Azure Traffic Manager, Front Door
- Application delivery patterns: CDN integration, traffic engineering
- Network security: Network Security Groups (NSGs), Azure Firewall, Web Application Firewall (WAF), DDoS Protection Standard
- Private access: Private Link, Private Endpoints, service endpoints, and secure connectivity to PaaS services
- Network monitoring and diagnostics: Network Watcher, Traffic Analytics, Connection Monitor, diagnostics tooling
- IP addressing and network segmentation: subnetting, CIDR, IPv6 readiness
- Azure networking SDKs and CLI tools
