## About

Introduced in 2022, AZ-305 validates expert-level skills in designing secure, scalable, and reliable solutions on Microsoft Azure for hybrid and cloud-native workloads. Holders advise stakeholders, translate business requirements into Azure architecture, and lead implementation across identity, governance, data platforms, and compute services. AZ-305 replaced older AZ-303 in 2022 and continues to evolve; the exam was most recently updated April 17, 2026. This is an expert-level credential requiring both technical depth (networking, security, data, DevOps, business continuity) and architectural decision-making acumen.

## Topics covered

The exam blueprint breaks into four domains with published weights:

- **Design Identity, Governance, and Monitoring Solutions (25–30%)**
  - Access control using Entra ID (formerly Azure AD), role-based access control (RBAC), conditional access policies
  - Governance frameworks: Azure Policy, Management Groups, blueprints
  - Monitoring and observability: Azure Monitor, Log Analytics, Application Insights, alert design
  - Audit and compliance: Azure Audit, regulatory requirements

- **Design Data Storage Solutions (25–30%)**
  - Relational databases: Azure SQL Database, Azure Database for PostgreSQL/MySQL
  - NoSQL: Azure Cosmos DB, table storage, blob storage
  - Storage access tiers (hot, cool, archive) and redundancy (LRS, ZRS, GRS, GZRS)
  - Data encryption, backup, and disaster recovery for data platforms
  - Data migration strategies (cloud-native vs. lift-and-shift)

- **Design Infrastructure Solutions (30–35%)**
  - Compute: App Service, Azure Container Instances, Azure Kubernetes Service (AKS), Azure Functions, Virtual Machines
  - Networking: virtual networks (vNets), subnets, Network Security Groups (NSGs), Application Gateway, Azure Front Door, Traffic Manager
  - Load balancing and traffic management strategies
  - Private Link, service endpoints, and hybrid connectivity (ExpressRoute, VPN)
  - Migration strategies: Rehost, Refactor, Rearchitect, Repurchase

- **Design Business Continuity Solutions (10–15%)**
  - High availability: availability zones, redundancy across regions, active-active vs. active-passive
  - Disaster recovery: RTO/RPO targets, backup strategies, failover planning
  - Azure Site Recovery, Azure Backup, and recovery vaults
  - Business continuity frameworks and testing

_Source:_ [Official Exam Objectives ↗](https://learn.microsoft.com/en-us/credentials/certifications/exams/az-305/)
