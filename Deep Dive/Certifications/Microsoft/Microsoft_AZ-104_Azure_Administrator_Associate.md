## About

The Microsoft Certified: Azure Administrator Associate (AZ-104) validates the skills to manage Azure subscriptions, identities, governance, storage, compute resources, networking, and monitoring at the Associate level. Released in 2018 and updated in 2025 to include Microsoft Entra ID (formerly Azure AD), Bicep infrastructure-as-code, and deployment stacks, this certification is the foundational credential for cloud operations roles. It is typically pursued after AZ-900 and often leads to AZ-305 (Solutions Architect Expert).

## Topics covered

The exam is divided into five domains with the following weightings:

- **Manage Azure Identities and Governance** (15–20%) — Microsoft Entra ID, RBAC, role assignments, subscriptions, management groups, policy
- **Implement and Manage Storage** (15–20%) — Storage accounts, redundancy, blob/file management, access controls, SAS tokens, encryption
- **Deploy and Manage Azure Compute Resources** (20–25%) — VMs, VM scale sets, Azure App Service, containers, ARM templates, Bicep, autoscaling
- **Implement and Manage Virtual Networking** (15–20%) — Virtual networks, subnets, routing, load balancing, VPN, Azure ExpressRoute, network security
- **Monitor and Maintain Azure Resources** (10–15%) — Azure Monitor, alerts, Log Analytics, backup, disaster recovery, Azure Site Recovery

*Source:* [Exam AZ-104 Skills Measured ↗](https://learn.microsoft.com/en-us/credentials/certifications/exams/az-104)

## Skills validated

*Cert-specific — what this exam actually tests, distinct from the shared "Common skills" above.*

- Microsoft Entra ID (formerly Azure AD) and identity management
- Azure role-based access control (RBAC) and custom roles
- Azure subscriptions, management groups, and governance policies
- Azure Storage (blob, file, queue, table services)
- Managed disks and unmanaged storage accounts
- Virtual machines (creation, scaling, patching, backup)
- Virtual Machine Scale Sets (VMSS)
- Azure App Service (web apps, function apps)
- Containers and Azure Container Instances (ACI)
- Azure Container Registry (ACR)
- Virtual networks, subnets, and network security groups (NSG)
- Network routing and Azure ExpressRoute
- Azure Load Balancer and Application Gateway
- Azure Backup and Azure Site Recovery
- Azure Monitor and Log Analytics
- Bicep and ARM template authoring
- Azure CLI and PowerShell resource management
- Cost management and Azure Advisor recommendations
