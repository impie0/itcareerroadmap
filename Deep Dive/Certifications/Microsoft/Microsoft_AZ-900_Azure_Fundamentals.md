## About

AZ-900 is Microsoft's entry-level cloud certification, launched in 2018. It validates foundational knowledge of cloud concepts, Azure services, governance, and compliance — pitched for non-technical personas (sales, business analysts, early-career IT). Unlike associate-level certs (AZ-104, AZ-204), AZ-900 requires zero hands-on experience and is primarily a conceptual exam. Earning AZ-900 is a prerequisite mindset for any Azure career path; it unlocks eligibility for specialist Azure roles and acts as a stepping stone to associate-level certifications. Updated in January 2026 with minor adjustments to identity, access, and resource management sections.

## Topics covered

- **Cloud Concepts (25–30%)**: Define cloud computing, shared responsibility model, cloud benefits (cost, scalability, reliability), cloud service models (IaaS, PaaS, SaaS), cloud deployment models (public, private, hybrid)
- **Azure Architecture and Services (35–40%)**: Core compute (VMs, App Service, Azure Functions), networking (VNets, DNS, Load Balancer), storage (Blob, File, Queue), databases (SQL Database, Cosmos DB), analytics (Synapse, Data Lake)
- **Azure Management and Governance (30–35%)**: Cost management (budgets, cost analysis), governance (Azure Policy, role-based access control), monitoring (Azure Monitor, Application Insights), compliance (standards, certifications, data residency), service-level agreements
