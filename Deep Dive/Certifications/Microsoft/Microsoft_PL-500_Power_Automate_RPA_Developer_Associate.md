## About

The PL-500 validates expertise in designing, developing, and deploying robotic process automation (RPA) solutions using Microsoft Power Automate for desktop and cloud flows. Launched in 2021 as part of the Microsoft Power Platform certification suite, this associate-level credential targets automation developers and citizen developers automating high-volume, repetitive business processes. **Retirement in June 2026 marks the end of this certification path; Microsoft has not published a direct replacement, signaling a shift toward AI-powered solutions (AB-100 Agentic AI Business Solutions Architect) rather than traditional RPA.**

## Topics covered

Exam blueprint weights per Microsoft Learn:

- **Design solutions (20–25%)** — Process analysis using Process Advisor, identify automation candidates, design cloud and desktop flow architectures, model exception handling and governance
- **Develop solutions (35–40%)** — Build cloud flows (automated, instant, scheduled), desktop flows and UI automation, integrate connectors (REST, SOAP, Dataverse), use AI Builder for document processing and form recognition, apply VBA/JavaScript for advanced logic
- **Deploy solutions (15–20%)** — Package and deploy solutions across environments, manage connections and gateways, set environment variables, implement solution pipelines
- **Support and troubleshooting (20–25%)** — Monitor flow performance, debug and troubleshoot failures, analyze logs and telemetry, implement retry policies and error handling

*Source:* [Study guide for Exam PL-500 ↗](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/pl-500)

## Skills validated

- Design cloud flows (automated, cloud-triggered, scheduled) and desktop flows (UI automation, image recognition)
- Develop multi-step workflows with branching, apply loops, handle errors with retry policies
- Integrate REST/SOAP APIs, use connectors for SaaS (Salesforce, Dynamics, SharePoint, Teams, email), on-premises gateways
- Apply AI Builder for document intelligence, form processing, object detection
- Model data transformation and JSON payload mapping
- Implement Dataverse record operations and queries
- Build approval workflows and human-in-the-loop patterns
- Deploy and manage solutions across dev/test/prod environments
- Monitor flow runs, analyze logs, troubleshoot performance bottlenecks
- Use Process Advisor for process mining and optimization metrics
- Apply governance policies (sharing, run-as identities, connection permissions)
