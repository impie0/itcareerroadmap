The AZ-204 certification validates the ability to participate in all phases of Azure application development — from requirements gathering and design through implementation, deployment, security, maintenance, performance tuning, and production monitoring. Targeted at developers (C#/.NET, Python, JavaScript, Java focus), cloud application engineers, and technical professionals transitioning to cloud-native architecture. **Critical status update:** This certification and exam retire on July 31, 2026, and will be replaced by AI-200 (Azure AI Cloud Developer Associate), signaling Microsoft's strategic pivot toward AI-integrated cloud development.

## Topics covered

| Topic | Weight |
|---|---|
| Develop Azure compute solutions (App Service, Functions, containers, scaling) | 25–30% |
| Develop for Azure storage (Blob, Table, Queue, Cosmos DB, data lifecycle) | 15–20% |
| Implement Azure security (Key Vault, Azure AD, encryption, managed identity) | 15–20% |
| Monitor, troubleshoot, and optimize Azure solutions (Application Insights, diagnostics, performance) | 10–15% |
| Connect to and consume Azure services and third-party services (service-to-service integration, APIs, webhooks) | 20–25% |

**Domains within each topic:**

**Azure Compute Solutions:**
- Create and deploy Azure App Service web apps and background services
- Implement containerized solutions (Docker, Azure Container Instances, Container Registry)
- Create and deploy Azure Functions (timer triggers, HTTP triggers, durable functions)
- Configure scaling policies and auto-scale rules
- Implement deployment slots and zero-downtime releases

**Azure Storage:**
- Work with Blob Storage, Table Storage, Azure Queues
- Query and manipulate data in Cosmos DB
- Implement caching strategies (Azure Cache for Redis)
- Manage data lifecycle policies and archival

**Azure Security:**
- Manage secrets and certificates in Azure Key Vault
- Implement identity and access control with Azure AD / Microsoft Entra
- Apply encryption for data at rest (Azure Storage encryption, Transparent Data Encryption)
- Implement encryption in transit (TLS/SSL, Always Encrypted)
- Secure API endpoints and implement managed identity

**Monitoring & Troubleshooting:**
- Configure Application Insights instrumentation
- Set up diagnostic settings and logging
- Create alerts and analyze performance metrics
- Troubleshoot production failures using logs and traces

**Service Integration:**
- Authenticate and consume Azure services (HTTP, managed identity)
- Integrate with third-party APIs and webhooks
- Implement event-driven architectures (Event Grid, Event Hubs, Service Bus)

*Source:* [Microsoft Learn: Study guide for Exam AZ-204 ↗](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/az-204)

## Skills validated

*Cert-specific competencies measured by the AZ-204 exam.*

- Azure App Service (web apps, API apps, function apps, deployment slots, scaling)
- Azure Functions (triggers, bindings, durable functions, timer-based automation)
- Azure Containers (Azure Container Instances, Container Registry, image management)
- Blob Storage, Table Storage, Azure Queues (creation, access policies, SAS tokens)
- Azure Cosmos DB (document databases, query language, consistency models, indexing)
- Azure Cache for Redis (caching strategies, cache-aside pattern, session state)
- Azure Key Vault (secret management, certificate management, access policies)
- Azure SQL Database and Azure Database for PostgreSQL (connection management, encryption, audit logging)
- Azure Application Insights (instrumentation, log queries, custom metrics, availability tests)
- Event Grid, Event Hubs, Service Bus (event-driven architectures, message routing)
- Azure Service Connectors and Managed Identity (passwordless authentication)
- Deployment patterns (Infrastructure-as-Code via ARM templates / Bicep, zero-downtime deployments)
- Performance tuning and cost optimization (resource sizing, reserved instances, autoscaling)
- Security best practices (least privilege, encryption in transit/at rest, network isolation)
