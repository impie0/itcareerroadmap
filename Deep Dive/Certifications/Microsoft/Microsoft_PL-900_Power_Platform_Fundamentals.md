## About

PL-900 is Microsoft's entry-level certification for the Power Platform ecosystem, introduced in 2019. It validates foundational knowledge of Power Platform components (Power Apps, Power Automate, Power BI, Dataverse, Power Pages, Copilot Studio) without requiring hands-on development experience. The cert does not expire and is widely recognized as the starting point for citizen developers, IT professionals, and business analysts evaluating low-code solutions. Unlike role-based certifications (PL-100, PL-200, PL-400), PL-900 is vendor-neutral in positioning—it tests conceptual understanding rather than deep technical proficiency.

## Topics covered

- **Describe the business value of Microsoft Power Platform (15–20%)**
  - Identify the core components and how they integrate
  - Describe business process automation, app development, and data analytics use cases

- **Identify core components of Microsoft Dataverse (15–20%)**
  - Tables, columns, relationships, and data types
  - Role-based access and security models
  - Difference between Dataverse and SQL database

- **Describe Microsoft Power Apps (20–25%)**
  - Canvas apps vs. model-driven apps
  - When to use Power Apps vs. other platforms
  - Key app-building capabilities (forms, galleries, controls)

- **Describe Microsoft Power Automate (15–20%)**
  - Cloud flows: automated, instant (button), scheduled
  - Triggers, actions, and connectors
  - Process advisor and process mining

- **Describe Copilot Studio and Power Pages (5–10%)**
  - Copilot Studio for building virtual agents
  - Power Pages for external-facing websites
  - Use cases and integration points
