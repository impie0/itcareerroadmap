## About

PL-7001 is Microsoft's Applied Skills credential for low-code canvas app development. Launched in 2023, it differs from traditional certifications by requiring hands-on completion of real-world app-building scenarios in an interactive lab rather than passing a multiple-choice exam. The credential demonstrates practical ability to design, build, and deploy canvas apps to solve business problems using Power Apps — the low-code platform within Microsoft Power Platform. It is free to attempt, with assessments hosted entirely on Microsoft Learn. No exam fees or external testing platform required. The credential carries no expiration and is appropriate for developers, makers, business analysts, and IT professionals transitioning into low-code development.

## Domain context — Low-Code Development

Low-code platforms abstract away traditional coding, enabling users to build applications through visual design, pre-built connectors, and formula-driven logic. Canvas apps (in Power Apps) represent the entry point for low-code citizen developers and junior app makers. This domain spans platforms like Power Apps, Mendix, OutSystems, and Salesforce Lightning. Microsoft's Applied Skills focus on Power Platform—a set of integrated tools (Power Apps, Power Automate, Power BI, Power Pages) for business applications and automation.

## Topics covered

The PL-7001 lab assessment evaluates practical competence across these areas (no formal percentage weights published by Microsoft, but all are equally emphasized):

- **Getting Started with Power Apps Studio** – launching the maker portal, understanding app structure, components, and the canvas-based UI designer
- **Connecting to Data Sources** – setting up connectors to SharePoint, Excel, SQL, Dataverse, and APIs; understanding data refresh and synchronization patterns
- **Building with Forms and Galleries** – creating data entry forms with built-in modes (View, Edit, New), configuring gallery controls for display, handling form submissions and validation
- **Microsoft Dataverse Integration** – working with Dataverse tables, relationships, security roles, and real-time sync within canvas apps
- **Authoring and Logic with Power Fx** – writing formulas for calculations, filtering, navigation, and conditional UI behavior (Power Fx is Microsoft's low-code expression language)
- **UI/UX Design** – responsive layouts, theming, branding, icons, images, accessibility, and multi-device considerations
- **Publishing and Sharing** – packaging the app, setting permissions, distributing to users, version control, and lifecycle management

## Skills validated

*Concrete technologies and practices tested in the PL-7001 interactive lab.*

- **Power Apps Studio** – visual designer, maker portal, app structure (screens, data sources, controls)
- **Canvas App Architecture** – component hierarchy, event-driven behavior, state management in Power Fx
- **Data Connectors** – SharePoint, Excel, SQL Server, Dataverse, common SaaS APIs (Outlook, Teams, etc.)
- **Microsoft Dataverse** – table structures, relationships, lookup fields, choice columns, formulas, security roles
- **Power Fx Formulas** – filter, search, sum, count, if/then/else, concatenation, date/time functions, regex
- **Form Controls** – form setup modes, field binding, validation rules, submit handlers, error display
- **Gallery and Data Grid Controls** – filtering, sorting, pagination, layout options, inline editing
- **Navigation and User Experience** – screen transitions, parameter passing, responsive design, accessibility
- **Publishing and Lifecycle** – versioning, sharing, environment/tenant management, admin settings
- **Performance and Limits** – canvas app request throttling, formula optimization, delegation awareness
