# Microsoft Certified: Windows Server Hybrid Administrator Associate

**`AZ-800 + AZ-801`** · ● **Retiring (Sept 30, 2026)** · **Associate** · _Microsoft_

> **Paired exams required:** Both AZ-800 (Administering Windows Server Hybrid Core Infrastructure) and AZ-801 (Configuring Windows Server Hybrid Advanced Services) must be passed to earn the certification. This cert is retiring September 30, 2026, in favor of the consolidated single-exam replacement, AZ-802.

## About

The Windows Server Hybrid Administrator Associate certification validates the ability to administer Windows Server in on-premises and hybrid cloud environments, with deep integration into Microsoft Azure services. Candidates must pass two exams (AZ-800 and AZ-801) covering core infrastructure, advanced services, high availability, disaster recovery, and server migration. This credential is retiring September 30, 2026, in favor of AZ-802, a single consolidated exam expected to release in August 2026 (beta June 2026). Existing cert holders need only pass AZ-802 to renew; new candidates should plan migration to AZ-802 by the retirement deadline.

## Topics covered

### AZ-800: Administering Windows Server Hybrid Core Infrastructure

- Deploy and manage Active Directory Domain Services (AD DS) in on-premises and cloud environments (30–35%)
- Manage Windows Servers and workloads in a hybrid environment (10–15%)
- Manage virtual machines and containers (15–20%)
- Implement and manage an on-premises and hybrid networking infrastructure (15–20%)
- Manage storage and file services (15–20%)

### AZ-801: Configuring Windows Server Hybrid Advanced Services

- Secure Windows Server on-premises and hybrid infrastructures (25–30%)
- Implement and manage Windows Server high availability (10–15%)
- Implement disaster recovery (10–15%)
- Migrate servers and workloads (20–25%)
- Monitor and troubleshoot Windows Server environments (20–25%)

## Skills validated

*Concrete technologies and systems this exam specifically measures — distinct from broader Associate-level skills.*

- **Active Directory Domain Services (AD DS)** — deployment, domain trusts, multi-forest, replication, Global Catalogs, schema
- **Azure Arc** — server registration, hybrid asset management, policy enforcement, updates via Azure
- **Windows Admin Center** — remote server management, cluster management, gateway deployment
- **PowerShell Core** — scripting, remote execution, module development, DSC (Desired State Configuration)
- **Hyper-V** — virtual machine provisioning, live migration, replication, cluster validation
- **Storage Spaces Direct (S2D)** — hyper-converged infrastructure, fault domains, tiering
- **Azure File Sync** — cloud tiering, sync group management, endpoint registration
- **Windows Server Failover Clustering** — quorum, role high availability, health monitoring
- **Azure Site Recovery** — replication policies, failover/failback, recovery plans
- **Azure Migrate** — server assessment, dependency mapping, replication/migration orchestration
- **Network configuration** — DNS, DHCP, network policies, virtual networks (Azure)
- **Disaster recovery** — RPO/RTO, backup/restore strategies, site recovery coordination
- **Monitoring and diagnostics** — Event Viewer, Performance Monitor, Azure Monitor, Log Analytics
- **Security hardening** — Windows Defender, firewall rules, AppLocker, UAC, credential guard
