## About

The Microsoft Certified: AI Agent Builder Associate validates your ability to design, build, and deploy enterprise-grade AI agents using Microsoft Copilot Studio and the Microsoft cloud platform. Released in beta (April 2026) and expected to reach general availability in June 2026, this certification targets IT app developers, consultants, and ISV partners building scalable, integrated AI solutions. Candidates demonstrate proficiency with Copilot Studio's agent framework, generative AI orchestration patterns (RAG, MCP, Agent2Agent), Power Fx, Microsoft Dataverse, and Microsoft 365 Copilot integration.

## Topics covered

Based on the official exam blueprint (partial disclosure during beta):

- **Agent Architecture & Design (25–30%)**
  - Agent types and capabilities in Copilot Studio
  - Multi-agent orchestration patterns (Agent2Agent protocol)
  - Agent state management and context handling
  - Security, compliance, and governance for agents
  
- **Generative AI Integration (20–25%)**
  - Retrieval-Augmented Generation (RAG) with knowledge sources
  - LLM selection and prompt engineering for agents
  - Model Context Protocol (MCP) integration
  - Custom actions and skill integration
  
- **Copilot Studio Configuration (20–25%)**
  - Topics, triggers, and conversation flow design
  - Channel integration (Teams, web, mobile, Office)
  - Power Fx and adaptive cards for UI
  - Foundry and custom data integration
  
- **Power Platform & Integration (15–20%)**
  - Power Automate workflows with agents
  - Microsoft Dataverse for agent data persistence
  - REST API integration and custom connectors
  - Microsoft 365 Copilot extensions
  
- **Deployment, Analytics & Operations (10–15%)**
  - Agent deployment to production channels
  - Analytics and monitoring via Copilot Studio insights
  - Rate limiting and performance tuning
  - Testing, troubleshooting, and iteration
