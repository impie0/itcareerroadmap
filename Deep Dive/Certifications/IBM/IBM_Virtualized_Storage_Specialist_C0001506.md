## About

The IBM Certified Specialist — Virtualized Storage v2 (C0001506) validates technical expertise in IBM's enterprise storage virtualization portfolio. This professional-level certification is intended for pre-sales technical specialists, solutions architects, and storage engineers who design and support IBM storage solutions in enterprise environments. The certification confirms knowledge of IBM Storwize family members, IBM FlashSystem V9000, IBM Spectrum Virtualize software, and related management tools across converged infrastructure and hybrid cloud deployments.

## Domain context — Storage Virtualization

Specialized expertise in enterprise storage abstraction, pooling, and management software that enables efficient resource utilization across heterogeneous storage systems. Focus: IBM SAN Volume Controller, IBM Spectrum Virtualize, and Storwize appliance family.

## Topics covered

- IBM Storwize family architecture and models (V5010, V5020, V5030, V5030F, V7000, V7000F)
- IBM FlashSystem V9000 design, deployment, and management
- Storage pool configuration, RAID selection, and tiering (Easy Tier functionality)
- IBM Spectrum Virtualize software and SAN Volume Controller capabilities
- Authentication and access control (LDAP, Active Directory, NTP protocols)
- Network protocols and connectivity (FCoE, iSCSI, Fibre Channel, TCP/IP)
- Block, file, and object data access functionality
- Replication and data consistency (crash consistency vs. application consistency)
- Storage software licensing and IBM Spectrum Control management
- Hybrid cloud integration and software-defined storage concepts
- IBM Spectrum Protect data protection integration
- Competitive positioning within IBM portfolio and against vendors
- Performance sizing tools and technical configuration design
- Maintenance, upgrades, and system life-cycle management
