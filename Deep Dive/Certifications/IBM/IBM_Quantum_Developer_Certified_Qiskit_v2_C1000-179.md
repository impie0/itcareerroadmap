## About

The IBM Certified Associate Developer — Quantum Computation using Qiskit v2.X validates hands-on competency in designing and executing quantum circuits, optimizing transpilation pipelines, applying quantum algorithms, and leveraging IBM's Qiskit Runtime for production quantum workloads. Launched March 31, 2025 alongside Qiskit SDK v2.0, this certification replaces the retired C1000-112 (Qiskit v0.2X). Candidates must demonstrate proficiency across the core Qiskit v2 architecture: QuantumCircuit construction, parameterized circuits, transpilation (PassManager, optimization levels 0–3), primitives-based execution (Sampler for measurement, Estimator for observables), noise modeling and error mitigation (zero-noise extrapolation, probabilistic error cancellation), and cloud execution via IBM Quantum Platform. The exam underwent significant structural changes to reflect Qiskit 1.0's (February 2024) API maturity and 2.0's breaking architectural improvements.

## Topics covered

Official exam blueprint (Qiskit v2.X focus):

- **Qiskit SDK architecture (v2.x)** (15–18%): QuantumCircuit API, Instruction set, Parameter binding, transpilation pipeline (PassManager, Target backend), transpiler optimization levels 0–3
- **Quantum circuit construction and manipulation** (20–25%): QuantumCircuit instantiation, gates and operations, controlled gates, custom gates, parameterized circuits, circuit composition and modular design
- **Transpilation and circuit optimization** (18–20%): PassManager workflow, layout and routing, gate optimization, target backend constraints, noise-aware transpilation
- **Qiskit Runtime and primitives** (20–25%): Sampler primitive (measurement statistics), Estimator primitive (observable expectation values), Batch and Session execution models, error mitigation configuration
- **Quantum algorithms and applications** (12–15%): Grover's algorithm, Quantum Approximate Optimization Algorithm (QAOA), Variational Quantum Eigensolver (VQE) basics, VQE + UCC for chemistry, phase estimation
- **Visualization and state inspection** (8–10%): Circuit diagrams, measurement histograms, Bloch sphere representation, statevector inspection
- **Noise modeling and error mitigation** (8–12%): Noise models (depolarizing, thermal, gate errors), zero-noise extrapolation (ZNE), probabilistic error cancellation (PEC), resilience levels in Runtime
- **IBM Quantum Platform and cloud execution** (5–8%): Cloud account setup, backend selection and queue management, Quantum Composer, parameter sweep jobs, monitoring and debugging

*Source:* [IBM Quantum Developer Certification (v2.X) Overview ↗](https://www.ibm.com/quantum/blog/qiskit-v2x-developer-certification)

## Skills validated

*Cert-specific — what this exam actually tests in Qiskit v2.X.*

- QuantumCircuit API (v2): instantiation, gate/instruction addition, parameter binding, circuit composition
- Parameterized circuits: Parameter objects, parameter expressions, binding workflows
- Transpilation pipeline: PassManager, custom pass design, optimization levels 0–3, target backend constraints
- Qiskit Runtime primitives: Sampler (shot-based measurement), Estimator (observable expectation values), Batch mode, Session mode
- Error mitigation: zero-noise extrapolation (ZNE), probabilistic error cancellation (PEC), resilience levels configuration
- Quantum algorithms: Grover's algorithm implementation, QAOA ansatz, VQE with parameterized circuits, basic phase estimation
- Visualization: circuit diagrams, measurement histograms, Bloch sphere plots, statevector inspection
- Noise models: depolarizing channels, thermal relaxation, gate error modeling
- IBM Quantum Platform: cloud account, backend selection, queue monitoring, Quantum Composer basics
- OpenQASM 3.0 basics: circuit syntax, pulse integration concepts
- Hybrid quantum-classical workflows: classical optimization loops, VQE/QAOA
