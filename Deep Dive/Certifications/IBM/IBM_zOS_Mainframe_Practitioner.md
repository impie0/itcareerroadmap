## About

The IBM z/OS Mainframe Practitioner Professional Certificate is a foundational credential for professionals entering mainframe computing. Delivered via Coursera and powered by IBM, this program provides hands-on experience on real IBM Z servers—not simulations. Learners gain entry-level skills in IBM Z hardware, z/OS system administration, and basic system programming. The certificate is designed for those targeting careers as mainframe application developers, system programmers, system administrators, or database practitioners. Released in 2019, it remains the primary entry point for mainframe careers globally and aligns with IBM's commitment to developing the next generation of mainframe professionals.

## Domain context — Mainframe / z/OS / Enterprise Computing

IBM Z mainframe systems power over 68% of worldwide transactions and process critical workloads in banking, insurance, government, and retail. z/OS is the primary operating system on IBM Z servers, and system administrators and programmers are in chronic shortage globally. This certificate is entry-level within the mainframe ecosystem.

## Topics covered

### Course 1: Introduction to Enterprise Computing
- IBM Z hardware architecture and families
- Operating system functions and concepts
- Sysplex concepts and clustered mainframes
- z/OS security methods and access control
- IOCDS and IOCP statements

### Course 2: Getting Started on Mainframe with z/OS Commands and Panels
- ISPF/PDF dialog interface and navigation
- TSO/E (Time Sharing Option/Extensions) commands
- JCL (Job Control Language) fundamentals
- JES (Job Entry Subsystem) job submission
- z/OSMF (z/OS Management Facility) interface
- ISHELL and HFS (Hierarchical File System) basics
- 7+ hands-on labs on real IBM Z hardware

### Course 3: Basic System Programming on IBM Z
- Advanced JCL and job processing
- VSAM (Virtual Storage Access Method) basics
- z/OS System Libraries and dataset management
- Generation Data Groups (GDGs)
- Language Environment and runtime
- RAIM (Redundant Array of Independent Mainframes)
- DB2 integration concepts
- UNIX System Services (USS) and file system
- z/OSMF advanced tasks

## Practice labs

The certificate includes 7+ hands-on labs executed on real IBM Z servers:

- z/OS environment navigation and system interaction
- ISPF/PDF dialog usage and TSO command execution
- JCL job creation and submission via JES
- Dataset and file system management
- System administration tasks in live z/OS environment
- z/OSMF interface operations

No separate practice exam vendors exist for this certificate; the integrated labs serve as the primary hands-on assessment.
