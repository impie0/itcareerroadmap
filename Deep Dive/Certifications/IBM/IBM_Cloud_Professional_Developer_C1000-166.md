## About

The IBM Certified Professional Developer — Cloud v6 validates the ability to design, build, deploy, and manage cloud-native applications on IBM Cloud. Candidates demonstrate expertise in containerized microservices, serverless functions, DevOps CI/CD, infrastructure-as-code, and integration of IBM Cloud data and AI services. This is the current version (released 2024); v5 (C1000-165) remains available but is not recommended for new candidates. The certification targets full-stack developers, cloud application engineers, and platform engineers building production workloads on IBM Cloud.

## Topics covered

Based on the official exam blueprint, C1000-166 assesses knowledge across core cloud-native development domains:

- **Cloud-Native Application Development (20%)** – 12-factor applications, microservices architecture, stateless service design, containerization best practices.
- **Container Deployment & Orchestration (18%)** – Docker and container registries (IBM Cloud Container Registry), Kubernetes concepts, IBM Kubernetes Service (IKS) deployment and scaling, pod networking and storage.
- **IBM Cloud Foundry & Container Platforms (15%)** – Cloud Foundry application deployment, buildpacks, application health checks, auto-scaling, IBM Red Hat OpenShift on IBM Cloud (RHOCP) fundamentals.
- **Serverless & Event-Driven Architecture (12%)** – IBM Cloud Code Engine (serverless container platform), Cloud Functions (IBM Cloud Functions), event sources, trigger configuration, cold start optimization.
- **CI/CD & DevOps Tools (15%)** – IBM Cloud Continuous Delivery service, Tekton pipelines, toolchains, Git integration, automated testing, deployment strategies (blue-green, canary).
- **Data & Integration Services (10%)** – IBM Cloud Object Storage SDK integration, Db2 on Cloud, Cloudant NoSQL, Event Streams (Kafka as a Service), API Connect integration.
- **Security, Monitoring & App ID (10%)** – IBM App ID (identity and access management for applications), encryption best practices, secret management, application logging, monitoring with Instana and LogDNA.

*Source:* [IBM Certified Professional Developer — Cloud v6 ↗](https://www.ibm.com/training/certification/ibm-certified-professional-developer-cloud-v6-C1000-166)
