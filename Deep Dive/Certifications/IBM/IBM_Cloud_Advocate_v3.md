## About

The IBM Certified Advocate - Cloud v3 is a functional, non-technical entry-level credential validating foundational cloud knowledge for business and emerging IT professionals. Building on the v2 foundation, v3 emphasizes IBM Cloud architecture patterns, Watsonx AI fundamentals, hybrid cloud and Red Hat OpenShift integration, security/compliance basics, and cost management. Released in 2024–2025, this certification bridges general cloud awareness and technical depth, serving as an entry point for career-changers, sales engineers, pre-sales consultants, business stakeholders, and cloud-curious IT professionals seeking IBM Cloud literacy.

## Topics covered

Based on the official IBM exam blueprint, v3 assesses knowledge across these domains:

- **IBM Cloud Fundamentals** (~20%) — IBM Cloud overview, service models (IaaS, PaaS, SaaS), global infrastructure, account and billing
- **IBM Cloud Services Overview** (~25%) — Compute (Virtual Servers, Kubernetes), Storage (Object, Block), Database (SQL/NoSQL), Networking, Integration, and AI/ML services portfolio
- **Watsonx Fundamentals** (~15%) — Watsonx.ai (foundation models, prompt engineering basics), Watsonx.data (data lakehouse), Watsonx.governance (responsible AI, compliance)
- **Hybrid Cloud & OpenShift** (~15%) — IBM Cloud Pak overview, Red Hat OpenShift on IBM Cloud, hybrid deployment patterns, multi-cloud strategy
- **Security, Compliance & Cost** (~15%) — Identity and Access Management (IAM), encryption, compliance frameworks (SOC 2, ISO 27001, HIPAA), cloud economics, cost estimation, RI/Savings Plans
- **IBM Cloud Adoption** (~10%) — Migration assessment, use cases by industry, organizational readiness, governance models

*Source:* IBM Training and Certification; official exam guidance
