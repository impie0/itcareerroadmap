## About

The IBM Certified watsonx Data Lakehouse Engineer v1 - Associate validates foundational knowledge and hands-on skills in designing, implementing, and managing data lakehouse architectures using IBM watsonx.data. Launched in 2024, this certification targets data engineers, data architects, and IT professionals transitioning into modern data platform roles. Candidates must demonstrate proficiency in data lakehouse principles, watsonx.data administration, data ingestion and integration, query optimization, and governance practices aligned with hybrid-cloud and AI workloads.

Enterprise data strategy, analytics infrastructure, machine learning platforms, and AI-driven decision making. Data lakehouse architecture unifies the governance of data warehouses with the flexibility of data lakes.

- Data Lakehouse Fundamentals (architecture, advantages over traditional data lakes and warehouses)
- watsonx.data Platform Architecture (components, deployment models, cloud integration)
- Data Integration & Ingestion (ingesting structured and unstructured data, connectors, data quality)
- Data Management & Governance (metadata management, data lineage, access control, retention policies)
- Query Optimization & Performance (SQL execution, indexing, query tuning, cost optimization)
- Analytics & Consumption (query engines, analytics tools integration, reporting)
- Security & Compliance (encryption, audit logging, compliance frameworks)
- Integration with watsonx.ai (model governance, collaborative AI workflows)

## Skills validated

*Exam-specific technical competencies — distinct from broader data/AI shared skills.*

- watsonx.data platform architecture and deployment (standalone, hybrid-cloud)
- Apache Iceberg and Delta Lake table format concepts and integration
- Apache Spark for distributed data processing and SQL on lakes
- Data ingestion connectors and CDC (Change Data Capture) patterns

- SQL query optimization and execution plan analysis
- Role-based access control (RBAC) and data governance policies
- Query engine selection and performance tuning (Spark, Presto, PrestoDB)
- Integration of watsonx.data with watsonx.ai for managed ML workflows
- Cost optimization and resource allocation in hybrid-cloud environments
