## About

The F1003500 credential is a combined professional-level certification for cloud architects and Site Reliability Engineers (SREs) working on IBM Cloud. It requires candidates to earn both the IBM Certified Professional Architect v6 and the IBM Certified Professional SRE - Cloud v2 credentials. An IBM Cloud Site Reliability Engineer operates services that sustain service level objectives and engineers scalable, secure, and highly reliable services by infusing automation, DevOps principles, proactive maintenance, and rigorous troubleshooting for workloads on IBM Cloud. This credential targets experienced cloud engineers and architects managing mission-critical services.

## Topics covered

The IBM Certified Professional SRE - Cloud v2 exam (C1000-119) covers nine weighted domains:

- Applying Site Reliability Engineering Principles (13%) — SLOs, error budgets, balancing risk and reliability
- Implementing and Managing Compute Resources (14%) — Virtual servers, bare metal, container orchestration
- Managing Container Platforms (15%) — Red Hat OpenShift on IBM Cloud, Kubernetes, pod scheduling, troubleshooting
- Managing Storage (12%) — Block, object, and file storage; provisioning and access policies
- Managing Networking (12%) — Network connectivity, security groups, network policies, load balancing
- Implementing Security and Compliance (14%) — IAM, encryption, secrets management, compliance automation
- Implementing Logging and Monitoring (13%) — Observability, metrics, alerting, incident response
- Implementing Automation and DevOps (13%) — CI/CD pipelines, infrastructure as code, GitOps
- Troubleshooting and Incident Management (8%) — Root cause analysis, incident response, service recovery

*Source:* [IBM Cloud Professional SRE Exam Syllabus ↗](https://www.edusum.com/ibm/ibm-cloud-professional-sre-exam-syllabus)
