## About

The IBM Certified Professional Architect — Cloud v6 validates the ability to design, plan, and architect cloud solutions on IBM Cloud. Candidates demonstrate expertise in hybrid cloud architecture, workload migration, security, networking, data management, and resiliency patterns. This is the current version (released 2023); v5 (C0001403) remains available but is not recommended for new candidates. The certification is a Professional-level credential targeting experienced cloud engineers and architects working with IBM Cloud infrastructure and services.

Hyperscale and enterprise cloud platforms (AWS, Microsoft Azure, Google Cloud, IBM Cloud, Oracle Cloud). Includes private cloud (Red Hat OpenStack) and specialty clouds (SAP, Financial Services, VMware).

## Topics covered

Based on the official exam blueprint, C9005800 assesses knowledge across six major domains:

- **Compute Options (19%)** – Virtual machines, containers, Kubernetes, serverless (Cloud Functions), bare metal. Sizing, selection, and deployment patterns.
- **Designing Cloud Solutions (16%)** – Solution architecture, multi-cloud strategy, workload assessment, migration planning, cost optimization.
- **Networking Options (15%)** – VPC, subnets, security groups, load balancers, DNS, VPN, Direct Link, edge networking.
- **Security (9%)** – Identity & Access Management (IAM), encryption at rest and in transit, compliance frameworks, network security, secrets management.
- **Data and Data Management (6%)** – Databases (relational, NoSQL), data warehouse, backup & recovery, data governance.
- **IBM Cloud Resiliency Features (8%)** – High availability, disaster recovery, failover, backup strategies, multi-region design.
- **Observability Capabilities (8%)** – Monitoring, logging, alerting, application performance management, troubleshooting.
