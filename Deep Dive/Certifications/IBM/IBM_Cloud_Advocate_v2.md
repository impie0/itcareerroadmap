## About

The IBM Certified Advocate - Cloud v2 is a functional, non-technical credential validating foundational cloud knowledge for business and emerging IT professionals. Released in 2023, this certification bridges the gap between general awareness and technical depth, emphasizing cloud economics, business value, and IBM Cloud positioning rather than hands-on implementation. No prerequisites exist; it serves as an entry point for career-changers, sales engineers, pre-sales consultants, and business stakeholders seeking IBM Cloud literacy.

## Topics covered

Based on official IBM exam scope, the C1000-142 assesses knowledge across these domains:

- **Cloud Concepts & Models** (~25%) — Public / private / hybrid deployment; IaaS / PaaS / SaaS service models; cloud economics and ROI
- **IBM Cloud Solutions** (~30%) — IBM Cloud core services (compute, storage, database, networking); service portfolio overview; integration and middleware
- **IBM Cloud Architecture** (~20%) — Reference architecture patterns; hybrid and multi-cloud design; scalability and resilience principles
- **Use Cases & Value Proposition** (~15%) — Industry-specific cloud adoption scenarios; business transformation with IBM Cloud
- **Adoption & Implementation** (~10%) — Migration strategies; governance; security and compliance frameworks; organizational readiness

*Source:* [IBM Training Certification ↗](https://www.ibm.com/training/certification/ibm-certified-advocate-cloud-v2-C9003700)
