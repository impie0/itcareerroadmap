## About

The Okta Certified Professional Administrator certification validates foundational expertise in Okta Identity Engine (OIE) and workforce identity management. Candidates demonstrate hands-on knowledge of user lifecycle management, directory integration, single sign-on (SSO) federation, authentication policies, multi-factor authentication (MFA), provisioning, and organizational administration. This Associate-level credential serves as the entry point into Okta's professional certification stack and is a prerequisite for pursuing the Administrator, Consultant, and Technical Architect certifications.

- Foundational Concepts (platform architecture, licensing, deployments)
- User Management (user creation, import, bulk operations, group management)
- Directory Integration (Active Directory, LDAP, cloud directories)
- Application Integration (SSO setup, SAML, OAuth, OIDC)
- Authentication Policies (sign-on rules, global policies, app-level policies)
- Multi-Factor Authentication (verification methods, MFA policies, factors)
- User Lifecycle Management (provisioning, deprovisioning, updates, lifecycle workflows)
- Organizational Administration (admin roles, API access, logging, reporting)
