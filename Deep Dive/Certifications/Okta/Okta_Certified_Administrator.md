## About

The Okta Certified Administrator certification validates expertise in managing and configuring Okta Identity Engine (OIE) for workforce identity management at scale. Candidates demonstrate advanced proficiency in user lifecycle management, directory integration, authentication policies, multi-factor authentication (MFA), application integration, organization administration, and troubleshooting. This Associate-level credential is typically pursued after the Professional Administrator certification and serves as a gateway to the Consultant and Technical Architect certifications. The exam combines multiple-choice questions with hands-on configuration exercises to assess practical operational competency.

## Topics covered

- **Identity Lifecycle Management** — User provisioning, deprovisioning, account recovery, lifecycle workflows, bulk operations
- **Authentication & Security** — Authentication policies, sign-on rules, password policies, risk-based authentication, adaptive MFA
- **Multi-Factor Authentication (MFA)** — Factor configuration, MFA policies, verification methods, policy-based enforcement
- **Directory & Integration** — Active Directory/LDAP synchronization, Universal Directory, directory integration troubleshooting
- **Application Integration** — SSO setup (SAML, OAuth 2.0, OIDC), application configuration, API access management
- **Authorization & Access Control** — Policy framework, attribute mapping, conditional access, application-level policies
- **Universal Directory** — User profiles, custom attributes, profile master configuration, data transformation
- **Administrative Functions** — Admin roles and permissions, API access management, logging and auditing, system administration
- **Troubleshooting & Support** — System logs, event monitoring, common configuration issues, user access problems
