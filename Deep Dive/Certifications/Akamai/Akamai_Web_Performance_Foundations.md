## About

The Akamai Web Performance Foundations certification validates foundational knowledge of Akamai's Intelligent Edge Platform and content delivery solutions. Earners complete either self-paced online modules or instructor-led training, then demonstrate understanding through course assessments covering Akamai CDN architecture, caching strategies, content provisioning (Property Manager), monitoring (mPulse RUM), and edge acceleration fundamentals. The credential is issued via Credly with lifetime validity and no renewal required. Released in 2020 in its current form, this associate-level cert is a gateway to Akamai's advanced web performance certifications and is prevalent among new hires at Akamai, Akamai resellers, DevOps teams, and internal support teams at enterprises using Akamai solutions.

---

## Domain context — CDN/Performance

Content Delivery Networks (CDNs) accelerate web and media delivery through globally distributed edge nodes, caching strategies, and performance optimization. Akamai is a primary CDN and edge compute vendor, serving approximately 30% of global web traffic. Web performance engineering bridges network design, application optimization, observability, and serverless edge computing.

## Topics covered

- **Edge Architecture & Ion Properties:** Understanding Akamai's global edge network, property creation, versioning, and activation workflow.
- **Caching Strategies:** Cache control headers, TTLs, cache key configuration, and invalidation patterns.
- **Request Routing & Global Traffic Management (GTM):** DNS-based load balancing, geographic routing, and traffic steering.
- **Image & Video Manager (IVM):** Responsive image delivery, format optimization, and adaptive video bitrate selection.
- **Adaptive Acceleration:** HTTP/2 server push, connection optimization, and performance prediction.
- **EdgeWorkers & EdgeKV:** Serverless JavaScript execution on the edge, KV storage for state, and request/response manipulation.
- **mPulse Real User Monitoring (RUM):** Collecting performance metrics, debugging slowdowns, and monitoring synthetic vs. real traffic.
- **Basic Troubleshooting:** Debugging cache hits/misses, header analysis, and traffic inspection via Akamai tools.

*Source:* [Akamai Web Performance Foundations course overview](https://www.akamai.com/content/dam/sdl-content/us/en/multimedia/documents/akamai/akamai-web-performance-foundations-course-overview.pdf)
