## About

The Akamai Web Security Specialist certification validates proficiency with Akamai's edge security platform covering web application firewalls (WAF), bot management, API protection, and account takeover prevention. Launched in 2020 as part of Akamai's Foundations track, it serves security professionals, architects, and DevOps engineers deploying Akamai Intelligent Edge Platform services. The cert is required or strongly preferred for many Akamai professional service roles and is recognized across organizations running Akamai in production.

## Topics covered

Based on Akamai University curriculum and published exam blueprint:

- **Kona Site Defender (WAF)** — rule sets, attack signatures, custom rules, bypass detection, log analysis
- **Bot Manager Premier** — bot classification, mitigation strategies, challenge types, credential stuffing defense
- **App and API Protector** — API endpoint security, rate limiting, threat patterns, GraphQL/REST protection
- **Account Takeover Prevention** — Account Protector, risk scoring, compromise detection, response actions
- **Audience Hijacking Protector** — account compromise signals, session anomalies, geographic indicators
- **Web & API Protection Bundles** — integration patterns, layered security, operational best practices
- **Akamai Cloud Wrapper** — origin offload, on-premise integration, hybrid deployment models
- **Incident response** — threat investigation, forensics, alert tuning, false-positive reduction
- **Configuration and tuning** — policy optimization, performance impact assessment, production deployment

*Source:* [Akamai University Course Curriculum ↗](https://learn.akamai.com/)
