## About

The New Relic Verified Foundation (NVF) is an entry-level certification for professionals new to observability or the New Relic platform. It validates foundational knowledge of modern observability concepts, the New Relic platform UI, telemetry data types, and agent configuration. No prerequisites are required. Launched in 2024 as part of New Relic's foundation-to-professional certification track, the NVF is not proctored and is available for free through New Relic University.

---

## Domain context — Observability & APM

Application Performance Monitoring (APM) and observability focus on monitoring, visualizing, and analyzing the health, performance, and behavior of software applications and their infrastructure in real time. Modern observability spans metrics, logs, traces, events, and user experience data across distributed cloud-native systems.

## Topics covered

Based on the official exam guide, the NVF exam assesses knowledge across these core areas:

- **Observability fundamentals** — Core concepts: monitoring vs. observability, telemetry data types (metrics, logs, traces, events).
- **New Relic platform overview** — UI navigation, account structure, dashboards, and workloads.
- **Data ingestion and agents** — Installing and configuring APM, Infrastructure, Browser, Mobile, and Synthetics agents.
- **NRQL (New Relic Query Language)** — Basic query syntax, data exploration, and analysis.
- **Alerts and notifications** — Setting up alert policies, conditions, and incident workflows.
- **Dashboards and visualizations** — Creating, customizing, and sharing dashboards.
- **Full-stack monitoring** — Understanding end-to-end observability across applications, infrastructure, logs, and user experience.
