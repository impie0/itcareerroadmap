## About

The Tenable Certified Security Engineer (TCSE) is a professional-level certification that validates hands-on expertise in Tenable vulnerability management platforms. The exam assesses the ability to plan, deploy, configure, and troubleshoot Tenable solutions in enterprise environments. The two-part structure (written exam + practical exam) ensures candidates demonstrate both theoretical knowledge and applied troubleshooting skills. The certification is active and part of Tenable's core professional-tier offerings, complementing vendor-specific specialist tracks for Tenable.io and Tenable.sc.

---

## Domain context — Vulnerability Management

Vulnerability management encompasses asset discovery, vulnerability scanning, risk assessment, remediation tracking, and integration with security infrastructure (SIEM, GRC, ticketing systems). Tenable is a primary vendor platform in this domain, alongside competitors like Qualys, Rapid7, and OpenVAS. The TCSE validates depth in Tenable-specific implementations.

## Topics covered

The TCSE exam assesses the following competency areas (per Tenable's certification program):

- **Tenable platform architecture and deployment** — Tenable.io cloud architecture, Tenable.sc on-premises deployment, licensing models
- **Asset discovery and management** — Asset identification, credential management, agent deployment, sensor configuration
- **Vulnerability scanning** — Nessus scan configuration, scan templates, policy design, plugin management, scan result interpretation
- **Scan result analysis and troubleshooting** — Plugin behavior interpretation, false positive identification, advanced scan data analysis, performance optimization
- **Risk scoring and vulnerability prioritization** — CVSS/CVSS3 scoring, risk-based remediation, SLA management, business context integration
- **Authentication and access control** — Role-based access control (RBAC), user and group management, API authentication
- **Integration with enterprise systems** — SIEM integration, GRC tool integration, ticketing system integration, API usage
- **Cloud security and compliance** — Tenable.io cloud-native scanning, container and serverless assessment (post-Ermetic), cloud compliance frameworks
- **Operational Technology (OT) security** — OT-specific scanning (post-Indegy acquisition), air-gapped environment assessment
