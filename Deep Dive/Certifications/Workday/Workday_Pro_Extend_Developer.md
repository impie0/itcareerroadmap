## About

The Workday Pro Extend Developer certification validates hands-on expertise in developing and managing custom applications using Workday Extend, Workday's low-code platform for building extensions to Workday HCM and Finance modules. This professional-level credential covers application architecture, custom object modeling, REST API integration, authentication, and deployment practices. The certification was reintroduced in 2024 as part of Workday's broader Pro certification program refresh, targeting developers, implementation partners, and customer development teams. Valid for two years with recertification required every two years via a closed-book proctored exam.

## Topics covered

- **Workday Extend Platform Overview** — Purpose of Workday Extend; navigating the developer site and locating resources; understanding when to extend vs. configure
- **Application Design & Use Cases** — Defining actors and their actions; crafting user narratives; identifying functional areas and integration touchpoints; designing for mobile and responsive UI
- **Custom Objects** — Effective-dated, multi-instance, and hierarchical custom object patterns; data model selection and field validation
- **API Integration** — Workday REST API authentication; OAuth and passwordless approaches; API Explorer; third-party REST vs. SOAP API considerations
- **Orchestration & Business Processes** — Orchestration Builder; PMD (Process Modeling) scripting; integrating custom logic into business workflows
- **User Interface Design** — Page structure and flow; Workday presentation layer design guidelines; custom pages and layouts
- **Authentication & Security** — Security domains; role-based access; custom task and permissions configuration; securing app-to-Workday communication
- **Custom Tasks & Related Actions** — Creating custom tasks; related actions; dashboards and announcements; search implementation
- **Testing & Deployment** — Development lifecycle; sandbox testing; promotion to production; rollback strategies and best practices
- **Workday Analytics & Reporting** — Reporting on custom objects; integration with Workday analytics; data visibility and audit trails

*Source:* [Workday Pro training community ↗](https://community-content.workday.com/en-us/public/learn/training/workday-pro.html); [Tech Pratham Workday Extend explainer ↗](https://www.techpratham.com/blog/workday-extend-explained-core-modules-architecture-use-cases-and-real-world-examples)
