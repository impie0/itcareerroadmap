## About

The CWAP is CWNP's Professional-level certification for wireless protocol analysts, packet inspectors, and advanced troubleshooters. Released in 2020 as CWAP-404, it validates hands-on mastery of 802.11 frame formats, authentication/association sequences, spectrum measurement tools (Wireshark, OmniPeek), and real-time PHY-layer analysis. Holders conduct deep packet inspection for network performance diagnostics, security incident response, and RF optimization. It is held by wireless engineers, network architects, support specialists, and analysts responsible for troubleshooting complex WLAN performance issues at enterprises and service providers. The exam is mandatory CWNA-108+ and serves as a steppingstone to CWNE alongside CWSP and CWDP certifications.

## Domain context — Networking / Wireless

Wireless networking is a specialized segment of enterprise networking focused on radio-frequency (RF) transmission, 802.11 standards compliance, WLAN infrastructure design, and integration with wired networks. It encompasses radio propagation, antenna theory, regulatory frameworks, security protocols, and troubleshooting methodologies. Protocol analysis — capturing and dissecting wireless frames at Layer 2 and PHY — is a core discipline within wireless engineering.

## Topics covered

The CWAP-404 exam blueprint covers wireless protocol analysis and frame-level troubleshooting with the following major knowledge domains:

- **802.11 Frame Formats and Structure** — MAC header (frame control, duration, addresses, sequence control), frame body (SSID, rates, parameters), FCS validation, frame types (management, control, data); understanding frame fields for diagnosis.
- **Management Frames: Authentication and Association** — Frame-by-frame analysis of Open System / Shared Key authentication, association request/response, reassociation, deauthentication, disassociation, and 802.11r (Fast Roaming) sequences; identifying authentication failures and delays.
- **Data Frame Exchange and Access Methods** — CSMA/CA operation, contention windows, inter-frame spacing (SIFS, DIFS, AIFS), RTS/CTS handshakes, 802.11ac/ax frame exchanges, aggregated MAC protocol data units (A-MPDU), and block acknowledgment; analyzing frame timing and contention.
- **Physical Layer (PHY) Analysis** — RSSI (Receive Signal Strength Indicator), SNR (Signal-to-Noise Ratio), noise floor measurement, data rates, channel bandwidth, spatial streams (MIMO), guard intervals, pilot tones, and interpretation of spectrum analyzer output; detecting PHY-layer errors.
- **802.11ac and 802.11ax (Wi-Fi 5/6) Frame Analysis** — Multi-user (MU) frame exchanges, OFDMA resource allocation, spatial multiplexing (MU-MIMO), beamforming frame sequences, trigger frames, and high-efficiency signaling; analyzing advanced multi-user scenarios.
- **Protocol Analysis Tools and Techniques** — Hands-on use of Wireshark, OmniPeek (packet capture and display), trace file generation, wireless NIC mode (monitor/promiscuous), driver compatibility, filtering, and export; building expertise with industry-standard tools.
- **Roaming and Mobility** — 802.11k (Neighbor Reports), 802.11v (BSS Transition Management), 802.11r (Fast Roaming), handoff triggers, pre-authentication, and transition disable; analyzing seamless roaming behavior.
- **Quality of Service (QoS) and VoWiFi Analysis** — EDCA (Enhanced Distributed Channel Access) queue analysis, WMM parameters, traffic classification, voice codec detection, jitter and latency measurement in packet captures; troubleshooting VoWiFi call quality.
- **Spectrum and Interference Analysis** — Channel utilization, co-channel interference (CCI), adjacent-channel interference (ACI), non-802.11 interference detection, duty cycle measurement, and spectrum analyzer integration; identifying interference sources.
- **Hidden Node and Performance Issues** — Capture-based detection of hidden nodes, near-far conditions, collision probability, throughput degradation analysis, and troubleshooting methodologies; using frames to diagnose medium access problems.
- **Encryption and Security Frame Analysis** — WPA2 (CCMP) and WPA3 (GCMP) frame examination, key exchange frame sequences, deauthentication attacks, and rogue AP detection; security incident analysis using packet data.
- **Troubleshooting Methodologies** — Frame-based root cause analysis, hypothesis testing using captures, establishing baseline behavior, identifying anomalies, and documenting findings; systematic packet-level problem-solving.
