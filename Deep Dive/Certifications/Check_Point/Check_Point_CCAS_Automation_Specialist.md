## About

The CCAS certification validates expertise in automating security operations and orchestrating Check Point management workflows via REST APIs, Gaia API, Ansible, and scripting tools. It targets security engineers, DevSecOps practitioners, and Check Point resellers who need to reduce manual overhead, standardize deployments, and integrate Check Point with third-party orchestration platforms. The exam covers API architecture, object modeling, JSON payload construction, troubleshooting via CLI, and real-world automation scenarios. Released in 2020 for R80.10 and actively maintained for R81.20, this Specialist-level credential bridges the gap between foundational CCSA/CCSE knowledge and advanced automation engineering.

## Topics covered

- Introduction to automation and orchestration (Check Point framework, use cases)
- Check Point Management API architecture and design patterns
- JSON, REST principles, HTTP methods (GET, POST, PUT, DELETE)
- Object creation and modification via API (policies, rules, objects, access control lists)
- Authentication and session management (login tokens, logout)
- API error handling, logging, and debugging
- Postman client: crafting and testing API requests
- jq and JSON parsing for payload inspection and filtering
- Gaia API fundamentals for device configuration and monitoring
- SmartConsole scripting and batch operations
- Ansible integration and playbook patterns for Check Point
- Policy deployment and task scheduling
- Real-world automation workflows (bulk user import, rule management, backup automation)
