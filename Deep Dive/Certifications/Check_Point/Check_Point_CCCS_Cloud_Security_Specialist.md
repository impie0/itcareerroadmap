## About

The Check Point Certified Cloud Security Specialist (CCCS) validates expertise in securing cloud workloads and infrastructure across AWS, Azure, and Google Cloud Platform using Check Point's CloudGuard platform. Designed for security professionals advancing from traditional network security to cloud-native environments, the CCCS covers cloud access control, compliance automation, threat detection, and application security in cloud deployments. Prerequisites recommend CCSE R82 foundational knowledge. The certification is positioned as a Professional-level credential suitable for cloud security engineers and architects transitioning from on-premises Check Point deployments.

---

## Domain context — Security

Cloud security for IaaS and PaaS workloads across major public cloud providers (AWS, Azure, GCP). Part of the broader Security domain alongside vendor-specific cloud certifications (AWS Security Specialty, Azure Security Engineer, GCP Cloud Security Professional). Check Point CloudGuard specializes in unified cloud-native security, posture management, threat prevention, and compliance automation.

## Topics covered

- CloudGuard Network Security: firewall and access control in AWS, Azure, GCP
- CloudGuard Posture Management (CSPM): compliance, misconfiguration detection, cloud security posture assessment
- CloudGuard Intelligence: threat detection, cloud workload monitoring, incident response
- CloudGuard Application Security: Web Application Firewall (WAF), API protection, runtime security
- Cloud workload protection: container and serverless function security
- Zero Trust access control and cloud identity integration
- SmartConsole multi-cloud management and unified policy

- Cloud compliance frameworks (AWS, Azure, GCP, PCI-DSS, HIPAA, SOC 2)
- Cloud infrastructure hardening (VM, container, Kubernetes)

- Cloud cost optimization and security orchestration
