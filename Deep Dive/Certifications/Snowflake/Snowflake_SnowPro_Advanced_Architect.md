## About

The SnowPro Advanced Architect (ARA-C01) is Snowflake's expert-level certification for data architects and solution architects. It assesses the ability to design end-to-end Snowflake architectures that meet complex business requirements at scale—including account strategy, security/governance, data loading/unloading, performance optimization, and disaster recovery. This cert requires a current SnowPro Core (COF-C02 or later) and is designed for candidates with 2+ years of hands-on production Snowflake experience. It is the flagship credential in Snowflake's advanced track and serves as the architecture baseline before pursuing specialized advanced certs (Data Engineer, Data Analyst, or other tracks).

## Topics covered

The exam blueprint covers five major domains (approximate weights inferred from candidate feedback and course syllabi):

- **Account & Security Architecture (~25%)** — Design account strategy, implement RBAC hierarchies, configure database roles, enable secure data sharing, audit and compliance frameworks
- **Snowflake Architecture (~20%)** — Object hierarchy (databases, schemas, tables, views, stages, streams, tasks), data recovery (time travel, cloning, fail-safe), table types and clustering
- **Data Loading & Unloading (~25%)** — Evaluate loading strategies (Snowpipe, CDC, bulk copy, Iceberg), external data sources (S3, GCS, Azure Blob), streaming ingest, unload patterns
- **Performance Optimization (~15%)** — Warehouse sizing, query execution tuning, clustering keys, caching strategies, materialized views, result caching
- **Data Lake & Modern Architecture (~15%)** — Iceberg format integration, metadata management, Medallion/Delta Lake patterns, schema-on-read design, open-table formats
