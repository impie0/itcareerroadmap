## About

The Snowflake SnowPro Core (COF-C02) validates foundational expertise in designing, deploying, and managing the Snowflake Data Cloud platform. This associate-level certification tests practical knowledge of Snowflake's architecture, security, data loading, transformations, and performance optimization. It is the entry certification in the SnowPro track and serves as a prerequisite or foundation for advanced SnowPro certifications (Architect, Data Engineer, Administrator, Data Analyst, Data Scientist). COF-C02 replaced COF-C01 and reflects the current state of Snowflake's platform and features.

## Topics covered

Snowflake SnowPro Core exam blueprint (approximate domain weights):

- **Snowflake Data Cloud Features and Architecture** (~25%)
  - Platform architecture (multi-cluster, multi-cloud, micro-partitions)
  - Snowflake editions and licensing model
  - Storage and compute separation
  - Database, schema, and object hierarchy

- **Account Access and Security** (~20%)
  - Authentication methods (username/password, SSO, MFA, OAuth)
  - Role-based access control (RBAC)
  - Network policies and IP allowlisting
  - Encryption (at rest, in transit, end-to-end)
  - Data sharing and secure shares

- **Performance Concepts** (~15%)
  - Query optimization and performance tuning
  - Warehouse sizing and scaling (auto-scaling, clustering)
  - Table design for performance (clustering keys, micro-partitions)
  - Query result caching
  - Cost optimization strategies

- **Data Loading and Unloading** (~10%)
  - COPY INTO command and data loading
  - Snowpipe (continuous data ingestion)
  - Data unloading (UNLOAD, external stages)
  - File formats and compression
  - Stage objects (internal, external)

- **Data Transformations** (~20%)
  - SQL and query optimization
  - Tasks and scheduling
  - Stored procedures and user-defined functions
  - Streams and dynamic tables (CDC concepts)
  - Semi-structured data (JSON, Parquet, Avro)

- **Data Protection and Data Sharing** (~10%)
  - Time Travel and Fail-safe
  - Cloning and Zero-Copy Clone
  - Iceberg tables and Apache Iceberg integration
  - Direct Share and Listings
  - Secure data sharing with external organizations
