# CompTIA Linux Network Professional (CLNP)

**`CLNP`** · ● **Active** · **Professional** · _CompTIA_

> **Stackable Certification:** The CLNP is earned by simultaneously holding both CompTIA Network+ and CompTIA Linux+. Neither exam is standalone; both certifications must remain current. The stackable badge is issued automatically upon completion.

## About

The CompTIA Linux Network Professional (CLNP) is a stackable credential combining CompTIA Network+ and Linux+ certifications. It is designed for IT professionals who administer, monitor, and troubleshoot critical Linux systems operating across enterprise networks and cloud platforms. Unlike single-exam credentials, CLNP requires simultaneous active status of both Network+ and Linux+; if either expires, the stackable title becomes inactive. The certification validates hands-on expertise in network architecture, Linux system administration, kernel internals, security hardening, and network troubleshooting—critical for roles spanning infrastructure, DevOps, cloud operations, and systems engineering. Launched formally as a stackable around 2023, CLNP reflects the industry reality that Linux powers ~98% of servers and cloud workloads, and modern infrastructure requires professionals fluent in both operating system and networking fundamentals.

## Topics covered

### Network+ (N10-009)

- Network Architecture and Connectivity (25%)
  - OSI and TCP/IP models, network topologies, Ethernet, IP addressing, DHCP, DNS, NAT
- Network Operations (23%)
  - Device configuration, routing protocols (BGP, OSPF, RIP), switching, VLANs, access control
- Network Security (23%)
  - Firewalling, authentication, encryption, VPN, intrusion detection, AAA
- Troubleshooting (18%)
  - Diagnostic tools (ping, traceroute, netstat, nslookup), packet analysis, log review
- Network Management (11%)
  - Monitoring, SNMP, syslog, baselines, performance metrics

### Linux+ (XK0-006)

- System Management (32%)
  - Package management, user/group administration, permissions, disk partitioning, kernel modules
- Troubleshooting (28%)
  - Boot diagnostics, log file analysis, service recovery, performance tuning
- Security (21%)
  - SELinux/AppArmor, firewall configuration, SSH hardening, encryption, vulnerability scanning
- Scripting, Containers, and Automation (19%)
  - Bash scripting, systemd, cron, container basics (Docker), infrastructure-as-code concepts

## Skills validated

*Specific technologies and practices this stackable certification tests.*

- TCP/IP networking, routing (BGP, OSPF), switching, VLANs, firewalling
- Linux kernel, package management (dnf, apt), user/group administration, permissions (Unix/Linux model)
- SELinux and AppArmor mandatory access controls
- SSH and cryptographic key management
- systemd service management, cron, boot diagnostics
- Bash scripting, basic automation, container fundamentals (Docker)
- Network troubleshooting tools (ping, traceroute, netstat, tcpdump, nslookup)
- Log file analysis and performance tuning
- DHCP, DNS, NAT configuration and troubleshooting
- VPN and intrusion detection concepts
