## About

CompTIA Data+ (DA0-001) is a vendor-neutral, entry-level certification validating foundational data analysis, visualization, and governance skills. Launched February 2022, it bridges the gap between business users and technical data roles, certifying that candidates can transform business requirements into actionable reports, mine and analyze datasets, build dashboards, and apply data quality controls across the full data lifecycle. Equivalent to 18–24 months hands-on experience in a junior data analyst or business intelligence analyst role. **Note:** This cert is being superseded by DA0-002 (launched October 2025); DA0-001 English retires April 14, 2026.

## Topics covered

The exam covers five domains with the following distribution:

- **Domain 1.0: Data Concepts and Environments** (15% weight) — Data types, structures, lifecycle, storage systems (databases, warehouses, lakes)
- **Domain 2.0: Data Mining** (25% weight) — ETL/ELT, data extraction, transformation, quality assurance, data pipeline fundamentals
- **Domain 3.0: Data Analysis** (23% weight) — Statistical methods, exploratory data analysis, hypothesis testing, basic SQL, descriptive and inferential stats
- **Domain 4.0: Visualization** (23% weight) — Dashboard design (Tableau, Power BI awareness), effective data storytelling, chart selection, user experience
- **Domain 5.0: Data Governance, Quality, and Controls** (14% weight) — Data quality frameworks, metadata, compliance, security, organizational data standards

*Source:* [Official CompTIA Data+ Exam Objectives ↗](https://comptiacdn.azureedge.net/webcontent/docs/default-source/exam-objectives/comptia-data-da0-001-exam-objectives-(2-0).pdf)

## Common skills at Data / AI · Entry

*Shared skills for entry-level roles in the Data/AI/ML domain — not specific to this cert.*

- SQL fundamentals (SELECT, JOIN, WHERE, GROUP BY, basic indexing)
- Data types and structures (relational, semi-structured JSON/XML, unstructured)
- ETL/ELT pipeline concepts and tools (hands-on awareness, not mastery)
- Statistical thinking (mean, median, std dev, correlation, p-values, confidence intervals)
- Data visualization best practices (choosing chart types, avoiding misleading visuals)
- Data quality and data governance fundamentals (accuracy, completeness, timeliness, lineage)
- Excel and spreadsheet proficiency for pivot tables, VLOOKUP, basic formulas
- Business domain awareness (understanding business KPIs, translating requirements to data queries)

## Skills validated

*Cert-specific skills this exam actually tests, distinct from generic "common skills."*

- Data lifecycle management and requirements translation (business to technical)
- SQL query writing and optimization for exploratory data analysis
- ETL/ELT pipeline design and data transformation validation
- Descriptive and basic inferential statistics (hypothesis testing, confidence intervals)
- Dashboard and visualization design (with awareness of Tableau, Power BI, Looker)
- Data quality assessment and remediation techniques
- Metadata and data lineage documentation
- Data governance policies, compliance frameworks (GDPR, HIPAA awareness)
- Structured and unstructured data format handling (JSON, CSV, databases, APIs)
- Performance-based scenario analysis (given real data, produce analysis output)

## Summary

**CompTIA Data+ (DA0-001)** is a vendor-neutral, entry-level certification ideal for career starters or career changers targeting junior data analyst roles. It validates foundational SQL, statistics, visualization, and data governance knowledge—the core skills needed to turn business questions into data-driven insights. At $269 (often discounted), 90 minutes, and 675/900 passing score, it is accessible and widely recognized.

**Key strengths:** No prerequisites, broad applicability (finance, healthcare, retail, tech), bridges business and technical worlds, integrates with career progression toward data engineering or analytics engineering.

**Key caveat:** DA0-001 English version retires **April 14, 2026**. CompTIA launched DA0-002 (V2) in October 2025; plan accordingly if certifying after mid-April. Both cert codes will remain valid for 3 years post-issue and for career use indefinitely.

**Typical timeline:** 6–12 weeks study with prior SQL exposure; 12–16 weeks from zero technical background.

**Salary impact:** Entry-level junior data analysts earn $65,000–$85,000 USD (2026); cert holders at $62,000–$75,000+ depending on geography and employer.
