## About

CompTIA IT Fundamentals+ (ITF+) is the entry point to IT careers for those without prior technical background. Launched in 2014 and maintained as an active credential, FC0-U61 tests foundational concepts only—hardware basics, operating system fundamentals, networking 101, security awareness, and troubleshooting methodology—at a level well below CompTIA A+. Widely used by employers as a filter for career changers and those seeking IT apprenticeships. Lifetime validity (no renewal) makes it a low-risk stepping stone. In 2026, CompTIA rebranded ITF+ to "Tech+" (FC0-U71), but FC0-U61 remains active and fully credible.

## Topics covered

Based on the official CompTIA ITF+ exam blueprint:

- **Computing Device Basics** (understanding hardware, form factors, peripherals, input/output devices)
- **Troubleshooting Methodology** (systematic problem-solving, asking diagnostic questions, common failure modes)
- **Memory and Storage** (bits, bytes, KB→MB→GB→TB conversions; RAM vs. storage; data persistence)
- **Network Concepts** (TCP/IP fundamentals, DNS, DHCP, IP addressing basics, network topology)
- **Security Risks and Prevention** (authentication, encryption, common threats, phishing, malware types, best practices)
- **Preventive Maintenance** (BIOS/UEFI, drivers, system updates, backup strategy, cooling and power)
- **Database Fundamentals** (concept of data organization, relational tables, keys, queries)
- **Basic Programming Concepts** (algorithms, variables, loops, conditional logic, software development life cycle)

*Source:* [Official exam guide ↗](https://www.comptia.jp/pdf/CompTIA%20IT%20Fundamentals%20FC0-U61%20Exam%20Objectives.pdf)

## Skills validated

*Cert-specific — what this exam tests, distinct from shared Foundation skills above.*

- Bit/byte/storage unit conversions and memory hierarchy
- Troubleshooting methodology and diagnostic flowcharts
- TCP/IP and basic network addressing (DHCP, DNS, IP classes)
- BIOS/UEFI, drivers, and system boot sequence
- Encryption and authentication concepts (symmetric vs. asymmetric)
- Mobile operating systems (iOS/Android) and device management
- Database structure (tables, keys, relationships) and SQL basics
- Programming fundamentals (loops, conditionals, variables, functions)
- Malware types and common attack vectors
- Performance optimization and preventive maintenance
