## About

CompTIA Cloud+ is a vendor-neutral Associate-level certification validating multi-cloud operations, deployment, security, and DevOps fundamentals. Launched September 24, 2024, CV0-004 is the current version; CV0-003 retires in early 2025. The exam targets systems administrators and early-career cloud engineers who manage infrastructure across AWS, Azure, GCP, and on-premises environments without depth in any single vendor ecosystem. Cloud+ is positioned as the operational companion to CompTIA's architecture and security offerings (Security+, CySA+, CASP+, Architecture+).

---

## Domain context — Cloud

Multi-cloud computing now dominates enterprise infrastructure: 94% of enterprises operate across two or more cloud providers to optimize cost, compliance, resilience, and geographic distribution. Vendor-neutral cloud skills—deployment patterns, shared responsibility models, Infrastructure-as-Code (IaC), container orchestration, and observability—are essential before specializing in AWS, Azure, or GCP deep tracks.

## Topics covered

The CV0-004 exam blueprint spans **six domains** with official CompTIA weightings:

- **Cloud Architecture (23%)** — Cloud models (public, private, hybrid, multi-cloud), virtualization, cloud-native design, deployment models
- **Deployment (19%)** — Infrastructure-as-Code (IaC), containerization, orchestration, automation, CI/CD, configuration management
- **Security (19%)** — Shared responsibility model, identity & access management (IAM), encryption, data protection, compliance frameworks (SOC 2, PCI DSS), data sovereignty
- **Operations & Support (17%)** — Monitoring, logging, alerting, performance tuning, cost optimization, resource management, disaster recovery
- **Troubleshooting (12%)** — Diagnostic tools, common failure modes, network/compute/storage troubleshooting methodologies
- **DevOps fundamentals (10%)** — Version control, CI/CD pipelines, infrastructure automation, observability, basic Kubernetes concepts

*Source:* [CompTIA Cloud+ Exam Objectives](https://www.comptia.org/en-us/certifications/cloud/)

## Skills validated

*Specific to CV0-004 exam; distinct from the shared "Common skills" above.*

- **Cloud deployment models** — Public, private, hybrid, multi-cloud selection and trade-offs
- **Virtualization & hypervisors** — VM concepts, hypervisor types, container vs. VM differences
- **Infrastructure-as-Code (IaC)** — Declarative vs. imperative, Terraform, CloudFormation, ARM templates, version control
- **Container management** — Docker fundamentals, container images, registries, Kubernetes orchestration basics
- **Cloud networking** — Virtual networks, subnets, routing, NAT, firewalls, peering, hybrid connectivity
- **Cloud storage** — Object storage, block storage, file storage, backup, replication, disaster recovery
- **Cloud compute** — Instances/VMs, scaling, load balancing, auto-scaling policies, high availability
- **Identity & Access Management (IAM)** — Role-based access control (RBAC), policies, federation, multi-factor authentication
- **Encryption & data protection** — At-rest encryption, in-transit encryption, key management, data classification
- **Compliance & data sovereignty** — SOC 2, PCI DSS, GDPR, data residency, audit logging
- **Cost optimization & FinOps** — Cost monitoring, reserved instances, spot pricing, right-sizing
- **Monitoring & observability** — Metrics, logging, tracing, alerting, performance baselines
- **Disaster recovery & business continuity** — RTO/RPO, backup strategies, failover, multi-region resilience
- **CI/CD pipelines** — Source control, build automation, deployment automation, testing integration
- **Troubleshooting methodologies** — Diagnostic tools, log analysis, performance tuning, vendor CLI/APIs
