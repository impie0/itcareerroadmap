# CompTIA A+ (220-1201 / 220-1202)

**Vendor:** CompTIA  
**Status:** Active (current series launched March 2025; valid through ~2028)  
**Cost:** USD $265 per exam (×2 exams = $530) — [CompTIA official pricing](https://www.comptia.org/en-us/blog/how-much-does-the-comptia-a-certification-cost/)  
**Renewal:** 3 years (CE program — 20 CEUs or earn higher CompTIA cert)  
**Validity:** Globally recognized, vendor-neutral foundation cert  
**Prerequisites:** None (CompTIA recommends 9–12 months hands-on experience)  
**Vendor URL:** [https://www.comptia.org/certifications/a](https://www.comptia.org/certifications/a)

---

## Exam Details

- **Format:** 90 questions max per exam (multiple-choice + performance-based tasks)
- **Duration:** 90 minutes each
- **Passing scores:** 
  - Core 1 (220-1201) = 675/900
  - Core 2 (220-1202) = 700/900
- **Languages:** English (other languages rolling out)
- **Delivery:** Pearson VUE testing center or OnVUE remote proctoring
- **Release date:** March 25, 2025
- **Estimated retirement:** 2028 (3.5-year window)

---

## Exam Objectives & Domains

### Core 1 (220-1201) — 5 Domains

1. **Mobile Devices** (13%)  
   Smartphones, tablets, BYOD, device synchronization, security issues specific to mobile.

2. **Networking** (23%)  
   IP addressing, TCP/IP, Wi-Fi, Ethernet, DNS, DHCP, VPNs, network troubleshooting.

3. **Hardware** (25%)  
   CPUs, RAM, motherboards, power supplies, storage drives, peripherals, BIOS/UEFI, cooling.

4. **Virtualization & Cloud Computing** (11%)  
   VM concepts, cloud models (IaaS/PaaS/SaaS), cloud provider comparison, resource allocation.

5. **Hardware & Network Troubleshooting** (28%)  
   Diagnostic processes, common failure modes, log analysis, hardware replacement workflows.

**Reference:** [CompTIA A+ Core 1 (220-1201) exam objectives](https://www.examcompass.com/comptia-certifications/a-plus/core-1/comptia-a-plus-220-1201-exam-objectives.pdf)

### Core 2 (220-1202) — 4 Domains

1. **Operating Systems** (31%)  
   Windows 10/11, macOS, Linux basics; file systems, user management, updates, boot sequences.

2. **Security** (28%)  
   Authentication, encryption, malware, access controls, firewalls, secure password practices.

3. **Software Troubleshooting** (23%)  
   Malware removal, application crashes, performance issues, OS-level diagnostic tools.

4. **Operational Procedures** (21%)  
   Backup/restore, documentation, asset management, change control, customer communication.

**Reference:** [CompTIA A+ Core 2 (220-1202) exam objectives](https://www.examcompass.com/comptia-certifications/a-plus/core-2/comptia-a-plus-220-1202-exam-objectives.pdf)

## Career Roles & Job Titles

### Entry-level roles typically held by A+ cert holders:

- **IT Support Specialist / Help Desk Technician** — Tier 1 support for end-user hardware/software issues
- **Desktop Support Technician** — On-site device support for corporate users
- **Field Service Technician** — Mobile hardware repair/replacement in customer locations
- **IT Service Desk Analyst** — Ticketed support via phone/email; may include some remote access
- **Retail Tech / Apple/Microsoft Store Genius** — In-store support and device setup
- **MSP Tier 1 Engineer** — Remote support for managed service provider clients
- **Junior System Administrator** — Entry pivot after A+ + Network+ (admin-track progression)

**Market demand:** Help desk and IT support roles are typically the largest job segment in IT; A+ is the dominant credential for breaking into these roles in North America and UK markets.

## Related Certifications

### Preceding (foundational alternatives to A+)

- **CompTIA IT Fundamentals+ (ITF+)** — Pre-A+; narrower scope; good for absolute beginners
- **Google IT Support Professional Certificate** — [Coursera](https://www.coursera.org/professional-certificates/google-it-support); cheaper entry point; gains traction in North America
- **Cisco Certified Support Technician (CCST) IT Support** — Single exam; vendor-specific focus; fewer job postings than A+ but growing

### Successor (stacking path)

CompTIA certifications follow a **stackable progression** for the **Secure Infrastructure Specialist (CSIS)** stack:

1. **CompTIA A+** (foundation; hardware/OS/basic networking)
2. **CompTIA Network+** (N10-009; networks, routing, wireless, VPN)
3. **CompTIA Security+** (SY0-701; information security, risk, compliance)

**Source:** [CompTIA Stackable Certifications Guide](https://www.comptia.org/en-us/resources/stackables/)

### Parallel alternatives for entry-level roles

- **Microsoft Azure Fundamentals (AZ-900)** — Cloud focus; lower bar; ~$99 exam fee
- **CompTIA Cloud Essentials+** — Cloud-specific; lighter than Security+
- **ISC² CC (Certified in Cybersecurity)** — Security pivot (requires no prior cert; cheaper than Security+)

## Predecessor & Retirement

### Previous version: 220-1101 / 220-1102

The CompTIA A+ 220-1101 (Core 1) and 220-1102 (Core 2) exams were the prior series. They **retired on September 25, 2025** (English version; non-English exams retired December 19, 2025).

**Equivalence note:** A candidate who earned A+ via the old exams holds the exact same credential as one certified via 220-1201/1202; version does not appear on the cert.

**Source:** [Vision Training Systems retirement notice](https://www.visiontrainingsystems.com/blogs/when-does-comptia-a-1101-retire-and-comptia-a-1102-expire/)

## Skills Validated

### Technical competencies covered by the A+ exams:

- **Hardware diagnosis & replacement** — Identifying failed components, safe removal/installation
- **OS installation & configuration** — Windows 10/11, macOS, Linux (basics), mobile platforms
- **Networking fundamentals** — IPv4/v6, DHCP, DNS, Wi-Fi, basic troubleshooting
- **Mobile device support** — iOS, Android, MDM, passcode resets, synchronization
- **Virtualization concepts** — VM architecture, hypervisor types, cloud deployment models
- **Basic security** — Password policies, encryption, malware detection, access control
- **Backup & disaster recovery** — Local backup strategies, cloud-based recovery
- **Ticketing & documentation** — Service desk workflows, change management, asset inventory
- **Customer communication** — Professional tone, active listening, remote support
- **Basic scripting** — PowerShell, bash fundamentals (not deep programming)
- **Troubleshooting methodology** — Diagnostic flowcharts, log interpretation, root-cause analysis

## Study Strategy & Timeline

### Recommended prep schedule (12 weeks)

1. **Weeks 1–3:** Watch free video courses (Professor Messer) to establish foundational knowledge
2. **Weeks 4–7:** Read study guide (Mike Meyers or Sybex); take hand-written notes on weak areas
3. **Weeks 8–10:** Practice exam drills; target 80%+ on MeasureUp/Jason Dion tests
4. **Weeks 11–12:** Intensive review; retake weak domains; sit one full practice exam per day

### Exam day logistics

- **Arrive 15 min early** at test center
- **Take Core 1 first** (220-1201) if scheduling both on same day; allow 2+ hours between exams
- **Bring photo ID** + proof of address
- **Online proctoring:** quiet room, webcam visible, desk uncluttered
