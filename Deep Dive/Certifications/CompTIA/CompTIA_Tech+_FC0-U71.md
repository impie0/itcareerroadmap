# CompTIA Tech+

**`FC0-U71`** · ● **New** · **Entry** · _CompTIA_

> CompTIA Tech+ (FC0-U71) is the newly redesigned foundational IT certification that replaces ITF+ (FC0-U61). Launched July 16, 2024, it covers computing fundamentals, IT infrastructure, applications, software development, databases, and security—designed as a stepping stone before A+ or other specialized certifications.

## About

CompTIA Tech+ (FC0-U71) is the modernized replacement for IT Fundamentals+ (ITF+ FC0-U61, retired July 31, 2025). It is purpose-built as a **pre-career foundational credential**, helping career changers and non-IT professionals validate digital literacy and decide if IT is the right path before pursuing deeper certifications like A+ or vendor-specific tracks. Released July 2024, Tech+ aligns with 2024–2026 IT realities, including cloud, security awareness, and basic development concepts.

## Topics covered

The CompTIA Tech+ exam covers 6 domains with published weights:

- **IT Concepts and Terminology (26%)** — Binary, hexadecimal, IT terminology, troubleshooting methodology, computing device purpose
- **Infrastructure (18%)** — Computing devices, internal components, storage types, peripheral device installation and configuration
- **Applications and Software (14%)** — OS fundamentals, software installation, application concepts
- **Software Development Concepts (10%)** — Programming basics, loops, conditionals, algorithm concepts
- **Database Fundamentals (13%)** — Database concepts, tables, queries, relational data
- **Security (19%)** — CIA triad, confidentiality, integrity, availability; password best practices; encryption; phishing, malware, and threat identification; digital privacy

*Source:* [Official exam objectives ↗](https://comptiacdn.azureedge.net/webcontent/docs/default-source/exam-objectives/comptia-tech-fc0-u71-exam-objectives-(2-0).pdf?sfvrsn=8bf4d474_4)

## Skills validated

*Cert-specific — what the FC0-U71 exam actually tests.*

- Binary and hexadecimal number systems and conversion
- IT troubleshooting methodology (define problem → research → implement → verify)
- Hardware component identification and purpose (RAM, storage drives, motherboards, CPUs, power supplies)
- Peripheral device installation (USB, networking, printing)
- Operating system concepts and installation basics
- Web browser configuration and safe browsing practices
- Network basics (IP addressing, DNS, DHCP, LAN/WAN, Internet concepts)
- Mobile device configuration and MDM concepts
- Cloud service models (IaaS, PaaS, SaaS) at foundational level
- Database tables, records, and basic query concepts
- Programming logic (loops, conditionals, variables, data types)
- Password management, encryption, and MFA
- Malware types (virus, worm, ransomware, spyware, adware)
- Phishing, social engineering, and threat identification
- Data privacy and GDPR basics
- Backup and disaster recovery concepts
