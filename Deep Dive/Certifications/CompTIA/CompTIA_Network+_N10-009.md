## About

CompTIA Network+ (N10-009) is the vendor-neutral entry-level certification for IT support technicians, network technicians, and junior network administrators. Launched June 20, 2024, it replaced the N10-008 exam (retired December 2024) with a modernized focus on cloud interconnect, zero-trust security architectures (SASE/SSE), and software-defined wide-area networking (SD-WAN). The exam validates foundational understanding of IP addressing, routing, switching, wireless access, network services (DNS, DHCP), basic firewalls and VPNs, cabling/infrastructure, monitoring, and troubleshooting methodologies. Widely required by government contractors, SMBs, and managed service providers for entry-level roles.

## Topics covered

The exam blueprint covers **five functional domains** with the following approximate weightings:

- **Networking Concepts (23%)** — OSI model, TCP/IP stack, IP addressing (IPv4/IPv6), subnetting, DNS/DHCP, routing fundamentals, switching basics
- **Network Implementation (20%)** — Ethernet standards, VLANs, spanning-tree, interface configuration, wireless (802.11), WAN/cloud connectivity, infrastructure resilience
- **Network Operations (19%)** — Network monitoring, SNMP, syslog, troubleshooting methodology, performance metrics, configuration management, disaster recovery
- **Network Security (14%)** — Firewalls, VPN (site-to-site, remote access), authentication (802.1X, TACACS+, RADIUS), zero-trust/SASE concepts, network segmentation
- **Network Troubleshooting (24%)** — Systematic troubleshooting process, common issues, packet analysis basics, cable/infrastructure diagnostics, command-line tools

**Key 2024/2026 additions:**
- SD-WAN and zero-trust security (SASE/SSE) as core topics
- Cloud interconnect and hybrid network design
- Intent-Based Networking (IBN) concepts
- Configuration management and automated deployment

## Skills validated

- IPv4 and IPv6 addressing and subnetting
- TCP/IP protocol stack and OSI model application
- DNS and DHCP configuration and troubleshooting
- VLAN trunking and inter-VLAN routing
- Spanning-tree protocol and RSTP
- Basic BGP, OSPF, and EIGRP concepts
- Ethernet standards (10BASE-T, 100BASE-TX, 1000BASE-T, fiber)
- 802.11 wireless (a/b/g/n/ac/ax) RF fundamentals and security (WPA2/WPA3)

- Site-to-site and remote-access VPN concepts
- Firewall rule sets and access control lists (ACLs)
- 802.1X and AAA (TACACS+, RADIUS)
- SD-WAN and zero-trust architecture (SASE/SSE)
- Network monitoring tools (Wireshark, tcpdump, netstat, traceroute)
- SNMP and syslog
- Cable management and infrastructure best practices
- Disaster recovery and business continuity concepts
