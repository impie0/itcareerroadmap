## About

CompTIA Server+ (SK0-005) validates vendor-neutral skills to **install, configure, manage, and troubleshoot physical and virtual servers** in on-premises, cloud, and hybrid environments. Released May 2021, it replaces the earlier SK0-004 version and represents the current standard for associate-level server administration. The cert is designed for IT professionals who operate datacenter infrastructure, including server hardware setup, storage management, security hardening, disaster recovery, and performance troubleshooting. **Critical distinction:** Server+ carries **lifetime validity** — once earned, it never expires and requires no renewal or continuing education, making it a permanent credential.

## Topics covered

**Domain 1: Server Hardware Installation and Management (18%)**
- Server form factors (1U, 2U, blade) and rack standards
- Power delivery (PSU redundancy, UPS integration, PDU management)
- Thermal management (airflow, cooling systems, monitoring)
- Network connectivity (Ethernet, NIC bonding, VLAN tagging)
- Storage hardware (HDDs vs. SSDs, hot-swap bays, RAID controllers)
- RAID levels (0, 1, 5, 6, 10) and hot-spare configuration
- Out-of-band management (IPMI, iLO, iDRAC, Redfish)
- Firmware and driver updates, hardware diagnostics

**Domain 2: Server Administration (30%)**
- Server deployment and initial configuration
- Operating system installation (Windows Server, Linux)
- User and group management, access control
- Storage and partition management (local, SAN, NAS)
- Virtual machine deployment and management (Hyper-V, ESXi, KVM)
- Service management and log monitoring
- Performance tuning (CPU, memory, disk I/O bottlenecks)
- Backup and snapshot strategies

**Domain 3: Security and Disaster Recovery (24%)**
- Authentication and authorization (local, domain, LDAP)
- Firewall rules and network segmentation
- Malware protection and patch management
- Backup and restore procedures
- Disaster recovery planning (RPO, RTO, failover mechanisms)
- High availability (clustering, load balancing)
- Compliance requirements (GDPR, HIPAA, PCI-DSS basics)
- Encryption at rest and in transit

**Domain 4: Troubleshooting (28%)**
- Hardware diagnostics and failure identification
- Boot and startup issues (BIOS/UEFI, boot order, POST)
- Network troubleshooting (connectivity, DNS, IP conflicts)
- Storage performance and capacity issues
- Virtualization and VM troubleshooting
- Service and daemon failures
- Log analysis and event monitoring
- Performance analysis and optimization

## Common skills at Systems/OS · Associate

*Shared competencies for the Systems/OS domain at Associate level — not specific to this cert.*

- **Command-line proficiency** — Windows Server CLI (PowerShell), Linux shell scripting basics
- **File systems and partitioning** — NTFS/ext4/XFS, MBR/GPT, LVM concepts
- **User/group management** — Local and domain accounts, permission inheritance, sudo/RunAs
- **Service and daemon control** — systemd (Linux), Services MMC (Windows), startup behavior
- **Networking fundamentals** — TCP/IP, DNS, DHCP, NIC configuration, basic routing
- **Log analysis** — Event Viewer (Windows), journalctl/rsyslog (Linux), troubleshooting via logs
- **Performance monitoring** — Task Manager, Resource Monitor (Windows), top/vmstat (Linux)
- **Storage basics** — Partitioning, mount points, disk quotas, RAID concepts

## Skills validated

*Cert-specific — what this exam actually tests, distinct from the shared "Common skills" above.*

- **Server hardware installation & racking** — Physical deployment, cable management, power distribution
- **RAID configuration & management** — Levels 0, 1, 5, 6, 10; hot-spare; rebuild scenarios
- **Storage technologies** — SAN vs. NAS distinction, shared storage provisioning, capacity planning
- **Power and thermal infrastructure** — PSU redundancy, UPS setup, cooling airflow, monitoring
- **Out-of-band management** — IPMI, iLO (HP/HPE), iDRAC (Dell), chassis management
- **Virtual machine deployment** — Hyper-V, ESXi, KVM basics; resource allocation; snapshots
- **Windows Server administration** — Roles/features, DNS, DHCP, Active Directory basics, PowerShell
- **Linux server basics** — Installation, user management, systemd services, file permissions
- **High availability & disaster recovery** — Clustering, failover, backup/restore, RPO/RTO concepts
- **Security hardening** — Firewall rules, patch management, authentication, encryption
- **Troubleshooting methodology** — Hardware diagnostics, boot issues, network connectivity, log analysis
- **Performance optimization** — CPU, memory, disk I/O tuning; bottleneck identification
