# SailPoint Certified Identity Security Cloud Professional
**`ISC-Professional`** · ● Active · Professional · _SailPoint_

## Overview

The **SailPoint Certified Identity Security Cloud Professional** is a knowledge-based credential demonstrating foundational competency in SailPoint's Identity Security Cloud platform. Launched in February 2025 as part of SailPoint's expanded credentialing program, this credential serves as an entry-to-intermediate level validation of skills in cloud-based identity governance, access management, and compliance operations.

Unlike the role-based **Identity Security Engineer** certification (which requires 1+ years hands-on experience and a proctored exam), the Professional credential is designed to validate learning through course completion and an unproctored exam, making it accessible to professionals transitioning into ISC roles or validating foundational knowledge.

## About

The SailPoint Identity Security Cloud Professional credential validates foundational knowledge across core ISC platform domains. This credential is designed for:

- **Career changers** entering identity governance and access management
- **Current professionals** seeking to validate ISC platform knowledge
- **IT administrators** expanding into identity security specialties
- **Consultants and integrators** preparing for deeper ISC engagement
- **Students** building foundational credentials before pursuing Engineer-level certification

### Credential Type vs. Certification

SailPoint distinguishes between two credential types:

- **Knowledge Credentials**: Validate learning through course completion + unproctored exam; require no hands-on experience; serve as stepping stones to advanced certifications
- **Role-Based Certifications**: Validate real-world expertise; require 1+ years hands-on experience; use proctored exams; represent mastery-level competency

The Professional credential is a **Knowledge Credential**, positioned as a progression pathway toward the **Identity Security Engineer** certification.

### Career Trajectory

- **Professional Credential** → Entry-level validation, career transition tool
- **Identity Security Engineer** → Advanced certification, role-based expertise validation
- **Identity Security Expert** → Advanced knowledge credential combining Professional + Engineer experience

## Domain Context — Identity / IGA

Identity Governance and Administration (IGA) is a critical cybersecurity domain addressing:

- **Who has access to what**: Mapping identities to entitlements across enterprise systems
- **Compliance**: Ensuring access policies align with regulatory requirements (SOX, GDPR, HIPAA, etc.)
- **Risk management**: Detecting and remediating access anomalies and excessive privileges
- **Identity lifecycle**: Managing identity creation, provisioning, updates, and deprovisioning

SailPoint's ISC platform automates these functions in cloud environments, replacing legacy on-premises IdentityIQ deployments. ISC enables:

- **Cloud-native architecture**: SaaS delivery with multi-tenant scalability
- **Rapid deployment**: Weeks vs. months for complex implementations
- **API-first integrations**: Native connectors to 300+ SaaS and enterprise applications
- **AI-driven governance**: Machine learning for anomaly detection and access recommendations

The Professional credential validates knowledge of ISC's core capabilities within this domain.
