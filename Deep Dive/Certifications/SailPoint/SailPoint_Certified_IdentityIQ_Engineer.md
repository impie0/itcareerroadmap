## About

The SailPoint Certified IdentityIQ Engineer credential validates professional-level competency in designing, developing, and troubleshooting identity governance solutions using the SailPoint IdentityIQ platform. First introduced in 2015, it has evolved through multiple versions to reflect current platform capabilities. This certification is ideal for IAM engineers, systems architects, and platform administrators who implement identity lifecycle management, provisioning workflows, and compliance controls in enterprise environments.

## Preparation tips

**Hands-on labs are non-negotiable.** SailPoint certifications prioritize practical platform experience over exam memorization. The IdentityIQ sandbox (available via partner programs or trial environments) is essential.

- **Workflow design:** Build 5–10 custom workflows covering request routing, approvals, and error handling.
- **Connector correlation:** Test identity matching algorithms with real-world data sets.
- **Report building:** Create 3–5 custom reports covering compliance, SoD violations, and access reviews.
- **SOD rules:** Design and test SOD rule chains for common compliance frameworks (SOX, GDPR, HIPAA).
- **Exam windows:** Allow 4–6 months from start to certification if new to IdentityIQ; 2–3 months if already working on the platform.
