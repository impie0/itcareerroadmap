## About

The Veritas Certified Specialist: NetBackup Administrator Advanced (VCS-284) is an advanced-level vendor credential validating the ability to configure, maintain, troubleshoot, and optimize NetBackup 10.x environments. Earners demonstrate expertise in master server administration, media server management, client configuration, storage unit setup, disaster recovery, replication, and security implementations. The exam targets IT professionals responsible for day-to-day NetBackup administration, capacity planning, performance tuning, and infrastructure optimization. As of December 2024, Veritas has merged with Cohesity; certification and training platforms are transitioning to Cohesity, but the VCS-284 credential remains active and widely recognized in the industry.

## Domain context — Data Protection

Data Protection encompasses technologies, strategies, and practices for safeguarding critical data through backup, archiving, replication, and recovery mechanisms. This domain addresses organizational requirements for RPO (Recovery Point Objective) and RTO (Recovery Time Objective) compliance, ransomware mitigation, and business continuity across physical, virtual, and cloud environments. Veritas NetBackup is an enterprise backup platform supporting diverse environments including physical servers, virtual machines, and cloud instances.

## Topics covered

The exam blueprint covers the following domains and topics:

- **NetBackup Architecture & Infrastructure** — Master server (command center), media servers (storage infrastructure), NetBackup client components, agent types and deployments
- **NetBackup Configuration** — Host properties, security certificates, network configuration, resource management, client registration
- **Primary Storage Setup** — Storage unit configuration, disk pools, tape device management, storage device properties
- **Media Management & Lifecycle** — Media pools, retention policies, media lifecycle management, storage tiering, archive media
- **Backup Job Management** — Policy creation and scheduling, backup types (full, incremental, synthetic full), job monitoring, job history
- **Catalog Backup & Disaster Recovery** — Catalog backup procedures, disaster recovery for master and media servers, catalog restoration
- **Security & Access Control** — RBAC (Role-Based Access Control) in NetBackup, user authentication, certificate management, data encryption
- **Application Agent Integration** — Oracle database backup (RMAN integration), SQL Server backup (VSS/native), VMware vCenter integration, Hyper-V protection
- **Virtual Machine Protection** — VMware snapshot management, Hyper-V backup, VM-level recovery, instant access (live mount)
- **Cloud Integration & Deduplication** — NetBackup CloudCatalyst (cloud deduplication), public cloud targets (AWS, Azure, GCP), cloud data protection
- **Replication Director** — Replication setup and management, disaster recovery copies to alternate NetBackup masters, failover procedures
- **Malware Scanning & Compliance** — Malware detection in backups, compliance reporting, audit trails
- **NetBackup REST API** — API-based automation, scripting, integration with third-party systems
- **Troubleshooting & Performance Optimization** — Failed job diagnosis, log analysis, performance tuning, capacity planning, diagnostic tools
- **Disaster Recovery Planning** — RTO/RPO requirements, failover strategies, recovery testing procedures

## Exam tips & study advice

1. **Hands-on lab environment** — Build a NetBackup lab (trial or training instance) to practice master server, media server, and client configurations
2. **Master NetBackup architecture** — Master server, media server, and client relationships are foundational; diagram these thoroughly
3. **Storage device management** — Understand disk pools, tape devices, and media management; heavily tested
4. **Disaster recovery scenarios** — Practice catalog backup, master server recovery, and media server failover procedures
5. **Job troubleshooting** — Study job logs, error codes, and diagnostic tools; real-world troubleshooting is emphasized
6. **Security & RBAC** — Understand role-based access control and certificate-based security
7. **Application-specific workflows** — Oracle RMAN, SQL Server VSS, VMware snapshot integration are regularly tested
8. **Time management** — 105 minutes for 75–85 questions; approximately 75 seconds per question
9. **Use official exam prep guide** — Veritas provides a detailed exam preparation guide with domain weighting
10. **Review release notes** — NetBackup 10.x feature updates and enhancements appear on the exam
