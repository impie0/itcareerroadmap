## About

VCS-284 is the current Veritas Certified Specialist (Associate-level) certification for NetBackup 10.x administrators. It validates hands-on competency in configuring primary and media servers, storage policies, backup jobs, deduplication, disaster recovery, and operational monitoring. Passing VCS-284 counts toward the VCP in Data Protection, which requires passing multiple VCS exams across Veritas' data protection product line. The exam launched to support NetBackup 10.x adoption and superseded VCS-282 (NetBackup 9.1) as the current standard. No formal prerequisites; practical experience administering NetBackup 10.x is strongly recommended.

## Domain context — Backup Administration

Backup Administration encompasses the design, deployment, configuration, and operational management of enterprise backup and recovery systems—including software, storage, policies, disaster recovery (DR) planning, and data protection compliance. This domain includes vendor-specific certifications for NetBackup, Commvault, Veeam, IBM Spectrum Protect, and cloud-native backup platforms.

---

## Topics covered

- **Configuration of primary, media, and client host settings** — Master server setup, media server configuration, client registration, security certificates, SSL/TLS configuration
- **Disk and cloud storage configuration** — Storage unit groups, MSDP (Media Server Deduplication Pool), cloud storage, Snapshot Manager, WORM storage, KMS options
- **Backup policy and protection plan design** — Full/incremental/differential policies, SLP (Storage Lifecycle Policy), multiplexing, resource scheduling, BOE (Backup Exec Optimization Engine)
- **NetBackup deduplication** — Media server deduplication, client-side deduplication, optimized duplication, dedup ratios, cloud dedup
- **Image and catalog management** — Backup image lifecycle, expiration, import, duplication, catalog backup and recovery
- **Operations and monitoring** — Job monitoring (backup, restore, duplication), prioritization, suspension, cancellation, reports, alerts, performance tuning
- **Restore and disaster recovery** — Initiating and monitoring restores, point-in-time recovery (PITR), bare-metal recovery, VM recovery, granular recovery
- **Troubleshooting and performance** — Interpreting logs, common failure modes, network tuning, dedup efficiency, capacity planning
