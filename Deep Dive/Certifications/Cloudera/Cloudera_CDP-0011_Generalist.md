## About

The CDP Generalist Exam validates broad foundational knowledge of the Cloudera Data Platform (CDP) architecture, components, and capabilities. This is not a role-specific certification—it tests general comprehension across multiple domains including system administration, development, data analysis, engineering, data science, and architecture perspectives.

The certification demonstrates that you understand the core CDP ecosystem and can navigate deployment scenarios across public cloud and on-premises environments.

## Domain context

The Cloudera Data Platform (CDP) is Cloudera's next-generation data platform that unifies data warehousing, data engineering, and analytics. It supports hybrid and multi-cloud deployments and provides unified governance, security, and data catalog capabilities.

With CDP, organizations move from traditional data silos to integrated, governed data environments. This generalist certification validates foundational knowledge across:
- Core CDP architecture and components
- Security models and governance frameworks
- Analytics and data experience capabilities
- Deployment models (Public Cloud, Private Cloud, Local)
- Integration with cloud providers (AWS, Azure, GCP)
