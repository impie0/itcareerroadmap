## About

The CDP Certified Administrator – Public Cloud certification validates professional expertise in deploying, managing, and operating Cloudera Data Platform environments on major public cloud infrastructures. This certification is ideal for system administrators, cloud engineers, and infrastructure specialists responsible for CDP cluster operations, security, and governance.

Certified administrators demonstrate mastery of cloud infrastructure integration, identity and access management, environment provisioning, data lake operations, and security policy administration—critical skills for maintaining production CDP environments.

## Domain context

Managing Cloudera Data Platform on public cloud requires understanding how CDP components interact with cloud-native infrastructure:

**Cloud Infrastructure Integration**: Deep knowledge of AWS, Azure, or GCP networking, storage, and identity services. Understanding how CDP provisions resources, manages networking (VPCs, subnets, security groups), and integrates with cloud storage (S3, Data Lake Storage, Cloud Storage).

**Identity and Access Management**: Configuring IAM policies, service accounts, cross-account access, and single sign-on integration. Understanding how Ranger policies in CDP layer on top of cloud IAM.

**Environment Lifecycle**: Onboarding new environments, configuring networking and connectivity, setting up clusters, managing scaling, backup/disaster recovery, and decommissioning.

**Operational Excellence**: Monitoring cluster health, managing resource utilization, troubleshooting issues, applying patches and updates, managing cost optimization.

**Security and Governance**: Implementing data encryption, managing sensitive data access, audit logging, compliance configurations, and security policy administration.
