## About

Trend Micro Vision One is an AI-powered extended detection and response (XDR) platform that unifies detection and response across endpoints, networks, servers, and cloud workloads. The Certified Associate level validates foundational knowledge of Vision One deployment, configuration, and operational use. This certification is designed for security professionals deploying Vision One in security operations centers (SOCs) and is part of Trend Micro's internal certification hierarchy, though a formal examination code has not been publicly assigned.

---

## Domain context — Cybersecurity/XDR Platform

Extended Detection and Response (XDR) represents next-generation security operations, consolidating threat signals from multiple security layers to reduce detection and response time. Vision One integrates Trend Micro and third-party products into a unified threat investigation and remediation platform.
