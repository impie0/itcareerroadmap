## About

The MTCSE is a professional-level certification for network engineers and security specialists who design, deploy, and maintain secure MikroTik RouterOS networks. Delivered only through official 2-day MikroTik training courses, the exam is invigilated by an approved trainer and tests hands-on knowledge of firewall configuration, attack mitigation, cryptography, VPN tunneling, and router hardening. The certification is widely recognized in ISP and enterprise network infrastructure roles.

The MTCSE builds directly on MTCNA (MikroTik Certified Network Associate) foundations and deepens expertise in the security-specific aspects of RouterOS deployments. Candidates who hold MTCSE are typically positioned for roles such as network security engineer, ISP infrastructure engineer, or security-focused network architect roles. The exam consists of 25 questions delivered in a 60-minute proctored session at the end of the official training course, with a 60% passing score (15 correct answers). A second-attempt opportunity is provided for candidates scoring between 50–59%, allowing a path to certification without immediate retesting costs.

## Topics covered

The MTCSE course outline covers six main modules:

- **Module 1: Introduction** — Security concepts, threats, vulnerabilities, risk assessment
- **Module 2: Firewall** — Stateful firewall configuration, packet flow, firewall chains (filter, NAT, mangle), RAW table operations
- **Module 3: OSI Layer Attacks & Mitigation** — ARP poisoning, DHCP starvation, SYN floods, ICMP attacks, MNDP threats, bridge filtering
- **Module 4: Cryptography & Authentication** — Encryption basics, SSL/TLS in RouterOS, user management, RADIUS, certificate-based access
- **Module 5: Router Hardening & Management Access** — Best practices, disabling unused services, API security, SSH hardening, detecting attacks on critical services
- **Module 6: Secure Tunnels** — IPsec fundamentals, L2TP + IPsec, SSTP with certificates, VPN deployment and troubleshooting

*Source:* [MTCSE Course Outline PDF ↗](https://i.mt.lv/cdn/training_pdf/mtcse_outline_2019181161836.pdf)

## Skills validated

**Firewall & Packet Processing:**
- RouterOS firewall chains (filter, NAT, mangle, raw tables) and rule ordering
- Stateful connection tracking and session state management
- Bridge filtering and Layer 2 security filtering
- Network Address Translation (NAT) configuration for single and dual-stack networks
- Packet marking and mangling for QoS and traffic management
- Raw table operations for defragmentation and connection tracking

**Attack Detection & Mitigation:**
- DoS and DDoS attack vectors (SYN floods, connection exhaustion, UDP floods)
- ARP poisoning and ARP table security
- DHCP starvation and rogue DHCP server detection
- MNDP (MikroTik Neighbor Discovery Protocol) attack prevention
- ICMP-based reconnaissance and filtering strategies
- Layer 2 attacks (MAC spoofing, bridge hijacking)

**VPN & Cryptography:**
- IPsec tunnel setup, policy configuration, and troubleshooting
- L2TP + IPsec for remote access scenarios
- SSTP (Secure Socket Tunneling Protocol) with certificate-based authentication
- Encryption algorithm selection and key management
- SSL/TLS certificate deployment in RouterOS

**Authentication & Access Control:**
- User role-based access control and permission models
- RADIUS integration for centralized authentication
- Certificate-based authentication and PKI concepts
- Hotspot security and captive portal configuration
- Wireless security (WPA/WPA2) and RADIUS-based 802.1X

**Router Hardening:**
- Disabling unnecessary services to reduce attack surface
- SSH hardening and key-based authentication
- API security and access restrictions
- Management access filtering by source IP
- Detecting and responding to attacks on critical infrastructure services
