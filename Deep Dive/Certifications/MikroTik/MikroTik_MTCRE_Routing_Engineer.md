## About

The **MikroTik Certified Routing Engineer** (MTCRE) is a professional-level routing certification for engineers working with MikroTik RouterOS across ISP, WISP, and enterprise networks. It builds on the MTCNA and covers advanced routing protocols (OSPF, BGP, RIP), tunneling technologies (PPP, EoIP, IPIP, GRE), VLAN segmentation, routing filters, and BGP path-selection mechanics. The cert is widely recognized in Eastern Europe, South Africa, and Latin America as a marker of hands-on routing expertise; it is particularly prominent among ISP/WISP engineers and network operators using MikroTik hardware or RouterOS in production environments.

## Topics covered

- **Static Routing** — route definition, administrative distance, gateway and interface configuration
- **Routing Information Protocol (RIP)** — RIP v1 and v2, metric calculation, passive/active interfaces, route distribution
- **Open Shortest Path First (OSPF)** — single-area and multi-area OSPF, neighbor election, LSA flooding, SPF calculation, area types (backbone, stub, not-so-stubby)
- **Border Gateway Protocol (BGP)** — BGP fundamentals, AS path, local preference, MED, route selection logic, basic filtering
- **Tunneling Technologies** — Point-to-Point Protocol (PPP), Ethernet over IP (EoIP), IP-in-IP (IPIP), Generic Routing Encapsulation (GRE)
- **VLAN Configuration** — 802.1Q tagging, trunk and access mode, native VLAN
- **Routing Filters and Policy** — prefix lists, route maps, route filtering, conditional routing
- **Traffic Control and Quality of Service** — queue discipline, priority queuing, traffic shaping (if included in current exam revision)

## Skills validated

- OSPF configuration across single and multi-area topologies
- BGP neighbor establishment, route advertisement, and AS path manipulation
- RIP v1/v2 protocol operation and metric calculation

- Practical diagnostics: traceroute, ping, route tracing, neighbor observation
