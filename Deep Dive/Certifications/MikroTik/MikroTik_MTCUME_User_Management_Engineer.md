## About

The MTCUME (MikroTik Certified User Management Engineer) is an advanced-level engineer certification in the MikroTik Academy track. It is designed for network professionals and service providers who manage large-scale, multi-user networks using RouterOS with centralized user management and authentication systems.

MTCUME validates expertise in:
- Advanced RouterOS user and access management
- Hotspot configuration and subscriber management
- PPP protocol variants (PPPoE, PPTP, L2TP, IPSec)
- VPN deployment and security
- RADIUS integration for centralized authentication
- Large-scale ISP/wireless deployments
- User Manager application (MikroTik's centralized user database)

The MTCUME is especially valuable for:
- **ISP Engineers**: Managing subscriber networks with thousands of users
- **Wireless Operators**: Hotspot and Wi-Fi access point management
- **Service Providers**: Multi-tenant network management
- **Enterprise Network Admins**: Large-scale VPN and user authentication deployments

Prerequisites require that candidates hold or have passed the MTCNA certification (or equivalent knowledge), ensuring foundational understanding of RouterOS before tackling advanced user management.

## Domain Context

The MTCUME sits within MikroTik's four-tier certification ecosystem:

1. **MTCNA** — Certified Network Administrator (entry level)
2. **MTCUME** — Certified User Management Engineer (advanced)
3. **MTCRE** — Certified Routing Engineer (advanced)
4. **MTCTCE** — Certified Training Center Engineer (instructor level)

MikroTik is a leading provider of RouterOS and networking hardware for ISPs, service providers, and enterprise networks. The certification emphasizes practical, hands-on skills in:

- **User Authentication & Management**: RADIUS, hotspot, and subscriber databases
- **Access Control**: Per-user bandwidth limiting, quota management, and traffic shaping
- **VPN & Encryption**: Secure remote access for distributed users
- **Large-Scale Deployments**: Managing networks with thousands of concurrent users
- **Service Provider Models**: Billing integration, user analytics, and reporting

MTCUME is particularly relevant in developing markets and service provider environments where MikroTik solutions dominate due to cost-effectiveness and powerful features.

## Topics Covered

### Core Areas

1. **MikroTik User Manager**
   - User database and directory setup
   - Multi-profile and multi-level user hierarchies
   - User provisioning and deprovisioning
   - User quotas and bandwidth limits
   - User lifecycle management

2. **Hotspot Configuration**
   - Hotspot profile creation and management
   - User authentication methods
   - Per-user data limits and time restrictions
   - Hotspot billing integration
   - MAC authentication
   - Walled-garden setup (free access services)

3. **PPP Protocol Suite**
   - PPPoE (PPP over Ethernet) configuration
   - PPTP (Point-to-Point Tunneling Protocol) setup
   - L2TP (Layer 2 Tunneling Protocol) with authentication
   - IPSec VPN fundamentals and configuration
   - PPP user profiles and IP pool management

4. **RADIUS Integration**
   - RADIUS server configuration on RouterOS
   - RADIUS client setup (connecting remote RADIUS servers)
   - User authentication via RADIUS
   - Accounting and logging via RADIUS protocol
   - RADIUS attribute mapping for routing policies

5. **VPN Technologies**
   - Site-to-site VPN configuration
   - Remote access VPN for mobile/distributed users
   - Tunnel encryption and algorithms
   - VPN troubleshooting and diagnostics

6. **User Access Control & Policies**
   - Bandwidth limiting and fair queuing
   - Per-user firewall rules
   - Traffic shaping and QoS
   - Access-list filtering
   - Time-based access restrictions

7. **ISP & Service Provider Scenarios**
   - Multi-tenant network isolation
   - Subscriber management workflows
   - Usage billing and quota enforcement
   - Network performance monitoring
   - SLA management

8. **Security & Monitoring**
   - User session monitoring
   - Login and access auditing
   - Threat detection (brute-force protection)
   - User data encryption
   - DDoS mitigation for user-facing services
