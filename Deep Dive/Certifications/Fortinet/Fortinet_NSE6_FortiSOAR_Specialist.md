## About

The **Fortinet NSE 6 — FortiSOAR Specialist** certification validates professional-level knowledge and hands-on expertise in deploying, managing, and optimizing Fortinet's FortiSOAR platform. FortiSOAR is an enterprise Security Orchestration, Automation, and Response (SOAR) solution that reduces dwell time, improves incident response efficiency, and minimizes manual SOC workload through playbook automation and threat intelligence integration. This credential is designed for security operations professionals, SOC analysts, and automation engineers who manage security incident workflows, integrate third-party security tools, and implement security response automation. The exam aligns with FortiSOAR 7.3 and is part of the broader Fortinet NSE 6+ Professional tier and FCP (Fortinet Certified Professional) Security Operations track.

---

## Domain context — SOAR / Security Automation

Security Orchestration, Automation, and Response (SOAR) platforms are critical infrastructure in modern Security Operations Centers (SOCs), enabling enterprises to automate repetitive alert triage and incident response workflows, integrate disparate security tools and threat intelligence sources, reduce mean time to response (MTTR) and analyst burnout, and scale incident handling without proportional headcount increases.

This domain bridges security operations, automation engineering, and incident response management.

## Topics covered

The **NSE6_FSR-7.3** exam blueprint covers five core knowledge domains:

- **SOC and SOAR Overview** — Role of security operations centers, SOAR platform capabilities, incident response workflows, threat intelligence integration, and automation benefits
- **System Configuration** — Connector setup and management, playbook authoring and logic flow, custom fields and data models, integration architecture, and API usage
- **Security Management** — Role-based access control (RBAC), permission inheritance, multi-tenancy, user provisioning, API key management, and least-privilege principles
- **System Operation** — Incident creation, assignment, escalation workflows, manual and automated response execution, dashboard navigation, reporting, and day-to-day SOC tasks
- **System Monitoring and Maintenance** — Performance tuning, log management and audit trails, backup and recovery, system health monitoring, troubleshooting, patch management, and update procedures
