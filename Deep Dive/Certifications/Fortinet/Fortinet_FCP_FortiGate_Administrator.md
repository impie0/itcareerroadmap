## About

The Fortinet Certified Professional (FCP) in Secure Network — FortiGate Administrator exam tests hands-on proficiency in deploying, configuring, and administering Fortinet FortiGate firewalls in production environments. This single-exam certification validates knowledge of FortiGate architecture, security policies, authentication, routing, VPN (IPsec and SSL), and threat prevention profiles. Released in its current form (NSE 4 — FortiOS 7.6) in July 2025, it replaced the 7.4 exam and is now the primary entry-level Fortinet network security credential at the Professional tier. Holders typically spend 4–8 weeks preparing and possess 1–2+ years of hands-on networking or firewall experience. As of May 2026, this certification remains under the "FCP" brand; it will rebrand to "NSE 4" effective July 15, 2026, as part of Fortinet's broader NSE program expansion.

## Domain context — Security / Networking

Firewall and network security policy management — a core competency within IT security operations and network defense. FortiGate devices are enterprise firewalls deployed at network perimeters, branch offices, and datacenters for threat prevention, traffic segmentation, and VPN termination. This cert validates production-grade FortiGate administration and pairs with other Fortinet certs (NSE 5 specialties, NSE 6–7 solutions architect roles) to form FCP and higher-tier credentials.

## Topics covered

The exam assesses the following competencies (per NSE 4 official blueprint):

- **FortiGate Architecture & Licensing** (~10%) — Hardware specs, licensing models, FortiCare, FortiGuard subscriptions, system initialization
- **Network Interfaces & Routing** (~15%) — Interface configuration, IP addressing, static/dynamic routing (OSPF, BGP), policy-based routing, SD-WAN basics
- **Firewall Policies & NAT** (~20%) — Policy syntax, destination NAT (DNAT), source NAT (SNAT), proxy-based inspection, policy logging
- **User Authentication & Identity** (~12%) — LDAP, RADIUS, local user databases, user groups, two-factor auth, SSO integration, captive portal
- **VPN** (~15%) — IPsec site-to-site VPN, SSL/TLS remote-access VPN, certificate management, VPN dial-up, redundancy
- **Security Profiles & Threat Prevention** (~12%) — Intrusion prevention (IPS), antivirus, web filtering (URL categories), application control, DNS filtering, sandbox (FortiSandbox)
- **Administration & Troubleshooting** (~10%) — CLI and GUI operations, log review, packet traces, HA (high availability) failover, backup/restore, firmware updates
- **Advanced Features** (~6%) — DNS security, advanced routing, multicast, traffic shaping, SSL inspection (certificate pinning, decryption limits)

## Skills validated

*Concrete FortiGate-specific technologies this exam tests, distinct from the shared "Common skills" above.*

- **FortiGate appliance administration (CLI & GUI)** — system config, firmware updates, HA setup
- **Firewall policy syntax** — ingress/egress rules, service objects, address objects, scheduling
- **FortiOS routing protocols** — OSPF, BGP, ECMP, reverse route injection (RRI)
- **IPsec VPN configuration** — phase 1/2 negotiation, encryption algorithms, IKE versions, PFS (perfect forward secrecy)
- **SSL/TLS remote-access VPN** — portal config, certificate installation, user authentication, LDAP/RADIUS backend
- **Security profiles (IPS, antivirus, web filtering, app control)** — profile tuning, block/monitor modes, signature updates
- **FortiGuard threat intelligence** — subscription activation, URL categorization, advanced threat protection (ATP)
- **LDAP/RADIUS integration** — user authentication against Active Directory, Okta, or third-party identity providers
- **DNS security (DNS filtering, DoH inspection)** — blocking malicious domains, encrypted DNS decryption
- **FortiSandbox & advanced threat detection** — file inspection, behavioral analysis, zero-day protection
- **Packet traces & log analysis** — troubleshooting connectivity, identifying dropped traffic, performance bottlenecks
- **High availability (HA) failover** — active-passive and active-active modes, VRRP, priority tuning
