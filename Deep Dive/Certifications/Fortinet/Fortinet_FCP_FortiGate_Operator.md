**`NSE3_FGT_OP-7.4`** · ● **Active** · **Associate** · _Fortinet_

> **Certification naming transition (2026):** Fortinet is transitioning from "FCP" (Fortinet Certified Professional) nomenclature to "NSE 3" (Network Security Associate) effective **July 15, 2026**. The exam code NSE3_FGT_OP remains consistent. This certification sits one level below the Professional-tier NSE 4 FortiGate Administrator and serves as an entry-level hands-on FortiGate operator credential. As of May 2026, both "FCP FortiGate Operator" and "NSE 3 FortiGate Operator" naming are valid; after July 15, 2026, the NSE 3 branding is official.

## About

The Fortinet Certified Professional (FCP) FortiGate Operator exam tests foundational proficiency in deploying, configuring, and operating Fortinet FortiGate firewalls in basic production environments. This single-exam certification validates core knowledge of FortiGate architecture, policy basics, authentication, simple routing, and fundamental threat prevention concepts. Released in its current form (NSE 3 — FortiOS 7.4) in April 2025, it positioned itself as a logical stepping stone between the free NSE 2 Associate-level knowledge and the hands-on Professional tier (NSE 4). Candidates typically spend 2–4 weeks preparing and possess entry-level to junior-level networking or systems administration experience. As of May 2026, this certification remains under the "FCP" brand; it will rebrand to "NSE 3" effective July 15, 2026, as part of Fortinet's broader NSE program expansion.

## Domain context — Security / Networking

Firewall and network security appliance operation — a core competency for entry-level IT operations and network support. FortiGate devices are enterprise firewalls deployed at network perimeters, branch offices, and datacenters for threat prevention and traffic control. This cert validates foundational FortiGate operation and pairs with NSE 2, NSE 4, and higher-tier Fortinet certs to form a complete career progression from awareness through expert.

## Topics covered

The exam assesses the following competencies (per NSE 3 official blueprint):

- **FortiGate Product Architecture & Licensing** (~12%) — Hardware SKUs, licensing models, FortiCare basics, system initialization, web-based admin console
- **Network Interfaces & IP Configuration** (~12%) — Interface types, IP addressing, VLAN basics, interface roles (LAN, WAN, DMZ)
- **Basic Firewall Policy Concepts** (~18%) — Policy syntax, source/destination objects, services, action (allow/deny), policy logging, implicit deny rule
- **User Authentication & Access Control** (~14%) — Local user accounts, LDAP basics, RADIUS basics, user groups, captive portal concept
- **Network Address Translation (NAT)** (~12%) — Source NAT, destination NAT, port translation, one-to-one NAT concepts
- **Basic VPN Operations** (~12%) — IPsec site-to-site VPN overview, SSL/TLS remote-access VPN overview, certificate basics, VPN state monitoring
- **Threat Prevention & Security Profiles** (~10%) — Antivirus basics, intrusion detection (IDS) overview, web filtering URL categories, application control basic concepts
- **Administration & Troubleshooting** (~10%) — CLI and GUI navigation, log viewing, basic connectivity diagnostics, firmware update process, password reset

_Source:_ [Fortinet NSE 3 Exam Blueprint ↗](https://training.fortinet.com/local/staticpage/view.php?page=nse_3)

## Skills validated

*Concrete FortiGate-specific technologies this exam tests, distinct from the shared "Common skills" above.*

- **FortiGate appliance operation (CLI & GUI basics)** — system access, password reset, system time, basic navigation
- **FortiGate firewall policy syntax** — policy order, source/destination address objects, service objects, policy action (allow/deny/monitor)
- **Static routing configuration** — default gateway, static routes, next-hop designation, route monitoring
- **IPsec VPN basics** — site-to-site VPN phase 1 and phase 2 overview, pre-shared keys vs. certificates, VPN status monitoring
- **SSL/TLS VPN basics** — remote-access VPN concept, portal configuration basics, certificate selection, user authentication
- **LDAP and RADIUS integration** — remote server configuration, user authentication flow, group mappings
- **Antivirus and IPS basics** — profile creation, signature updates, monitoring blocked threats
- **Web filtering basics** — URL category selection, blocking policy application, monitoring blocked sites
- **Log review & monitoring** — policy logs, traffic logs, security event logs, basic log filtering and search
- **Interface and VLAN configuration** — interface naming, IP assignment, VLAN tagging, interface status monitoring
- **System administration basics** — firmware updates, configuration backup/restore, HA awareness (basic failover concept)
