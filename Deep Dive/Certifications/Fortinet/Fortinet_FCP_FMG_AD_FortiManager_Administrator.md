## About

The FCP - FortiManager Administrator certification validates applied knowledge of configuring, deploying, and managing Fortinet FortiManager for centralized administration of multiple FortiGate security appliances. Candidates demonstrate expertise in ADOM management, policy packages, device registration, firmware distribution, and multi-site troubleshooting. This Associate-level credential is designed for network and security analysts responsible for day-to-day FortiManager operations across enterprise environments. The FCP track launched in 2022–2023 as Fortinet's modernized certification path, replacing the legacy NSE 4/5 stack with role-specific credentials.

## Topics covered

Per official Fortinet exam blueprint:

- **Administration (15%):** User accounts, roles, permissions, system settings, backup/restore, licensing, high-availability configuration.
- **Device Manager (20%):** FortiGate device registration, script management, revision history, firmware upload and push.
- **Policy and Objects (30%):** Policy packages, ADOM structure, shared vs. dynamic objects, policy installation, workspace mode, revision control, policy importation.
- **Advanced Configuration (20%):** VPN policies, authentication schemes, traffic shaping, security profiles, multi-site deployments, FortiGuard distribution.
- **Troubleshooting (15%):** Log interpretation, connectivity diagnosis, policy enforcement verification, system performance metrics, common error resolution.
