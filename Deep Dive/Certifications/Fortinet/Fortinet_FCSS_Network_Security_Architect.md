## About

The **Fortinet Certified Solution Specialist — Network Security Architect (FCSS NSA)** validates expertise in designing, deploying, and managing enterprise-grade network security solutions using Fortinet's Security Fabric ecosystem. This certification replaces the legacy NSE 7 Network Security path (retiring July 2026) and combines knowledge of enterprise firewalls, SD-WAN, network access control, and advanced security orchestration. It is designed for network architects, senior security engineers, and solutions consultants who architect security-first infrastructure at scale. The FCSS credential structure launched in 2024 to align with modern hybrid-cloud and zero-trust security architectures.

## Topics covered

The FCSS NSA exam combines four legacy NSE 7 exam domains into a unified credential. Topics include:

**Enterprise Firewall (NSE 7 Enterprise Firewall legacy)**
- FortiGate cluster architecture, failover, and high availability (HA)
- IPSec VPN site-to-site and remote access configuration
- SSL/TLS VPN and user authentication frameworks
- FortiManager centralized management and policy deployment
- FortiAnalyzer logging, reporting, and compliance
- NGFW (Next-Generation Firewall) features: intrusion prevention, advanced threat protection, sandboxing
- Policy optimization and traffic inspection workflows

**SD-WAN (NSE 7 SD-WAN legacy)**
- FortiGate SD-WAN controller configuration and overlay deployment
- Application-aware routing and performance SLAs
- Hybrid WAN integration (MPLS, broadband, 4G/5G failover)
- ADVPN (Automatic Dynamic Mesh VPN) topology
- Link health checks and priority routing
- Traffic shaping and QoS policies for branch optimization

**Network Access Control (NSE 7 NAC legacy)**
- FortiNAC deployment and authentication (802.1X, portal-based, SSO)
- Endpoint compliance assessment and device profiling
- Quarantine and remediation workflows
- Integration with FortiGate firewalls for segmentation
- Inventory management and hardware lifecycle tracking

**Advanced Threat Detection & Response (NSE 8 Expert concepts)**
- FortiSIEM correlation and alert prioritization
- Threat intelligence feeds and IOC automation
- Incident response procedures within Security Fabric
- Behavioral analytics and anomaly detection
- Integration of sandbox, threat emulation, and reverse-proxy appliances

*Source:* [Official FCSS NSA Exam Blueprint ↗](https://training.fortinet.com/resource/fcss-nsa-exam-blueprint)
