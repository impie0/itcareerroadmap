## About

Fortinet's NSE 6 - OT Security 7.6 Architect (NSE6_OTS_AR-7.6) is a specialist-level professional certification for network and security architects responsible for securing industrial and operational technology environments. Unlike IT-only security certifications, this exam tests OT-specific concepts including industrial protocol inspection (Modbus, DNP3, BACnet), IEC/ISA 62443 zone and conduit architecture, and availability-first incident response strategies. Released as part of Fortinet's transition to the FCP (Fortinet Certified Professional) ecosystem, NSE6_OTS remains the current standard for OT security architects validating Fortinet deployments. Passing requires demonstrated competency in both Fortinet product configuration and industrial cybersecurity architecture principles.

## Domain context — OT Security / Industrial Cybersecurity

Operational Technology (OT) security focuses on protecting SCADA, ICS, and industrial networks that control physical systems (power grids, manufacturing, water treatment, chemical plants, oil & gas). Unlike IT networks, OT environments prioritize **availability first** and cannot tolerate unplanned downtime or disruptive patches. This exam validates security architects' ability to secure these mission-critical systems using Fortinet's integrated platform approach.

## Topics covered

Based on official Fortinet exam objectives, NSE6_OTS_AR-7.6 covers:

- **OT Network Architecture & Design** — Zone/conduit topology per ISA/IEC 62443; defense-in-depth for industrial networks; segmentation of critical systems
- **Industrial Protocol Security** — Modbus, DNP3, BACnet, IEC 60870-5-104 protocol inspection and filtering; anomaly detection for protocol violations
- **FortiGate OT Configuration** — Industrial protocol enablement; virtual patching for unpatched/legacy systems; policy design for OT workflows
- **Threat Detection & Response** — SCADA-specific threat indicators; operational technology asset discovery without traffic disruption; incident response with availability prioritization
- **FortiNAC Integration** — Passive asset discovery and device profiling for OT networks; NAC policy enforcement for device compliance; operational MAC filtering
- **FortiSIEM & FortiAnalyzer** — Centralized logging and forensics for OT events; alert correlation for industrial attacks; compliance reporting (IEC 62443, NERC CIP)
- **System Integration** — Multi-product orchestration (FortiGate + FortiNAC + FortiSIEM); API integration for third-party OT systems; zero-trust architecture application to industrial networks
