## About

The FCSS_SOC_AN-7.4 exam validates expertise in designing, deploying, administering, monitoring, and troubleshooting Fortinet security operations solutions using FortiAnalyzer, FortiSIEM, and FortiSOAR. Candidates demonstrate proficiency in detecting, investigating, and responding to cyber threats at a professional (NSE 6) level. The certification is part of the FCSS in Security Operations track and is intended for SOC analysts and security operations professionals seeking vendor recognition of hands-on SOC platform expertise.

---

## Domain context — Security Operations / SOC

Security Operations Centers (SOCs) manage real-time threat detection, investigation, and incident response across enterprise networks. This domain spans SIEM (Security Information and Event Management), SOAR (Security Orchestration, Automation and Response), log aggregation, threat hunting, and incident response automation. The FCSS_SOC_AN exam focuses specifically on Fortinet's SOC platform ecosystem.

## Topics covered

Based on the official FCSS Security Operations exam blueprint, the FCSS_SOC_AN-7.4 exam covers:

- **Security Operations fundamentals** — SOC architecture, roles, and SIEM/SOAR concepts
- **FortiAnalyzer administration** — Log collection, normalization, correlation, and event management
- **Threat detection and analysis** — Event classification, outbreak detection, threat intelligence integration
- **Incident response and playbooks** — Automated playbooks, event handlers, incident investigation workflows
- **FortiSOAR integration** — SOAR orchestration, incident case management, automation integration
- **Reporting and compliance** — Security reporting, compliance frameworks (PCI-DSS, NIST, ISO 27001)
- **Troubleshooting and optimization** — Log pipeline tuning, performance optimization, common issues
- **FortiAI and advanced analytics** — GenAI-assisted detection, anomaly detection, behavioral analytics

## Common skills at Security Operations · Professional

Shared competencies for security operations professionals at the professional (NSE 6 / FCSS) level:

- Configuring and maintaining centralized log collection and normalization
- Analyzing security events and identifying indicators of compromise (IOCs)
- Designing and tuning threat detection rules and correlation logic
- Responding to and remediating security incidents using standardized procedures
- Integrating and managing multiple security tools within a unified SOC platform
- Creating and documenting security runbooks and incident response procedures
- Performing threat hunting using log analysis and behavioral patterns
- Reporting security metrics, KPIs, and compliance status to stakeholders
