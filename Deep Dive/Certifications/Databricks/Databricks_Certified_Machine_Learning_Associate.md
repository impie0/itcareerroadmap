## About

The Databricks Certified Machine Learning Associate is a vendor-specific certification that validates practical ability to perform core machine learning tasks on the Databricks Lakehouse platform. Launched in 2023, this certification targets data engineers and data scientists with foundational machine learning experience who want to demonstrate competency in data preparation, feature engineering, model training, and deployment using Databricks tools including MLflow, AutoML, and Feature Store. The exam assesses hands-on proficiency in building and deploying production-grade ML workflows within the Databricks environment.

## Topics covered

- **Databricks Machine Learning (38%)** — AutoML, MLflow, Unity Catalog, Feature Store, Spark ML fundamentals
- **ML Workflows (19%)** — End-to-end ML lifecycle, experiment tracking, model registry, reproducibility
- **Model Development (31%)** — Feature engineering, data preparation, model training, hyperparameter tuning, model evaluation
- **Model Deployment (12%)** — Model serving, batch inference, monitoring, CI/CD for ML pipelines
