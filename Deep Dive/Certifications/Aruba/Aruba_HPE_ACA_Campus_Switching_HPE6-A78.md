## About

The **HPE6-A78 — HPE Aruba Networking Associate (Campus Switching)** certifies foundational knowledge of Aruba CX switching, campus network design, switching protocols, and Layer 2/3 fundamentals using Aruba AOS-CX operating system. Intended for network technicians, junior network engineers, and deployment professionals with at least one year of hands-on experience in enterprise networking environments. First offered in 2024, this cert reflects Aruba's post-Juniper acquisition strategy (July 2025) to clarify and consolidate its Associate-level credential pathway.

## Topics covered

*Based on official HPE certification blueprint (2026):*

- **ArubaOS-CX Fundamentals & Architecture** (~15%)
  - Distributed OS architecture, forwarding engine concepts
  - VLAN and trunk configuration basics
  - Port types (access, tagged, untagged)
  - Interface configuration and monitoring

- **Campus Network Design** (~20%)
  - Layer 2 switching principles and topology design
  - Redundancy and resiliency concepts (STP, RSTP, MSTP)
  - Link aggregation (LAG) and multi-link trunking
  - Power over Ethernet (PoE) and power budgeting

- **Layer 2 Switching Protocols** (~25%)
  - Spanning Tree Protocol (STP/RSTP/MSTP) configuration and troubleshooting
  - VLAN tagging (802.1Q) and VLAN management
  - MAC address learning and aging
  - Storm control and port security

- **Layer 3 Routing Essentials** (~20%)
  - IPv4 and IPv6 addressing, subnetting, and routing fundamentals
  - Static routing and dynamic routing protocol introduction (OSPF, BGP at high level)
  - Default gateway configuration
  - Route manipulation and policy-based routing basics

- **Aruba Central & Management** (~10%)
  - Cloud-managed provisioning via Aruba Central
  - Device templates and zero-touch provisioning (ZTP)
  - Dashboard navigation and monitoring
  - Role-based access control (RBAC) for Central

- **Security & Access Control** (~10%)
  - Port security and 802.1X basics
  - ACL (Access Control List) fundamentals
  - VLAN segmentation for security
  - Introduction to ClearPass integration concepts

*Source:* [HPE6-A78 Official Blueprint ↗](https://certification-learning.hpe.com/en/datacard/certification/ACA-CampusSwitch)

## Skills validated

*Specific technologies and protocols this exam actually tests:*

- **Aruba CX Switches** — Configuration, troubleshooting, and operation of CX 6300, 6400, 8000 series
- **AOS-CX operating system** — Command-line interface (CLI), configuration modes, stateful services
- **VSX (Virtual Switching Extension)** — High-availability active-active switching architecture
- **Layer 2 protocols** — RSTP, MSTP, STP convergence, rapid spanning tree timers
- **VLAN tagging** — 802.1Q, trunk configuration, VLAN membership, allowed VLANs
- **Link aggregation** — LAG (Link Aggregate Groups), LACP, static trunks, load balancing
- **Port security** — MAC address limits, learn modes, dynamic/static binding
- **802.1X authentication** — Port-based network access control (PNAC), supplicant/authenticator roles
- **Power over Ethernet (PoE)** — Power classes, inline power delivery, powered device detection
- **Access Control Lists (ACLs)** — Stateless filtering, numbered/named ACLs, filter application to ports
- **Inter-VLAN routing** — SVI (Switched Virtual Interface) configuration, default gateway setup
- **Aruba Central provisioning** — Zero-touch provisioning (ZTP), template deployment, device groups
- **IPv6 fundamentals** — Basic IPv6 addressing, link-local, multicast group membership
- **Troubleshooting tools** — `show` commands, packet capture, logging, diagnostic outputs
