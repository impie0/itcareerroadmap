## About

The Dynatrace Certified Associate is the foundational level in Dynatrace's three-tier certification hierarchy, designed for professionals with theoretical knowledge and approximately six months of hands-on experience. This certification validates understanding of the Dynatrace platform's core components, including OneAgent, ActiveGate, data collection mechanisms, monitoring capabilities, dashboarding, and foundational analysis techniques using AI-driven problem diagnostics (Davis). The exam emphasizes practical navigation of the Dynatrace user interface, awareness of monitoring scope across hosts, processes, services, and traces, and recognition of when to use specific features for troubleshooting and observability scenarios.

## Domain context — Application Performance Monitoring

Application Performance Monitoring (APM) focuses on real-time visibility into application behavior, infrastructure health, and end-user experience through automated collection, analysis, and visualization of performance data. Dynatrace is a leading vendor in full-stack observability, combining APM, infrastructure monitoring, log analytics, and AI-driven anomaly detection.

## Topics covered

The Dynatrace Associate exam covers foundational platform knowledge across six primary areas:

- **Platform Overview & Architecture** — What Dynatrace is, business/technical problems it solves, major platform components (OneAgent, ActiveGate, Grail, Smartscape), and release cycles
- **Data Collection & Monitoring** — How Dynatrace collects data, supported monitoring types (APM, infrastructure, log analytics), OneAgent/ActiveGate deployment models, and data retention policies
- **Core Concepts & Entities** — Hosts, processes, services, traces, metrics, events, topology, PurePath technology, and relationship mapping via Smartscape
- **Platform Navigation & Apps** — Navigating the Dynatrace user interface, accessing core views (Services, Hosts, Processes), using pre-built dashboards, and overview of Dynatrace App ecosystem
- **Analysis & Diagnostics** — Identifying problems, interpreting metrics and traces, understanding Davis AI recommendations, using problem-detection features, and basic troubleshooting scenarios
- **Integration & Extensibility** — OpenPipeline, custom metrics ingestion, webhook notifications, and third-party integrations overview

## Skills validated

*Specific to Dynatrace Associate exam — what this certification actually tests.*

- Installation and initialization of OneAgent on various platforms (Windows, Linux, Kubernetes)
- Configuration of ActiveGate for on-premise monitoring and secure communication
- Understanding of Smartscape topology engine and automatic service dependency discovery
- Interpretation of PurePath traces for request flow analysis
- Reading and analyzing metrics, events, and log streams in the Dynatrace UI
- Use of Davis AI-driven anomaly detection and problem diagnosis
- Navigation of Services, Hosts, Processes, and Technology Process Group views
- Configuration of alerting thresholds and notification channels
- Understanding of data retention policies and Grail data platform
- Troubleshooting scenarios involving application latency, resource bottlenecks, and infrastructure issues
