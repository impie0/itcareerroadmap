## About

The Commvault Certified Professional (CCP) is a professional-level vendor credential validating the skills required to install, configure, and administer a CommCell environment using the Commvault Command Center. Earners demonstrate proficiency in CommCell administration, storage administration, deduplication configuration, and CommCell implementation. As of January 2024, Commvault restructured its certification path to require a single Professional Foundations exam rather than multiple exams, streamlining the professional-level certification process. The credential is held by systems administrators, backup engineers, and storage professionals seeking formal validation of Commvault platform expertise.

## Domain context — Data Protection

Data Protection encompasses technologies, strategies, and practices for safeguarding critical data through backup, archiving, replication, and recovery mechanisms. This domain addresses organizational requirements for RPO (Recovery Point Objective) and RTO (Recovery Time Objective) compliance, ransomware mitigation, and business continuity across physical, virtual, and cloud environments. Commvault provides an integrated data management platform combining backup, archiving, disaster recovery, and compliance capabilities.

## Topics covered

The exam blueprint covers the following domains and topics:

- **CommCell Architecture & Components** — CommServe (command center), MediaAgent (media management), iDataAgent (client-side protection), storage policy orchestration
- **CommCell Infrastructure Setup** — Installation and configuration of CommServe, MediaAgent deployment, iDataAgent installation on diverse platforms (Windows, Linux, Unix)
- **Storage Policy & Data Path Design** — Storage policy creation, copy management, data routing, storage optimization for deduplication and tiering
- **Backup Sets & Subclients Configuration** — Subclient definition for granular backup control, backup set organization, filter policies, content exclusion
- **Auxiliary Copy & Deduplication** — Auxiliary copy workflows for backup distribution, global deduplication engine (GDE) configuration, storage pool deduplication
- **Disaster Recovery & Replication** — DR copy (backup to alternate MediaAgent), vault copy (long-term archival), replication to alternate CommCells, snapshot management
- **Application Protection** — Oracle RMAN integration, SQL Server VSS/native backup, SAP HANA backup, Microsoft Exchange agent, SharePoint protection
- **Virtual Machine Protection** — VMware vCenter integration with snapshots, Hyper-V integration, VM-level backup and recovery, snapshot management
- **Cloud Backup Integration** — AWS, Azure, GCP storage targets, object storage (S3, Azure Blob), cloud-native deduplication, archive tiering
- **Commvault Command Center Administration** — Web-based console navigation, policy management, multi-tenant administration (if applicable)
- **Monitoring, Reporting & Alerts** — Job monitoring dashboards, backup compliance reporting, alert configuration, health checks
- **Recovery Operations** — File-level recovery, system-level recovery, granular recovery (application items), live mount and instant recovery workflows
- **Troubleshooting & Performance Optimization** — Job failure diagnosis, log analysis, performance tuning, capacity planning

## Exam tips & study advice

1. **Hands-on lab practice** — Build a CommCell lab environment (trial or training account) to practice all exercises
2. **Focus on storage policy design** — This is a heavily tested area; master copy creation, deduplication, and tiering
3. **Study the architecture diagram** — CommServe, MediaAgent, and iDataAgent relationships are foundational
4. **Application agent specifics** — Understand Oracle, SQL Server, and Exchange backup workflows
5. **Review job logs** — Examine actual job output, error codes, and troubleshooting steps in test labs
6. **Disaster recovery workflows** — Practice DR copy, vault copy, and replication scenarios
7. **Time management** — 90 minutes for 65–75 questions; approximately 1–1.5 minutes per question
8. **Use Readiverse Academy** — Official labs and exam prep materials are aligned with current product version
