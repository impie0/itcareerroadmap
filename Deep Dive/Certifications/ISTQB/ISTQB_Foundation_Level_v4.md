## About

The ISTQB Certified Tester Foundation Level (CTFL) is the entry-level vendor-neutral software testing certification, recognized globally across 150+ countries. Released in v4 (2024), it validates foundational knowledge of testing principles, techniques, and tools aligned with the ISO/IEC/IEEE 29119 software testing standard. No prerequisites; open to anyone entering QA or test-focused roles. Common progression path to ISTQB Advanced Level certifications (Test Manager, Test Analyst, Technical Test Analyst, Mobile Testing, Security Testing, or Performance Testing).

## Topics covered

The CTFL v4 exam blueprint spans six major knowledge areas:

- **Fundamentals of Testing** (12%) — testing principles, terminology, defect types, test objectives
- **Testing Throughout SDLC** (15%) — testing in waterfall, iterative/agile, DevOps; test levels (unit, integration, system, acceptance)
- **Static Testing** (10%) — reviews, walkthroughs, inspection; static analysis tools
- **Test Analysis and Design** (32%) — test case design techniques (boundary value, equivalence partitioning, decision tables, state transitions), traceability
- **Managing Test Activities** (28%) — test planning, resource management, risk-based testing, defect lifecycle, test metrics and reporting
- **Test Tools** (3%) — tool classification, tool selection, test automation frameworks

*Source:* [ISTQB CTFL v4 Syllabus ↗](https://www.istqb.org/documents)

## Skills validated

*Concrete competencies this exam validates, distinct from broader QA domain skills.*

- Test case design using multiple techniques (boundary value analysis, equivalence partitioning, decision table testing)
- Black-box vs. white-box testing terminology and application
- Defect lifecycle and categorization (severity, priority, status)
- Test execution and defect reporting
- Test planning and estimation concepts
- Risk-based test prioritization
- Understanding of test levels (unit, integration, system, acceptance) and test types (functional, non-functional)
- Static testing practices (reviews, inspections, walkthroughs)
- Awareness of test automation tool categories and when to automate
- Alignment with ISO/IEC/IEEE 29119 testing standards

## Career path: CTFL → Advanced Level specialization

**Test Manager track** (leadership focus) → ISTQB Advanced Test Manager (ATM)

**Test Analyst track** (design & execution focus) → ISTQB Advanced Test Analyst (ATA)

**Technical Test Analyst track** (automation & code focus) → ISTQB Advanced Technical Test Analyst (ATTA)

**Specialty tracks:**
- Mobile testing (iOS/Android automation) → ISTQB Mobile Application Testing
- Security testing (vulnerability, penetration) → ISTQB Security Tester
- Performance & load testing → ISTQB Performance Testing

No prerequisites between specialties; choose based on career interest.
