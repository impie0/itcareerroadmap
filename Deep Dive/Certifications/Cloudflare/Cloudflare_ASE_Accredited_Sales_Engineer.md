## About

The Cloudflare Accredited Sales Engineer (ASE) certification is designed for channel partners, resellers, and presales engineers who sell and implement Cloudflare solutions. Launched as part of Cloudflare's expanded partner training program in 2024–2025, the ASE validates knowledge of Cloudflare's core services—DNS, SSL/TLS, WAF, DDoS mitigation, Workers, and Zero Trust—in a presales context. Unlike the Accredited Configuration Engineer (ACE), which focuses on implementation depth, the ASE targets sales engineers who need to present architectural options, cost-benefit analysis, and feature selection to customers. It sits at the Associate level and is a stepping stone to the Accredited Services Architect (ASA) for engineers moving into post-sales roles.

## Domain context — Security & CDN

Security and CDN (Content Delivery Network) span edge network protection, application-layer security, and global content distribution. Cloudflare operates at the edge, providing DNS resolution, DDoS protection, Web Application Firewall (WAF), bot mitigation, SSL/TLS termination, and caching. This domain overlaps with both cloud security (protecting web applications and APIs) and networking (DNS, routing, performance optimization).

## Topics covered

Cloudflare's ASE curriculum is delivered in modular format. While the vendor does not publish a formal exam blueprint, the training modules cover:

- **Introduction to Cloudflare** — Platform overview, history, edge network architecture
- **DNS and Nameserver Management** — DNS fundamentals, zone setup, record types, Cloudflare DNS routing
- **SSL/TLS Certificates and HTTPS** — Certificate management, TLS versions, HTTPS enforcement, certificate transparency
- **Web Application Firewall (WAF)** — Rule sets, managed rulesets, custom rules, bot management
- **DDoS Protection** — Attack types, mitigation strategies, advanced DDoS protection
- **Caching and Performance** — Page Rules, Cache Rules, performance optimization, analytics
- **Cloudflare Workers and Serverless** — Serverless functions, API integration, Worker deployment
- **Cloudflare One and Zero Trust** — Identity-based access, Secure Web Gateway, Gateway policies
- **Sales Engineering Approach** — Presales methodology, ROI calculation, competitive positioning
