# Cloudflare Accredited Configuration Engineer

**`ACE`** · ● **Active** · **Professional** · _Cloudflare_

> Cloudflare's Accredited Configuration Engineer (ACE) is a hands-on certification for implementation engineers, systems engineers, and technical partners responsible for deploying, configuring, and optimizing Cloudflare services. The ACE validates practical expertise in implementing Cloudflare's edge network, DNS, SSL/TLS, WAF, DDoS mitigation, caching, and operational troubleshooting. Launched in 2024–2025 as part of Cloudflare's expanded partner training program, ACE is the technical implementation counterpart to the Accredited Sales Engineer (ASE) and the prerequisite foundation for the Accredited Services Architect (ASA).

## About

The Cloudflare Accredited Configuration Engineer (ACE) is a hands-on implementation certification designed for technical professionals—implementation engineers, solutions engineers, systems administrators, and resellers—who deploy and configure Cloudflare services in production environments. Unlike the sales-focused Accredited Sales Engineer (ASE), ACE emphasizes practical configuration, operational best practices, and troubleshooting. The certification consists of three modules delivered over instructor-led or self-paced training: Implementing Cloudflare (core architecture, onboarding, user administration, DNS and SSL setup), Optimizing Cloudflare (performance tuning, caching strategies, advanced routing, analytics), and Troubleshooting Cloudflare (diagnostics, debugging, incident response, edge cases). Valid for 2 years, the ACE serves as a foundational credential for implementation teams and can lead to the advanced Accredited Services Architect (ASA) for architects and pre-sales engineers.

## Domain context — Security & CDN

Security & CDN spans edge network protection, application-layer security, and global content delivery. Cloudflare operates at the network edge, providing DNS resolution, DDoS protection, Web Application Firewall (WAF), bot mitigation, SSL/TLS termination, intelligent caching, and Zero Trust access control. The ACE focuses on the technical implementation side of these capabilities—how to configure DNS failover, enable HTTPS, deploy WAF rules, optimize cache performance, and troubleshoot real-world issues. This domain overlaps with cloud security (application and API protection), networking (DNS, routing, BGP), and systems engineering (operational excellence).

## Topics covered

Cloudflare's ACE curriculum is delivered as three hands-on modules. The vendor does not publish a formal exam blueprint, but training materials reveal the following scope:

- **Module 1: Implementing Cloudflare** (30% of assessment weight)
  - Platform overview, edge network architecture, Cloudflare's global footprint
  - Account setup, user administration, role-based access control
  - DNS configuration: zone setup, record types, CNAME flattening, DNS failover
  - SSL/TLS certificate deployment, certificate types (Universal, Advanced, Custom), HTTPS enforcement
  - Initial onboarding and integration workflows

- **Module 2: Optimizing Cloudflare** (40% of assessment weight)
  - Cache Rules vs. Page Rules: decision trees and configuration
  - Cache behavior: TTL settings, cache purging, cache variants
  - Performance optimization: compression, image optimization, prefetching
  - WAF rules: managed rulesets, custom rules, rule priorities, sensitivity tuning
  - Cloudflare Workers basics: serverless functions, routes, deployment
  - Rate limiting and DDoS protection configuration
  - Cloudflare One (Zero Trust): Access policies, Gateway rules, Secure DNS
  - Analytics, logging, and real-time monitoring

- **Module 3: Troubleshooting Cloudflare** (30% of assessment weight)
  - Diagnostic workflows and common issues
  - DNS troubleshooting: resolution, delegation, propagation
  - SSL/TLS debugging: certificate errors, cipher negotiation, mixed content
  - Cache behavior troubleshooting: unexpected cache misses, stale content
  - WAF and security rule impact analysis
  - Performance and latency investigation
  - Log analysis and SIEM integration
  - Incident response and edge cases

## Skills validated

- Cloudflare zone setup, DNS records (A, AAAA, CNAME, MX, SRV, TXT), and CNAME flattening
- SSL/TLS certificate provisioning (Universal, Advanced, Custom), renewal, and HTTPS enforcement
