## About

The Securing Email with Cisco Email Security Appliance (SESA 300-720) exam validates expertise in administering, configuring, and troubleshooting Cisco Secure Email Gateway (formerly known as Cisco Email Security Appliance). This 90-minute professional-level concentration exam tests knowledge of email security administration, spam and malware filtering, encryption, data loss prevention, directory integration, and message routing. Passing this exam earns the Cisco Certified Specialist - Email Content Security credential and satisfies one of the concentration exam requirements for the Cisco Certified Network Professional (CCNP) Security certification.

---

## Domain context — Security

Email represents one of the most prevalent attack vectors in enterprise environments. Organizations face threats from phishing, malware distribution, data exfiltration, and business email compromise (BEC). The SESA 300-720 certification addresses the critical need for security professionals who can design, deploy, and maintain email security solutions that protect organizational assets while maintaining business continuity. This concentration is essential for security architects and administrators responsible for securing corporate email infrastructure against evolving threats.

## Topics covered

The 300-720 SESA v1.1 exam blueprint evaluates candidates across six primary knowledge domains:

| Domain | Weight | Topics |
|--------|--------|--------|
| **Cisco Email Security Appliance Administration** | 15% | System setup, initial configuration, administrative interfaces, policy configuration, operational management, appliance health and performance monitoring |
| **Spam Control with Talos SenderBase and Antispam** | 15% | Anti-spam policy configuration, sender reputation systems (Talos), detection algorithms, spam scoring, whitelisting and blacklisting strategies |
| **Encryption and Secure Email Transport** | 15% | Email encryption mechanisms, TLS/SMTPS configuration, encrypted connection setup, certificate management, secure transport protocols |
| **Directory Integration and Authentication** | 15% | LDAP integration, directory services configuration, user authentication methods, directory synchronization, user management |
| **Email Threat Protection and Content Filtering** | 15% | Antivirus and anti-malware filtering, URL reputation filtering, advanced malware detection, content filtering policies, attachment handling |
| **Quarantine Management and Message Delivery** | 10% | Quarantine management, message routing, delivery policies, system quarantine configuration, non-delivery reports (NDRs) |

**Key competency areas:**
- Email security architecture and Secure Email Gateway operation
- Threat prevention (malware, spam, phishing)
- Data loss prevention (DLP) and content inspection
- Email encryption and secure communications
- Compliance and regulatory requirements (GDPR, HIPAA, etc.)
- System administration and troubleshooting

## Common job-ready skills

Passing the 300-720 SESA exam demonstrates readiness for:

- **Email Security Administration:** Configure and manage Cisco Secure Email Gateway appliances for enterprise deployments
- **Threat Detection and Prevention:** Implement and tune anti-spam, antivirus, and anti-malware policies
- **Encryption Management:** Deploy email encryption and establish secure transport channels (TLS)
- **DLP Implementation:** Configure data loss prevention policies to prevent unauthorized information disclosure
- **Directory Integration:** Integrate email security systems with LDAP/Active Directory for authentication and user management
- **Policy Optimization:** Develop and maintain email security policies balancing protection with user productivity
- **Troubleshooting:** Diagnose and resolve email delivery issues, quarantine management, and system performance problems
- **Compliance Monitoring:** Ensure email security compliance with regulatory requirements and organizational standards
- **Log Analysis:** Review and analyze email security logs for threat investigation and forensics
