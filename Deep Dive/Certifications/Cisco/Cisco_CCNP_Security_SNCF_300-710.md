## About

The **300-710 SNCF** (Securing Networks with Cisco Firepower) is a concentration exam for the Cisco Certified Network Professional (CCNP) Security certification. Candidates must hold or earn CCNP Security core (350-701 SCOR) and pass one concentration exam to earn the full CCNP Security designation. SNCF focuses exclusively on Cisco Firepower NGFW (Next-Generation Firewall) architecture, policies, and operations. The exam covers Firepower Threat Defense (FTD) in multiple deployment modes (routed, transparent, inline), Firepower Management Center administration, Access Control Policy design, IPS tuning, SSL/TLS decryption, VPN configuration, high-availability clustering, and troubleshooting via CLI and FMC UI. This cert is ideal for security engineers working with Cisco firewalls in enterprise or cloud environments.

## Topics covered

Based on official Cisco exam blueprint:

- **Cisco Firepower NGFW Architecture & Licensing** (10%)
  - Firepower Threat Defense (FTD) architecture and components
  - NGFW licensing and feature enablement
  - Management Center (FMC) architecture and high availability

- **Access Control Policy (ACP) Framework** (20%)
  - ACP rule design, rule order, and evaluation
  - Security zones (trust, allow, block, interactive, logging)
  - Rule actions and conditions (allow, block, monitor, interactive block)
  - URL filtering and application filtering rules

- **Intrusion Prevention System (IPS)** (15%)
  - Snort rules and rule sets
  - IPS policy configuration and deployment
  - Inline and monitoring modes
  - Rule tuning and alert suppression

- **SSL/TLS Decryption Policy** (10%)
  - SSL inspection architecture
  - Certificate handling and trust stores
  - Decryption policies and exceptions
  - Performance impact and best practices

- **Firepower Management Center (FMC) Operations** (10%)
  - FMC interface and navigation
  - Device registration and management
  - Policy deployment and updates
  - Dashboards, reports, and health monitoring

- **Threat Defense (FTD) Deployment Modes** (10%)
  - Routed, transparent, inline, ERSPAN modes
  - Mode selection and transition
  - Interface configuration and data flow

- **Network Address Translation (NAT) on FTD** (8%)
  - Static, dynamic, and dynamic PAT
  - Policy-based NAT
  - NAT policy design and troubleshooting

- **VPN on Firepower** (8%)
  - Site-to-site IPsec VPN configuration
  - Remote-access VPN (AnyConnect, clientless)
  - VPN policy and encryption domain design

- **High Availability & Clustering** (5%)
  - Active/passive failover
  - Active/active clustering
  - Health monitoring and failover triggers

- **Security Intelligence, File Inspection, and AMP** (2%)
  - IP/URL/DNS reputation feeds
  - File inspection policies
  - Advanced Malware Protection (AMP) integration

- **Troubleshooting Firepower** (2%)
  - CLI diagnostic show commands
  - Packet capture and analysis
  - Common misconfigurations and resolution

*Source:* [Cisco Learning Network SNCF Exam Blueprint ↗](https://learningnetwork.cisco.com/s/sncf-exam-topics)
