## About

The Cisco Certified DevNet Specialist - Security Automation and Programmability (SAUTO 300-735) is a specialized certification launched in 2019 that validates hands-on expertise in implementing security automated solutions using Cisco platforms. The exam tests candidates on network programmability foundations (10% weight), network security automation (35%), advanced threat and endpoint security (30%), and cloud, web, and email security APIs (25%). Released in v1.1 as of 2024, it serves as a concentration exam for both the CCNP Security and Cisco Certified DevNet Professional career tracks, targeting security engineers and network programmers who work at the intersection of DevOps culture and security operations.

## Topics covered

Exam blueprint topics and approximate weights (per official exam guide):

- **Network Programmability Foundation (10%):** Programming basics (Python, REST), data models (YANG), protocols (NETCONF, RESTCONF), APIs, authentication/authorization, JSON/XML
- **Network Security (35%):** Firewall automation (ASA, Firepower), intrusion prevention (Snort), configuration management, device hardening, access control policy automation
- **Advanced Threat & Endpoint Security (30%):** Cisco Secure Endpoint (AMP) API, Cisco Threat Grid, threat hunting automation, endpoint detection & response (EDR) integration, incident response workflows
- **Cloud, Web & Email Security (25%):** Cisco Umbrella DNS filtering API, web proxy (WSA) automation, email security (ESA), Secure Cloud Access architecture, SecureX/XDR platform orchestration

*Source:* [Official Exam Topics v1.1 ↗](https://learningcontent.cisco.com/documents/marketing/exam-topics/300-735-SAUTO-v1.1.pdf)

## Skills validated

*This exam tests hands-on, verifiable competencies distinct from general security knowledge.*

- Cisco ASA / Firepower configuration via REST API
- Cisco Identity Services Engine (ISE) API integration and policy automation
- Cisco Secure Endpoint (AMP) threat intelligence API usage
- Cisco Umbrella DNS API for dynamic policy and reporting
- Cisco SecureX / XDR API for threat correlation and orchestration
- RESTful API design patterns (OAuth, API keys, rate limiting)
- Python scripting for security device automation (requests library, JSON parsing)
- Webhook and syslog event automation for SOC alerting
- Container security basics (Cisco Secure Container Image Scanning)
- Threat hunting and incident response workflow automation
