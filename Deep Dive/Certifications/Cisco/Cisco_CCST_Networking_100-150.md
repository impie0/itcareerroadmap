## About

Cisco Certified Support Technician Networking (CCST Networking, exam code 100-150) is Cisco's entry-level networking certification, launched in December 2023. It validates foundational skills required for help desk technicians, network support specialists, junior network operations center (NOC) analysts, and junior network engineers working in production networks.

**Market positioning:** CCST Networking sits deliberately between CompTIA Network+ (vendor-neutral, broader IT context) and Cisco CCNA 200-301 (advanced, requires deeper lab experience). Unlike CCNA, which typically requires 200–300 hours of study, CCST demands 150–200 hours, making it accessible to career starters with minimal networking background. The certification directly maps to real job roles: most NOC Tier 1 technician and junior network support roles list CCST or equivalent as preferred.

**Practical focus:** The exam emphasizes real-world troubleshooting, command-line interface (CLI) operations, physical-layer diagnostics, and hands-on configuration of Cisco switches and routers. Performance-based lab questions simulate actual technician tasks: connecting a cable correctly, interpreting show command output, identifying a VLAN misconfiguration, or troubleshooting ping failures. This is not theoretical; CCST tests what you'll do on day one as a network support tech.

**Career trajectory:** CCST Networking is explicitly designed as the on-ramp to Cisco's professional track. After CCST + 3–6 months production experience, candidates pursue CCNA 200-301, then CCNP Enterprise within 1–3 years. Cisco actively markets CCST as a feeder program to CCNA, with Networking Academy integration and direct pathways in job postings.

## Topics covered

The CCST Networking 100-150 exam blueprint covers practical network support skills organized around six functional domains:

- **Standards and Concepts (15–20%)** — OSI model (all 7 layers), TCP/IP stack, encapsulation, protocols (TCP, UDP, ICMP), port numbers, MAC vs. IP addressing, unicast/multicast/broadcast
- **Addressing and Subnet Formats (15–20%)** — IPv4 and IPv6 addressing, CIDR notation, subnetting fundamentals, address classes (legacy), DHCP operation and troubleshooting, NAT basics, private vs. public addressing
- **Endpoints and Media Types (15–20%)** — Ethernet standards (802.3), cabling types (Cat5e, Cat6, Cat6A, single-mode fiber, multimode fiber), connector types (RJ45, fiber optics), wireless fundamentals (802.11a/b/g/n/ac/ax), media testing, cable termination
- **Infrastructure Components (20–25%)** — Cisco device types (switches, routers, firewalls), interface types and status, VLAN creation and trunking, Spanning-Tree Protocol (STP/RSTP) concepts, static routing, basic dynamic routing (OSPF, EIGRP overview)
- **Diagnostics and Troubleshooting (15–20%)** — Systematic troubleshooting methodology, ping/traceroute/ipconfig/netstat commands, show commands output interpretation, LED status lights, packet flow analysis, common network problems (wrong VLAN, duplex mismatch, bad cable)
- **Security and Network Operations (10–15%)** — Access Control Lists (standard and extended), firewall basics, VPN concepts, device access (SSH vs. Telnet), physical security, CDP/LLDP protocols, syslog basics

**Exam format note:** The 75 questions include scenario-based labs and drag-and-drop tasks. You may be asked to configure a VLAN, troubleshoot ping failures, identify correct cabling, or interpret show command output in real time.

## Skills validated

*Concrete technologies and tools tested on the CCST Networking 100-150 exam — what you must demonstrate hands-on.*

**Core CLI and Device Management**
- Cisco IOS command-line interface (CLI): navigation, configuration mode vs. user mode, `enable`, `configure terminal`
- Show commands: `show ip interface`, `show vlans`, `show running-config`, `show version`, `show status`
- Device access via console cable (RJ45 serial), SSH (encrypted), Telnet (insecure)
- Device modes: User EXEC mode, Privileged EXEC mode, Global Configuration mode, Interface Configuration mode

**Routing and Switching**
- VLAN creation (`vlan 10`), tagging, trunk ports (`switchport mode trunk`)
- Inter-VLAN routing via static routes or router-on-a-stick
- Static routing configuration on Cisco routers (`ip route 10.0.0.0 255.255.255.0 192.168.1.1`)
- Spanning-Tree Protocol (STP) basic operation (BPDU, election, ports roles)
- OSPF and EIGRP: overview only; not deep configuration

**IP Addressing and Subnetting**
- IPv4 addresses, classes, CIDR notation, subnet masks, network/host calculation
- IPv6 addresses, hexadecimal, IPv6 subnetting basics
- DHCP scopes, leases, address pools, exclusion ranges
- DNS resolution (A records, reverse DNS), nslookup, host troubleshooting

**Security and Access Control**
- Access Control Lists (ACLs): standard vs. extended, permit/deny rules, implicit deny
- Firewall basics: stateful inspection, access control, port translation
- VPN fundamentals: site-to-site, remote access, encryption overview
- Port security on switches (MAC address limits)

**Diagnostics and Troubleshooting**
- Ping (ICMP echo), traceroute (TTL decrement), IPv4 vs. IPv6 variants
- `ipconfig` / `ifconfig` (Windows/Linux), address configuration, lease renewal
- `netstat`, `arp -a`, `show mac-address-table`
- Packet capture concepts (Wireshark introduction)
- Systematic troubleshooting: OSI layer-by-layer methodology
- Common issues: mismatched VLAN, duplex mismatch, wrong subnet mask, down interfaces

**Physical Layer and Cabling**
- Ethernet standards: 10 Mbps, 100 Mbps (Fast Ethernet), 1 Gbps (Gigabit), 10 Gbps
- Cable types: unshielded twisted-pair (UTP) Cat5e, Cat6, Cat6A; fiber (single-mode, multimode)
- RJ45 connector pinout and crimping (568A, 568B standards)
- Fiber optic connectors: SC, LC, MTP
- Cable testing: continuity, length, signal loss
- Interface LED indicators (link up/down, activity, speed)

**Discovery and Monitoring**
- Cisco Discovery Protocol (CDP): device discovery, neighbor information
- Link Layer Discovery Protocol (LLDP): cross-vendor neighbor discovery
- Syslog basics: log monitoring, severity levels
- SNMP overview: monitoring network devices

## Typical study timeline

**Full-time study (career change scenario):**
- Weeks 1–4: Network fundamentals, OSI model, TCP/IP, IP addressing (Networking Academy course)
- Weeks 5–7: Cisco IOS CLI, VLAN configuration, switching basics (hands-on labs)
- Weeks 8–10: Routing, troubleshooting, practice exams (Boson or Cisco Press companion)
- Week 11: Review weak areas, take final practice tests
- Week 12: Exam day

**Part-time study (working technician, 5–8 hrs/week):**
- Months 1–2: Fundamentals (YouTube + Networking Academy modules)
- Months 2–3: Hands-on labs (GNS3 or real Cisco equipment)
- Months 3–4: Practice exams, review
- Month 5: Exam

**Prerequisite knowledge:** CompTIA A+ or Network+ significantly accelerates this timeline (can reduce by 2–4 weeks). Candidates with zero networking background should plan 4–6 months.

## Career progression (CCST → CCNA → CCNP)

**Typical timeline from CCST Networking to CCNP Enterprise (5–7 years):**

| Milestone | Cert | Timeline | Salary (USD) | Prerequisites |
|---|---|---|---|---|
| **Entry support** | CCST Networking (100-150) | 1–3 months study | $40k–$55k | None |
| **Junior engineer** | CCNA 200-301 | 3–6 months after CCST | $65k–$85k | CCST or equivalent |
| **Network engineer** | CCNP Enterprise Core (ENCOR 350-401) | 6–12 months on the job | $95k–$130k | CCNA + 3+ yrs production |
| **Senior engineer** | CCNP Enterprise Specialist (ENARSI, ENSDWI, or ENSLD) | 12–18 months | $110k–$145k | ENCOR passed + 4+ yrs |
| **Architect/senior specialist** | CCIE Enterprise (written + lab) | 24+ months intensive study | $140k–$200k+ | CCNP + 6+ yrs; $4k–$8k exam costs |

**Fast-track option:** Candidates with strong network operations background (e.g., NOC technicians with 2+ years hands-on experience) can sometimes skip CCST and pursue CCNA directly (6–9 months vs. 3 months), reducing total time to junior engineer role.

**Industry context:** Cisco CCST Networking has become the default on-ramp into enterprise networking roles since its launch in late 2023. Most corporate recruiting pipelines now see CCST → CCNA → CCNP as the standard progression, with CCNA no longer viewed as an entry cert but rather a mid-career consolidation cert. CCST adoption is accelerating globally, particularly in MSPs, ISPs, and Fortune 500 NOCs.
