## About

The CCNP Service Provider SPRI (300-510) concentration exam is one of four electives available to candidates pursuing the full CCNP SP professional certification. SPRI focuses on advanced routing technologies deployed in service provider core networks, including multi-area OSPF, IS-IS, advanced BGP (path selection, route reflection, confederation), MPLS TE with FRR, Segment Routing, and multicast routing in carrier environments. Passing SPRI, combined with the mandatory 350-501 SPCOR core exam, grants the "Cisco Certified Network Professional Service Provider" credential.

## Domain context — Networking

Service Provider Networking (ISP/Carrier backbone operations) is a specialized branch of enterprise networking focused on high-scale, mission-critical infrastructure: MPLS core networks, BGP routing at internet scale, VPN services, QoS, and automation of carrier-grade systems. CCNP SP SPRI validates expertise in the routing layer of these architectures.

## Topics covered

The 300-510 SPRI exam blueprint covers advanced routing technologies in service provider networks:

- **Advanced OSPF** — Multi-area design, OSPFv3, OSPF traffic engineering (TE), stub areas, super-backbone
- **IS-IS** — Level 1/Level 2 (L1/L2) design, ISIS-TE, wide metrics, adjacency optimization
- **Advanced BGP** — Route reflection, confederation, communities, path selection, multipath, load balancing, BGP security
- **MPLS Traffic Engineering (MPLS TE)** — RSVP-TE signaling, Explicit Routes (EROs), Auto-route, Fast Reroute (FRR) mechanisms
- **Segment Routing** — SR-MPLS architecture, SRv6 fundamentals, segment lists, steering policies
- **Multicast in Service Provider Networks** — PIM modes (Dense Mode, Sparse Mode), mLDP, Multicast VPN (mVPN), SSM
- **BGP-Free Core Architecture** — Simplification of core networks using tunneling and segment routing
- **Inter-AS VPN** — Option A, B, and C; ASBRs; inter-domain VPN scaling
- **BGP Confederation** — Confederation structure, confederation members, scaling BGP in service provider cores

## Skills validated

*Cert-specific — what this exam actually tests.*

- Advanced OSPF design and optimization in multi-area service provider networks
- OSPFv3 deployment for IPv6 in carrier environments
- OSPF Traffic Engineering (TE) for dynamic path optimization
- IS-IS Level 1/Level 2 hierarchical design and ISIS-TE configuration
- Advanced BGP: route reflection, confederation, path selection, load balancing, community-based policy
- BGP multipath and equal-cost load distribution in core networks
- MPLS Traffic Engineering (MPLS TE) with RSVP-TE signaling
- Explicit Routing Objects (EROs) and constraint-based routing
- Fast Reroute (FRR) for sub-50ms failover in backbone networks
- Segment Routing (SR-MPLS) deployment and simplification
- SRv6 (Segment Routing over IPv6) fundamentals
- Multicast routing: PIM modes, mLDP, Multicast VPN (mVPN)
- BGP-free core architecture principles and implementation
- Inter-AS VPN options (Option A, B, C) and ASBR design
- BGP confederation for scale in large service provider networks
- Troubleshooting routing convergence and failover in carrier environments
