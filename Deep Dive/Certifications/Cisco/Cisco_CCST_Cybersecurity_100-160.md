## About

**CCST Cybersecurity** (100-160) is Cisco's flagship entry-level cybersecurity certification, designed to validate foundational security knowledge for aspiring support technicians, SOC analysts, help desk professionals, and junior security operations roles. Launched in March 2023 as part of the Cisco Certified Support Technician (CCST) program—a trilogy alongside CCST IT Support (100-140) and CCST Networking (100-150)—this exam bridges formal cybersecurity education and hands-on operational security work. 

The credential validates ability to identify security vulnerabilities, explain the CIA triad and defense-in-depth principles, recognize common threats (malware, ransomware, social engineering), configure basic endpoint hardening, perform vulnerability assessment interpretation, and execute foundational incident response procedures. It assumes no prerequisites—entry-level IT professionals, help desk staff, career changers, and bootcamp graduates are equally eligible. Valid for 3 years; renewal via retake or progression to CyberOps Associate (200-201) or CCNA Cybersecurity. The exam typically takes 50–70 minutes, contains 40–50 questions, and uses scenario-based and simulation-style formats to assess practical security knowledge.

## Topics covered

The exam validates essential cybersecurity concepts across five primary domains with approximate weighting:

- **Essential Security Principles (20%)** — CIA triad (confidentiality, integrity, availability), defense-in-depth architecture, security controls (preventive, detective, corrective, compensating), threat classification, attack vectors, vulnerability vs. risk, types of attackers, motivations for attacks, professional ethics and responsible disclosure
- **Basic Network Security Concepts (25%)** — Firewalls and stateful inspection, VPNs and tunneling, intrusion prevention systems (IPS) vs. intrusion detection systems (IDS), network segmentation (VLAN, DMZ), secure communication protocols, symmetric and asymmetric encryption, public key infrastructure (PKI), digital certificates, hashing algorithms, SSL/TLS handshake basics
- **Endpoint Security Concepts (20%)** — Antivirus and antimalware technologies, endpoint detection and response (EDR) tools, device hardening practices (OS patching, disabling unnecessary services), disk encryption (BitLocker, FileVault), access control lists (ACLs), mobile device management (MDM), biometric authentication
- **Vulnerability Assessment and Risk Management (20%)** — Identifying and classifying vulnerabilities, CVSS score interpretation, vulnerability scanning tools, risk assessment methodologies (qualitative vs. quantitative), risk prioritization and remediation strategies, risk mitigation vs. risk acceptance, business impact analysis
- **Incident Handling (15%)** — Incident response lifecycle phases (identification, containment, eradication, recovery, post-incident analysis), evidence preservation and chain of custody, roles in incident response (responders, managers, stakeholders), escalation procedures, playbook execution, post-mortem documentation

## Skills validated

*Cert-specific — what the 100-160 exam tests, distinct from shared entry-level skills.*

- **Threat identification and classification:** Recognizing common attack patterns (malware, ransomware, worms, trojans, botnets), social engineering tactics (phishing, spear-phishing, vishing, smishing, tailgating), denial-of-service (DoS/DDoS) attacks, zero-day vulnerabilities
- **Network defense operations:** Firewall stateful inspection and rules, VPN tunnel configuration and encryption, IPS/IDS signature detection vs. anomaly-based detection, alert triage and false-positive filtering, network segmentation architecture (VLAN, DMZ, subnet isolation)
- **Endpoint hardening practices:** OS patching and update procedures, antivirus/antimalware tool operation, endpoint detection and response (EDR) dashboard interpretation, disk encryption (BitLocker, FileVault) verification, access control (local admin removal, principle of least privilege)
- **Vulnerability assessment and interpretation:** Scanning tool output analysis, CVSS score understanding (severity ratings), remediation prioritization based on business impact, risk register documentation, evidence collection for audit trails
- **Incident response execution:** Alert triage severity assessment, containment procedures (isolating affected systems, revoking credentials), evidence preservation (chain of custody, volatile data capture), escalation decision-making, post-incident documentation and root cause analysis
- **Compliance and governance awareness:** Recognizing regulatory drivers (GDPR, HIPAA, PCI-DSS basics), audit log review, security policy adherence verification, incident reporting requirements
- **Cryptography application:** Symmetric vs. asymmetric encryption use cases, hashing algorithms (SHA-256, MD5), SSL/TLS handshake phases, digital certificate validation, public key infrastructure (PKI) roles

## Study approach (8–12 week timeline)

**Week 1–2: Foundations**
- Register for official Cisco Networking Academy Cybersecurity Essentials course.
- Review exam blueprint from Cisco Learning Network; identify knowledge gaps.
- Skim Cisco Press Official Cert Guide chapters 1–3 (security principles, threat landscape, CIA triad).
- Take free Cisco Learning Network baseline quiz; assess weak domains.

**Week 3–5: Core content mastery**
- Complete Cisco Networking Academy modules 1–4 (security concepts, network fundamentals).
- Watch CBT Nuggets or professor Messer videos on CIA triad, encryption, firewalls, IPS.
- Read Cisco Press chapters 4–8 (network security, endpoints, cryptography, PKI).
- Lab: Configure basic firewall rules (GNS3), test VPN connections, analyze traffic with Wireshark.

**Week 6–8: Application and deep-dive**
- Complete Cisco Networking Academy modules 5–8 (vulnerability assessment, incident response).
- Work through practice questions on Cisco Learning Network; target 75%+ accuracy.
- Read Cisco Press chapters 9–12 (vulnerability management, incident response procedures).
- Lab: Interpret CVSS scores, practice incident response playbook execution, document chain of custody.

**Week 9–10: Practice exams and review**
- Take first full MeasureUp or Cisco Press practice exam (target 70%+).
- Review incorrect answers; re-read corresponding Cisco Press chapters.
- Take second full-length practice exam one week later (target 75%+).
- Flashcard review: key definitions, tool names, incident response steps, cryptographic concepts.

**Week 11: Final prep**
- Review exam blueprint one final time; ensure all five domains covered.
- Take any remaining practice quizzes on Cisco Learning Network.
- Re-do one lab scenario (vulnerability assessment or incident response simulation).
- Get 8+ hours sleep; review exam policies.

**Week 12: Exam day**
- Arrive early (15 min before scheduled time).
- Carefully read all questions; flag uncertain items for final review.
- Manage time: 50 minutes for 40–50 questions ≈ 1 minute per question.

**Study time estimate:** 80–120 hours (coursework, reading, labs, practice exams). Realistic timeline for working professionals: 10–14 weeks part-time (6–8 hrs/week).
