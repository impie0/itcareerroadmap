## About

The **300-715 SISE** (Implementing and Configuring Cisco Identity Services Engine) is a concentration exam required to earn Cisco Certified Network Professional (CCNP) Security certification alongside the mandatory 350-701 SCOR core exam. Launched in 2020, it tests hands-on configuration and troubleshooting of Cisco ISE, the market-leading network access control and identity platform. Candidates must demonstrate expertise in ISE deployment architectures, 802.1X authentication flows, certificate management, policy frameworks, guest workflows, BYOD integration, TrustSec Security Group Tags (SGT), and integration with Cisco Software-Defined Access (SDA). This exam is ideal for identity and access management engineers, network access control specialists, and security architects implementing Zero Trust network strategies.

## Domain context — Security

Identity Services Engine (ISE) is Cisco's core platform for network access control (NAC), policy enforcement, and identity management in enterprise networks. Covers wired/wireless authentication, guest lifecycle, BYOD workflows, and TrustSec-based segmentation.

## Topics covered

Based on official Cisco exam blueprint:

- **ISE Architecture & Deployment** (20%)
  - Policy Service Nodes (PSN), Admin Node, Monitoring Node
  - High availability and deployment models
  - pxGrid architecture and Cisco ecosystem integrations
  - ISE personas (Admin, Policy Service, Monitoring)

- **802.1X Authentication & Protocols** (25%)
  - EAP-TLS, PEAP, EAP-FAST, EAP-MD5 configuration
  - Certificate-based authentication and PKI integration
  - Authentication policy creation and enforcement
  - Wired and wireless 802.1X deployment

- **Policy Sets & Authorization** (20%)
  - Authentication and authorization policies
  - Network Device Groups and Conditions
  - Policy results and profiling integrations
  - Enforcement actions (CoA, quarantine)

- **Guest Management & BYOD** (15%)
  - Guest portal and sponsor portal configuration
  - Guest lifecycle workflows (creation, notification, expiry)
  - BYOD device registration and certificate provisioning
  - Personal Device Management (PDM)

- **Device Profiling & Posture Assessment** (10%)
  - Network device profiling rules and conditions
  - Client posture assessment and remediation
  - Integration with Third-party tools

- **TrustSec & SDA Integration** (10%)
  - Security Group Tags (SGT) and Scalable Group Policies (SGP)
  - Cisco Software-Defined Access (SDA) workflows
  - TrustSec enforcement and monitoring
