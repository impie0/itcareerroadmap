## About
The 300-740 SCAZT exam certifies a candidate's knowledge of designing and implementing cloud security architectures that protect users, devices, and endpoints. The exam emphasizes Zero Trust Network Access (ZTNA), Security Service Edge (SSE), identity-based access control, and cloud-delivered security services. This concentration exam is part of the CCNP Security certification track and requires no formal prerequisites, making it accessible to IT professionals at various experience levels.

The exam represents a strategic shift from theoretical security knowledge to practical implementation of Zero Trust frameworks and cloud security services, particularly using Cisco solutions like Secure Access Service Edge (SASE), Duo for MFA, and cloud-based firewall capabilities.

## Domain context
**Cloud Security & Zero Trust Architecture**

The 300-740 SCAZT exam is positioned at the intersection of cloud adoption, remote work expansion, and Zero Trust security paradigms. Organizations are moving away from perimeter-based security to identity-centric, device-aware security models. This exam validates the ability to:

- Design and implement cloud-native security architectures
- Apply Zero Trust principles to user and device access
- Integrate identity, device posture, and application security
- Implement SASE (Secure Access Service Edge) components
- Enable secure cloud application access
- Provide visibility and threat response in distributed environments

The exam aligns with industry frameworks including NIST Cybersecurity Framework, CISA guidelines, and MITRE ATT&CK threat intelligence.

## Topics covered

### 1.0 Cloud Security Architecture (10%)
- Components of the Cisco Security Reference Architecture (threat intelligence, security operations, user/device security, network security, workload/application/data security)
- Use cases and capabilities within integrated architectures (common identity, converged multicloud policy, SASE integrations, zero-trust network access)
- Industry security frameworks (NIST, CISA, DISA)

### 2.0 User and Device Security (20%)
- User and device authentication via identity certificates
- Multifactor authentication (MFA) for users and devices via Cisco Duo
- Endpoint posture policies (device compliance, security posture assessment)
- Device onboarding and identity-based access control

### 3.0 Network and Cloud Security (20%)
- Security policies for endpoints accessing SaaS applications (Office 365, Workday, Salesforce)
- VPN and application-based remote access policies
- Security services edge (SSE) enforcement
- Cisco Secure Firewall policy and application filtering
- Zero Trust Network Access (ZTNA) implementation

### 4.0 Application and Data Security (25%)
- MITRE ATT&CK framework and attacker defense mitigation techniques
- Cloud security attack tactics and mitigation strategies
- Web application firewall (WAF) and DDoS attack protection
- Cisco Secure Workload for lateral movement prevention
- Microsegmentation and workload isolation
- Application policy enforcement and anomaly detection

### 5.0 Visibility and Assurance (15%)
- Cisco XDR (Extended Detection & Response) solution
- Visibility and assurance automation use cases
- SIEM, OpenTelemetry, and Cisco Secure Network Analytics
- Traffic flow validation and telemetry reporting
- Baseline and compliance behavior analysis
- Troubleshooting user application and workload access
- Application dependency mapping

### 6.0 Threat Response (10%)
- Response automation use cases
- Actions based on telemetry reports
- Policy determination based on security audit reports
- Response to user or application compromise (containment, reporting)

## Common job-ready skills
Upon passing the 300-740 SCAZT exam, professionals are ready to:

- **Design Zero Trust architectures** for cloud-connected organizations

- **Deploy secure access policies** for SaaS application protection
- **Manage endpoint security** using Cisco Secure Client and endpoint detection
- **Implement microsegmentation** using Cisco Secure Workload
- **Monitor and respond** to cloud security threats using XDR and SIEM
- **Validate compliance** and enforce security policies across multicloud environments
- **Troubleshoot cloud access issues** using visibility and analytics tools
- **Apply MITRE ATT&CK** framework to threat modeling and mitigation
