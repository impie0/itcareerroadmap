## About

CLICA (Implementing Cisco Collaboration Applications) is a concentration exam within the CCNP Collaboration track, launched September 2020. To earn CCNP Collaboration, candidates must first pass the core exam CLCOR (350-801), then pass CLICA (300-810) or one of three alternative concentration exams: CLACCM (Call Manager, 300-815), CLCEI (Cloud & Edge, 300-820), or CLAUTO (Collaboration Automation, 300-835). CLICA specifically validates deep expertise in Cisco collaboration applications—UC messaging, presence services, instant messaging clients, and modern hybrid/cloud deployments. This exam targets collaboration engineers and architects managing messaging infrastructure, mobility solutions, and cloud-hybrid integrations in large enterprises.

## Topics covered

Official exam blueprint (subject weights):

- **Cisco Unity Connection (30%)** — Unified messaging architecture, mailbox configuration, class-of-service (CoS), user profiles, directory integration, partition models, and CUCM integration.
- **Unity Connection Integration with CUCM (20%)** — Call routing to Unity, voice mail trigger setup, directory handlers, message-waiting indicators (MWI), and troubleshooting integration points.
- **IM and Presence Service (CUPS) (15%)** — CUPS architecture, presence federation, instant messaging topology, client connectivity, XMPP protocol basics, and IM routing.
- **Jabber Client Deployment (15%)** — Cisco Jabber for Windows/Mac/Mobile, device profiles, certificate-based authentication, presence status, do-not-disturb modes, and endpoint integration.
- **Mobile & Remote Access (10%)** — Cisco Single Number Reach (SNR), Mobile Remote Access (MRA), Webex hybrid calling, secure remote connectivity, and unified notification.
- **Cisco Meeting Server & Webex Integration (5%)** — Meeting server basics, Webex hybrid features, calendar integration, and meeting room management.
- **Security, Troubleshooting & Administration (5%)** — TLS/SRTP security for collaboration apps, LDAP directory integration, user provisioning, and common troubleshooting scenarios.

*Source:* [Cisco 300-810 CLICA Exam Blueprint ↗](https://learningnetwork.cisco.com/s/ccnp-collaboration-300-810-exam-topics)
