## About

The Cisco Certified Technician (CCT) Routing & Switching certification validates skills for onsite installation, maintenance, and troubleshooting of Cisco routers, switches, and IOS environments. Entry-level field technicians and support engineers use this credential to demonstrate hardware identification, CLI proficiency, and TAC collaboration competencies. The exam costs $125, lasts 90 minutes, and contains 60–70 multiple-choice and multiple-response questions. No prerequisites; valid for three years.

## Topics covered

- **Network Fundamentals** (15–20% estimated)
  - OSI model, TCP/IP stack basics, Ethernet and IP addressing
  - Common network protocols and packet flow

- **LAN Switching Technologies** (20–25% estimated)
  - VLAN configuration and trunking
  - Spanning Tree Protocol (STP) fundamentals
  - Switch port security and access control lists

- **Routing Technologies** (20–25% estimated)
  - Static and dynamic routing fundamentals
  - OSPF and EIGRP basic operation
  - Route selection and path determination

- **Infrastructure Services** (10–15% estimated)
  - DHCP, DNS, and IP address management
  - NTP and logging basics

- **Cisco Equipment & Hardware** (15–20% estimated)
  - Router and switch model identification
  - Interfaces, ports, and cabling types
  - Module recognition and slot configurations

- **Cisco IOS Software Operation** (10–15% estimated)
  - Boot process and image loading
  - Device startup configuration
  - Software version identification

- **Service-Related Knowledge** (5–10% estimated)
  - RMA (Return Merchandise Authorization) procedures
  - TAC (Technical Assistance Center) escalation
  - Field technician support workflows

## Common skills at Hardware/Maintenance · Entry

*Shared entry-level competencies for field technicians supporting physical IT infrastructure.*

- Identify network devices by model number and hardware revision
- Recognize cabling standards (UTP, fiber, serial) and connector types
- Navigate command-line interfaces (CLI) via terminal or console
- Interpret error messages and system logs for basic troubleshooting
- Document hardware inventory and asset information
- Coordinate with vendor support and escalate unresolved issues
- Apply basic safety practices (ESD, power, physical handling)

## Skills validated

*Specific technologies and tasks tested in this exam, distinct from general hardware-support competencies.*

- Cisco router model families (Catalyst, ISR, ASR) and switch platforms
- Console port access and terminal emulation (SSH, Telnet, serial)
- IOS image identification and boot sequence
- Basic router and switch configuration via CLI
- Spanning Tree Protocol (STP) state identification
- VLAN trunking and access port configuration
- DHCP address pool and lease management
- Static route and default gateway configuration
- Ethernet cable types (Cat 5e, Cat 6, Single-mode/Multimode fiber)
- Hardware RMA workflow and TAC escalation procedures
- Event log interpretation and syslog configuration
