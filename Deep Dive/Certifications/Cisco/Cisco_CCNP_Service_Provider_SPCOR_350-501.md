## About

The CCNP Service Provider (CCNP SP) certification validates expertise in implementing and operating Cisco service provider network core technologies. A candidate passing the 350-501 SPCOR core exam earns the "Cisco Certified Specialist — Service Provider Core" credential and must then pass one of four concentration exams (SPRI, SPVI, SPAUTO, or SPCNI) to achieve the full CCNP SP professional-level certification. This path is designed for network engineers deployed in ISP, carrier, and service provider backbone environments running MPLS, segment routing, and multi-service architectures.

---

## Domain context — Networking

Service Provider Networking (ISP/Carrier backbone operations) is a specialized branch of enterprise networking focused on high-scale, mission-critical infrastructure: MPLS core networks, BGP routing at internet scale, VPN services, QoS, and automation of carrier-grade systems. CCNP SP is the vendor-certified professional standard for this domain.

## Topics covered

The 350-501 SPCOR exam blueprint covers:

- **Architecture (15%)** — Service provider network topology, modular design, scalability, redundancy
- **Networking (30%)** — BGP, OSPF, IS-IS, MPLS/Segment Routing, IPv6, IP addressing, multicast, forwarding
- **Services (20%)** — Layer 2 VPN (VPLS, E-Line), Layer 3 VPN (MPLS L3VPN, 6VPE), EVPN
- **Automation (10%)** — NETCONF/YANG, programmability, infrastructure automation for service provider operations
- **Security (10%)** — BGP security, MPLS security, uRPF, control-plane protection, DDoS mitigation
- **Network Assurance (15%)** — monitoring, logging, analytics, SLA verification, telemetry, troubleshooting

## Skills validated

*Cert-specific — what this exam actually tests.*

- Implementing and scaling BGP in service provider networks
- MPLS Label Switching and LDP/RSVP signaling
- Segment Routing (SR) for simplified MPLS operations
- Layer 3 VPN (MPLS L3VPN) and 6VPE (IPv6 over MPLS)
- EVPN (Ethernet VPN) for multi-service delivery
- Layer 2 VPN services (VPLS, E-Line, Ethernet over MPLS)
- IP Core network design and optimization
- Quality of Service policy implementation (DiffServ, traffic shaping, policing)
- Service Provider security: uRPF, BGP authentication, control-plane ACLs
- Network automation using NETCONF/YANG
- Operational metrics and telemetry (Streaming Telemetry, Flexible NetFlow)
- Troubleshooting at scale using logs, syslog, SNMP, and analytics
