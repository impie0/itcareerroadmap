## About

The CCNA is Cisco's entry-level networking certification, validating foundational knowledge and skills in network fundamentals, switching, routing, security, and automation. Launched in its current form (200-301) in February 2020 and updated to v1.1 in August 2024 to include cloud and AI/ML topics, the CCNA is the most widely recognized credential for junior and mid-career network engineers globally. It serves as the prerequisite foundation for all advanced Cisco certifications (CCNP, CCIE tracks) and is often required or strongly preferred by employers in network operations, support, and engineering roles.

## Topics covered

The CCNA 200-301 v1.1 exam blueprint covers six weighted domains:

- **Network Fundamentals (20%)** — OSI & TCP/IP models, media types, cabling, topologies, virtualization, switching mechanics, device roles (router, switch, firewall, access point, PoE)
- **Network Access (20%)** — VLANs, trunking (802.1Q), Spanning Tree Protocol (STP), EtherChannel, wireless LAN (802.11, WPA), switch security
- **IP Connectivity (25%)** — IPv4 & IPv6 addressing, static routing, OSPF, EIGRP, default gateway, first-hop redundancy protocols (HSRP, VRRP)
- **IP Services (10%)** — DHCP, DNS, NAT/PAT, NTP, QoS, syslog, SNMP
- **Security Fundamentals (15%)** — Access control lists (ACLs), AAA, device authentication (SSH, Telnet), port security, threats & mitigation
- **Automation and Programmability (10%)** — REST APIs, JSON/XML, Python basics, network automation concepts, Cisco DNA Center, IOS XE

*Source:* [Official exam guide ↗](https://learningcontent.cisco.com/documents/200_301_CCNA_v1.1.pdf)

## Skills validated

*Cert-specific — what this exam actually tests, distinct from the shared "Common skills" above.*

- Cisco IOS CLI operation and device configuration (switches, routers)
- IPv4 and IPv6 addressing, subnetting, and routing
- Ethernet switching, VLAN tagging, and Spanning Tree Protocol
- OSPF and EIGRP dynamic routing protocol configuration
- DHCP, DNS, NAT, and IP services
- Access Control Lists (ACLs) and basic firewall concepts
- Device authentication (SSH, password security)
- Wireless LAN fundamentals (802.11, WPA2/WPA3)
- QoS (Quality of Service) and traffic management
- SNMP, syslog, and device monitoring
- REST APIs and JSON/XML for network automation
- Python scripting for network tasks (basics)
- Cisco DNA Center and SD-WAN concepts
