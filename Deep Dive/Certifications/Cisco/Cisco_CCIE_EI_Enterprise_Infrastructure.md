## About

The **CCIE Enterprise Infrastructure** is Cisco's highest-level enterprise networking certification, designed for senior network engineers and architects who design, deploy, and optimize complex enterprise network infrastructures. First released in 2016 (replacing the legacy CCIE Routing & Switching), the current v1.1 launched in February 2026 and was updated in April 2026 to include Cisco Crosswork Network Automation and Secure Firewall Threat Defense scenarios. This certification is recognized globally as one of the most prestigious in IT networking; CCIEs are in high demand and command premium salaries. Passing requires both a written core exam (350-401 ENCOR) and an 8-hour hands-on lab exam.

## Topics covered

**ENCOR (350-401) Blueprint Topics:**

- Architecture, Components, and Concepts (15% weight)
  - Dual stack (IPv4/IPv6) design
  - Virtualization and overlay/underlay concepts
  - Infrastructure as Code / programmability
  - Hybrid cloud connectivity

- Routing (20% weight)
  - BGP routing protocol and design patterns
  - OSPF, IS-IS fundamentals
  - Multicast and MCAST design
  - Route filtering and policy

- Switching (15% weight)
  - VLANs, inter-VLAN routing
  - Spanning Tree Protocol (STP/RSTP/MSTP)
  - EtherChannel aggregation
  - Switch hardware and software architecture

- Infrastructure Services (13% weight)
  - DHCP, DNS, NTP configuration
  - QoS design and implementation
  - High availability and redundancy
  - First Hop Redundancy (FHRP)

- Network Assurance (12% weight)
  - Device monitoring and management (SNMP, NetFlow, syslog)
  - Troubleshooting tools and methodologies
  - Network telemetry and analytics
  - Performance baselining

- Security (13% weight)
  - Access Control Lists (ACLs) and filtering
  - Network Address Translation (NAT)
  - IPsec VPN design
  - Firewall concepts and integration

- Automation, Programmability, and DevOps (12% weight)
  - REST APIs and NETCONF/YANG
  - Python scripting for network automation
  - Ansible, Terraform IaC frameworks
  - Model-Driven Telemetry (MDT)

**CCIE EI Lab Exam (v1.1) Domains:**

- **Design Module (3 hours)**: Network architecture, service requirements mapping, topology design, technology selection
- **Deploy/Operate/Optimize Module (5 hours)**:
  - Network Infrastructure: Routing, switching, multicast deployment
  - Software-Defined Infrastructure: SD-WAN, SD-Access, intent-based networking
  - Transport Technologies: MPLS, segment routing, segment routing for traffic engineering
  - Network Services: QoS, multicast, first-hop redundancy, load balancing
  - Network Security: Firewalls, VPNs, threat defense, encryption
  - Network Infrastructure Automation: Configuration management, provisioning, Crosswork Network Automation (new in v1.1 April 2026)
  - Network Assurance: Monitoring, troubleshooting, analytics, AI-ops integration

## Study plan outline (12–18 months, typical)

**Months 1–3: Foundation (CCNP level)**
- Complete CCNP Enterprise (ENCOR 350-401) if not already certified
- Read Narbik's *Foundation* 2nd ed., chapters 1–15
- Set up Cisco Modeling Labs (CML) or GNS3
- Practice ENCOR mock exams (Boson, Whizlabs)

**Months 4–8: Core lab topics**
- Follow INE or Micronics structured learning path
- Labs: BGP, OSPF, IS-IS, MPLS, VRF, multicast (100–200+ hours)
- Focus on design and troubleshooting, not just "make it work"
- Practice exams every 2 weeks

**Months 9–12: Integration and lab format**
- Study v1.1 updates (Crosswork, Secure Firewall Threat Defense)
- Full mock lab exams (8-hour timed) every 2 weeks
- Narbik workbooks (design + deploy tasks)
- Track weak topics and iterate

**Months 13–18: Lab attempt readiness**
- Daily mock labs and topic deep-dives
- Speed practice (complete design in 3 hours, deploy in 5)
- Join study groups (NetworkKings, INE community, Reddit r/ccna)
- Register for lab exam once confidence reaches 85%+

**Total cost estimate:** $3,000–$10,000 (courses $1,000–$3,000, labs $1,000–$2,000/year, exam attempts $2,300+, travel/materials $500+)
