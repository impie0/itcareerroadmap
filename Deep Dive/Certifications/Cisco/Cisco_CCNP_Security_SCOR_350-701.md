## About

The **350-701 SCOR** (Implementing and Operating Cisco Security Core Technologies) is the mandatory core exam for Cisco Certified Network Professional (CCNP) Security certification. Introduced in January 2019, it replaces the legacy Security exam and tests hands-on implementation and operation of Cisco security technologies spanning network security (firewalls, IDS/IPS), cloud security, content filtering, endpoint detection, secure access, and visibility. A candidate must pass 350-701 SCOR plus one concentration exam (choice of six: SNCF, SISE, SESA, SWSA, SVPN, or SAUTO) to earn CCNP Security. The exam is a qualifying exam for CCIE Security and carries forward the professional-level rigor expected in security engineering roles.

## Topics covered

Based on official Cisco exam blueprint v1.1:

- **Domain 1: Security Concepts** (25%)
  - Threat landscape and attack vectors
  - Cryptography, SSL/TLS, certificates
  - Authentication and authorization frameworks
  - Security policies and compliance

- **Domain 2: Network Security** (20%)
  - Firewalls and firewall rule design
  - IDS/IPS concepts and deployment
  - DDoS mitigation
  - VPN technologies (site-to-site, remote-access)

- **Domain 3: Securing the Cloud** (15%)
  - Cloud security architecture
  - AWS, Azure, GCP security considerations
  - Cloud IAM and access control
  - Secure cloud migrations

- **Domain 4: Content Security** (10%)
  - Web security and SSL inspection
  - Email security and malware filtering
  - Data loss prevention (DLP)
  - Web application firewalls

- **Domain 5: Endpoint Protection and Detection** (15%)
  - Antimalware and anti-ransomware
  - Endpoint detection and response (EDR)
  - Mobile device management
  - Patch and vulnerability management

- **Domain 6: Secure Network Access, Visibility, and Enforcement** (15%)
  - Identity Services Engine (ISE) architecture and policy
  - Network access control (NAC)
  - Secure access service edge (SASE)
  - Threat visibility and telemetry

*Source:* [Cisco Learning Network SCOR Exam Blueprint ↗](https://learningnetwork.cisco.com/s/scor-exam-topics)
