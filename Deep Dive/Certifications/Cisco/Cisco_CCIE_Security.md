## About

The Cisco Certified Internetwork Expert (CCIE) Security certification is the vendor's highest individual technical credential in network security. It comprises two sequential exams: a 120-minute written exam (350-701 SCOR) and an 8-hour hands-on lab where candidates configure, optimize, and troubleshoot enterprise security infrastructure. First introduced in 2012, the current v6 syllabus (launched February 2019) reflects modern threat landscapes, cloud-hybrid networking, and automation practices. CCIE Security holders are typically 5–10 years into infrastructure and security roles, pursuing roles as senior architects, distinguished engineers, or independent security consultants.

## Written exam (350-701 SCOR) — Topics covered

**Exam weight distribution:**

- **Cisco Secure Firewall (ASA, FTD, FMC)** — 20% — Threat prevention, access control, IPS/IDS, NAT, HA/failover, policy architecture
- **Secure Connectivity (Site-to-Site & Remote Access VPN)** — 15% — IPsec, SSL/TLS VPN, FlexVPN, high availability, scalable VPN design
- **Cisco Identity Services Engine (ISE)** — 15% — Authentication (802.1X, MAB, guest access), authorization policies, profiling, guest portals, TrustSec
- **Secure Network Telemetry & Analytics** — 12% — NetFlow, Security Monitoring, threat hunting, event correlation, SIEM integration
- **Threat Defense Intelligence** — 10% — Talos threat feeds, reputation scoring, URL filtering, DNS security, sandboxing
- **Cloud Security & APIs** — 10% — AWS/Azure security integration, API security, container security, cloud egress controls
- **Advanced Threat Protection** — 10% — Malware detection, ransomware defense, advanced persistent threat (APT) hunting, endpoint integration
- **Network Security Infrastructure & Automation** — 8% — Ansible/Terraform for security, Python scripting, DevSecOps, infrastructure-as-code security

*Source:* [Cisco Learning Network Exam Blueprint ↗](https://learningnetwork.cisco.com/s/ccie_security_exam_topic_details)

## Lab exam — Modules and topology

The 8-hour hands-on lab is conducted at authorized Cisco testing centers (Pearson VUE Advanced Testing Centers) and covers real-world configuration and troubleshooting scenarios. The lab typically includes 3–4 major sections:

- **Perimeter Security** — Configure Cisco Secure Firewall (ASA/FTD), implement access control lists (ACLs), set up threat prevention policies, tune IPS/IDS signatures, configure NAT, and design failover scenarios.

- **Secure Connectivity** — Design and implement IPsec site-to-site VPN, SSL/TLS remote-access VPN, configure high-availability VPN deployments, troubleshoot encryption and tunnel state issues.

- **Identity & Access Management** — Configure Cisco ISE for 802.1X authentication, set up guest portal workflows, implement authorization policies with TrustSec segmentation, and profile network endpoints.

- **Threat Defense & Telemetry** — Configure malware defense policies, enable advanced threat protection, set up NetFlow/sFlow monitoring, integrate SIEM or analytics platforms, hunt for indicators of compromise.

- **Automation & Programmability** — Write Python scripts to interact with Cisco security APIs, automate policy deployment with Ansible or Terraform, configure webhook integrations, and manage security infrastructure programmatically.

*Candidates may face real-world troubleshooting scenarios where devices are intentionally misconfigured; identifying and remedying problems is part of the assessment.*

## Skills validated

*Concrete technologies, protocols, and tools this exam assesses.*

- **Cisco Secure Firewall (ASA & Threat Defense)** — Stateful inspection, access control lists, threat prevention, intrusion prevention/detection, NAT/PAT, high-availability failover
- **IPsec VPN** — Tunnel negotiation, encryption/authentication algorithms, Dead Peer Detection (DPD), split tunneling, dynamic routing over VPN
- **SSL/TLS VPN** — Clientless portal VPN, AnyConnect secure mobility client, certificate management, session lifecycle
- **Cisco Identity Services Engine (ISE)** — 802.1X authentication, MAC authentication bypass (MAB), guest access portals, authorization policies, profiling, TrustSec enforcement
- **Advanced Threat Protection (ATP)** — Malware detection, ransomware defense, sandbox detonation, URL filtering, DNS Security (Umbrella integration)
- **Network Telemetry & SIEM** — NetFlow/sFlow collection, security event correlation, threat hunting workflows, integration with third-party SIEMs (Splunk, ArcSight)
- **Cisco AMP (Advanced Malware Protection)** — Endpoint detection and response (EDR), file reputation scoring, behavioral analysis, incident response integration
- **Secure Connectivity Protocols** — FlexVPN, Internet Key Exchange (IKE) v1/v2, Elliptic Curve Cryptography (ECC), Perfect Forward Secrecy (PFS)
- **Policy as Code & Automation** — Ansible playbooks for security, Terraform for infrastructure, Python scripting with Cisco APIs (REST, XML-RPC), CI/CD integration
- **Cloud Security** — AWS VPC security groups, Azure Network Security Groups (NSGs), hybrid cloud egress controls, API gateway security
- **Zero-Trust & Microsegmentation** — TrustSec Scalable Group Tags (SGTs), segment routing (SR), policy-based access control

## Career progression

**Before CCIE Security:**
- Typically 5–10 years in infrastructure/security roles: network engineer, security engineer, firewall administrator, identity management specialist
- CCNA and/or CCNP (or equivalent CompTIA Security+ / EC-Council CEH) mastery
- Hands-on experience troubleshooting ASA/FTD, ISE deployments, VPN connectivity, threat prevention policies

**After CCIE Security:**
- Senior Network Security Architect roles (large enterprises)
- Distinguished Security Engineer or Staff Security Architect (tech-leader track)
- Independent consulting: fractional CISO, security assessment teams, design review authority
- Product security specialist (Cisco TAC, threat research labs)
- Security solutions architect (Cisco partner / reseller)

## Study timeline

**Full-time bootcamp:** 8–12 weeks (40+ hours/week, including labs)  
**Part-time study:** 6–12 months (10–15 hours/week, labs + written practice)  
**Lab practice:** 100–200 hours minimum on real or virtualized topologies  

The written exam (350-701 SCOR) is the faster gate (typically 3–4 months prep); the lab is the real differentiator (6–12 months). Most candidates pass the written, then spend additional months refining lab troubleshooting speed and accuracy.
