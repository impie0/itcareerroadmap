## About

CLACCM (Call Manager Advanced) is a concentration exam for CCNP Collaboration certification, focusing on advanced Cisco Unified Communications Manager (CUCM) administration, call control, and enterprise mobility features. Released in 2020 as part of the modernized CCNP Collaboration track, CLACCM validates deep expertise in dial plan architecture, advanced routing, CUCM clustering, media resources, gateway protocols, and mobility services. Candidates must hold or concurrently earn CLCOR (350-801) to obtain CCNP Collaboration. CLACCM targets senior collaboration engineers and UC architects managing enterprise-scale CUCM deployments, multisite environments, and advanced call routing scenarios.

## Topics covered

Official exam blueprint (estimated subject weights):

- **Cisco Unified CM Dial Plan & Routing (25%)** — Route patterns, translation patterns, partitions, calling search spaces (CSS), prefix digits, external phone number masks, and digit manipulation.
- **CUCM Clustering & Architecture (15%)** — Cluster design, intra-cluster communication, publisher/subscriber roles, redundancy, failover, and disaster recovery.
- **CUCM Media Resources (10%)** — Media Termination Point (MTP), transcoder configuration, conference bridges, hardware vs. software resources, and resource allocation.
- **Gateway Protocols & Trunk Configuration (20%)** — H.323, MGCP, SIP trunk design, gateway dial peers, translation rules, PSTN integration, and call routing through gateways.
- **Cisco Unified Border Element (CUBE) (10%)** — SIP normalization, SIP manipulation, trunk security, inter-cluster communication, and enterprise edge deployment.
- **Call Admission Control (CAC) & Locations (10%)** — Locations-based CAC, bandwidth management, gatekeeper functions, and bandwidth reservations.
- **Cisco Emergency Responder (CER) & E911 (5%)** — Emergency call routing, E911 dial plan considerations, and civic address integration.
- **Mobility Features (5%)** — Extension Mobility, Cisco Unified Mobility Services, device pools, remote destination profiles, and IP phone mobility.

*Source:* [Cisco 300-820 CLACCM Exam Blueprint ↗](https://learningnetwork.cisco.com/s/ccnp-collaboration-claccm-exam-topics)
