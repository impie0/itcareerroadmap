## About

The 300-630 DCACIA is the advanced concentration exam for the CCNP Data Center certification. This exam is a companion to the 300-620 DCACI (associate-level ACI exam) and targets network engineers, data center architects, and SDN specialists working with advanced Cisco Application Centric Infrastructure implementations. It validates expertise in multi-site ACI deployments, orchestration, automation, and enterprise-scale fabric management using Nexus Dashboard Orchestrator and APIC controllers.

## Domain context — Data Center / SDN

Policy-based data center networking using Cisco Application Centric Infrastructure (ACI). Multi-tenant SDN fabric management via APIC (Application Policy Infrastructure Controller), Nexus Dashboard Orchestrator, and Nexus 9000 Series switches in ACI mode. Focus on enterprise deployments spanning multiple sites, external routing, service graph integration, and infrastructure automation.

## Topics covered

The 300-630 exam blueprint covers the following major topic areas:

1. **ACI Packet Forwarding** (~15%)
   - VxLAN encapsulation and packet flow within the fabric
   - Role of spine and leaf switches
   - Forwarding tables and lookup mechanisms
   - Server NIC teaming and endpoint learning optimization
   - Intra-pod and inter-pod forwarding (multi-pod architecture)

2. **Advanced ACI Policies and Integrations** (~20%)
   - Tenant policies, application profiles, and endpoint groups
   - Contracts, filters, and policy rules
   - Layer 4–7 policy-based redirect and service graphs
   - VMM integration (vSphere, Hyper-V, OpenStack)
   - Kubernetes integration and container networking
   - Configuration of advanced policy models

3. **External Connectivity and Routing** (~20%)
   - L3Out configuration for external BGP and OSPF peering
   - VRF route leaking and inter-VRF communication
   - OSPF, BGP, EIGRP external routing protocols
   - Layer 2 external connectivity (transit routing)
   - Route summarization and redistribution in multi-site
   - External endpoints and border leaf configurations

4. **Multi-Pod and Multi-Site ACI** (~20%)
   - Multi-pod fabric architecture and inter-pod network (IPN)
   - Spine-to-spine peering and super-spine topologies
   - Multi-site implementation using Nexus Dashboard Orchestrator (NDO)
   - Stretched VRFs, contracts, and policies across sites
   - Multi-site tunnel setup and LISP control plane
   - Traffic forwarding between pods and sites
   - Disaster recovery and redundancy in multi-site

5. **ACI REST API and Automation** (~10%)
   - APIC REST API structure and authentication
   - Python and Ansible for ACI automation
   - Terraform for infrastructure-as-code (IaC)
   - API-based policy configuration and management
   - Script development for fabric operations
   - Integration with CI/CD pipelines

6. **APIC Administration and Operations** (~10%)
   - APIC high availability and backup/restore
   - Role-based access control (RBAC)
   - Monitoring, telemetry, and dashboards
   - Fabric health and fault reporting
   - Firmware upgrades and fabric maintenance
   - Certificate management and security
   - Logging and syslog configuration

7. **ACI Troubleshooting** (~5%)
   - Contract troubleshooting and policy validation
   - Fabric health checks and diagnostics
   - Endpoint and bridge domain troubleshooting
   - Connectivity issues in multi-site deployments
   - Common misconfigurations and resolution
   - Packet capture and flow analysis in ACI

## Common job-ready skills

- Configure and manage advanced ACI policies, contracts, and service graphs
- Design and deploy multi-site ACI fabrics using Nexus Dashboard Orchestrator
- Implement external connectivity (L3Out, BGP/OSPF) for traditional network integration
- Automate ACI operations using REST API, Python, Ansible, and Terraform
- Troubleshoot ACI fabric issues, endpoint reachability, and policy enforcement
- Manage APIC controllers in high-availability clusters with proper security and RBAC
- Integrate ACI with virtualization platforms (vSphere, Hyper-V) and container orchestrators (Kubernetes)
- Optimize ACI packet forwarding, VxLAN encapsulation, and multi-pod routing
