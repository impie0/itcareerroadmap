## About

The Cisco Certified DevNet Specialist - Webex (300-920 DEVWBX) validates intermediate-level competency in building production applications on Webex APIs and integrations. Launched in 2017 and refreshed to v1.1 in 2024, this exam tests real-world scenarios: scheduling meetings programmatically, controlling room devices via xAPI, building bots and webhooks, handling authentication flows, and ensuring compliance with data-retention policies. It sits at the **Specialist level** in Cisco's DevNet portfolio—higher than DevNet Associate (200-901) but distinct from the DevNet Professional core (350-901 DEVCOR), which requires passing both a core exam and a concentration exam (like this one). Webex specialists are in steady demand across enterprise collaboration teams, system integrators, and ISVs building custom workflows for hybrid work.

## Topics covered

The exam blueprint spans six core areas, with emphasis on practical API usage and Webex Meeting / Device integration:

- **Webex API Foundations** (15% weight)  
  – Get access to Webex APIs and developer accounts  
  – Authentication methods for Teams, Devices, Meetings  
  – OAuth token lifecycle and refresh mechanisms  
  – REST API error handling (rate limits, auth failures, pagination)  
  – JavaScript SDK promises and requests  

- **Webex Meetings** (20% weight)  
  – Meeting scheduling via API  
  – Recording retrieval and download  
  – Meeting metadata and participant lists  
  – XML API for user and meeting management  
  – Meeting webhooks  

- **Webex Devices** (20% weight)  
  – xAPI over SSH, REST, and WebSocket protocols  
  – Device configuration and status queries  
  – Macro development for room automation  
  – Gesture and event-driven actions on endpoints  
  – Device clustering and call control  

- **Webex Messaging** (15% weight)  
  – Webex messaging APIs and SDKs  
  – Bot development and command handling  
  – Attachment and rich-text card formatting  
  – Message webhooks and real-time event streams  

- **Embedding Webex** (15% weight)  
  – Embedded meeting clients in third-party apps  
  – SDK initialization and configuration  
  – Custom branding and SSO integration  
  – Embedded messaging and presence APIs  

- **Administration & Compliance** (15% weight)  
  – Admin APIs for user/organization management  
  – Data retention policies and GDPR compliance  
  – Audit logs and reporting  
  – License allocation and user provisioning  

*Source:* [Official Exam Topics PDF ↗](https://www.cisco.com/c/dam/en_us/training-events/le31/le46/cln/marketing/exam-topics/300-920-DEVWBX.pdf)
