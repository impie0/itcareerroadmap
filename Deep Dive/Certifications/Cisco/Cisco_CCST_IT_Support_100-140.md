## About

The **Cisco Certified Support Technician (CCST) IT Support** certification validates foundational knowledge and practical skills required for entry-level IT support roles such as Help Desk Technician, IT Support Specialist, or Computer Support Specialist. Released in June 2024, the CCST IT Support exam (100-140) is the newest addition to Cisco's CCST program, following the Cybersecurity and Networking certifications launched in 2023. The certification tests candidates on IT support job tasks, hardware troubleshooting, connectivity diagnostics, operating systems management, security awareness, and customer communication—covering both technical hands-on skills and professional support workflow practices.

## Topics covered

The CCST IT Support exam covers six primary domains based on the official Cisco exam blueprint:

- **IT Support Job Tasks and Responsibilities** — Queue management, ticketing systems, SLAs, KPIs, time management, documentation best practices
- **Troubleshooting and Problem Solving** — Problem-solving methodology, symptom analysis, root cause identification, resolution planning, change implementation, result observation, documentation
- **Hardware and Peripherals** — Printer connectivity, multifunction devices, port identification, component troubleshooting, peripheral diagnostics
- **Connectivity** — LAN (wired) troubleshooting, WLAN (wireless) connectivity issues, network access diagnosis, connectivity diagnostics tools
- **Operating Systems** — Windows, macOS, and Linux basics; software installation; OS troubleshooting; file systems; user management
- **Security** — Security threat recognition, user education, incident escalation procedures, company security policies, confidentiality guidelines, data protection

**Interpersonal skills** are assessed throughout (professional communication, active listening, technical explanation to non-technical users, appropriate vocabulary).

## Skills validated

*Concrete technologies and practices this exam tests.*

- Windows, macOS, and Linux operating system basics and troubleshooting
- Hardware components (CPU, RAM, storage, peripherals) and device identification
- Network connectivity (Ethernet, Wi-Fi, TCP/IP basics)
- Remote access and support tools (VPN, remote desktop, chat/ticketing systems)
- Printer and multifunction device troubleshooting
- User account management and permission basics
- Email and collaboration application support
- Malware, phishing, and social engineering threat recognition
- Password security and two-factor authentication awareness
- Incident documentation and ticket system workflows
- Active Directory and user identity basics
- BIOS/UEFI and boot sequence troubleshooting
- File sharing and permission models (NTFS, POSIX)
- Command-line basics for Windows (CMD, PowerShell) and Linux (bash)
- VoIP phone system basics
- Mobile device (iOS, Android) support fundamentals
