## About

The Cisco Certified CyberOps Professional (350-201 CBRCOR) certification validates expertise in performing cybersecurity operations using Cisco security technologies. Launched in 2023, this exam serves SOC analysts, threat hunters, incident responders, and detection engineers working in security operations centers. CBRCOR is the core exam; candidates may also pursue a concentration exam (300-215 CBRFIR for forensics or 300-220 CBRTHD for threat hunting) to specialize. Passing CBRCOR earns the Cisco Certified Specialist – Cybersecurity Core; adding a concentration yields the full Cisco Certified CyberOps Professional credential.

## Topics covered

The 350-201 CBRCOR exam blueprint covers cybersecurity operations through the lens of Cisco technologies:

- **Fundamentals (20%):** Security principles, threat landscapes, asset types, attack vectors, CIA triad, security tools integration
- **Techniques (30%):** Hands-on threat detection, malware analysis, intrusion identification, traffic analysis, log examination
- **Processes (30%):** Incident handling, evidence collection, forensics workflows, containment, eradication, post-incident review
- **Automation (15%):** Security automation tools, workflow orchestration, response automation, Cisco Orchestration, integration principles
- **Threat Hunting & AI (5%):** Proactive threat hunting methodology, AI-powered data analytics, threat intelligence integration

## Skills validated

- Cisco Secure endpoint detection and response (EDR)
- Cisco Secure firewall configuration and log interpretation
