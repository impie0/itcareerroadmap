## About

The HashiCorp Certified: Vault Associate (002) validates foundational knowledge of Vault, the industry-standard secrets management and encryption-as-a-service platform. Introduced in 2020 to replace version 001, the exam tests your ability to deploy, configure, authenticate, and manage Vault in production. This is the entry-level certification in HashiCorp's Vault track and serves as a prerequisite for the Vault Operations Professional (expert) exam.

## Topics covered

Official exam blueprint weighted by domain:

- **Authentication Methods (15%)** — Userpass, LDAP, AppRole, JWT/OIDC, AWS IAM, Azure AD; selecting appropriate method by use case; human vs. system authentication.
- **Auth Methods Configuration (15%)** — Configuring and troubleshooting authentication mechanisms; managing auth method lifecycles; token generation workflows.
- **Vault Encryption & Policies (15%)** — Vault policy syntax; path and capability definitions; policy testing; root token; token lifecycle; leases and renewal.
- **Secrets Engines (15%)** — Static and dynamic secrets; enabling/disabling engines; KV v1/v2; database dynamic secrets; PKI; SSH; AWS secrets; transit engine basics.
- **Run a Vault Cluster (10%)** — Vault architecture; high availability; storage backends; Raft; unsealing; initialization; disaster recovery.
- **Configure Vault for AAA (15%)** — Authentication, authorization, accounting; audit logging; compliance requirements; multi-tenant configuration.
- **Manage Vault Tokens (10%)** — Token types (service, batch); token lifecycle; TTL/max-TTL; orphan tokens; token policies.
- **Read Security Models (5%)** — Vault's security model; threat model; attack surface; least-privilege design patterns.
