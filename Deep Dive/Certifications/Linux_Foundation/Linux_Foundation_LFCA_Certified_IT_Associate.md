# Linux Foundation Certified IT Associate (LFCA)

**`LFCA`** · ● **Active** · **Entry** · _Linux Foundation_

> The Linux Foundation Certified IT Associate (LFCA) is a pre-professional entry-level certification designed for individuals new to IT or considering a career transition into IT administration or engineering roles. It validates foundational knowledge across Linux fundamentals, system administration basics, cloud computing concepts, security fundamentals, and DevOps principles.

## About

The LFCA certification was originally launched in 2020 and underwent significant updates in 2024–2025 to align with current industry practices in cloud computing, containerization, and DevOps. It is designed as a stepping stone for career changers and IT newcomers, validating hands-on competency across operating systems, command-line administration, networking fundamentals, security best practices, and open-source software principles. The certification establishes readiness for entry-level system administration, IT support, and junior cloud operations roles. No prerequisites or prior certification experience is required; the exam is vendor-neutral and platform-agnostic.

## Topics covered

The refreshed 2024–2025 LFCA exam covers six primary domains and competencies:

- **Linux Fundamentals** — File systems, command-line navigation, user management, file permissions and ownership, package installation and updates
- **System Administration Fundamentals** — User and group administration, system services and daemons, basic troubleshooting, log analysis, disk and partition management
- **Cloud Computing Fundamentals** — Public, private, and hybrid cloud models, cloud service types (IaaS, PaaS, SaaS), cloud platforms (AWS, Azure, Google Cloud overview)
- **Security Fundamentals** — User authentication and access control (ACLs), firewall basics, password policies, secure communication (SSH), vulnerability assessment overview
- **DevOps Fundamentals** — Infrastructure as Code (IaC) principles, containerization concepts (Docker, Kubernetes overview), CI/CD pipeline basics, configuration management overview
- **Supporting Applications and Developers** — Package management, scripting fundamentals (bash basics), application deployment, monitoring and logging, open-source licensing

*Source:* [Linux Foundation LFCA Program Changes (2025) ↗](https://training.linuxfoundation.org/lfca-program-changes-2025/)
