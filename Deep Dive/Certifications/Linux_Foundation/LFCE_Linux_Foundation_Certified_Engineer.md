## About

The Linux Foundation Certified Engineer (LFCE) is a professional-level hands-on certification for Linux systems engineers with 2–5+ years of production experience. Released in 2015, LFCE complements the LFCS entry-level credential and establishes expertise in advanced system administration, cluster architecture, container orchestration, security hardening, and infrastructure automation. The exam is a live Linux lab environment (no multiple-choice questions), where candidates configure, troubleshoot, and deploy real systems under time pressure.

---

## Topics covered

Official exam blueprint domains (2026):

- **Network Configuration** (25%) — IP routing, firewalls, DNS, IPv4/IPv6, bonding, bridges, virtual networking, troubleshooting connectivity
- **System Design and Deployment** (20%) — Kernel configuration, boot process, grub/bootloader, system initialization, cloud-init, automated deployments
- **Linux Filesystems and Storage Management** (12%) — LVM, RAID, filesystem tuning (ext4, XFS, btrfs), partitioning, quota, snapshots
- **Containers** (10%) — Docker, container networking, image management, security contexts, container registries
- **Cluster and High Availability** (15%) — Kubernetes architecture, node setup, etcd, cluster networking, StatefulSets, DaemonSets, service mesh concepts
- **Security** (12%) — SELinux/AppArmor, cryptography basics, certificate management, user/group security, audit logging, intrusion detection
- **Configuration Management** (6%) — Ansible, Puppet, or Chef — infrastructure as code patterns, idempotency, templating
