## About

AWS Certified Cloud Practitioner (CLF-C02) is the foundational, vendor-neutral entry point to AWS certification. Launched September 19, 2023, CLF-C02 replaced CLF-C01 and tests broad AWS cloud knowledge across service categories, shared responsibility, pricing, and compliance — not deep technical implementation. Ideal for IT professionals, operations staff, pre-sales engineers, and career changers entering cloud, requiring 6 months exposure to AWS or general cloud knowledge.

## Topics covered

Official exam blueprint (4 domains with published weights):

- **Domain 1: Cloud Concepts (24%)** — AWS value proposition, cloud economics, migration benefits, design principles (scalability, elasticity, resilience, high availability).
- **Domain 2: Security and Compliance (30%)** — Shared responsibility model, IAM fundamentals, encryption, AWS Organizations, compliance frameworks (HIPAA, PCI-DSS, FedRAMP), AWS Trusted Advisor.
- **Domain 3: Cloud Technology and Services (34%)** — Compute (EC2, Lambda, elastic scaling), storage (S3, EBS), databases (RDS, DynamoDB), networking (VPC, CloudFront), monitoring (CloudWatch).
- **Domain 4: Billing, Pricing, and Support (12%)** — AWS pricing models, Reserved Instances, Savings Plans, Spot Instances, AWS Free Tier, Cost Explorer, AWS Budgets, support plans (Basic, Developer, Business, Enterprise).

*Source:* [Official exam guide PDF ↗](https://d1.awsstatic.com/training-and-certification/docs-cloud-practitioner/AWS-Certified-Cloud-Practitioner_Exam-Guide.pdf)
