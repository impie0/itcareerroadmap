## About

AWS Certified DevOps Engineer – Professional (DOP-C02) validates your ability to design, implement, and maintain DevOps practices and infrastructure-as-code at enterprise scale on AWS. Launched March 7, 2023, DOP-C02 replaced the older DOP-C01 and is the professional-tier certification for engineers beyond the associate level. Targeted at senior DevOps engineers, SREs, and platform engineers who own end-to-end CI/CD pipelines, infrastructure automation, monitoring and observability, and incident response strategies.

---

## Domain context — DevOps / SRE / Cloud Infrastructure

Automation, reliability, and continuous delivery across the cloud infrastructure and platform engineering spectrum. **DevOps** emphasizes speed and collaboration through CI/CD and infrastructure-as-code; **SRE** applies engineering discipline to operational reliability via observability and error budgets; **Platform Engineering** builds internal developer platforms for self-service deployments.

## Topics covered

**Content Domains from Official Exam Blueprint (DOP-C02):**

- **SDLC Automation (22%)** — CodePipeline, CodeBuild, CodeDeploy, CodeArtifact, GitHub Actions integration, deployment strategies (blue/green, canary), approval processes.
- **Configuration Management and Infrastructure as Code (17%)** — CloudFormation, AWS CDK, Terraform/Pulumi, StackSets, Service Catalog, AWS SAM, configuration drift detection.
- **Resilience and Observability (20%)** — CloudWatch (metrics, logs, dashboards, alarms), X-Ray distributed tracing, Container Insights, Synthetics, EventBridge automation, Application Auto Scaling.
- **Incident and Event Response (15%)** — Incident Manager, on-call routing, runbooks, post-incident reviews, automated remediation, AWS Systems Manager OpsCenter.
- **Security and Compliance Automation (14%)** — KMS rotation, Secrets Manager, AWS Config rules, security automation via Lambda, GuardDuty, Security Hub integration.
- **Cost and Performance Optimization (12%)** — Tagging strategies, FinOps, performance tuning, Reserved Instances/Savings Plans automation, workload optimization.

*Source:* [AWS DOP-C02 Exam Guide ↗](https://d1.awsstatic.com/training-and-certification/docs-devops-pro/AWS-Certified-DevOps-Engineer-Professional_Exam-Guide.pdf)

## Skills validated

*Cert-specific — what this exam actually tests, distinct from the shared "Common skills" above.*

- AWS CodePipeline orchestration and multi-stage deployments
- AWS CodeBuild build automation and custom build environments
- AWS CodeDeploy strategies (in-place, blue/green, canary)
- AWS CodeArtifact artifact repository management
- GitHub Actions and Jenkins integration with AWS services
- AWS CloudFormation (templates, parameters, outputs, stack policies, StackSets)
- AWS Cloud Development Kit (CDK) and Infrastructure-as-Code at scale
- AWS Service Catalog for self-service infrastructure
- AWS Serverless Application Model (SAM)
- ECS / EKS / Fargate / App Runner deployment patterns and scaling
- Lambda blue/green deployments and canary traffic shifting
- CloudWatch metrics, logs, dashboards, and alarms at scale
- AWS X-Ray distributed tracing and service mapping
- Container Insights for ECS/EKS observability
- CloudWatch Synthetics for synthetic monitoring
- EventBridge automation and event-driven orchestration
- AWS Systems Manager (Patch Manager, OpsCenter, Incident Manager, Run Command, Automation, Session Manager)
- AWS Config rules for compliance and configuration drift detection
- AWS Backup for disaster recovery and RPO/RTO validation
- AWS Health Dashboard and personal health notifications
- GuardDuty and Security Hub for security observability
- AWS Inspector for automated vulnerability scanning
- KMS key rotation and key policy automation
- AWS Secrets Manager rotation and Lambda integration
- Auto Scaling lifecycle hooks and custom scaling policies
- AWS Distro for OpenTelemetry (ADOT) collector setup
- Application load balancing and traffic shifting patterns
- Disaster recovery architectures (RTO, RPO, backup strategies)
- Cost analysis tools (Cost Explorer, Trusted Advisor, finops automation)
