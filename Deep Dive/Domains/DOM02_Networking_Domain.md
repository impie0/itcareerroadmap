## Domain Overview

The **NETWORKING domain** spans the design, deployment, operation, and troubleshooting of enterprise and service-provider networks. It encompasses:

- **Core protocols:** IPv4, IPv6, BGP, OSPF, EIGRP, IS-IS, MPLS, QoS
- **Hardware platforms:** Routers, switches, firewalls, load balancers, SD-WAN appliances
- **Specialties:** Wireless, data center, service provider, campus/LAN, security, cloud interconnect
- **Automation:** Ansible, Terraform, network configuration management, Intent-Based Networking (IBN)

Networking professionals span roles from:
- **Help Desk / Support Technician** (CCST) — $45k–$65k
- **Network Technician / Junior Admin** (CCNA, CompTIA Network+) — $55k–$75k
- **Network Engineer / Administrator** (CCNP, JNCIS, ACP) — $90k–$130k
- **Senior Engineer / Architect** (CCIE, JNCIE, ACE-Professional) — $120k–$180k+
- **Principal / Distinguished Engineer** — $150k–$250k+

## Vendor-Neutral Certifications

### CompTIA Network+ (N10-009)

The entry-level, vendor-neutral **CompTIA Network+** is widely recognized in IT operations, government contracting, and SMB environments.

| Attribute | Details |
|-----------|---------|
| **Exam code** | N10-009 |
| **Duration** | 90 minutes |
| **Questions** | 80 multiple-choice, drag-and-drop, and performance-based |
| **Cost** | $369 USD (2026 pricing) |
| **Valid for** | 3 years |
| **Retirement date** | Approx. 2027 |
| **Objectives URL** | [CompTIA Network+ official page](https://www.comptia.org/en-us/certifications/network/) |

**Topics covered:**
- Network fundamentals (OSI model, IP addressing, DNS, DHCP)
- Network access (Ethernet, VLANs, STP)
- Network architecture (topologies, WAN, cloud connectivity)
- Network operations (monitoring, troubleshooting, performance)
- Security basics (firewalls, VPN, authentication)
- Network resilience and disaster recovery

**Typical progression:** Help Desk → Network Technician → CCNA or Juniper track.

## 2026 Strategic Notes

### Major Changes in Effect

1. **CCNA 200-301 v1.1** (Feb 3, 2026): AI, cloud, automation deeply integrated
2. **CCNP ENCOR 350-401 v1.2** (March 19, 2026): Wireless removed to specialist exams
3. **DevNet → CCNA/CCNP/CCIE Automation** (Feb 3, 2026): Automation now standalone CCIE track
4. **CCIE Enterprise Infrastructure v1.1** (Feb 2026): Crosswork, YAML/Jinja2, cloud-edge scenarios
5. **No CCNA prerequisite for CCNP** (effective March 2026): Fast-track possible for experienced IT pros
6. **Juniper Open Learning** (ongoing): Best-value entry into service provider networking
7. **Arista ACE program** (no prerequisites): Rapid data center specialist path
8. **Mikrotik MTCNA** (strong in Africa): Cost-effective for ISP/edge skills

### Emerging Opportunities

- **Automation expertise** now carries equal weight to routing/switching
- **SD-WAN + SASE** becoming mandatory architecture skills
- **Cloud networking** (AWS/Azure direct connect, overlay architectures) heavily tested
- **AI/ML for networking** (intent-based networking, anomaly detection) increasing in visibility
