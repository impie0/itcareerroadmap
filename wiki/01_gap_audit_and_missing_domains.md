# Deep Gap Audit: Missing Domains & Career Paths in IT Roadmap

**Date:** April 2026  
**Version:** 1.0  
**Status:** Comprehensive Analysis

---

## Executive Summary

The current 10-domain IT roadmap (Foundation, Networking, Systems, Cloud, DevOps, Security, Identity & Endpoint, Data/AI, Architecture, IT Management) provides solid coverage of traditional IT operations and infrastructure. However, significant career paths and specialization ecosystems are missing.

The most critical gaps are:

1. **Software Engineering / Application Development** — Backend, frontend, mobile, and full-stack development are distinct career tracks with entirely different skill requirements from infrastructure roles.
2. **Quality Engineering / SDET** — Test automation and QA engineering have evolved into specialized engineering disciplines with 40%+ year-on-year demand growth.
3. **Database Administration (DBA)** — Separate from Data Engineering; focuses on operational stability, performance tuning, backup/recovery, and compliance.
4. **Specialty Enterprise Ecosystems** — Salesforce, SAP, ServiceNow, Oracle, Workday represent multi-billion-dollar platforms with mature certification paths and significant hiring demand.
5. **Healthcare IT / EHR Systems** — Epic and Cerner migrations drive substantial opportunity; represents 48% and 26% of US hospital beds respectively.
6. **Embedded Systems / IoT / Firmware** — Distinct from software engineering; driven by automotive, medical devices, and consumer IoT explosive growth.
7. **Privacy & Data Protection (IAPP)** — CIPP certifications (CIPP/E, CIPP/US, CIPM) represent a growing regulatory-driven career path.
8. **Digital Forensics & Incident Response** — GCFA, GCFE, GREM represent specialized security niches beyond general security engineering.
9. **Adjacent Technical Roles** — DevRel, Sales Engineer, TPM, Technical Writer, and IT Trainer represent viable alternatives to traditional IC or management tracks.
10. **Emerging/Niche Domains** — Blockchain/Web3 (real, but volatile); Quantum Computing (still nascent); AR/VR (limited enterprise adoption); Mainframe (declining but stable).

**Recommendation:** Add 8–12 new domains to v2 of the roadmap in priority order. See Section 7 for detailed recommendation.

## 2. Missing Major Domains

### 2.1 Software Engineering / Application Development

**Why it deserves its own column:**

Software engineering and IT infrastructure are fundamentally different career tracks. A backend engineer and a systems administrator share almost no skillsets despite both being "IT professionals."

**Market validation:**
- 26% of tech roles now require AI expertise (98% YoY surge)
- AI/ML skills command a 17.7% salary premium
- Frontend, backend, full-stack, mobile, and game development are distinct specializations
- Average job search takes 5–6 months; competitive field with high barrier to entry

**Key specializations:**
- **Backend Engineering** — API design, databases, microservices, scaling; languages: Python, Go, Rust, Java
- **Frontend Engineering** — React, Vue, Angular, responsive UI; CSS, accessibility, performance
- **Full-Stack Development** — End-to-end responsibility; common entry point
- **Mobile Development** — iOS (Swift), Android (Kotlin), React Native, Flutter
- **Game Development** — Unity, Unreal Engine, specialized physics/graphics pipelines
- **AI/ML Engineering** — Specializing in large language models, computer vision, or data pipelines

**Key skills:**
- Programming (Python, JavaScript, Java, Go, Rust, C++, Swift, Kotlin)
- Data structures and algorithms
- Version control (Git)
- Testing and CI/CD
- Cloud platforms (AWS, GCP, Azure)
- DevOps fundamentals
- Communication and Agile methodology

**Certifications:**
- No universal certifications; vendor-neutral: AWS Certified Developer Associate
- Language-specific: Oracle Java Programmer, Python Institute PCAP
- Bootcamp completions (General Assembly, Flatiron School, Springboard)

**Source:** [Refonte Learning — Software Engineering Career Path 2026](https://www.refontelearning.com/blog/how-to-become-a-successful-software-engineer-in-2026-skills-and-career-path)

### 2.2 Quality Engineering / SDET / Test Automation

**Why it deserves its own column:**

QA has transformed from manual testing to engineering-centric test automation and SDET (Software Development Engineer in Test) roles. Demand surged 40%+ YoY and salaries now match senior developers.

**Market validation:**
- Over 60% of organizations integrating AI/ML into testing
- Demand for SDET skills up 40%+ year-on-year
- SDET median starting salary: $110,000 USD
- AI tools augment SDETs rather than replacing them

**Key specializations:**
- **QA Automation Engineer** — Selenium, Cypress, Playwright, Appium
- **SDET (Software Development Engineer in Test)** — Code-focused; API testing, performance testing, chaos engineering
- **Performance/Load Testing** — JMeter, LoadRunner, Gatling
- **Mobile QA Automation** — Appium, XCTest, Espresso
- **API Testing** — REST Assured, Postman, SoapUI
- **Security Testing / Pentesting** — OWASP, burp suite, vulnerability scanning

**Key skills:**
- At least one test automation framework (Selenium, Cypress, Playwright)
- Programming (Python, Java, JavaScript, C#)
- API testing fundamentals
- Database querying (SQL)
- CI/CD integration (Jenkins, GitHub Actions)
- Code review and version control
- Problem-solving and debugging

**Certifications:**
- ISTQB Certified Tester (Foundation, Advanced SDET)
- No single vendor-neutral SDET cert exists yet
- Bootcamps: TripleTen, General Assembly, Code Bootcamp

### 2.3 Database Administration (DBA) — Distinct from Data Engineering

**Why it deserves its own column:**

DBA is an operational specialty focused on availability, performance, backup/recovery, and compliance. Data Engineers build pipelines. These are entirely different career tracks often reporting to different teams.

**Market validation:**
- BLS occupational outlook shows DBAs as distinct role from data engineer
- 2026 job market separates "DBA" from "Data Engineer" in 70%+ of postings
- DBAs command $90K–$150K depending on platform (Oracle, SQL Server, PostgreSQL)

**Key specializations:**
- **Operational DBA** — Daily operations, monitoring, backups, disaster recovery
- **Performance/Tuning DBA** — Query optimization, indexing, capacity planning
- **Strategic DBA** — Long-term planning, architecture, database consolidation
- **Security DBA** — Access control, encryption, compliance (GDPR, HIPAA)
- **Cloud DBA** — RDS, Cloud SQL, managed services (AWS, Azure, GCP)
- **Specific platforms:** Oracle, SQL Server, PostgreSQL, MySQL, MongoDB, Redis

**Key skills:**
- SQL (intermediate to expert)
- Database design and normalization
- Backup and recovery procedures
- Performance monitoring and tuning
- Security and access control
- Platform-specific tools (SQL Server Management Studio, Oracle Enterprise Manager)
- Scripting (T-SQL, PL/SQL, Python)

**Certifications:**
- Oracle Database Administrator (OCP) — https://www.oracle.com/education/
- Microsoft SQL Server (MCSA, MCSE) — https://learn.microsoft.com/en-us/certifications/
- PostgreSQL (EDB Certified Associate) — https://www.enterprisedb.com/
- Cloudera Certified Associate Data Analyst — https://www.cloudera.com/
- Exin SUSE CLE (SUSE Linux Enterprise) includes DB admin — https://www.exin.com/

**Source:** [BLS Occupational Handbook](https://www.bls.gov/ooh/computer-and-information-technology/database-administrators.htm); [Indeed — DBA vs Database Engineer](https://www.indeed.com/career-advice/finding-a-job/database-engineer-vs-database-administrator)

### 2.7 Embedded Systems / IoT / Firmware

**Why it deserves its own column:**

Distinct from software engineering. Embedded engineers develop firmware for devices: automotive, medical, consumer electronics, industrial IoT. Skills are specialized (RTOS, hardware interaction, power optimization).

**Market validation:**
- Demand vastly exceeds supply
- Embedded engineer salaries: $80K–$160K USD
- EV and autonomous vehicle boom drives automotive firmware hiring
- Medical device and IoT device proliferation drives growth

**Key specializations:**
- **Automotive Firmware** — ECU programming, AUTOSAR, functional safety (ISO 26262)
- **Medical Device Firmware** — Real-time OS, regulatory compliance (IEC 62304)
- **IoT Firmware** — Connectivity (Wi-Fi, Bluetooth, Zigbee, LoRaWAN), low power
- **RTOS Engineering** — FreeRTOS, QNX, VxWorks
- **Hardware-Software Integration** — PCB design knowledge, debugging with oscilloscopes
- **Embedded Linux** — Yocto, Buildroot, device drivers

**Key skills:**
- C/C++ (primary)
- Embedded Python (for scripting)
- Rust (emerging for safety)
- Microcontroller architectures (ARM, RISC-V, x86)
- Real-time operating systems
- Hardware debugging (JTAG, SWD)
- Understanding of electrical concepts

**Certifications:**
- Certified Embedded Systems Engineer (CESE) — https://www.iseek.org/
- Certified IoT Professional (CIoTP) — https://www.iiot-world.com/
- ARM Embedded Systems Fundamentals — https://www.arm.com/
- TI (Texas Instruments) Embedded Systems certifications — https://www.ti.com/
- NXP Semiconductor training programs — https://www.nxp.com/

**Source:** [Piestech Systems — Embedded Systems Career Roadmap 2026](https://piestsystems.com/blog/embedded-systems-career-roadmap-2026/); [The Tutor Bridge — Embedded Systems Engineer Roadmap](https://www.thetutorbridge.com/roadmap/embedded-systems)

## 7. Recommendations — Priority Order for v2 Roadmap

Based on market demand, career stability, and job market size, here are the recommended additions to the IT roadmap in priority order:

### **Tier 1: High Priority (Add Immediately)**

1. **Software Engineering / Application Development**
   - Justification: Distinct from IT ops; 26% of tech roles require AI expertise; massive job market
   - Include: Frontend, Backend, Full-Stack, Mobile, Game dev, AI/ML specializations
   - Est. audience: 30–40% of IT professionals considering pivot

2. **Quality Engineering / SDET / Test Automation**
   - Justification: 40%+ YoY demand growth; specialized role; career ladder clear
   - Include: QA Automation, SDET, Performance/Load testing, Security testing
   - Est. audience: 10–15% of IT professionals

3. **Database Administration (separate from Data Engineering)**
   - Justification: Operational DBA is different from Data Engineer; 70%+ of job market treats as separate
   - Include: Operational DBA, Performance DBA, Cloud DBA, Platform-specific (Oracle, SQL Server, PostgreSQL)
   - Est. audience: 15–20% of IT professionals

4. **Specialty Enterprise Ecosystems (Consolidated Domain)**
   - Justification: Salesforce, SAP, ServiceNow, Oracle, Workday serve 80%+ of Fortune 500
   - Include: Separate tracks for each major platform
   - Est. audience: 20–30% of IT professionals (depending on industry)

5. **Healthcare IT / EHR Systems**
   - Justification: Epic + Cerner power 71% of US hospital beds; 15–20% of enterprise IT market; high salaries
   - Include: Epic Admin/Developer, Cerner/Oracle, Meditech, HL7/FHIR integration
   - Est. audience: 15–20% of IT professionals in healthcare sector

### **Tier 2: Medium Priority (Add in Next Release)**

6. **Embedded Systems / IoT / Firmware**
   - Justification: Demand exceeds supply; EV/medical/IoT boom; distinct from software engineering
   - Include: Automotive firmware, Medical devices, IoT, RTOS, Embedded Linux
   - Est. audience: 5–10% of IT professionals

7. **Privacy & Data Protection (IAPP)**
   - Justification: Regulatory-driven career path; CIPP certifications growing fast
   - Include: CIPP variants (E/US/C/A/CN), CIPM, data protection engineering
   - Est. audience: 5–8% of IT professionals

8. **Digital Forensics & Incident Response (Specialization under Security)**
   - Justification: GCFA, GCFE, GREM highly valued; specialized niche with good compensation
   - Include: Forensic Examiner, Forensic Analyst, Reverse Engineering/Malware
   - Est. audience: 3–5% of IT professionals

9. **Adjacent Career Lanes (Consolidated Domain)**
   - Justification: Sales Engineer, DevRel, TPM, Technical Writer, IT Trainer, Open-Source maintainer viable alternatives
   - Include: Roles and career paths for non-traditional tracks
   - Est. audience: 10–15% of IT professionals (seeking alternatives to IC/management)

### **Tier 3: Emerging / Monitor (Add in Future Release)**

10. **Hardware / Datacenter / Physical Infrastructure**
    - Justification: Hybrid cloud drives continued demand; edge computing growing
    - Include: Datacenter ops, Storage engineering, Hyperconvergence, Power/cooling
    - Est. audience: 5–10% of IT professionals
    - Note: Will grow as edge/on-prem infrastructure remains relevant

11. **Telecom / Networking Physical (5G, Fiber, RF)**
    - Justification: 5G rollout ongoing; fiber expansion; distinct from network engineering
    - Include: 5G RAN, Fiber optics, Wireless engineering, RF design
    - Est. audience: 2–5% of IT professionals

12. **Mainframe / Legacy Systems**
    - Justification: Declining but stable; strong pay; career security
    - Include: COBOL, DB2, CICS, JCL, System Programmer
    - Est. audience: 1–3% of IT professionals (but high loyalty to the platform)

13. **AR/VR / Spatial Computing** (Emerging)
    - Justification: Apple, Meta investing; still niche but growing
    - Include: AR app dev (ARKit, ARCore), VR (Unity, Unreal)
    - Est. audience: <1% currently; monitor for 2027+

14. **Blockchain / Web3** (Volatile, Supplement)
    - Justification: Real market with 8,000–12,000 jobs globally; but volatile
    - Include: Smart contract dev, Infrastructure, Security, Data analytics
    - Est. audience: 1–2% of IT professionals; recommend as specialization, not main track
    - Note: Revisit annually; do not position as primary career path
